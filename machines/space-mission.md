## ⚙️ Core Mechanics & Scoring

### 🎯 Swinging Target System

The swinging target is the heart of Space Mission's scoring system [^2]:

| Target Position     | Value          | Spinner Lit? |
| ------------------- | -------------- | ------------ |
| A positions         | 1,000 or 5,000 | No           |
| 5,000 between A & B | 5,000          | ✅ Yes        |
| B positions         | 1,000 or 5,000 | No           |
| 5,000 between B & C | 5,000          | No           |
| C positions         | 1,000 or 5,000 | No           |
| 5,000 right of C    | 5,000          | ✅ Yes        |

The target value cycles one step right each time you collect it (by hitting it directly OR via the lit top lanes) [^1].

> ⚠️ **Important:** The spinner is only lit ~20% of the time! The current target value carries over between players, not between balls [^1]

### 🌀 Spinner Scoring

| Spinner State      | Points per Spin   |
| ------------------ | ----------------- |
| Unlit              | 100 points        |
| Lit (1,000 Points) | 1,000 points [^2] |

> 💰 **Pro Tip:** A good spinner shot when lit can be worth 20,000+ points! [^1]

### 📊 Bonus System

| Setting            | Bonus Advance                  |
| ------------------ | ------------------------------ |
| Liberal            | 5 advances per trigger [^2]    |
| Conservative       | 3 advances per trigger [^2]    |
| Points per advance | 100 or 1,000 (adjustable) [^2] |

**Double Bonus:** When activated, your end-of-ball bonus is DOUBLED! [^1]

## 🔤 The A-B-C System

The A-B-C letter system is crucial for maximizing your score:

### How to Complete Letters

1. **Shoot the swinging target** when it's on an A, B, or C value
2. **OR** roll through the **lit top rollover lanes** to collect the current letter [^2]

### What Completing A-B-C Does

| Achievement                               | Reward                                              |
| ----------------------------------------- | --------------------------------------------------- |
| Complete any letter (both lights)         | Lights side saucers for Double Bonus [^1]           |
| Complete A, B, or C                       | Lights eject holes [^2]                             |
| Complete A, B, or C (with certain lights) | Lights Extra Ball lane [^2]                         |
| Complete A, B, and C                      | Lights bottom outlanes alternately for Special [^2] |

## 🎱 Skill Shot

**Objective:** Plunge to hit the **lit "collect swinging target value" lane** [^1]

| Situation             | Strategy                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------- |
| Lane is lit on plunge | Plunge with controlled strength to hit it                                                   |
| Lane is NOT lit       | Aim for an **odd number** of hits on the top right slingshot (left side has no switch) [^1] |

> 💡 **Note:** The collect lanes cycle on/off with slingshot hits

## 🏆 Strategy: The UTAD Method

The winning strategy for Space Mission is **UTAD** — and sometimes, the winning move is **NOT to flip!** [^1]

### What is UTAD?

| Letter | Meaning     | Purpose                                  |
| ------ | ----------- | ---------------------------------------- |
| **U**  | Upkickers   | Let ball roll into kickers near flippers |
| **T**  | Targets     | Hit the swinging target                  |
| **A**  | A-B-C lanes | Complete letters via top lanes           |
| **D**  | Doubler     | Activate double bonus                    |

### The "Don't Flip" Strategy

> 🎬 *"Ever see the movie War Games? 'The only winning move is not to play.' Well, here, playing is okay, but a winning move is often not to flip."* [^1]

**How it works:**
1. **Hold flippers UP** and let the ball roll into the upkickers [^1]
2. The kickers tend to hit the center swinging target
3. The ball often drops back toward the flippers
4. Repeat the process!

### When TO Flip

| Situation                         | Action                                  |
| --------------------------------- | --------------------------------------- |
| Spinner is LIT                    | Shoot up the left spinner [^1]          |
| Side saucers lit for Double Bonus | Shoot a side saucer to activate it [^1] |
| Ball heading for outlane          | Save it!                                |

> 💡 **Pro Tip:** The backstop behind the side saucers makes them easier shots than most saucers — rim-outs are rare! [^1]

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots & Drain Paths

| Danger Zone         | Risk Level | Notes                              |
| ------------------- | ---------- | ---------------------------------- |
| Center drain        | 🔴 High     | No center post on this machine     |
| Outlanes            | 🔴 High     | No inlanes on this playfield! [^7] |
| Wild flipper shots  | 🟡 Medium   | Can send ball to outlanes          |
| Missed spinner shot | 🟡 Medium   | Ball can come back fast            |

### ✅ Safe Shots & Ball Control

| Safe Zone     | Why It's Safe                             |
| ------------- | ----------------------------------------- |
| Upkickers     | Ball returns predictably to flippers [^1] |
| Side saucers  | Backstop prevents rim-outs [^1]           |
| Held flippers | Ball rolls to upkickers naturally         |

### 🎯 Key Feeds to Know

| Kickout Source       | What to Do                                                    |
| -------------------- | ------------------------------------------------------------- |
| Side saucer kickouts | Capture to upkicker OR dead bounce to shoot left spinner [^1] |
| Top saucer kickouts  | Be ready — can be unpredictable                               |

## ⚙️ Setup Recommendations

### Pitch & Level

| Setting            | Recommendation            |
| ------------------ | ------------------------- |
| Playfield pitch    | 6.0° - 6.5° (standard EM) |
| Side-to-side level | Perfectly level           |

| Spinner 1,000 pts | More positions lit | Fewer positions lit |

## ✅ Quick-Start Checklist

### 🚀 Before Your First Ball

- [ ] Note if spinner is currently lit (check swinging target position)
- [ ] Check which top lanes are lit
- [ ] Observe the swinging target's current value

### 🎮 During Play

- [ ] **Plunge** for the lit collect lane (skill shot)
- [ ] **Hold flippers up** — let ball find the upkickers
- [ ] **Build bonus** through repeated upkicker/target cycles
- [ ] **Complete a letter** (A, B, or C) to light double bonus saucers
- [ ] **Shoot side saucer** when lit to activate double bonus
- [ ] **Shoot spinner** when lit (look for 1,000 pts light)
- [ ] **Survive** to collect your (doubled!) bonus

### 🎯 Key Reminders

| Do This                 | Don't Do This                        |
| ----------------------- | ------------------------------------ |
| ✅ Hold flippers up      | ❌ Flip wildly                        |
| ✅ Use upkickers         | ❌ Ignore the swinging target         |
| ✅ Shoot lit spinner     | ❌ Shoot unlit spinner (only 100 pts) |
| ✅ Activate double bonus | ❌ Drain before bonus collect         |

## 🎮 Final Thoughts

Space Mission may not have the modes, multiballs, and wizard modes of modern machines, but it offers something equally valuable: **pure, skill-based pinball**. The UTAD strategy rewards patience and ball control over frantic flipping.

**Your realistic goal:** Clock the machine at 199,990 and keep going! The top players have achieved 500,000+ through multiple rollovers. That's the true "high score" challenge on this classic EM.

*Good luck, astronaut!* 🚀
