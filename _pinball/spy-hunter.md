## 🎯 Core Mechanics

### Skill Shot 🎱

**How it works:** When you plunge, lights strobe downward in the shooter lane at increasing speed.

| Timing         | Award                                    |
| -------------- | ---------------------------------------- |
| Hit lit switch | **50,000 points**                        |
| Bonus          | Qualifies right saucer for "Bonus Value" |

> 💡 **Pro Tip:** The skill shot also closes one of the left outlane gates! This is CRITICAL for survival.

---

### Drop Targets (Main Playfield) 🎯

Complete the 4 drop targets on the main playfield:

| Completion | Points       | Bonus                           |
| ---------- | ------------ | ------------------------------- |
| 1st time   | 10,000       | Opens left gate, +1X multiplier |
| 2nd time   | 20,000       | +1X multiplier                  |
| 3rd time   | 30,000       | +1X multiplier                  |
| 4th time   | 40,000       | +1X multiplier                  |
| 5th+ time  | Up to 70,000 | +1X multiplier (max 5X)         |

> ⚠️ **Strategy Note:** Don't actively shoot these - they're not worth the risk. You'll complete them incidentally. Focus on the saucer and upper playfield.

---

### S-H-O-T Targets 🔫

| Action              | Award                                             |
| ------------------- | ------------------------------------------------- |
| Hit flashing target | 5,000 points + lights target                      |
| Complete S-H-O-T    | 25,000 points                                     |
| Bonus               | Qualifies saucer + lights spinner (one shot only) |

---

### The Saucer (Primary Scoring Shot) 💰

**Qualification:** Skill shot OR complete S-H-O-T targets

| Shot # | Award                |
| ------ | -------------------- |
| 1st    | 25,000               |
| 2nd    | 50,000               |
| 3rd    | 75,000               |
| 4th    | 100,000              |
| ...    | Increases to 250,000 |

> 🏆 **THIS IS YOUR GO-TO SHOT!** The saucer is the highest-value repeatable shot in the game.

**The Saucer Loop:**
1. Shoot saucer
2. Trap kickout on right flipper
3. Backhand the saucer again
4. Repeat ad nauseum

---

### Upper Playfield (7 Standups) 🔝

| Completion                 | Award               |
| -------------------------- | ------------------- |
| Complete 7 standups (1st)  | 70,000              |
| Complete 7 standups (2nd+) | 140,000             |
| Hit 4+ standups            | Lights captive ball |

**Captive Ball Awards:**

| Shot # | Points  | Playfield Multiplier |
| ------ | ------- | -------------------- |
| 1st    | 25,000  | 2X playfield         |
| 2nd    | 50,000  | 3X playfield         |
| 3rd    | 100,000 | —                    |

> 💡 **Pro Tip:** Keep the ball in the upper playfield as long as possible! But once it leaves, don't waste shots trying to get back up there.

---

### Warehouse Rollovers (Upper Left Bagatelle) 🏭

When the ball enters the upper left "warehouse" area:

| Feature            | Details                      |
| ------------------ | ---------------------------- |
| Lane change        | Right flipper button         |
| Each lit rollover  | 10,000 bonus (end of ball)   |
| Maximum            | 90,000 bonus                 |
| With 5X multiplier | **450,000 potential bonus!** |

> 💡 **Pro Tip:** NUDGE aggressively to hit as many different rollovers as possible. Use lane change to target unlit lanes.

## 🏆 Scoring Strategy

### Priority Order

1. **🥇 Nail the Skill Shot** - 50K + qualifies saucer + closes outlane gate
2. **🥈 Saucer Loop** - Highest repeatable value (up to 250K per shot)
3. **🥉 Upper Playfield** - 70K-140K completions + captive ball multipliers
4. **4️⃣ Warehouse Rollovers** - Build bonus for end of ball
5. **5️⃣ Drop Targets** - Build multiplier (but don't actively shoot)

### Path to Rolling the Score (10,000,000+)

Based on player reports, rolling the score requires:

1. **Consistent skill shots** to keep outlane gates closed
2. **Master the saucer loop** - This is where big points come from
3. **Maximize bonus multiplier** to 5X
4. **Fill warehouse rollovers** for 450K bonus per ball
5. **Earn extra balls** to extend play
6. **Avoid the left side** at all costs!

## ⚙️ Setup Recommendations

### Pitch/Level

| Setting | Recommendation                                                   |
| ------- | ---------------------------------------------------------------- |

## 🎮 Quick-Start Checklist for New Players

- [ ] **Plunge timing is everything** - Watch the strobing lights, hit when lit for 50K
- [ ] **Stay RIGHT** - The left side of the playfield is a death trap
- [ ] **Learn the saucer loop** - Shoot saucer → trap kickout → backhand repeat
- [ ] **Use lane change** - Right flipper changes warehouse rollover lights
- [ ] **Nudge in the warehouse** - Get as many rollovers as possible
- [ ] **Build your multiplier** - Drop targets increase bonus multiplier to 5X
- [ ] **Don't chase drop targets** - They're not worth the risk
- [ ] **Watch the pop bumper** - It's positioned to drain you left

## 🎵 Theme & Atmosphere

The game features the iconic **"Peter Gunn" theme** from the Spy Hunter video game. The artwork by Tony Ramunni captures the 1980s spy aesthetic with a sleek, dangerous feel.

> 📝 **Historical Note:** Designer Greg Kmiec originally planned this as an Elvis jukebox-themed game! The warehouse rollover area was inspired by Ted Zale's JOUST game. "AGENT GK" on targets refers to Greg Kmiec himself.
