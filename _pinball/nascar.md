# 🏁 NASCAR - Strategy Guide

*Stern Pinball, 2005 — Designed by Pat Lawlor & Louis Koziarz*

> **"Spell RACE. Lock the hauler. Chase the Championship."**

NASCAR builds toward Race Multiball through spelling R-A-C-E via drop targets, then escalates through Garage modes and the five main challenges toward the Champion Challenge final wizard mode. Our goal: spell RACE, lock balls via the hauler, complete all five main challenges to spell CHAMP, and reach Champion Challenge.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Complete the drop targets repeatedly to light R-A-C-E letters** — each completion lights one letter shot
- [ ] **Spelling RACE lights LOCK at the left ramp**
- [ ] **Lock 2 balls in pit road, then shoot the third ball through the hauler** (which locks at the back of the playfield) to start **Race Multiball**
- [ ] **Complete all the Garage modes** to unlock the **Super Party** sub-wizard mode
- [ ] **Complete the game's five main challenges** to spell **CHAMP** and unlock **Champion Challenge**, the final wizard mode

---

## 🏎️ Core Mechanics

### Spelling RACE & Race Multiball

1. Complete the **drop targets** — each completion **lights one R-A-C-E letter shot**
2. Once **RACE is fully spelled**, the **left ramp lights LOCK**
3. **Lock 2 balls in pit road**
4. Shoot the **third ball through the hauler**, which locks at the back of the playfield, to **start Race Multiball**

### Super Party (Sub-Wizard Mode)

- **Requirement:** Complete all the **Garage modes**
- **Gameplay:** 3-ball multiball — hit the **pop bumpers enough times to count down to zero**, then knock down the **drop targets** for a **Super Jackpot**

### Champion Challenge (Final Wizard Mode)

- **Requirement:** Spell **CHAMP** by completing the game's **five main challenges**
- **Gameplay:** **4-ball multiball** with **all targets lit for maximum value**

---

## ⚠️ Dangerous Shots & Drain Patterns

| Risk | Mitigation |
|---|---|
| **Captive ball / Newton ball mechanics** | Unique physics elements that can produce unexpected behavior | Learn their specific rebound patterns before relying on them for precision shots |
| **Drop target rebounds** (two separate banks) | Standard rebound risk, compounded by needing repeated completions for RACE letters | Approach with controlled, deliberate shots |
| **Pop bumper countdown grinding (Super Party)** | Extended bumper time increases overall exposure | Treat as a deliberate, sustained objective rather than rushing |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Start completing drop targets toward your first RACE letters
2. Work toward locking your first balls in pit road
3. Begin progress on Garage modes

**Target Score:** 10M–30M

### Mid Game (Ball 2)

1. Complete RACE and start Race Multiball via the hauler lock sequence
2. Continue Garage mode progress toward Super Party
3. Work on the five main challenges toward spelling CHAMP

**Target Score:** 30M–80M

### Late Game (Ball 3+)

1. Run Super Party if Garage modes are complete
2. Push to finish all five main challenges for CHAMP and Champion Challenge
3. Maximize the 4-ball Champion Challenge if you've reached it

**Target Score per ball:** 40M–120M+

---

## ⚙️ Setup Recommendations

| Setting | Recommendation | Reason |
|---|---|---|
| **Playfield pitch** | 6.5°–7.0° | Stern factory standard for mid-2000s machines |
| **Hauler/pit road lock mechanism** | Verify smooth, reliable lock and release | Central to the entire Race Multiball start sequence |
| **Drop target banks (both)** | Check for clean, reliable resets | Needed for both RACE letter progress and Super Party's drop target finish |

---

## 📋 Quick Reference: Strategic Priorities

**Ball 1:** Complete drop targets toward RACE, start Garage mode progress.
**Ball 2:** Spell RACE and run Race Multiball, keep working Garage modes and challenges.
**Ball 3+:** Run Super Party if qualified, finish the five challenges, chase Champion Challenge.

**Target High-Score Ballpark:** **40–120 million points** is a strong, competitive game; reaching Champion Challenge represents full completion of this NASCAR-licensed ruleset.
