## 🎮 Core Mechanics

### Character Selection
At game start, choose one of four characters using the flippers. Each has unique benefits:

| Character      | Special Power                                         | Best For                           |
| -------------- | ----------------------------------------------------- | ---------------------------------- |
| 🗡️ **Luke**     | 2x scoring during Lightsaber Duel                     | Beginners, Lightsaber Duel focus   |
| 👸 **Leia**     | +3x shot multiplier during final planet modes         | Multiplier stacking                |
| 🤖 **R2-D2**    | Easier shot multipliers (fewer hits, longer duration) | Consistent play, stacking          |
| 🔫 **Han Solo** | 2x scoring during Escape from Boba Fett               | Advanced players, high-risk/reward |

> 💡 **Pro Tip:** R2-D2 is the most beginner-friendly choice. His multiplier benefits make maintaining high multipliers much easier.

### Skill Shots

| Skill Shot            | How to Execute                                        | Reward                              |
| --------------------- | ----------------------------------------------------- | ----------------------------------- |
| **3-Bank Target**     | Plunge and hit the flashing center 3-bank target      | Bonus X (middle target scores most) |
| **FORCE Target**      | Plunge and hit the roving FORCE drop target           | Spots next FORCE letter             |
| **Double Skill Shot** | Time center 3-bank shot to ricochet into FORCE target | Both rewards!                       |

> ⚠️ **Warning:** The plunge can be unpredictable. A short plunge may validate the ball and send it SDTM (straight down the middle). Practice your plunge strength!

### Planet Modes

Shoot any planet shot **3 times** to qualify modes for that planet. Each planet has 2-3 modes (varies by character) plus a final multiball mode.

#### 🔴 Death Star (Left Mini-Loop)

| Mode                              | Characters    | Objective                                                          | Tips                                      |
| --------------------------------- | ------------- | ------------------------------------------------------------------ | ----------------------------------------- |
| **Rescue the Princess**           | Luke, R2, Han | Complete 4 hurry-up shots (left orbit, mini-loops, right ramp)     | Value increases per hurry-up, caps at 25M |
| **Escape the Death Star**         | Luke, Leia    | Shoot 3 mini-loops, then an orbit before timer expires             | Keep ball under control                   |
| **Escape the Death Star**         | Han           | Pilot Falcon via mini-loops/orbits, finish at left ramp            | Different version for Han                 |
| **Fight TIE Fighters**            | Luke, Han     | Shoot flashing purple shots to light TIE target, hit it 4x         | 10-second windows                         |
| **FINAL: Destroy the Death Star** | All           | Shoot ramps to light 10-sec hurry-up at left mini-loop, collect 3x | Starts Victory Multiball                  |

#### 🟢 Endor (Left Ramp)

| Mode                          | Characters    | Objective                                                          | Tips                                |
| ----------------------------- | ------------- | ------------------------------------------------------------------ | ----------------------------------- |
| **Speeder Bike Chase**        | Luke          | Shoot flashing green shots within 15-sec timers                    | Fast reflexes needed                |
| **Throne Room Duel**          | Luke          | Make combo shots to both ramps 3x, right mini-loop increases value | Timers apply                        |
| **Knockout Shield Generator** | Leia, R2, Han | Complete 4 flashing green shots to unlock, then same 4 to finish   | 8 total shots                       |
| **FINAL: Escape from Endor**  | All           | Complete 3 shots before timer, then shoot left mini-loop           | 3-bank adds time; starts Victory MB |

#### 🔵 Hoth (Right Mini-Loop)

| Mode                        | Characters | Objective                                                                   | Tips                                      |
| --------------------------- | ---------- | --------------------------------------------------------------------------- | ----------------------------------------- |
| **Navigate Asteroid Field** | Leia, Han  | Can start Video Mode                                                        | If Video Mode already lit, advances level |
| **Cloud City Escape**       | Leia       | Shoot both ramps, both orbits, then right mini-loop (10-sec timer)          | Methodical approach                       |
| **Duel on Cloud City**      | Luke       | Make 5 FORCE target shots, ramps increase values, finish at right mini-loop | Build value first                         |
| **Repair the Hyperdrive**   | R2         | Shoot any blue shot to light left mini-loop hurry-up                        | Quick mode                                |
| **TIE Fighter Escape**      | Leia       | Complete 3-bank targets to light left orbit (10-sec timer)                  | Focus on 3-bank                           |
| **FINAL: Escape from Hoth** | All        | Shoot left orbit 3x, any 3-bank shot, then mini-loops 4x to down AT-AT      | Starts Victory MB                         |

#### 🟠 Tatooine (Right Scoop)

| Mode                            | Characters    | Objective                                                                       | Tips                           |
| ------------------------------- | ------------- | ------------------------------------------------------------------------------- | ------------------------------ |
| **Get to the Escape Pod**       | Leia, R2      | Complete 5 orange shots (orbits, mini-loops, left ramp)                         | Straightforward                |
| **Escape from Mos Eisley**      | Luke, R2, Han | Shoot either ramp to light left mini-loop; ramps build value                    | Build value first              |
| **The Rancor**                  | Luke          | Shoot ramps to light 3-bank targets to damage Rancor                            | Second hit requires both ramps |
| **FINAL: Escape from Tatooine** | All           | Alternate left ramp and right mini-loop; 3-bank/right ramp/scoop increase value | Starts Victory MB              |

> 💡 **Pro Tip:** Modes do NOT time out! Take your time and focus on accuracy over speed.

## 🎱 Multiballs

### TIE Fighter Multiball

| Aspect                  | Details                                                                      |
| ----------------------- | ---------------------------------------------------------------------------- |
| **Qualification**       | Destroy 35 TIE Fighters (110 for subsequent starts)                          |
| **How to Destroy TIEs** | Shoot TIE Fighter target for hurry-up, mash action button during 7-sec timer |
| **Balls in Play**       | 3 balls                                                                      |
| **Jackpots**            | All major shots lit for 750K each                                            |
| **Super Jackpot**       | Left ramp after all jackpots collected; worth 4M + 100K per advance          |
| **Building Super**      | Shoot flashing red 3-bank targets                                            |

> 💡 **Pro Tip:** TIE Fighter MB has relatively low scoring. Use it as backup/ball save during planet multiballs rather than a primary scoring opportunity.

### Hyperspace Multiball

| Aspect            | Details                                                                         |
| ----------------- | ------------------------------------------------------------------------------- |
| **Qualification** | Complete inlane rollovers to light right ramp/left loop for Hyperspace Hurry-Up |
| **Starting**      | Collect 2 hurry-ups at left mini-loop (+1 per subsequent start)                 |
| **Ball Lock**     | Left mini-loop (Pro) or wireform (Premium/LE)                                   |
| **Jackpots**      | Yellow shots based on collected hurry-up value                                  |
| **Relight Super** | Shoot enough jackpots to light left mini-loop again                             |

> 💡 **Pro Tip:** Like TIE Fighter MB, Hyperspace is best used as backup during planet multiballs for ball save extension.

### Victory Multiball

| Aspect               | Details                                                            |
| -------------------- | ------------------------------------------------------------------ |
| **Qualification**    | Complete any planet's final multiball mode                         |
| **Balls Added**      | 3 additional balls                                                 |
| **Objective**        | Complete 6 major shots for big points                              |
| **Medals**           | Bronze → Silver → Gold (determines Jedi Multiball starting points) |
| **After Completion** | Planet is disqualified until wizard mode is played                 |

> ⚠️ **Important:** Victory Multiball medals are crucial for Jedi Multiball scoring. Aim for Gold medals!

### Add-a-Ball Mechanic

During planet multiballs, knock down FORCE targets to light the action button orange. Hold it for ~2 seconds to add a ball.

> ⚠️ **Warning:** Don't hold the action button too long or you'll restart everyone's game!

## 📈 Progression Systems

### Shot Multipliers (CRITICAL!)

The shot multiplier system is the **key to high scores**.

| Mechanic              | Details                                                           |
| --------------------- | ----------------------------------------------------------------- |
| **Starting Value**    | 2x                                                                |
| **Maximum Value**     | 20x (40x with FORCE doubler)                                      |
| **Increasing**        | Hit any 3-bank target (+1x per hit)                               |
| **Toggle On/Off**     | Action button (green = on, red = off)                             |
| **Moving Multiplier** | When off, use flippers to move left/right                         |
| **Timeout**           | Resets to 2x if 3-bank not hit; higher multipliers timeout faster |

**Multiplier Shot Availability:**

| Multiplier Level | Available Shots | R2-D2 Threshold |
| ---------------- | --------------- | --------------- |
| < 5x             | 3 shots         | < 7x            |
| < 10x            | 2 shots         | < 14x           |
| 10x+             | 1 shot          | 14x+            |

> 💡 **Pro Tip:** Backhanding the center 3-bank target is a reliable way to reset the multiplier timer. Make this a habit!

### FORCE Targets

Complete F-O-R-C-E targets (in any order) to earn awards in sequence:

| Completion | Award                                          |
| ---------- | ---------------------------------------------- |
| 1st        | 💫 Light Mystery                                |
| 2nd        | 🛡️ Light Escape (Ball Save)                     |
| 3rd        | 🎮 Light Video Mode                             |
| 4th        | ✨ **Double Multipliers** (until timer expires) |
| 5th        | ⚔️ Light Lightsaber Duel                        |

> 💡 **Pro Tip:** The 4th completion (Double Multipliers) is extremely valuable! This can push your multiplier to 40x.

### Jedi Training

| Level           | Requirement                           | Benefit                         |
| --------------- | ------------------------------------- | ------------------------------- |
| Youngling       | Complete all blue standups in ~30 sec | Increased target scores         |
| Padawan         | Complete again                        | Higher bonus                    |
| Jedi            | Complete again                        | Even higher bonus               |
| **Jedi Master** | Complete again                        | **Required for Jedi Multiball** |

Each level also completes FORCE targets automatically.

### Character Path Awards

At the start of each ball, select from 4 awards using flippers. Awards are character-specific:

**Luke's Path:**
- Jedi Training, Light Death Star Missions, +5 TIE Fighters, Hyperspace Hurry-Up, Max Time, Light Tatooine Missions, +5% Score, +3X Shot Multiplier, Light Video Mode, Light Endor Missions, Light Escape, +3X Bonus Multipliers, Bonus Held

**Leia's Path:**
- Light Tatooine Missions, Light Death Star Missions, Light Mystery, +3X Bonus Multipliers, Max Time, +5% Score, Jedi Training, Light Endor Missions, Light Hoth Missions, Hyperspace Hurry-Up, Light Escape, +3X Shot Multiplier, Bonus Held

**R2-D2's Path:**
- +3X Shot Multiplier, Light Death Star Missions, Light Tatooine Missions, +3X Bonus Multipliers, Max Time, Light Video Mode, +5% Score, Light Hoth Missions, +5 TIE Fighters, Light Endor Missions, Jedi Training, Bonus Held

**Han's Path:**
- Light Boba Fett Escape, Light Endor Missions, +5 TIE Fighters, Hyperspace Hurry-Up, Max Time, Light Tatooine Missions, Light Video Mode, Light Escape, +5% Score, Light Death Star Missions, Bonus Held, +3X Bonus Multipliers, Jedi Training, +3X Shot Multiplier

> 💡 **Pro Tip:** "Bonus Held" is very far down the list but can compound to massive bonuses (2+ billion!) if collected multiple times during Lightsaber Duel with Han.

## 🔗 Stacking & Combos

### ✅ What Stacks Well

| Combination                       | Why It Works                                               |
| --------------------------------- | ---------------------------------------------------------- |
| **Death Star + Endor modes**      | R2-D2 can light both quickly; leads to stacked Victory MBs |
| **Planet MB + TIE Fighter MB**    | TIE MB provides ball save backup                           |
| **Planet MB + Hyperspace MB**     | Hyperspace provides ball save backup                       |
| **High multipliers + Video Mode** | Video Mode scoring benefits from shot multipliers          |
| **Double Multipliers + Any mode** | 40x multiplier on mode shots = massive points              |

### ❌ What NOT to Stack

| Avoid                         | Reason                                    |
| ----------------------------- | ----------------------------------------- |
| **Planet Final MBs together** | Must be played solo (game enforced)       |
| **Lightsaber Duel + Modes**   | LSD takes priority and pauses other modes |
| **Modes in Competition Mode** | Stacking disabled in tournament settings  |

### Optimal Timing

1. **Build multipliers FIRST** during ball save time
2. **Light Double Multipliers** (4th FORCE completion) before big modes
3. **Start planet MB** with TIE Fighter or Hyperspace ready as backup
4. **Save Lightsaber Duel** until you have Double Multipliers active

## 🧙 Wizard Modes

### Lightsaber Duel (Mini-Wizard)

| Aspect            | Details                                                                |
| ----------------- | ---------------------------------------------------------------------- |
| **Qualification** | Complete FORCE targets 5 times, no other modes active                  |
| **Start**         | Left ramp                                                              |
| **Gameplay**      | Timed mode; shoot blue shots to help allies escape, red shots add time |
| **FORCE Targets** | Increase award values                                                  |
| **Completion**    | Shoot right loop after helping all allies                              |
| **Luke Bonus**    | 2x scoring during this mode                                            |
| **Required For**  | Jedi Multiball                                                         |

> ⚠️ **Warning:** Avoid "Lightsaber Duel jail" - don't start this without Double Multipliers active or you'll waste time for minimal points.

### Escape from Boba Fett (Mini-Wizard)

| Aspect            | Details                                                                                            |
| ----------------- | -------------------------------------------------------------------------------------------------- |
| **Qualification** | Complete 4 white arrow shots (from combos); last 2 must be in non-mode play                        |
| **Gameplay**      | Timed hurry-up with 5 sequences: left ramp → right mini-loop → right ramp → left orbit → left ramp |
| **Adding Time**   | Shoot 3-bank targets                                                                               |
| **Han Bonus**     | 2x scoring during this mode                                                                        |
| **Required For**  | Jedi Multiball                                                                                     |

### Jedi Multiball (Main Wizard Mode)

| Requirement                      | Details                           |
| -------------------------------- | --------------------------------- |
| ⚔️ Become Jedi Master             | Complete Jedi Training 4 times    |
| 🗡️ Win Lightsaber Duel            | Complete the mode                 |
| 🚀 Complete Escape from Boba Fett | Complete the mode                 |
| 💥 Destroy 100 TIE Fighters       | Cumulative across game            |
| 🌍 Start all 4 Planet Final MBs   | Death Star, Endor, Hoth, Tatooine |

**Gameplay:**
1. Battle Emperor Darth Sidious
2. Starts with 2 balls
3. Complete all blue shots to add-a-ball
4. If timer expires, red shots light for new timer
5. Four phases to win
6. Mode continues on single ball but drains if lost

**Rewards:**
- Starting points based on Victory Multiball medals collected
- Winning shows ending sequence
- After completion, returns to character selection (no super-wizard mode)

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                  | Danger                     | Mitigation                     |
| --------------------- | -------------------------- | ------------------------------ |
| **Left Orbit (fast)** | Can return SDTM            | Soft plunge, be ready to nudge |
| **Right Scoop Eject** | Fast eject down the middle | Anticipate and catch           |
| **Center 3-Bank**     | Unpredictable rebounds     | Backhand for more control      |
| **Slingshots**        | Can send ball to outlanes  | Adjust slingshot gaps if owner |

### ✅ Safe Shots

| Shot                        | Why Safe                      | Best Use                         |
| --------------------------- | ----------------------------- | -------------------------------- |
| **Left Ramp**               | Clean return to right flipper | Mode starts, multiplier building |
| **Right Ramp**              | Returns to left flipper       | Jackpots, mode progression       |
| **Mini-Loops (controlled)** | Predictable returns           | Hurry-ups, mode completion       |

### Flipper Selection

| Target          | Flipper | Notes                           |
| --------------- | ------- | ------------------------------- |
| Left Orbit      | Left    | Full swing                      |
| Left Mini-Loop  | Left    | Short flip                      |
| Left Ramp       | Left    | Standard timing                 |
| 3-Bank Targets  | Either  | Backhand from right is reliable |
| Right Mini-Loop | Right   | Short flip                      |
| Right Ramp      | Right   | Standard timing                 |
| Right Scoop     | Right   | Aim carefully                   |

### Ball Control Positions

- **Left Flipper Cradle:** Safe position for planning shots
- **Right Flipper Cradle:** Good for right ramp/scoop shots
- **Post Pass:** Use to transfer ball between flippers
- **Dead Bounce:** Let ball bounce off dead flipper to other side
