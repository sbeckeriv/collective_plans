# 🎯 James Bond 007 Pro - Complete Strategy Guide
## *Your Mission: 1 Billion Points*

> **Scoring Context:** Good tournament scores break 200M, but great scores surpass **1 billion** <kcite ref="53"/>. The key to massive scores is mastering the **7x playfield multiplier** through Bond Women collection <kcite ref="59"/>.

---

## 📋 Quick-Start Checklist for New Players

- [ ] **Plunge:** Hold left flipper → full plunge → shoot any purple shot for Super Skill Shot (+1 missile)
- [ ] **Start a mode:** Right ramp = Villain mode, Side ramp = Henchmen mode
- [ ] **Build toward multiball:** Hit drop targets → lock balls at center lane (Bird One)
- [ ] **Collect Bond Women:** Shoot upper loop when flashing
- [ ] **Use 007 Scoring:** Press action button (pink) to activate playfield multiplier
- [ ] **Use Smart Missiles:** Press action button (blue) to auto-collect hardest lit shot

## 🎰 Core Mechanics

### Skill Shots

There are **8 skill shots** available at ball start and after certain modes end <kcite ref="22"/>.

| Skill Shot             | How to Execute                                | Value             | Reward      |
| ---------------------- | --------------------------------------------- | ----------------- | ----------- |
| **Classified**         | Plunge for flashing MI6 lane (no lane change) | 500k + 200k/shot  | +1x Bonus   |
| **MI6**                | Plunge for flashing MI6 lane (can change)     | 300k + 100k/shot  | +1x Bonus   |
| **Commander Bond**     | Medium plunge → left ramp                     | 1.25M + 250k/shot | Points only |
| **Licensed To Kill** ⭐ | Full plunge → left eject                      | 2.5M + 500k/shot  | +1 Missile  |
| **Undercover** ⭐       | Hold left flipper → full plunge → right ramp  | 2.5M + 500k/shot  | +1 Missile  |
| **Double "O"** ⭐       | Hold left flipper → full plunge → side ramp   | 2.5M + 500k/shot  | +1 Missile  |
| **Eyes Only** ⭐        | Hold left flipper → full plunge → DB5 eject   | 2.5M + 500k/shot  | +1 Missile  |
| **Top Secret** ⭐       | Hold left flipper → full plunge → bumper lane | 2.5M + 500k/shot  | +1 Missile  |

⭐ = Super Skill Shots (missiles only awarded once per unique shot per game)

> **💡 Pro Tip:** The **MXV Super Skill Shot** awards 2 missiles: Full plunge → right orbit → upper loop → side ramp <kcite ref="22"/>.

## 🎱 Multiballs

### Bird One Multiball 🚀

**Qualify:** Complete drop targets → lock ball at center lane. Lock 3 balls to start <kcite ref="27"/>.

| Phase             | Objective                                                              | Value                                 |
| ----------------- | ---------------------------------------------------------------------- | ------------------------------------- |
| **Start**         | Shoot 3 upper flipper shots (side ramp/upper loop) for double jackpots | Qualifies **multiball extension**     |
| **Jackpots**      | All shots lit green                                                    | 1M + 25k per SPECTRE target hit       |
| **Super Jackpot** | After 3+ jackpots, shoot rocket                                        | Sum of jackpot base values (max 300M) |
| **Progression**   | Clear singles → doubles → triples → super → repeat                     | Values multiply                       |

> **💡 Pro Tip:** At multiball start, let balls drain below upper flipper (ball save active) and focus on the 3 upper flipper shots. This earns the **multiball extension** - when you get down to 1 ball, 2 more auto-launch! <kcite ref="27"/>

---

### Jetpack Multiball 🚁

**Qualify:** Left eject → left ramp lights to start. Subsequent starts need more shots <kcite ref="28"/>.

| Phase                   | Objective                                  | Value                            |
| ----------------------- | ------------------------------------------ | -------------------------------- |
| **Hurry-up**            | Shoot center lane to lock in jackpot value | 3M (+1M per MB started)          |
| **Jackpots**            | 5 shots at any flashing shot               | Locked hurry-up value            |
| **Double Jackpot**      | Left ramp                                  | 2x jackpot value                 |
| **Super Jackpot (Pro)** | Roving shot (R→L, starts at DB5)           | 10M (+5M per super) + add-a-ball |

> **💡 Pro Tip:** Subsequent jackpots must be at **unique shots** - plan your route!

> ⚠️ **Can only be played once per game!**

### On Her Majesty's Secret Service (Final Wizard Mode) 👑

**Requirements:** Play all 4 villain mini-wizard modes <kcite ref="39"/>

**Start:** Right ramp or side ramp (side ramp = +50 flips)

This is a **flip-count based** multi-stage wizard mode:

| Resource           | Details                                         |
| ------------------ | ----------------------------------------------- |
| **Starting Flips** | 50 (100 from side ramp) + 10 per completed mode |
| **Flip Gains**     | +5 per advance shot, +35 per stage completed    |
| **Flip Loss**      | -5 per drain (single-ball only)                 |

**Six Film Stages** (played in order you completed films):

| Stage               | Film                                                              | Objective |
| ------------------- | ----------------------------------------------------------------- | --------- |
| Dr. No              | Alternate top lanes ↔ Dragon Tank. 3 Dragon Tank hits to complete |
| From Russia         | R orbit lights combos. 5 lit R orbit shots                        |
| Goldfinger          | Goldfinger target lights shots. 6 target hits                     |
| Thunderball         | Left scoop lights roving shots. 4 scoop shots                     |
| You Only Live Twice | Rocket lane → rocket behind drops. 4 rocket shots                 |
| Diamonds            | Spinner lights shots. 5 spinner rips                              |

**MI6 Multiball (Final Phase):**
- 3-ball multiball
- 6 jackpots = totals from each film stage
- Side ramp = **Super Jackpot** (combined totals) + 10 flips
- Game ends after bonus is awarded

> ⚠️ **007 Scoring and Smart Missiles are disabled during OHMSS** <kcite ref="39"/>

## 📈 Progression Systems

### Bond Women & 007 Scoring (THE KEY TO 1 BILLION) 💃

**Qualify:** Make shots to specific features. **Collect:** Shoot upper loop when flashing <kcite ref="31"/>

| Bond Woman       | Film                | Qualifying Requirement |
| ---------------- | ------------------- | ---------------------- |

**007 Scoring Activation:**
- Press action button (pink) to activate all collected women
- Each woman = +1x playfield multiplier
- **Maximum: 7x playfield for 70 seconds + 30 second ball save!**

| Women Collected | Multiplier | Duration            |
| --------------- | ---------- | ------------------- |
| 1               | 2x         | 20s                 |

> **💡 Pro Tip:** Collected women reset on drain. Qualified women carry between balls. If you've played all 4 modes for a film, that woman **persists across balls** until used <kcite ref="31"/>.

### Smart Missiles 🚀

**Use:** Press action button (blue) to auto-collect hardest lit shot <kcite ref="32"/>

**Earn missiles from:**
- Game start: 1 missile
- Super skill shots: 1 per unique shot
- Completing any mode: 1 missile
- Martini shot combo: 1 missile
- Swimming With Sharks perk: +2 per ball
- Upgraded mystery: random award

> ⚠️ **Missiles disabled when super jackpot is lit!** Max stack: 4 missiles.

## 🔄 Stacking & Combos

### What Stacks Well ✅

| Stack                                | Notes                                     |
| ------------------------------------ | ----------------------------------------- |
| Villain/Henchmen modes + Bird One MB | Mode continues after MB <kcite ref="87"/> |
| Villain/Henchmen modes + Jetpack MB  | Mode continues after MB                   |
| Single-ball Q modes + Main MBs       | Radioactive Recon & Q's Ring stack        |
| 007 Scoring + Any mode/MB            | **This is your scoring multiplier!**      |
| SPECTRE perks + Everything           | Permanent benefits                        |

### What Does NOT Stack ❌

| Restriction                        | Reason                               |
| ---------------------------------- | ------------------------------------ |
| Q Branch 2-ball MBs + Anything     | Mutually exclusive <kcite ref="80"/> |
| SPECTRE Weapons + Multiballs       | Cannot start during MB               |
| Feeding Frenzy + Bird One progress | Uses same targets                    |
| Swipe-a-Ball + Add-a-Ball          | One per multiball (Prem/LE)          |

### Optimal Stacking Strategy

1. **Start a villain or henchmen mode**
2. **Qualify Bird One or Jetpack during the mode**
3. **Start multiball** - mode pauses but resumes after
4. **Activate 007 Scoring during multiball** for multiplied jackpots
5. **Use Smart Missiles** on hard shots during modes

## 🎯 Shot Analysis

### Dangerous Shots ⚠️

| Shot                 | Risk                                     | Mitigation                                                               |
| -------------------- | ---------------------------------------- | ------------------------------------------------------------------------ |
| **Left Eject**       | Hidden under upper flipper, easy to miss | Hold upper flipper up, shoot under it, release at apex <kcite ref="77"/> |
| **Right Orbit**      | Fast return, can SDTM                    | Soft shot or be ready for catch                                          |
| **Center Lane**      | Feeds upper flipper awkwardly            | Anticipate the feed                                                      |
| **Side Ramp (weak)** | Can reject to right outlane              | Commit to full shots                                                     |

### Safe Shots ✅

| Shot           | Why Safe                    | Best Use              |
| -------------- | --------------------------- | --------------------- |
| **Left Ramp**  | Clean feed to right flipper | Mode starts, combos   |
| **Right Ramp** | Controlled return           | Villain modes, combos |
| **DB5 Eject**  | Predictable kickout         | Q Branch qualifying   |
| **Upper Loop** | Feeds upper flipper         | Bond Women collection |

### Flipper Selection Guide

| Target      | Flipper | Notes                         |
| ----------- | ------- | ----------------------------- |
| Left Ramp   | Right   | Standard backhand             |
| Right Ramp  | Right   | Straight shot                 |
| Side Ramp   | Right   | Slight angle right            |
| Left Eject  | Upper   | Hold up, shoot under, release |
| DB5 Eject   | Left    | Backhand from left flipper    |
| Right Orbit | Left    | Backhand or right forehand    |
| Center Lane | Upper   | From upper flipper position   |
| Rocket      | Upper   | Through drop targets          |

## 📦 Additional Information

### Extra Ball Opportunities

Extra balls light at the left eject after <kcite ref="35"/>:
- 5 upper loop shots (can be as low as 1 based on percentaging)
- 3, 6, 12, or 18 total modes played
- Mystery award (percentage-based)
- Completing Bond...James Bond super jackpot

> If extra balls are disabled, 25M compensation is awarded (can be multiplied by 007 Scoring).

### High Stakes Mystery Awards

**Qualify:** Complete BOND return lanes (every 2nd completion) <kcite ref="30"/>

**Standard Awards:**
- Start Villain/Henchmen Mode
- Advance SPECTRE Letter (+2)
- Light Bird One Lock
- Advance Jetpack
- Bond Women Multiplier Lit
- Dragon Tank Lit
- Light Outlane Ball Save
- Light Extra Ball
- Bonus Multiplier +1x
- 007 Points (7,007,000)

**Upgraded Awards (after Death By Derby):**
- Bond Women Multiplier **Awarded**
- Hold Bond Women Multipliers
- +1 Smart Missile
- Outlane Ball Save Plus (flipper toggle)
- Hold Bonus X

| Setting              | Recommendation                                   |
