## 🎯 Core Mechanics

### Skill Shots

| Skill Shot Type      | How to Execute                                  | Value               | Notes                  |
| -------------------- | ----------------------------------------------- | ------------------- | ---------------------- |
| **Standard**         | Plunge into flashing rollover                   | 200K (+50K each)    | Right lane = 2x value  |
| **Super Skill Shot** | Hold left flipper + plunge + hit any major shot | 2x standard (+100K) | Disables orbit up-post |

#### 🌟 Super Skill Shot Bonus Awards

| Target                 | Bonus Award                                      |
| ---------------------- | ------------------------------------------------ |
| Dead End Target        | Immediately qualifies 2x Playfield               |
| Scoop                  | Awards Mystery                                   |
| Blue Spider Arrow Shot | Starts Coffin Hurry-Up + 200K to next skill shot |
| Rollover Lanes         | 2x value (4x if right lane!)                     |

> **💡 Pro Tip:** On Ball 1 in competition mode, the blue spider arrow is always on the left orbit. Use this for a guaranteed Coffin Hurry-Up start!

---

### Mode Overview

Metallica features **four main Crank It Up modes**, **six multiball modes**, and numerous **side modes** that can stack together for massive scoring.

#### Crank It Up Modes (Main Single-Ball Modes)

| Mode                        | Song                    | Objective                                              | Completion Bonus    |
| --------------------------- | ----------------------- | ------------------------------------------------------ | ------------------- |
| **For Whom the Bell Tolls** | For Whom the Bell Tolls | Light all 15 item inserts (3 per shot × 5 shots)       | 15M + 2.5M per mode |
| **Fade to Black**           | Fade to Black           | Collect switch hits, advance 12 levels                 | 15M + 2.5M per mode |
| **Battery**                 | Battery                 | Shoot Sparky 12 times, build value with other shots    | 15M + 2.5M per mode |
| **Enter Sandman**           | Enter Sandman           | Light all 15 inserts solid (alternating with switches) | 15M + 2.5M per mode |

> **💡 Pro Tip:** These modes last as long as the song (~5-8 minutes each). Take your time! Ball save only lasts the first few seconds.

#### Starting Crank It Up
1. Collect **12 items** of each type (Electric Chairs, Grave Markers, Snakes, Coffins)
2. Return to single-ball play
3. Shoot the **scoop** to select your mode

---

### Collecting Items

Items are collected by:
- **Direct shots** to features (Electric Chair, Grave Markers, Snake, Coffin captive ball)
- **Shooting lit inserts** at major shots (placed by the blue spider arrow)
- **Self-Destruct Button** (collects all lit inserts + 1 coffin per complete column)

| Item             | Feature             | Direct Shot Value             |
| ---------------- | ------------------- | ----------------------------- |
| ⚡ Electric Chair | Sparky              | 1 hit (2 for center target)   |
| ✝️ Grave Marker   | Inline drop targets | 1 per target                  |
| 🐍 Snake          | Snake mouth         | 1 per shot                    |
| ⚰️ Coffin         | Captive ball only   | 1 per hit (2 with lit inlane) |

> **💡 Pro Tip:** Combos ending at the captive ball add +1 coffin hit per shot in the combo!

### ⚡ Electric Chair Multiball (James)

**Songs:** Creeping Death (odd), Ride the Lightning (even)

**Jackpot Structure:**

| Award         | Value                       |
| ------------- | --------------------------- |
| Jackpot       | 250K + (2.5K × total items) |
| Super Jackpot | 1.5M + (5K × total items)   |

**How to Play:**
1. Fill Jacob's Ladder by hitting jackpot shots (orbits + Sparky)
2. Shoot Sparky for Super Jackpot when full
3. Electrocute 7 Sparkys to light ALL shots for supers

> **💡 Pro Tip:** Each jackpot counts as 10 switch hits toward filling the ladder!

---

### ✝️ Grave Marker Multiball (Robert)

**Song:** Master of Puppets

**Jackpot Structure:**

| Award             | Value                                          |
| ----------------- | ---------------------------------------------- |
| Jackpot           | 200K + (2.5K × total items)                    |
| Grave Marker Shot | 250K + relights all jackpots                   |
| Double Jackpot    | 2× Grave Marker value                          |
| Super Jackpot     | 1.25M + (2.5K × items) × multiplier (up to 5x) |

**How to Play:**
1. Shoot 4 jackpots to light Double Jackpot at grave marker
2. Score Double Jackpot → ball held on magnet for 20 seconds
3. Hit 3 unique shots while held to increase Super Jackpot multiplier
4. Knock ball off magnet for Super Jackpot

> **💡 Pro Tip:** The Super Jackpot multiplier is PERMANENT for the rest of the game!

---

### 🐍 Snake Multiball (Kirk)

**Song:** Sad But True

**Jackpot Structure:**

| Award         | Value                         |
| ------------- | ----------------------------- |
| Jackpot       | 300K + (3K × total items)     |
| Snake Combo   | Jackpot × multiplier (max 3x) |
| Super Jackpot | 750K + (2.5K × total items)   |

**How to Play:**
1. Hit any shot to build Snake Combo multiplier (+1x per shot)
2. Collect at Snake for multiplied jackpot
3. Score 5 Snake Combos to light all shots for Super Jackpots

---

### ⚰️ Coffin Multiball (Lars)

**Song:** For Whom the Bell Tolls

**Jackpot Structure:**

| Award         | Value                     |
| ------------- | ------------------------- |
| Jackpot       | 400K + (4K × total items) |
| Super Jackpot | 3M + (10K × total items)  |

**How to Play:**
1. Alternate between captive ball and flashing shots
2. Hit all flashing shots once to light Super Jackpot at captive ball

> **🔥 CRITICAL:** Coffin Multiball is UNIQUE — you can start ONE other multiball during it! This is essential for stacking strategies.

---

### 🎸 Frantic Multiball

**Song:** Frantic

**Qualification:** Collect 10 Setlist Combo pairs

**How to Play:**
1. 4-ball multiball with color-changing shots (white → yellow → blue → green → pink)
2. Match 2+ shots of same color for jackpots (10× shot value)
3. Match 3+ shots to light Super Jackpot + add-a-ball at piston target
4. Super Jackpot multiplier = matching shots - 2

---

### ⚖️ And Justice for All Multiball (Topper Only)

**Song:** ...And Justice for All

**Qualification:** Spell METALLICA on topper during Lady Justice

**How to Play:**
1. Alternate left/right shots for jackpots (single = 1 letter, double = 2 letters)
2. Score 3 jackpots to light Super Jackpot at ramps
3. Spell METALLICA again to increase Super Jackpot multiplier
4. First 3 Super Jackpots add a ball

---

### 🔓 Multiball Add-a-Ball (All Multiballs)

During ANY multiball:
1. Complete 4 **Pick targets** (around left ramp area)
2. Lock ball at **Snake** within 20 seconds
3. Awards: **Add-a-ball** + **2x Playfield for 20 seconds** + ball save

> **💡 Pro Tip:** Hit 3 of 4 Pick targets BEFORE starting multiball, then complete the 4th immediately after!

## 📈 Progression Systems

### Band Member Levels

Band members level up (max level 8) by playing multiballs and scoring jackpots:

| Member     | Multiball      | Level Up Methods                                               |
| ---------- | -------------- | -------------------------------------------------------------- |
| **James**  | Electric Chair | +1 per start, +1 per Sparky fried                              |
| **Robert** | Grave Marker   | +1 per start, +1 per double jackpot, +1 per super × multiplier |
| **Kirk**   | Snake          | +1 per start, +1 per snake combo/super jackpot                 |
| **Lars**   | Coffin         | +1 per start, +1 per captive ball jackpot                      |

**Max Level (8) Benefits in Blackened:**

| Max Members | Bonus                                         |
| ----------- | --------------------------------------------- |
| 1           | 1 add-a-ball button                           |
| 2           | 1 add-a-ball button + 1 extra starting ball   |
| 3           | 2 add-a-ball buttons + 1 extra starting ball  |
| 4           | 2 add-a-ball buttons + 2 extra starting balls |

> **💡 Pro Tip:** Band member levels carry over the ENTIRE game. Focus on maxing them early!

### Item Collection Progress

| Progress Type           | Behavior                                            |
| ----------------------- | --------------------------------------------------- |
| Items collected         | Cumulative per game                                 |
| Lit inserts at shots    | Reset each ball (unless locked via Coffin Hurry-Up) |
| Crank It Up requirement | 12 items (+1 per mode played)                       |

### Coffin Hurry-Up System

1. Light all 3 item inserts at one shot
2. Shoot that shot to start Coffin Hurry-Up
3. Collect at captive ball to **LOCK** those inserts permanently
4. Complete all 5 shots → Start **Seek & Destroy**

**Hurry-Up Value:** 750K + (50K × shot levels of items used)

## 🔗 Stacking & Combos

### ✅ What Stacks Well

| Combination                      | Why It's Good                           |
| -------------------------------- | --------------------------------------- |
| **Coffin MB + Any Other MB**     | Only multiball that allows stacking!    |
| **Side Modes + Crank It Up**     | Side modes continue during CIU          |
| **FUEL 2x + Snake 2x**           | 4x Playfield scoring!                   |
| **72 Seasons + Multiball**       | 72 Seasons continues through everything |
| **Seek & Destroy + Crank It Up** | All shots award items during S&D        |

### ❌ What NOT to Stack

| Avoid                                     | Reason                                     |
| ----------------------------------------- | ------------------------------------------ |
| **Electric Chair + Grave Marker + Snake** | Mutually exclusive (only one at a time)    |
| **Crank It Up + New Side Modes**          | Can't START new side modes during CIU      |
| **And Justice for All + Other MBs**       | Can only stack with Coffin (started first) |
| **Blackened + Crank It Up**               | Must choose one at scoop                   |

### Optimal Stacking Strategy

1. **Start Coffin Multiball first**
2. **Qualify another multiball** during Coffin
3. **Start the second multiball** for double multiball action
4. **Complete Pick targets** for 2x playfield + add-a-ball
5. **Have FUEL 2x ready** for potential 4x scoring

## 🏆 Wizard Modes

### ⚫ Blackened (Mini-Wizard Mode)

**Qualification:** Play all 4 multiballs (Electric Chair, Grave Marker, Snake, Coffin)

**Starting Value:** 800K + (100K × total band member levels)

**Gameplay:**
1. 4-ball multiball (+ extras based on max-level members)
2. All scoring multiplied by (balls in play - 1)
3. Red shots build Super Jackpot value
4. Purple shot scores jackpot and "blackens" that shot
5. Captive ball moves the purple shot
6. Blacken 3 shots → add-a-ball
7. Blacken ALL shots → Super Jackpot + add-a-ball + Self-Destruct + UNFORGIVEN letter

> **💡 Pro Tip:** Max out band members BEFORE starting Blackened for massive starting bonuses!

---

### 🔚 End of the Line (Main Wizard Mode)

**Qualification:** Play all 4 Crank It Up modes + collect items again

**Starting Bonus:** 200K × total items collected in game

**Gameplay:**
1. 2-ball multiball, 1 shot lit for jackpot
2. Collect jackpot → roving Super Jackpot lights
3. Collect Super Jackpot → add-a-ball + ball save
4. Each Super requires +1 more jackpot shot
5. Super Jackpot values = 25% of your CIU scores (lowest to highest)

**Super Jackpot Calculation Example:**
| CIU Mode                | Score | 25% Value |
| ----------------------- | ----- | --------- |
| Enter Sandman           | 150M  | 37.5M     |
| For Whom the Bell Tolls | 200M  | 50M       |
| Fade to Black           | 400M  | 100M      |
| Battery                 | 600M  | 150M      |

- Super 1: 37.5M
- Super 2: 37.5M + 50M = 87.5M
- Super 3: 87.5M + 100M = 187.5M
- Super 4: 187.5M + 150M = 337.5M
- Super 5: 337.5M + 37.5M = 375M (wraps around)

> **🔥 CRITICAL:** Both FUEL 2x AND Snake 2x work during End of the Line! You can 4x these Super Jackpots!

---

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots & Drain Paths

| Danger Zone                           | Risk Level | Description                                       |
| ------------------------------------- | ---------- | ------------------------------------------------- |
| **Left Outlane Area**                 | 🔴 EXTREME  | Balls bounce off inlane rail into sling → outlane |
| **Center Drain (Sparky funnel)**      | 🔴 HIGH     | Bumper exits feed center drain                    |
| **FUEL Lane**                         | 🟠 HIGH     | Feeds directly into left outlane area             |
| **Slingshots**                        | 🟠 MEDIUM   | Keep ball away — very hungry outlanes             |
| **Electric Chair from Right Flipper** | 🟠 MEDIUM   | Nearly impossible angle, high brick rate          |

### ✅ Safe Shots & Ball Control

| Shot              | Safety   | Best Flipper       | Notes                                                |
| ----------------- | -------- | ------------------ | ---------------------------------------------------- |
| **Snake**         | 🟢 Safe   | Either             | Easiest of the main features                         |
| **Sparky**        | 🟢 Safe   | Left               | Relatively safe from left flipper                    |
| **Grave Markers** | 🟢 Safe   | Left (backhand)    | Accessible from both, safer as backhand              |
| **Left Ramp**     | 🟢 Safe   | Right              | Standard ramp shot                                   |
| **Right Ramp**    | 🟢 Safe   | Left               | Standard ramp shot                                   |
| **Captive Ball**  | 🟡 Medium | Either             | Graze right → feeds Snake; graze left → feeds Sparky |
| **Orbits**        | 🟡 Medium | Backhand preferred | Good for combos                                      |

### Ball Control Positions

| Position                 | Use For                                                   |
| ------------------------ | --------------------------------------------------------- |
| **Left flipper cradle**  | Sparky, right ramp, grave markers (backhand), right orbit |
| **Right flipper cradle** | Left ramp, Snake, captive ball, left orbit                |
| **Post pass**            | Transfer between flippers safely                          |
| **Dead bounce**          | Let ball bounce to opposite flipper                       |

> **💡 Pro Tip:** The captive ball can be "grazed" to feed either the Snake (from right) or Electric Chair (from left). Master this shot!

## ✅ Quick-Start Checklist

### 🎸 Before You Play
- [ ] Check pitch is ~6.5°
- [ ] Verify game is level side-to-side
- [ ] Note if topper is installed (enables And Justice for All MB)

### 🎯 Ball 1 Goals
- [ ] Make skill shot (hold left flipper for super skill shot)
- [ ] Start Grave Marker Multiball (safest first MB)
- [ ] Hit Pick targets during multiball for add-a-ball
- [ ] Light Resurrection ball save
- [ ] Begin collecting items toward Crank It Up

### 🔥 Every Ball Goals
- [ ] Complete FUEL targets → light 2x Playfield
- [ ] Work toward Coffin Hurry-Ups (lock item inserts)
- [ ] Build band member levels through multiballs
- [ ] Complete rollover lanes for Mystery + Super Pistons

### 🏆 Path to Billion
1. [ ] Play all 4 multiballs → Qualify Blackened
2. [ ] Max band member levels → Better Blackened bonuses
3. [ ] Play Blackened → Score Super Jackpot
4. [ ] Collect items → Start Crank It Up modes
5. [ ] Complete all 4 CIU modes with 2x active
6. [ ] Qualify End of the Line
7. [ ] Stack FUEL 2x + Snake 2x during EOTL
8. [ ] Collect all Super Jackpots at 4x
9. [ ] 🎉 **BILLION ACHIEVED!**
