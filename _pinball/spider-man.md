# 🕷️ Spider-Man - Strategy Guide

*Stern Pinball, 2007 — Designed by Steve Ritchie, Software by Lyman F. Sheats Jr.*

> **"With great power comes great jackpots."**

Spider-Man (Stern, 2007) is a Stern Whitestar/SAM-era machine featuring four iconic Marvel villains — Doc Ock, Green Goblin, Venom, and Sandman — each with a dedicated toy and unique mode mechanics. Doc Ock's magnet holds balls for a "Fusion Malfunction" event, Green Goblin hovers above pumpkin bomb targets, Venom's ramp feeds back immediately to the left flipper, and Sandman's motor-driven 3-bank spins in the center. Our goal: **300M–1B** by defeating all four villains and stacking their modes with Web Multiball.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Shoot the Green Goblin pumpkin bomb targets** to start Green Goblin mode
- [ ] **Complete the Doc Ock arms** (left orbit shots) to qualify Doc Ock multiball
- [ ] **Shoot Venom's ramp** — the ball returns immediately to the left flipper for fast combos

- [ ] **Lock balls for Web Multiball** via the right scoop (lit by villain progress)
- [ ] **Stack an active villain mode with Web Multiball** for doubled jackpot values

### Four Villains

**Doc Ock (Left Orbit — Arms)**
- Shoot the left orbit repeatedly to extend Doc Ock's arms (each orbit = one arm)
- Extend all 4 arms → Doc Ock captures the ball via the magnet ("Fusion Malfunction")
- During Fusion Malfunction, press the action button (or wait) to release → bonus award
- Completing Doc Ock's mode = significant score + Web Multiball lock credit

**Green Goblin (Pumpkin Bomb Targets — Upper Right)**
- Three pumpkin bomb targets above the right ramp
- Hit all 3 targets to activate Green Goblin mode
- Mode: shoot the lit shots while the Goblin hovers overhead; each successful shot detonates a bomb for 5M–25M
- Completing the mode knocks the Goblin down (huge victory animation)

**Venom (Left Ramp — "Venom's Ramp")**
- Shoot Venom's ramp (left side) — the ball rides a wireform and returns immediately to the left flipper
- The return is very fast; be ready to catch and shoot again
- Venom's mode: chain consecutive left ramp shots for escalating combo multipliers
- Venom is the best combo-farming shot on the machine

**Sandman (Motor 3-Bank — Center)**
- The Sandman 3-bank physically rotates via a motor — the drop targets spin to different positions
- Timing the shot to hit the lit target when it's aligned is key
- Completing the bank starts Sandman's mode: timed target sequence for escalating jackpots

### Web Multiball (Primary Multiball)

**How to start:** Accumulate Web locks by completing villain milestones (typically 2 villain events = 1 lock credit). Lock 2 balls via the right scoop.

**Jackpot structure:**
- 3-ball multiball
- All four villain shots are active jackpots simultaneously
- **Super Jackpot:** Complete all four villain jackpots in sequence → shoot the right scoop → 50M–200M+
- Villain modes started before Web Multiball continue during multiball, doubling income

### Dangerous Areas

- **Venom's ramp return** — ball comes back very fast; a slow left flipper will drain
- **Post-Doc Ock magnet** — when the magnet releases, direction varies; both flippers ready
- **Sandman timing** — shooting the motor bank at the wrong rotation can deflect sideways

## 💡 Pro Tips

> **Venom's ramp is your combo engine.** During multiball, stay on Venom's ramp and let the combo multiplier ride — 4× multiball jackpots are possible with just this one shot.

> **Don't ignore Sandman timing.** The rotating bank looks intimidating but aligns predictably; watch it for two rotations and shoot when the lit target faces you.

> **Stack before locking.** Start at least 2 villain modes before triggering Web Multiball — the overlap pays enormously.
