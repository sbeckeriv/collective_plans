# 🖖 Star Trek Pro (Stern, 2013) - Ultimate Strategy Guide
## 🎯 Goal: 1 Billion Points

> **"Dammit Jim, I'm a doctor, not a pinball wizard!"** — Bones Combo Quote

This is Stern's first all-LED Pro Model, designed by legendary pinball designer Steve Ritchie <kcite ref="69"/><kcite ref="42"/>. Released October 1, 2013, this fast-flowing game features the 2009/2013 J.J. Abrams Star Trek films <kcite ref="43"/>.

## 🎱 Core Mechanics

### Skill Shots

There are **3 different skill shots** available <kcite ref="94"/>:

| Skill Shot                | How to Execute                                                                 | Value                                |
| ------------------------- | ------------------------------------------------------------------------------ | ------------------------------------ |
| **Warp Ramp Skill Shot**  | Short plunge so ball trickles to upper right flipper, then shoot warp ramp     | Variable                             |
| **Super Skill Shot**      | Hold LEFT flipper during plunge, shoot lit Vengeance target OR right ramp      | 1M first time, +250K each subsequent |
| **Super Warp Skill Shot** | Hold RIGHT flipper, full plunge to reject ball from rollovers, shoot warp ramp | 1.5M                                 |

> 💡 **Pro Tip:** The Super Skill Shot value increases throughout the game. Save it for later balls when it's worth more!

---

### Mission System

Your main objective is completing **6 missions**, each with **3 phases (I, II, III)** <kcite ref="101"/>:

| Mission                   | Color      | Shots Lit (Phase I)         |
| ------------------------- | ---------- | --------------------------- |
| 🔵 **Prime Directive**     | Dark Blue  | All shots                   |
| 🔴 **Klingon Battle**      | Red        | Orbits & Ramps              |
| 🩵 **Destroy the Drill**   | Light Blue | Left side shots             |
| 🟣 **Nero**                | Purple     | Orbits & Ramps              |
| 🟢 **Save the Enterprise** | Green      | Left Orbit, Left Ramp, Warp |
| 🟡 **Space Jump**          | Yellow     | Orbits & Ramps              |

**Starting Missions:**
- On ball start: Choose from offered missions (use flipper buttons to select, Fire button to launch)
- Scoop (when lit blue): Choose and start a mission
- Galactic Away Team shot: Choose and start a mission

> ⚠️ **Important:** You cannot stack multiple missions simultaneously, but you CAN stack a mission with multiballs!

### Mission Details

#### 🔵 Prime Directive
- **Phase I:** All shots lit blue. Make shots to advance. GAT lights Galactic Lanes (top rollovers + orbits for big points, +20 sec) <kcite ref="102"/>
- **Phase II:** 30-second timer to clear all lit shots. Yellow/blue blinking shot scores double. Progress saves if you fail <kcite ref="71"/>
- **Phase III:** Similar structure, increased difficulty

#### 🔴 Klingon Battle
- **Phase I:** Orbits and ramps lit. Cannot make same shot twice in succession. GAT lights Galactic Ramps (alternating left/right ramps for huge points) <kcite ref="103"/>
- **Phase II:** Left ramp repeatedly. Klingons approach - shoot them when far away for more points <kcite ref="72"/>
- **Phase III:** One shot lit for big points. If missed, second shot lights for less, and so on <kcite ref="71"/>

#### 🩵 Destroy the Drill
- **Phase I:** Left scoop, left orbit, left ramp, warp lit. Shots retract left when made. GAT lights Galactic Spinner (75K per spin) <kcite ref="104"/>
- **Phase II:** Shots light clockwise/counterclockwise. Make 6 of 7 shots. Precision required <kcite ref="71"/>
- **Phase III:** VUK, Vengeance, Right orbit. Multi-colored shot lights periodically for 5 sec - big points <kcite ref="71"/>

#### 🟣 Nero
- **Phase I:** All orbits and ramps lit. GAT lights Galactic Targets (all black hole targets for big points) <kcite ref="105"/>
- **Phase II:** 3 pairs of shots. Make a pair to shoot a Romulan. Destroy all 3 Romulans before another appears for bonus jackpot <kcite ref="72"/>
- **Phase III:** 2 roving shots controlled by flipper buttons. Timed mode <kcite ref="71"/>

#### 🟢 Save the Enterprise
- **Phase I:** Left Orbit, Left Ramp, Warp lit. GAT lights Galactic Warp Ramp <kcite ref="106"/>
- **Phase II & III:** Not timed - good for stacking with multiballs <kcite ref="71"/>

#### 🟡 Space Jump
- **Phase I:** Orbits and ramps lit. GAT lights Galactic Vengeance (single center shot ends mode) <kcite ref="107"/>
- **Phase II & III:** Focus on orbit/ramp combos. Completing all 3 lights right orbit 2x (not the best reward) <kcite ref="71"/>

## 🎰 Multiballs

### ⚔️ Klingon Multiball

**How to Qualify:** Hit the 3 green Klingon standup targets to light locks at Vengeance and/or left scoop <kcite ref="95"/>

**How to Start:** Lock 3 balls

**Gameplay:**
- 3-ball multiball (can become 4-ball via Black Hole award)
- All shots lit RED for Klingon Jackpots

| Jackpot                                 | Value            |
| --------------------------------------- | ---------------- |
| 1st Jackpot                             | 200,000          |
| 2nd Jackpot                             | 375,000          |
| 3rd Jackpot                             | 750,000          |
| (continues increasing)                  | ...              |
| **Super Jackpot** (after 6 jackpots)    | 6.25M            |
| **Double Super Jackpot** (after 6 more) | 20M + Gold Medal |

> 🎯 **Strategy:** The Double Super Jackpot shot travels around the playfield quickly. Stack with Warp 9.1 double scoring for massive points! <kcite ref="137"/>

---

### 🚀 Vengeance Multiball

**How to Qualify:** Hit center Vengeance drop target 5 times <kcite ref="111"/>

**How to Start:** Shoot behind the dropped target to hit Vengeance

**Gameplay:** <kcite ref="96"/>
1. Ball held on magnet, second ball served
2. 15-second timer to free the captive ball
   - Success = 3-ball multiball
   - Fail = 2-ball multiball
3. Destroy Vengeance by hitting it or using Photon Torpedoes
4. Destroying Vengeance = 5M (10M as combo!)
5. **Victory Laps:** All shots lit, completing a round scores 1M, 2M, 3M...

> 💡 **Pro Tip:** Black Hole target can add a 4th ball during multiball!

---

### 🎯 Vengeance Scoring (Single Ball Mode)

**How to Start:** After Vengeance Multiball, hit Vengeance 6 more times <kcite ref="115"/>

**Gameplay:** Timed hurry-up - hit Vengeance 5 times before points drop to 500K

| Shot | Starting Value | Minimum |
| ---- | -------------- | ------- |
| 1st  | 1,000,000      | 500,000 |
| 2nd  | 1,100,000      | 500,000 |
| 3rd  | 1,200,000      | 500,000 |
| 4th  | 1,300,000      | 500,000 |
| 5th  | 1,400,000      | 500,000 |

> 🎁 **Reward:** Completing all 5 shots awards an **Extra Ball** instantly!

---

### ⚡ Vengeance Battle (The Big Money Shot!)

**How to Start:** After Vengeance Scoring, hit Vengeance 5 more times <kcite ref="116"/>

**Gameplay:** "Sudden death" mode
- Hit the target once more, it drops briefly
- Shoot into Vengeance = **10% of your current score!**

| Multiplier                 | Total Award       |
| -------------------------- | ----------------- |
| Base                       | 10% of score      |
| With Combo                 | 20% of score      |
| With Double Scoring (9.1)  | 20% of score      |
| **Combo + Double Scoring** | **40% of score!** |

> 🚨 **BILLION POINT STRATEGY:** At 300M score, a perfect Vengeance Battle with combo + double scoring = **120M jackpot!** <kcite ref="138"/>

## 📈 Progression Systems

### Warp Speed Levels

Shoot the Warp Ramp repeatedly to increase warp level <kcite ref="98"/>:

| Warp Level        | Award                                     |
| ----------------- | ----------------------------------------- |
| Each shot         | 250K × 2 (doubles with consecutive shots) |
| ~6-7 (adjustable) | **Extra Ball lit**                        |
| **9.1**           | **Double Scoring for 60 seconds!**        |
| **9.9**           | **Double Scoring + Add-a-Ball!**          |

After 9.9, warp resets to 1. Extra Ball only lights once per cycle on factory settings.

> 🎯 **Critical Strategy:** Save Warp 9.1 and 9.9 for wizard modes, Klingon MB, or Vengeance Battle for MASSIVE scores!

---

### Three in a Row → Captain's Chair

Complete 3 missions in a row (horizontally on the mission grid) for Captain's Chair awards <kcite ref="108"/>:

| Row      | Award             | Objective           | Points    |
| -------- | ----------------- | ------------------- | --------- |
| Left 3   | **Super Pops**    | 100 pop bumper hits | 90K each  |
| Bottom 3 | **Super Spinner** | 100 spins           | 75K each  |
| Right 3  | **Super Ramps**   | 9 ramp shots        | 700K each |

> ✅ These awards have **no time limit** and carry over between balls!

---

### Three in a Column → Permanent 2x Multiplier

Complete all 3 phases of one mission for a permanent shot multiplier <kcite ref="109"/>:

| Mission Completed            | 2x Shot     |
| ---------------------------- | ----------- |
| Prime Directive I-II-III     | Saucer      |
| Klingon Battle I-II-III      | Left Ramp   |
| Space Jump I-II-III          | Right Orbit |
| Destroy the Drill I-II-III   | Warp Ramp   |
| Nero I-II-III                | Left Orbit  |
| Save the Enterprise I-II-III | Right Ramp  |

> 💡 **Pro Tip:** Prioritize Nero or Klingon Battle for the most useful 2x shots!

### TREK Rollovers

## 🧙 Wizard Modes

### 🏆 Kobayashi Maru (Tier 1 Wizard Mode)

**Requirement:** Complete all 6 Level I missions <kcite ref="97"/>

**Gameplay:**
- 2-ball multiball (3-ball with Black Hole)
- 6 shots lit (one per mission)
- 1st shot of each = base points
- 2nd shot of each = points earned in that mode (up to 7M)
- 12 total shots to complete

**Medal Bonus at Start:**
- Full Bronze set = 10M
- Full Silver set = 20M
- Full Gold set = 35M

> 🎯 **Strategy:** Delay starting KM to earn Level 2 and 3 medals first for bigger bonuses! <kcite ref="139"/>

---

### 🏆 Enterprise Amok (Tier 2 Wizard Mode)

**Requirement:** Complete all 6 Level II missions <kcite ref="97"/>

**Gameplay:**
- 2-ball multiball (up to 4-ball)
- Switch-based scoring - every switch counts!

**Medal Bonus at Start:**
- Full Bronze set = 15M
- Full Silver set = 30M
- Full Gold set = 50M

---

### 🏆 Five Year Mission (Final Wizard Mode)

**Requirement:** Complete all 6 Level III missions

**Gameplay:** The ultimate challenge - details vary but represents the culmination of all progression.

## 🔄 Stacking & Combos

### ✅ What Stacks Well

| Stack                                      | Why It's Good                                                          |
| ------------------------------------------ | ---------------------------------------------------------------------- |
| **Mode + Multiball**                       | Complete mode objectives while collecting jackpots                     |
| **Warp 9.1 + Klingon MB**                  | Double all jackpots for 60 seconds                                     |
| **Warp 9.1 + Vengeance Battle**            | Potential 40% of score award                                           |
| **GAT (Galactic Targets) + Klingon MB**    | Balls bouncing off red matter targets = huge points <kcite ref="140"/> |
| **Captain's Chair + Any Mode**             | Passive scoring while focusing on objectives                           |
| **Save the Enterprise II/III + Multiball** | Not timed, easy to complete                                            |

### ❌ What NOT to Stack

| Avoid                             | Why                                              |
| --------------------------------- | ------------------------------------------------ |
| **Space Jump GAT**                | Immediately ends mode with single Vengeance shot |
| **Timed modes without multiball** | Too much pressure, likely to fail                |
| **Multiple missions**             | Not possible - only one mission at a time        |

### ⏱️ Optimal Timing

> 💡 **Pro Tip:** Start a multiball JUST AFTER starting a mode. This gives you multiple balls to complete mode objectives while also collecting jackpots! <kcite ref="140"/>

**Ideal Sequence:**
1. Lock 2 balls for Klingon MB
2. Start a mode (Nero I is good for GAT stacking)
3. Lock 3rd ball → Klingon MB starts
4. Complete mode while collecting jackpots

### ⚠️ Dangerous Shots

| Shot                   | Risk                                  | Mitigation                      |
| ---------------------- | ------------------------------------- | ------------------------------- |
| **Right Orbit**        | Can rattle back SDTM                  | Nudge right if ball hesitates   |
| **Vengeance (center)** | Rebounds unpredictably                | Be ready for quick flipper save |
| **Weak Warp Ramp**     | Rolls back to right flipper or drains | Full power shots only           |
| **Pop Bumpers**        | Random exits toward outlanes          | Nudge to guide ball             |

### ✅ Safe Shots

| Shot                   | Why Safe                             | Best Use                   |
| ---------------------- | ------------------------------------ | -------------------------- |
| **Left Ramp**          | Clean feed to right flipper          | Mode shots, combos         |
| **Left Orbit (full)**  | Feeds to right flipper via habitrail | Building warp, combos      |
| **Warp Ramp (strong)** | Returns to left flipper              | Warp building, skill shots |

### 🎮 Flipper Selection

| Target                  | Use                                |
| ----------------------- | ---------------------------------- |
| **Left Flipper**        | Warp Ramp, Right Orbit, Right Ramp |
| **Right Flipper**       | Left Orbit, Left Ramp, Vengeance   |
| **Upper Right Flipper** | Warp Ramp (skill shot), Away Team  |

> 💡 **Pro Tip:** Don't flip both flippers at the same time! <kcite ref="141"/>

## 💰 Scoring Strategy: Path to 1 Billion

### Early Game (0-100M)

| Priority | Action                        | Expected Points |
| -------- | ----------------------------- | --------------- |
| 1        | Complete 2-3 Level I missions | 5-15M           |
| 2        | Build warp toward 9.1         | 2-5M            |
| 3        | Light Klingon locks           | Setup           |
| 4        | Collect skill shots           | 1-3M            |
| 5        | Start Vengeance progression   | Setup           |

### Mid Game (100-400M)

| Priority | Action                                     | Expected Points |
| -------- | ------------------------------------------ | --------------- |
| 1        | Complete Level I missions → Kobayashi Maru | 30-70M          |
| 2        | Stack modes with Klingon MB                | 20-50M          |
| 3        | Complete "Three in a Column" for 2x shot   | Multiplier      |
| 4        | Collect Captain's Chair awards             | 9-15M each      |
| 5        | Progress through Vengeance modes           | 10-30M          |

### Late Game (400M-1B)

| Priority | Action                                   | Expected Points |
| -------- | ---------------------------------------- | --------------- |
| 1        | **Vengeance Battle with 9.1 + Combo**    | 80-160M         |
| 2        | Enterprise Amok with medals              | 50-100M         |
| 3        | Klingon MB Double Super Jackpot with 9.1 | 40-80M          |
| 4        | Five Year Mission                        | Variable        |

### 🎯 The Billion Point Formula

1. **Build score to 300M+** through normal play
2. **Save Warp 9.1** for Vengeance Battle
3. **Combo into Vengeance Battle** for 40% of score
4. **Repeat** with subsequent Vengeance cycles

> 🚨 **Key Insight:** Vengeance Battle's percentage-based scoring means it gets MORE valuable as your score increases. A 40% award at 500M = 200M! <kcite ref="138"/>
