# 🛩️ F-14 Tomcat Pinball Strategy Guide

> **"It's fast. It's mean. It's F-14 Tomcat."**

## 📊 Scoring Reality Check

F-14 Tomcat uses **7-digit displays**, meaning the maximum displayable score is **9,999,999** per display segment <kcite ref="12"/>. The game does NOT support billion-point scoring. 

| Score Level          | Points            | Skill Level          |
| -------------------- | ----------------- | -------------------- |
| 🥉 Beginner           | 1-3 Million       | Learning the ropes   |
| 🥈 Intermediate       | 3-8 Million       | Consistent multiball |
| 🥇 Expert             | 8-15 Million      | Multiple jackpots    |
| 🏆 **Grand Champion** | **15-35 Million** | Elite play           |

## 🎯 Core Mechanics

### T-O-M-C-A-T Targets

The heart of F-14 Tomcat gameplay revolves around spelling TOMCAT to unlock features <kcite ref="21"/>.

| Target Set      | Location                     | Best Flipper   |
| --------------- | ---------------------------- | -------------- |
| **TOM** (upper) | Between orbit and jet bumper | Upper flippers |
| **CAT** (upper) | Between orbit and jet bumper | Upper flippers |
| **TOM** (lower) | Left side, mid-playfield     | Lower left     |
| **CAT** (lower) | Right side, mid-playfield    | Lower right    |

> 💡 **Pro Tip:** Focus on the **upper TOMCAT targets** and largely ignore the lower ones. The upper targets are safer and also advance Hot Streak bonus <kcite ref="42"/>.

**Completing TOMCAT lights:**
- 🔴 **LOCKS** - Ball lock for multiball
- 🎱 **MB** - Multiball ready
- 🚀 **RELEASE** - Start multiball shot
- 💰 **RIPOFF** - Alternate multiball start

### Skill Shot / Plunge

F-14 uses a **manual plunger** with no traditional skill shot <kcite ref="69"/>. The ball launches into a wireframe that delivers it to the upper playfield <kcite ref="31"/>.

**Plunge Strategy:**
- The machine will **auto-fill any lit locks** from plunger launches before putting a ball into play <kcite ref="46"/>
- Use this to your advantage when locks are lit
- Multiple plunges may be needed

### Launch Bonus

A **left inlane → right VUK** combo awards Launch Bonus <kcite ref="27"/>:

| Combo # | Value   |
| ------- | ------- |
| 1st     | 50,000  |
| 2nd     | 100,000 |
| 3rd     | 150,000 |
| 4th     | 200,000 |
| 5th+    | 250,000 |

This shot also **spots a TOMCAT letter** and awards a lock if lit <kcite ref="103"/>.

### Hot Streak

1. Shoot the **Flyby orbit** to start Hot Streak <kcite ref="21"/>
2. Hit **TOMCAT targets repeatedly** to advance Hot Streak Bonus
3. Collect via subsequent Flyby shots

## 🎱 Multiball

### Qualifying Multiball

F-14 features a **4-ball multiball** - one more than its predecessor High Speed <kcite ref="60"/>.

**Steps to Start Multiball:**

| Step | Action                    | Notes                       |
| ---- | ------------------------- | --------------------------- |
| 1️⃣    | Spell TOMCAT              | Lights first lock           |
| 2️⃣    | Shoot Right VUK           | Lock ball #1                |
| 3️⃣    | Spell TOMCAT again        | Lights second lock          |
| 4️⃣    | Shoot Right VUK           | Lock ball #2                |
| 5️⃣    | Spell TOMCAT again        | Lights third lock           |
| 6️⃣    | Shoot Right VUK           | Lock ball #3                |
| 7️⃣    | Spell TOMCAT again        | Lights RELEASE              |
| 8️⃣    | Shoot Right VUK (Release) | **START 4-BALL MULTIBALL!** |

> 🎪 **Light Show:** On a properly working machine, locking the third ball triggers the **backbox beacon lights** (red, white, blue) to spin, creating an impressive light show while waiting for your Release shot <kcite ref="23"/>.

### Jackpot Structure

During multiball, shoot the **Right VUK (Landing shot)** to score Landings <kcite ref="23"/>:

| Landing     | Value                   |
| ----------- | ----------------------- |
| 1st Landing | 150,000                 |
| 2nd Landing | 250,000                 |
| 3rd Landing | **~703,850 (JACKPOT!)** |

> ⚡ **Critical:** Landings are **cumulative across multiballs**! If you only get 2 landings in your first multiball, you only need 1 more in your next multiball to collect the jackpot <kcite ref="23"/>.

### Multiball Special (Secret!)

**Top Secret Feature:** If you light **ALL TOMCAT targets (both upper AND lower)** while in multiball, **SPECIAL** is lit on both outlanes, awarding an Extra Ball <kcite ref="24"/>.

## 📈 Progression Systems

### Yagov Kills

You are "Hitman," a Navy pilot hunting **General Yagov** <kcite ref="30"/>. Kill his planes to earn rewards!

**How to Score Kills:**
1. Roll through an **inlane** (lights KILL on center targets) <kcite ref="27"/>
2. Hit the **lit center target** to register a kill
3. OR shoot the **Yagov Kicker** (left kickback area) directly <kcite ref="65"/>

| Kill # | Callsign  | Reward          |
| ------ | --------- | --------------- |
| 1      | Alpha     | Progress        |
| 2      | Bravo     | Progress        |
| 3      | Charlie   | Progress        |
| 4      | Delta     | Progress        |
| 5      | Echo      | Progress        |
| 6      | Fox       | Progress        |
| 7      | Golf      | Progress        |
| 8      | **YAGOV** | **EXTRA BALL!** |

> ⚠️ **Warning:** After your first Yagov kill, inlanes **no longer light** the center KILL targets <kcite ref="26"/>.

### Bonus Multiplier

Build your bonus multiplier via the **Flyby orbit** and **1-6 targets** <kcite ref="25"/>:

| Step | Action                                                       |
| ---- | ------------------------------------------------------------ |
| 1    | Hit a **lit 1-6 target** (cycles 1→6→1)                      |
| 2    | This lights **Bonus X** on the Flyby orbit                   |
| 3    | Shoot the **Flyby orbit** to collect and increase multiplier |
| 4    | Repeat to reach **8X maximum**                               |
| 5    | After 8X: **EXTRA BALL** lit on left kickback!               |

> 📝 **Note:** Bonus multipliers are **NOT carried** between balls - you must earn them each ball <kcite ref="67"/>.

### Spinner

The spinner on the right side of the Flyby orbit awards **1,000 points per spin** when lit <kcite ref="28"/>. Light it by hitting a lit 1-6 target.

## 🔄 Stacking & Combos

F-14 Tomcat is a **System 11A game** from 1987, predating complex stacking mechanics. However, some synergies exist:

### ✅ Good Combinations

| Combo                         | Why It Works                                    |
| ----------------------------- | ----------------------------------------------- |
| **Multiball + Landing Focus** | Jackpot is the biggest single score             |
| **Inlane → Right VUK**        | Scores Launch Bonus + spots TOMCAT + locks ball |
| **Flyby → Upper Targets**     | Builds bonus X while spelling TOMCAT            |
| **Lit Spinner + Flyby**       | Extra points on every orbit                     |

### ❌ Avoid These

| Situation                          | Why                                                             |
| ---------------------------------- | --------------------------------------------------------------- |
| **Shooting Yagov without EB lit**  | Extremely fast return, high drain risk <kcite ref="43"/>        |
| **Center targets straight-on**     | Ball goes **SDTM (Straight Down The Middle)** <kcite ref="26"/> |
| **Dead bouncing on soft flippers** | Ball won't return properly <kcite ref="40"/>                    |

**Ultimate Achievement:** Defeat General Yagov by killing all 8 planes, which awards an Extra Ball and triggers Yagov's defeat speech <kcite ref="65"/>.

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                         | Danger Level | Notes                                                                                   |
| ---------------------------- | ------------ | --------------------------------------------------------------------------------------- |
| **Yagov Kicker**             | 🔴 EXTREME    | Fastest shot in pinball history - fires ball back at incredible speed <kcite ref="80"/> |
| **Center Targets (dead-on)** | 🔴 HIGH       | Straight-on hits = SDTM drain <kcite ref="71"/>                                         |
| **Upper Right Flipper Tip**  | 🟡 MEDIUM     | Missed orbit shots can bounce dangerously <kcite ref="44"/>                             |

### ✅ Safe Shots

| Shot                         | Safety Level | Notes                                                           |
| ---------------------------- | ------------ | --------------------------------------------------------------- |
| **Right VUK (Lock/Landing)** | 🟢 SAFE       | Ball is captured, no drain risk                                 |
| **Upper TOMCAT Targets**     | 🟢 SAFE       | Ball returns to upper flippers                                  |
| **Flyby Orbit**              | 🟢 SAFE       | Returns to lower flippers controllably                          |
| **Center Targets (angled)**  | 🟢 SAFE       | Hit the **sides** of targets, not straight-on <kcite ref="26"/> |

### 🎮 Flipper Techniques

| Technique                                 | When to Use                                               |
| ----------------------------------------- | --------------------------------------------------------- |
| **Hold right flipper up** for right orbit | Creates safer return if you shoot early <kcite ref="44"/> |
| **Backhand ricochet** off 4-5 targets     | Alternative way to hit right VUK <kcite ref="42"/>        |
| **Cradle and aim**                        | Essential on this fast game                               |
| **Avoid dead bounces**                    | Especially with soft flippers <kcite ref="40"/>           |

> 💡 **Pro Tip:** Master **multiple ways** to hit the right VUK shot - it's the key to the entire game <kcite ref="42"/>.

## 📋 Additional Information

### Extra Ball Opportunities

| Method                  | Requirement                                                         |
| ----------------------- | ------------------------------------------------------------------- |
| 🎖️ **Yagov Defeat**      | 8 kills (Alpha through Golf + Yagov) <kcite ref="29"/>              |
| 📈 **Bonus Multiplier**  | Reach 8X+ on a single ball, shoot left kickback <kcite ref="67"/>   |
| ⭐ **Multiball Special** | Light ALL TOMCAT (upper + lower) during multiball <kcite ref="24"/> |

### Flight Insurance (Ball Save)

F-14 Tomcat was the **first pinball machine with a ball save feature** <kcite ref="80"/>!

- Called "**Flight Insurance**" or "**Fly Again**"
- If the kickback drains your ball during its grace period, you get another ball <kcite ref="29"/>
- The kickback is **very fast** on a clean machine <kcite ref="29"/>

### Kickback (Rescue)

- Left outlane has a **kickback** that saves drains <kcite ref="37"/>
- **Relight** by hitting the lit center RESCUE target <kcite ref="68"/>
- Target alternates on a timer between the two center targets
- This shot is easy enough that you should **almost never have a left drain** <kcite ref="68"/>

### Setup Recommendations

| Setting              | Recommendation               | Notes                                                  |
| -------------------- | ---------------------------- | ------------------------------------------------------ |
| **Pitch**            | 6.5° - 7.0°                  | Standard to slightly steep                             |
| **Level**            | Perfectly level side-to-side | Critical for fair play                                 |
| **Flipper Strength** | Strong                       | Weak flippers = dead bounce problems <kcite ref="40"/> |
| **Kickback Speed**   | Properly maintained          | Should fire ball to upper playfield                    |

> ⚠️ **Speed Warning:** A properly maintained F-14 plays **EXTREMELY fast** <kcite ref="71"/>. Ball times become long once skilled because there are no "drain on near miss" shots.

## ⚡ Quick-Start Checklist

```
□ 1. Plunge ball into play
□ 2. Cradle ball - DO NOT dead bounce
□ 3. Shoot upper TOMCAT targets (upper flippers)
□ 4. When TOMCAT complete → Shoot Right VUK to lock
□ 5. Repeat steps 3-4 until 3 balls locked
□ 6. Complete TOMCAT one more time
□ 7. Shoot RELEASE (Right VUK) → 4-BALL MULTIBALL!
□ 8. During MB: Shoot Right VUK for Landings
□ 9. 3 Landings = JACKPOT (~703,850)
□ 10. Keep kickback lit (hit center RESCUE target)
□ 11. Work bonus multiplier via Flyby orbit
□ 12. Avoid Yagov shot unless EB is lit
□ 13. Repeat multiball for more jackpots!
```
