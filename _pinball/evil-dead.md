# 🪓 Evil Dead - Strategy Guide

*Spooky Pinball, 2024*

> **"Groovy."**

Evil Dead adapts both the 1981 original and 1987's Evil Dead II into a movie-mode structure: complete 4 modes from whichever film you've selected, and the game switches you to the other movie before switching back. Layered on top are three multiballs (Cheryl/Deadites, Cabin, Boomstick), a Good Linda/Bad Linda mechanic, and a growing set of sub-wizard modes added throughout 2025. Our goal: cycle through both movies' modes, qualify all three multiballs, and earn Spooky Tokens toward the final wizard mode.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Take the skill shot** (standard or super, depending on plunge) for an early points/setup boost
- [ ] **Complete 4 modes from your selected movie** to switch the game to the other film — then complete 4 more to switch back
- [ ] **Spell G-R-O-O-V-Y to light Mystery** — a reliable secondary objective throughout the game
- [ ] **Complete two modes from the first film to qualify Cheryl/Deadites Multiball**
- [ ] **Watch for Good Linda vs. Bad Linda** — this mechanic affects "Linda tricks" and changes how certain shots play out
- [ ] **Bank Spooky Tokens during normal play** — they're your currency for accessing sub-wizard and wizard modes later

## 🎬 Core Mechanics

### Movie Mode Structure

- Each film (**The Evil Dead** and **Evil Dead II**) has **4 modes**
- **Each mode is split into two phases**
- Complete your selected movie's full set of 4 modes, and the game **switches you to the other movie**; complete that set, and it **switches back**
- Modes draw on different playfield elements: some require hitting **the hand**, some route through the **basement/lower playfield via ramps**, and some use the **pop bumpers**

### Spelling GROOVY

Spell **G-R-O-O-V-Y** to light **Mystery** — a steady, repeatable secondary objective alongside your mode progress.

### Good Linda / Bad Linda

A binary state mechanic that affects "Linda tricks" — which version is active changes how certain interactions and shots play out. Pay attention to which Linda is currently active before committing to Linda-related shots.

## 🎁 Multiballs

| Multiball | Qualification |
|---|---|
| **Cheryl / Deadites Multiball** | Complete **2 modes from the first film** (The Evil Dead) |
| **Cabin Multiball** | Mode-driven, tied to cabin-related shot progress |
| **Boomstick Multiball** | Mode-driven, tied to Boomstick-related shot progress |

There's also a **Blood Flood** feature and **Video Mode** that supplement the core mode/multiball loop.

## 🏆 Sub-Wizard & Wizard Modes

- **Spooky Tokens**, earned during normal gameplay, are spent to access **sub-wizard modes**
- Sub-wizards added through 2025 include **"Hero From The Sky"** and the **"Mirror, Mirror"** Sanity mode
- **Sanity Mode** is a distinct mode layered on top of the core movie-mode structure
- The **final wizard mode** culminates in battling the **flying deadite** in the past

> 🪓 **Pro Tip:** Since this is a actively-evolving modern ruleset with content added throughout 2025, expect your specific machine's code revision to affect exactly which sub-wizards are available — check your code version if something described here doesn't match what you see on the phone/DMD display.

## ⚠️ Dangerous Shots & Drain Patterns

| Risk | Mitigation |
|---|---|
| **Basement/lower playfield ramp shots** | Modern Spooky ramps can be tight — mistimed shots risk a stall or drain | Practice each ramp's specific angle before relying on it under mode pressure |
| **Pop bumper-routed modes** | Less predictable than direct ramp/standup shots | Treat bumper-based mode objectives as semi-passive rather than forcing them |
| **Mid-mode movie switches** | Losing track of which movie/mode set you're working can waste shots | Check the display before committing to a shot during transition periods |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Take the skill shot, then start your first movie's mode set
2. Begin spelling GROOVY for Mystery alongside mode progress
3. Work toward Cheryl/Deadites Multiball via your first 2 modes

**Target Score:** Build steadily — this is a deep, mode-driven modern ruleset rather than a single big-number game

### Mid Game (Ball 2)

1. Complete your first movie's 4-mode set to switch films
2. Qualify and run Cabin or Boomstick Multiball as they become available
3. Bank Spooky Tokens for later sub-wizard access

### Late Game (Ball 3+)

1. Complete the second movie's mode set to switch back
2. Spend banked Spooky Tokens on sub-wizard modes (Hero From The Sky, Mirror Mirror)
3. If you've built enough progress, push toward the final wizard mode and the flying deadite battle
