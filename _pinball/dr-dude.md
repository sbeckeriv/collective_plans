# 🎸 Dr. Dude and His Excellent Ray - Ultimate Strategy Guide

> **Goal:** Become a **Super Dude** and hit the **Gazillion Point Shot** for maximum scoring!

Dr. Dude is a notoriously aggressive table that rewards precise shot-making and ball control <kcite ref="70"/>. This guide will help you master the path to coolness!

## 🎯 Core Mechanics

### Skill Shot

| Target                | Value       | Notes                                                                |
| --------------------- | ----------- | -------------------------------------------------------------------- |
| "Way Cool" (top lamp) | **100,000** | Time plunge when light changes from "Geek" to "OK" <kcite ref="87"/> |
| "75K" lamp            | 75,000      | Also lights bumpers for increased value                              |
| Other lamps           | 25K-50K     | Lesser values                                                        |

> **💡 Pro Tip:** Consider **soft-plunging** instead! The pop bumpers frequently dump the ball straight down the middle for a drain. A soft plunge brings the ball safely to a flipper <kcite ref="74"/>.

### Elements of Coolness

Collect all three to light the **Excellent Ray** for Multiball:

| Element                    | Location            | Shots to Collect |
| -------------------------- | ------------------- | ---------------- |
| 🧲 **Magnetic Personality** | Right magnet target | 3 shots          |
| 🎸 **Heart of Rock & Roll** | Center target       | 3 shots          |
| 🗣️ **Gift of Gab**          | Right VUK           | 3 shots          |

**Scoring per Element:**
- 1st shot: 5,000 points <kcite ref="21"/>
- 2nd shot: 25,000 points <kcite ref="56"/>
- 3rd shot: 50,000 points + **Element Collected**

> **💡 Pro Tip:** You can also spot Elements by hitting the corresponding colored targets in the **Mixmaster** 10 times each <kcite ref="72"/>. This is MUCH safer than shooting the Elements directly!

**Progress is saved between balls** - Element values don't reset until collected <kcite ref="21"/>.

### REFLEX 1-2-3

A timed combo sequence for big points:

1. Complete all 6 **REFLEX** standup targets → Lights **REFLEX 1** at left saucer
2. Shoot left saucer → Lights **REFLEX 2** at Gift of Gab (10 sec timer)
3. Shoot Gift of Gab → Lights **REFLEX 3** at Magnetic Personality (10 sec timer)
4. Shoot Magnetic Personality → **Collect bonus!**

| Condition                                 | Value                       |
| ----------------------------------------- | --------------------------- |
| Base value                                | 1,000,000 <kcite ref="51"/> |
| Each REFLEX completion without collecting | +250,000                    |
| Value carries over entire game            | ✅                           |

> **⚠️ Warning:** The REFLEX standup targets are **dangerous shots** with high drain risk <kcite ref="80"/>. Only attempt when you're confident!

## 🎱 Multiball

### How to Start Multiball

1. Collect all 3 **Elements of Coolness** (or spot via Mixmaster)
2. Shoot the **Mixmaster ramp** to light the lock
3. Shoot the **left orbit (Excellent Ray)** to lock ball and start 2-ball Multiball <kcite ref="42"/>

### Jackpot Structure

| Jackpot # | Targets Needed | Base Value            |
| --------- | -------------- | --------------------- |
| 1st       | 10 hits        | 1,000,000 - 5,000,000 |
| 2nd       | 20 hits        | 1,000,000 - 5,000,000 |
| 3rd+      | +10 each       | 1,000,000 - 5,000,000 |

**Building Jackpot Value:**
- Starts at **1,000,000** points
- Each Mix Master target bank hit adds **100,000** <kcite ref="46"/>
- Maximum jackpot: **5,000,000** points

### Super Jackpot (2x Jackpot)

Complete the **drop targets** during Multiball to **double** the current jackpot value! <kcite ref="53"/>

| Normal Jackpot | With Drop Targets |
| -------------- | ----------------- |
| 1M - 5M        | **2M - 10M**      |

> **⚠️ Warning:** Drop targets are dangerous - rebounds often drain <kcite ref="80"/>. Only attempt if you have good ball control!

### Post-Multiball Bonus

**2x Playfield Scoring** activates when Multiball ends and lasts for the rest of that ball <kcite ref="77"/>. This is HUGE - even if you don't get a jackpot, ending Multiball is valuable!

### Multiball Strategy

1. **Soft-plunge** the second ball to avoid pop bumpers <kcite ref="74"/>
2. Try to **cradle one ball** on the left flipper
3. Shoot the other ball into the **Mixmaster** from the right flipper
4. Use **cradle separation** if both balls end up on one flipper <kcite ref="75"/>
5. Keep shooting the ramp for jackpots!

> **💡 Pro Tip:** There is **no ball saver** during Multiball <kcite ref="73"/>. Play conservatively!

## 🌟 Wizard Mode: Super Dude & Gazillion Shot

### Requirements

1. Start **5 Multiballs** to reach **Super Dude** status <kcite ref="45"/>
2. Start your 6th Multiball

### Gazillion Point Shot

When Super Dude is active, the **Gazillion Shot** lights at the Mixmaster ramp:

**Scoring:** 500,000 × (number of targets hit in Mixmaster) <kcite ref="54"/>

| Targets Hit       | Points         |
| ----------------- | -------------- |
| 1                 | 500,000        |
| 5                 | 2,500,000      |
| 10                | 5,000,000      |
| 10 + 2x Playfield | **10,000,000** |

> **💡 Pro Tip:** The Gazillion shot is somewhat random - the ball may hit many targets or fall out quickly <kcite ref="79"/>. Don't stress if it doesn't pay off big!

### Advanced Gazillion Strategy

For maximum scoring <kcite ref="78"/>:

1. Build Dude-O-Meter to **Major Dude** (one below Super Dude)
2. Start a new game (meter persists!)
3. Get to Multiball → Reach **Super Dude**
4. **Don't shoot** the Gazillion shot - let one ball drain
5. This returns you to Major Dude, but **2x Playfield** is now active
6. Get another Multiball → Super Dude again
7. NOW shoot Gazillion with **2x scoring** for double points!

## 🎯 Shot Analysis

### Dangerous Shots ⚠️

| Shot                   | Risk     | Notes                                            |
| ---------------------- | -------- | ------------------------------------------------ |
| Drop Targets           | **HIGH** | Rebounds drain frequently <kcite ref="80"/>      |
| Element Shots (direct) | **HIGH** | Magnetic Personality, Heart of Rock, Gift of Gab |
| REFLEX Standups        | **HIGH** | White targets along sides                        |
| Pop Bumpers            | **HIGH** | Often dump ball down center <kcite ref="74"/>    |

### Safe Shots ✅

| Shot                       | Risk    | Notes                                     |
| -------------------------- | ------- | ----------------------------------------- |
| Mixmaster Ramp             | **LOW** | Primary scoring shot <kcite ref="72"/>    |
| Left VUK (Bag of Tricks)   | **LOW** | Safe shot, good rewards <kcite ref="80"/> |
| Left Orbit (Excellent Ray) | **LOW** | Locks ball for Multiball                  |

### Flipper Selection

| Ball Position | Best Flipper    | Shot                                         |
| ------------- | --------------- | -------------------------------------------- |
| Right flipper | Right           | Mixmaster ramp (primary)                     |
| Right flipper | Right           | Left orbit (Excellent Ray)                   |
| Left flipper  | Left (backhand) | Mixmaster (harder)                           |
| Left flipper  | -               | Post pass to right flipper <kcite ref="81"/> |

> **💡 Pro Tip:** Both main shots (Mixmaster and Ray) are on the left side, so you generally want the ball on the **right flipper** <kcite ref="81"/>.

### Ball Control Techniques

**Post Pass:** Release and re-engage flipper very quickly to scoot ball to other flipper <kcite ref="83"/>

**Dead Pass:** Leave flipper down - ball bounces to other side (usually safe on this table)

**Nudge Pass:** Nudge right when ball comes down from Ray kickout to stop it on left flipper <kcite ref="82"/>

**Cradle Separation:** Quick flipper flick to separate two balls on same flipper <kcite ref="75"/>

### Handling Specific Returns

| Ball From            | Technique                                                             |
| -------------------- | --------------------------------------------------------------------- |
| Ray kickout          | Hold left flipper up → bounces to right flipper tip <kcite ref="82"/> |
| Magnetic Personality | Hold left flipper up for easy catch <kcite ref="84"/>                 |
| Incomplete Ray shot  | Hold left flipper up for catch <kcite ref="85"/>                      |
| Incomplete Mixmaster | Hold left flipper up (dead pass is risky here) <kcite ref="86"/>      |
