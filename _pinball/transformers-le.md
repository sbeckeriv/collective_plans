### 🔺 Shot Multipliers (Cyber Multipliers)

1. Complete your faction's 4 insignia inserts
2. This lights one of 6 triangular "X" multiplier inserts on major shots
3. Shooting a lit "X" shot applies **2× multiplier**
4. Complete all 6 multipliers → **Roving 3× multiplier** activates <kcite ref="44"/><kcite ref="76"/>

| Multiplier Stage | Effect                                   |
| ---------------- | ---------------------------------------- |
| Individual 2×    | Doubles that specific shot               |
| All 6 Complete   | 3× roving multiplier moves between shots |

## 🧩 Stacking & Combos

### ✅ Stack These Together

| Combination                      | Why It Works                     |
| -------------------------------- | -------------------------------- |
| Mode + 2× Bumblebee Scoring      | Doubles all mode scoring         |
| Multiball + Active Mode          | Jackpots AND mode shots score    |

### ❌ Avoid Stacking

| Combination                                  | Why It's Bad                                               |
| -------------------------------------------- | ---------------------------------------------------------- |
| AllSpark during mode (if wanting EB)         | You get extra time instead of extra ball <kcite ref="91"/> |
| Two modes simultaneously                     | Not possible - one at a time                               |
| Starting multiball with nearly complete mode | May lose mode progress                                     |

### Optimal Timing

1. **Build multipliers first** before starting high-value modes
2. **Start multiball** when a mode is already running
3. **Collect AllSpark** outside of modes for best extra ball chance
4. **Save Sentinel Prime** until multipliers are maxed

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                 | Danger                                                       | Mitigation                                   |
| -------------------- | ------------------------------------------------------------ | -------------------------------------------- |
| **Left Loop**        | Can return SDTM (straight down the middle) <kcite ref="61"/> | Only shoot with clean hit, be ready to nudge |
| **Center Lane**      | Fast return to flippers                                      | Prepare for catch or controlled bounce       |
| **Weak Orbit Shots** | Reject to outlanes                                           | Commit to full-power shots                   |

### ✅ Safe Shots

| Shot                         | Why Safe                          | Best Use                           |
| ---------------------------- | --------------------------------- | ---------------------------------- |
| **Right Ramp**               | Controlled return to left flipper | Mode shots, building rhythm        |
| **Megatron Scoop**           | Ball held, controlled release     | Lock balls, relight jackpots       |
| **AllSpark Saucer**          | Ball held                         | Mystery awards, breathing room     |
| **Captive Ball (Bumblebee)** | No ball movement                  | Safe points, 2× scoring activation |

### Ball Control Positions

| Position                 | How to Achieve                   | Use For                   |
| ------------------------ | -------------------------------- | ------------------------- |
| **Left Flipper Cradle**  | Dead bounce from right           | Planning next shot        |
| **Right Flipper Cradle** | Catch from left return           | Megatron lock shots       |
| **Post Pass**            | Bounce off post between flippers | Switching flipper control |
