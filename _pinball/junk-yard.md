## 🎮 Core Mechanics

### 🎱 Skill Shot

The skill shot uses a **spinner in the plunger lane** - each spin advances the lit award on the DMD <kcite ref="26"/>.

| Award                | Effect                                                                | Value                               |
| -------------------- | --------------------------------------------------------------------- | ----------------------------------- |
| **Collect Junk**     | Awards flashing junk item (use left flipper to cycle before plunging) | Junk + ~10-25K                      |
| **Super Skill Shot** | Lights Sewer for escalating points                                    | 250K → 500K → 750K → ... → 2.5M max |
| **Bonus X**          | Increases end-of-ball multiplier (cumulative all game)                | Up to 6X                            |
| **Light Extra Ball** | Lights Sewer for EB (stays lit until collected)                       | Extra Ball                          |
| **Spot D-O-G**       | Instantly lights scoop for video mode                                 | Video Mode                          |
| **Points**           | Direct point award (50K, 100K, 150K, 250K, 500K)                      | Varies                              |

> 💡 **Pro Tip:** A **weak plunge** gives only 1-2 spins, letting you reliably land on "Collect Junk" - the first award in the sequence <kcite ref="93"/>.

> 💡 **Pro Tip:** Use the **left flipper before plunging** to cycle which junk item is flashing - "Collect Junk" becomes as good as "Choose Junk" <kcite ref="93"/>.

### 🎬 Video Modes (D-O-G Scoop)

Complete D-O-G letters by hitting the scoop, then shoot it when lit <kcite ref="62"/>.

#### Run From Spike 🐕
*Available before building Toaster Gun*

| Objective      | How to Play                  | Reward    |
| -------------- | ---------------------------- | --------- |
| Escape the dog | Mash flipper buttons rapidly | Junk item |

> 💡 **Pro Tip:** Delay collecting the Hair Dryer to keep this easier mode available longer <kcite ref="93"/>.

#### Save the Girl 👧
*Available after building Toaster Gun*

| Objective                     | How to Play                                             | Reward    |
| ----------------------------- | ------------------------------------------------------- | --------- |
| Hit dog 3 times in 12 seconds | Shoot toast at left/right openings with flipper buttons | Junk item |

- Minimum 12 toast ammo first time
- **Gets harder each completion** - dog/girl patterns become erratic <kcite ref="62"/>
- If all junk collected: awards 3 Fireworks instead

## 🎰 Multiballs

### Crane Multiball (2-Ball)

#### How to Qualify

| Method                              | Progress                 |
| ----------------------------------- | ------------------------ |
| Hit crane ball into car targets     | +1 MULTIBALL letter each |
| Complete either lower standup bank  | +1 MULTIBALL letter      |
| "Sneak in" center shot (crane down) | +3 MULTIBALL letters     |
| Magic Bus "Light Multiball"         | Instant qualification    |

Complete **MULTIBALL** → crane raises → shoot center shot to start <kcite ref="64"/>.

#### Crane Hurry-Up (Alternative Start)
Complete **both** lower standup banks → inlanes light for Crane Hurry-Up <kcite ref="64"/>:
- Starts at 50K, counts down to 25K
- Hit crane ball into car target to collect + start 2-ball multiball

#### Jackpot Structure

| Phase           | Shot                        | Value                 |
| --------------- | --------------------------- | --------------------- |
| 1️⃣ Jackpot       | Center shot (crane raised)  | ~1M+ (buildable)      |
| 2️⃣ Super Jackpot | Center shot (crane lowered) | **3 Million**         |
| 3️⃣+              | Keep shooting center        | 3M each Super Jackpot |

> 💡 **Pro Tip:** At multiball end, if Super Jackpot was lit, crane lifts briefly for a **last-chance Super Jackpot** - no indication except crane movement <kcite ref="65"/>!

> 💡 **Pro Tip:** If you never got the first jackpot, center shot lights for **Multiball Restart** instead <kcite ref="65"/>.

#### During Multiball
- Adventures, video modes, and ball-holding features are **disabled** <kcite ref="65"/>
- **Exception:** You CAN start multiball during Radar or Great Toilet adventures - they run simultaneously!

### 🎆 Fireworks System

Fireworks are collected during Adventures and contribute to:
- End-of-ball bonus (25K each) <kcite ref="45"/>
- Outer Space entry bonus (50K × Fireworks, +10M if 20+) <kcite ref="67"/>

**Toaster Gun Target:** During adventures, costs 5 toast to fire, awards Fireworks <kcite ref="67"/>.

## 🔀 Stacking & Combos

### ✅ What Stacks Well

| Combination                            | Why It Works                                                             |
| -------------------------------------- | ------------------------------------------------------------------------ |
| **Multiball + Radar Adventure**        | Both run simultaneously - double scoring! <kcite ref="65"/>              |
| **Multiball + Great Toilet Adventure** | Same as above                                                            |
| **Multiple Extra Balls**               | All EBs stack and stay lit until collected <kcite ref="47"/>             |
| **Ball Recycles**                      | Both outlanes can be lit independently, don't time out <kcite ref="76"/> |
| **Bonus X + Long Games**               | Multiplier is cumulative all game, maxes at 6X <kcite ref="45"/>         |

### ❌ What NOT to Stack

| Situation                                | Why to Avoid                                                     |
| ---------------------------------------- | ---------------------------------------------------------------- |
| **Starting Adventures during Multiball** | Adventures disabled during multiball (except if already running) |
| **Video Modes during Multiball**         | Disabled                                                         |
| **Multiple Adventures**                  | Only one can run at a time                                       |
| **Collecting Hair Dryer early**          | Switches to harder "Save the Girl" video mode                    |

### ⏱️ Optimal Timing

1. **Build Jalopy first** - unlocks most junk collection opportunities
2. **Stack Ball Recycles** before risky shots
3. **Start Multiball during an Adventure** if possible (Radar/Toilet)
4. **Save Outer Space** for when you have 20+ Fireworks for 10M bonus

## 🚀 Wizard Mode: Outer Space

### Requirements
1. ✅ Complete all 4 Adventures (Radar, Jalopy Race, Toilet, Air Tactical)
2. ✅ Collect ALL 10 junk items
3. ✅ Start at Sewer when lit

### Entry Bonus
- 50,000 × Fireworks collected
- **+10 Million** if 20+ Fireworks <kcite ref="67"/>

### Gameplay (4-Ball Multiball)

| Feature       | Details                                     |
| ------------- | ------------------------------------------- |
| Balls         | 4 balls with continuous autofire            |
| Objective     | Clear all 5 major shots to defeat Crazy Bob |
| Jackpots      | 1M+ per lit shot                            |
| Fireworks     | Toaster Gun target + Crane award more       |
| Timer         | Fireworks count down over time              |
| End Condition | Run out of Fireworks OR defeat Crazy Bob    |

### Scoring Potential
- Each lit shot: **1M+**
- Clearing all 5 shots: Defeats Crazy Bob
- Can potentially score **20-50M+** in a good Outer Space

### After Outer Space
- Fireworks and Adventures **reset** <kcite ref="67"/>
- Junk stays collected
- All adventures re-light - you can reach Outer Space again!

## 🎯 Shot Analysis

### 🔴 Dangerous Shots

| Shot                         | Risk                          | Mitigation                            |
| ---------------------------- | ----------------------------- | ------------------------------------- |
| **Outlanes**                 | Very wide openings            | Light Ball Recycles via standup banks |
| **Center Shot (crane down)** | Blocked, unpredictable bounce | Wait for crane to raise               |
| **Weak Bus Ramp**            | Falls through to dog hole     | Full-power shots only                 |
| **Crane Ball**               | Wild bounces                  | Controlled flipper timing             |

### 🟢 Safe Shots

| Shot                      | Why Safe                 | Reward                          |
| ------------------------- | ------------------------ | ------------------------------- |
| **Toilet Ramp**           | Returns to left inlane   | Builds Toilet Jackpot           |
| **Bus Ramp (full power)** | Returns to right inlane  | Bus Rides, Magic Bus            |
| **D-O-G Scoop**           | Returns via habitrail    | D-O-G letters, video mode       |
| **Lower Standup Banks**   | Ball returns to flippers | MULTIBALL letters, Ball Recycle |

### 🎯 Shot Selection by Flipper

| Left Flipper        | Right Flipper      |
| ------------------- | ------------------ |
| Toilet Ramp         | Bus Ramp           |
| Bus Ramp (backhand) | D-O-G Scoop        |
| Left standup bank   | Right standup bank |
| Sewer (difficult)   | Crane ball         |
|                     | Center shot        |

### 🎯 The Sewer Shot

> ⚠️ **Critical Shot:** The Sewer is one of the hardest shots but most important - it collects Adventures, Extra Balls, Time Machine, and Super Skill Shot <kcite ref="93"/>.

Practice this shot! The Super Skill Shot exists specifically to encourage Sewer practice.

**Target Breakdown:**

- **10+ Super Jackpots:** 3M each = 30M+
- **Time Machine modes:** 50-100M
- **Adventures completion:** 50-100M
- **End-of-ball bonuses (6X):** 50-100M
- **General play:** 100M+

**Key:** Reach Outer Space multiple times. After completing it, all adventures re-light, so you can loop back through the progression!

### 📊 End-of-Ball Bonus

Calculated as <kcite ref="45"/>:

**(Junk items × 10-25K each) + (Completed contraptions bonus) + (Fireworks × 25K) + (Modes played bonus) × Bonus Multiplier (1X-6X)**

## 🏆 Summary: The Billion Point Path

1. **Master the soft plunge** - Control skill shots for Collect Junk
2. **Delay Hair Dryer** - Keep easy "Run From Spike" mode
3. **Build Jalopy early** - Unlocks most progression
4. **Practice the Sewer shot** - Critical for Adventures and Time Machine
5. **Stack Ball Recycles** - Both outlanes, they persist
6. **Chain Super Jackpots** - 3M each in multiball
7. **Reach Outer Space with 20+ Fireworks** - 10M bonus + huge multiball
8. **Loop Outer Space** - Adventures reset, repeat the cycle!
