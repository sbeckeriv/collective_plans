## 🎮 Core Mechanics

### Game Overview

The Mandalorian has three main progression branches that must be completed to reach the final wizard mode <kcite ref="21"/>:

| Branch       | Requirement    | Wizard Mode Unlocked |
| ------------ | -------------- | -------------------- |
| 🎬 Missions   | Start all 5    | I Like Those Odds    |
| 🚀 Multiballs | Start all 3    | Precious Cargo       |
| ⚔️ Encounters | Complete all 3 | You Have What I Want |

Complete all three mini-wizard modes to unlock **This Is The Way** <kcite ref="21"/>.

---

### 🎱 Skill Shots

#### Regular Skill Shot
**How:** Plunge the ball so it rolls through one of the two lit adjacent lanes in the CHILD rollover area <kcite ref="21"/>.

| Award            | Value                   |
| ---------------- | ----------------------- |
| Points           | 1M (+1M per skill shot) |
| Bonus Multiplier | +2X                     |
| Light The Child  | ✅                       |
| Super Pops       | +1 bumper lit           |

#### Super Skill Shot ⭐
**How:** Hold the LEFT flipper while plunging until the ball passes the spinner <kcite ref="21"/>.

**Sequence (3 shots, ~5 seconds each):**
1. **Right Scoop** (Pro: U-Turn)
2. **Right Ramp**
3. **Final Shot** (choose one):

| Final Shot  | Award                                             |
| ----------- | ------------------------------------------------- |
| Left Scoop  | Start Mission (or Whistling Birds if already lit) |
| Left Ramp   | Start Encounter (or Jetpack if already lit)       |
| Center Ramp | Start Multiball (or Amban Rifle if already lit)   |

> 💡 **Pro Tip:** Each completed Super Skill Shot step adds +1X to your end-of-ball bonus multiplier <kcite ref="21"/>.

### 🎬 Missions

**How to Start:** Shoot both ramps, then the left scoop <kcite ref="21"/>.

> 💡 **Pro Tip:** For the first 2 missions, you can shoot the left scoop 3 times to start (it spots ramp shots) <kcite ref="21"/>.

All missions are **60-second timed modes**. Completing a mission awards a timer bonus <kcite ref="22"/>.

### ⚔️ Encounters

**How to Qualify:** Shoot the left ramp until the insert flashes red <kcite ref="22"/>:
- 1st Encounter: 1 ramp shot
- 2nd Encounter: +2 ramp shots
- 3rd Encounter: +2 ramp shots

**Gameplay:** Played on the mini-playfield (helmet-shaped). Premium/LE has a tilting mini-playfield with two flippers <kcite ref="22"/>.

Each encounter has **3 waves** of targets. After each wave, a **Bonus Level** begins where all shots score based on targets hit <kcite ref="22"/>.

| Encounter | Episode | Theme                              |
| --------- | ------- | ---------------------------------- |
| 1         | S2:E6   | Protect The Child (Storm Troopers) |
| 2         | S1:E2   | Catch the Jawas                    |
| 3         | S1:E8   | Escape to the Sewer (Moff Gideon)  |

> 💡 **Pro Tip:** Progress carries over between attempts! You don't lose wave progress on drain <kcite ref="22"/>.

## 🚀 Multiballs

### Razor Crest Multiballs

**How to Start:** Hit the center Razor Crest target until lit <kcite ref="21"/>.

**Changing Multiball:** Hit the Boba Fett targets to cycle which multiball is next <kcite ref="21"/>.

> ⚠️ **Important:** After your first multiball, Razor Crest is disabled until you complete a Hunter mission OR start a new ball <kcite ref="21"/>.

---

#### 🕷️ Ice Spider Multiball (S2:E2)
*Escape from the Spiders!*

| Mechanic      | Description                                            |
| ------------- | ------------------------------------------------------ |
| Jackpots      | Each lit shot is a spider on a timer (value decreases) |
| Waves         | Spiders come in waves                                  |
| Super Jackpot | Lights after completing a wave (timed, repeatable)     |
| Location      | U-Turn                                                 |

---

#### 🚀 Jetpack Multiball (S1:E8)
*Bring down Moff Gideon's TIE fighter*

| Mechanic        | Description                                             |
| --------------- | ------------------------------------------------------- |
| Jackpots        | All purple shots score jackpots                         |
| Upper Playfield | Complete targets for add-a-ball and boost Super Jackpot |
| Super Jackpot   | **Lights AFTER multiball ends** (single ball play!)     |
| Max Value       | 50M base                                                |

> 💡 **Pro Tip:** This is the BIG MONEY multiball. Build value during multiball, then collect the Super Jackpot in single ball with multipliers active for up to **600M** <kcite ref="71"/>!

---

#### 🏴‍☠️ Pirates Multiball (S2:E7)
*Stop the pirates from exploding the Rhydonium*

| Mechanic        | Description                               |
| --------------- | ----------------------------------------- |
| Rhydonium Meter | Shows current jackpot value               |
| Stabilize       | Shoot spinner to increase stability/value |
| Waves           | Complete all waves of lit shots           |
| Super Jackpot   | Lights after completing all waves         |

---

### Razor Crest Hurry-Up

**When:** Lights after 1st and 3rd multiballs <kcite ref="21"/>.

**How:** Shoot center ramp when lit, then hit Razor Crest target X times.

| Active +1X Multipliers | Shots Required |
| ---------------------- | -------------- |
| 0                      | 2              |
| 1                      | 4              |
| 2                      | 6              |
| 3                      | 8              |

> 💡 **Pro Tip:** Each collected hurry-up value adds to the next one. With all 3 multipliers (6X playfield) and Flamethrower (2X), your final shot can be worth **12X** <kcite ref="21"/>!

---

### Dank Farrik! Multiball

**How:** Random award from the "?" Foundry item <kcite ref="21"/>.

A 2-ball frenzy multiball where every switch scores points.

## 📈 Progression Systems

### 💰 Beskar Currency

Beskar is collected and spent at the Foundry <kcite ref="21"/>.

**Ways to Collect:**
- Hunter missions (primary)
- Scope feature
- The Child mystery award

---

### 🏭 Foundry

**How to Light:** Collect Beskar from Scope, The Child, or complete Hunter. Also lights if you have 200+ Beskar and complete the left 3-bank <kcite ref="21"/>.

| Item                 | Cost | Effect                              |
| -------------------- | ---- | ----------------------------------- |
| Armor                | 200  | Extended ball save times            |
| Whistling Birds      | 175  | Spots mission shots                 |
| Rising Phoenix       | 175  | Spots an Encounter level            |
| Flamethrower         | 175  | Sets shots on fire (2X each)        |
| Amban Rifle          | 175  | Add-a-ball during multiballs        |
| Start Mission        | 175  | Starts next mission                 |
| Light Razor Crest    | 300  | Lights next multiball               |
| Light Encounter      | 100  | Lights next encounter               |
| Light Scope          | 75   | Lights Scope feature                |
| Light Heal           | 200  | Ball save on outlane                |
| Light Extra Ball     | 600  | One time only per game              |
| Light +1X Multiplier | 500  | Lights one playfield multiplier     |
| ? (Mystery)          | 100  | Random item (can hit greyed items!) |
| Pass                 | 0    | Save Beskar for later               |

> ⚠️ **Note:** Each purchase increases that item's cost by ~50% <kcite ref="21"/>.

---

### ❓ Mystery "?" Award

Can award ANY item including greyed-out ones! If it lands on "?" again, you get a hidden random award <kcite ref="21"/>:
- Dank Farrik Frenzy Multiball
- Light Child
- Start Ambush
- **ALL THREE +1X multipliers (6X total!)**
- Start Hunter

---

### 🔧 Equipment

One-time use items activated with the **Action Button** <kcite ref="21"/>:

| Equipment       | Button Color | Effect                             |
| --------------- | ------------ | ---------------------------------- |
| Armor           | N/A          | Passive: Extended ball saves       |
| Amban Rifle     | 🟢 Green      | Add-a-ball during multiball        |
| Whistling Birds | 🟡 Yellow     | Collect all lit mission shots      |
| Flamethrower    | 🟠 Orange     | Sets shots on fire (2X each)       |
| Jetpack         | 🔴 Red        | Complete Encounter level instantly |

> 💡 **Pro Tip:** Use Amban Rifle just as your second-to-last ball is about to drain. You'll continue multiball with extra balls AND a fresh ball saver <kcite ref="21"/>!

> 💡 **Pro Tip:** You can hold Flamethrower AND another equipment. Long press = Flamethrower. Tap = other equipment <kcite ref="21"/>.

### 🎯 Hunter

**How to Light:** Complete the left 3-bank targets <kcite ref="22"/>.

**How to Start:** Ball through lit return lane (change with flipper buttons).

**Gameplay:**
1. Two shots light: one ramp, one orbit (opposite the starting lane)
2. Shoot the orbit
3. Opposite orbit and ramp light
4. Shoot the second orbit to complete

**Rewards:**
- Beskar (increased by shooting ramps during Hunter)
- +1X Playfield Multiplier (75 seconds)
- Re-enables Razor Crest

---

### 🔭 Scope

**How to Light:** Shoot all 3 blue arrow shots <kcite ref="22"/>:
- Right Scoop (Premium/LE) or U-Turn (Pro)
- Left 2X target
- Right Ramp

**How to Start:** Ball through left return lane when lit (5-second timer).

| Shot              | Award                   |
| ----------------- | ----------------------- |
| Left Scoop/U-Turn | Two pieces of equipment |
| Target            | Beskar + Light Foundry  |
| Right Ramp        | Points                  |

> ⚠️ **Note:** After completing all 3 Scope awards, it's disabled until you play a wizard mode <kcite ref="22"/>.

---

### 🔥 Playfield Multipliers

Three +1X multipliers stack up to **6X total** (3+3 bonus for having all three) <kcite ref="22"/>:

| Color    | How to Light                      |
| -------- | --------------------------------- |
| 🟢 Green  | Complete Boba Fett 2-bank targets |
| 🟠 Orange | Complete Hunter                   |
| 🔵 Blue   | Complete Scope                    |

**Permanent +1X:** Completing "This Is The Way" wizard mode <kcite ref="21"/>.

> 💡 **Pro Tip:** Hitting either Boba Fett target adds time to most running timers <kcite ref="21"/>!

---

### 🚨 Ambush

**How to Start:** Shoot the U-Turn when nothing else is running. "Try to Escape" starts <kcite ref="21"/>:
- 6 shots blink for ~6 seconds
- 2 random shots start Ambush
- Other shots = "Escaped!" award

**Gameplay (Two Stages):**

| Stage | Color | Objective                                                                     |
| ----- | ----- | ----------------------------------------------------------------------------- |
| White | ⚪     | Value counting down. Shoot U-Turn to lock value. Other shots add time/points. |
| Blue  | 🔵     | Right Ramp collects value. Other shots add value and return to White stage.   |

> 💡 **Pro Tip:** Build value with multipliers and Flamethrower before collecting for massive scores <kcite ref="21"/>!

## 🔀 Stacking & Combos

### ✅ What Stacks Well

| Combination                 | Why It Works                                                                     |
| --------------------------- | -------------------------------------------------------------------------------- |
| Mission + Multiball         | Quick, safe points. Mission shots often overlap with jackpots <kcite ref="66"/>. |
| All 3 Playfield Multipliers | 6X scoring on everything <kcite ref="67"/>!                                      |
| Multipliers + Flamethrower  | 12X on first shot, 6X thereafter <kcite ref="70"/>.                              |
| Jetpack MB + Multipliers    | Build Super Jackpot during MB, collect at 12X in single ball <kcite ref="71"/>.  |

---

### 🔥 The Fire Triangle Strategy

The ultimate scoring strategy <kcite ref="69"/>:

1. **Before Multiball:**
   - Complete Scope (Blue +1X)
   - Start/Complete Hunter (Orange +1X)

2. **During Multiball:**
   - Complete Boba Fett targets (Green +1X)
   - Now at 6X playfield!

3. **Activate Flamethrower:**
   - All shots now worth 12X (first hit) then 6X

---

### ❌ What NOT to Stack

| Avoid                  | Reason                                                          |
| ---------------------- | --------------------------------------------------------------- |
| Ambush + Anything      | Ambush only runs when nothing else is active <kcite ref="21"/>. |
| Multiple Missions      | Only one mission runs at a time.                                |
| Scope during Multiball | Hard to control timing with multiple balls.                     |

---

### ⏱️ Optimal Timing

| Feature          | Best Time to Start                                        |
| ---------------- | --------------------------------------------------------- |
| Multiball        | After Scope and Hunter are complete/started               |
| Flamethrower     | Right before collecting Super Jackpot or during multiball |
| Ambush           | When you have multipliers ready but no modes running      |
| Jetpack MB Super | Single ball with all multipliers + Flamethrower           |

## 🧙 Wizard Modes

### Mini-Wizard Modes

#### 🎲 I Like Those Odds (S1:E3 – The Sin)
**Requirement:** Start all 5 Missions <kcite ref="21"/>

**How to Start:** Same as starting a mission (both ramps → left scoop)

**Gameplay:** Three phases culminating in a 4-ball multiball and 60-second wizard frenzy <kcite ref="22"/>.

---

#### 📦 Precious Cargo (S1:E6 – The Prisoner)
**Requirement:** Start all 3 Razor Crest Multiballs <kcite ref="21"/>

**How to Start:** Same as starting a multiball (Razor Crest target)

---

#### 🎯 You Have What I Want (S2:E8 – The Rescue)
**Requirement:** Complete all 3 Encounters <kcite ref="21"/>

**How to Start:** Shoot left ramp 7 times <kcite ref="22"/>

---

### 🏆 This Is The Way (Final Wizard Mode)

**Requirement:** Complete all 3 mini-wizard modes <kcite ref="21"/>

**How to Start:**
1. Complete 6 "This Is The Way Ready" shots
2. Shoot left scoop

**Gameplay:** 3 phases, each followed by a timed Bonus Round <kcite ref="21"/>:
- Complete each phase as quickly as possible
- Unused time adds to Bonus Round duration

**Rewards:**
- 10M points
- Your Signet
- **Permanent +1X Playfield Multiplier**
- All missions/multiballs/encounters reset

---

### 🎩 Topper Exclusive Modes

#### Mandalorian Madness
**Requirement:** Complete one of each <kcite ref="21"/>:
- Episode (complete one mission)
- Razor Crest Multiball (collect one Super Jackpot)
- Encounter (complete one)
- Ambush (complete final shot)
- Scope (complete one shot)
- Hunter (complete one)

**Gameplay:**
- Starts as 2-ball multiball
- Timed mode (complete right 2-bank for more time)
- 5 areas cycle, each can light Super Jackpot for 10 seconds
- Collecting 3 Super Jackpots adds balls

---

#### Beskar Bonanza
**Requirement:** Cumulative 20,000 Beskar collected (across all players/games) <kcite ref="21"/>

**Reward:** One of EVERY Foundry item!

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                | Risk                                                | Mitigation                         |
| ------------------- | --------------------------------------------------- | ---------------------------------- |
| Left Orbit          | Can dump straight down the middle <kcite ref="53"/> | Adjust ball guide (see setup tips) |
| Weak Right Ramp     | Rollback to right outlane                           | Commit to full power shots         |
| Center Ramp Reject  | SDTM drain                                          | Nudge right on reject              |
| Mini-Playfield Exit | Unpredictable bounce                                | Be ready for either flipper        |

---

### ✅ Safe Shots

| Shot               | Why Safe                              |
| ------------------ | ------------------------------------- |
| Right Ramp         | Clean feed to left flipper            |
| Left Ramp          | Feeds right flipper or mini-playfield |
| Right Scoop/U-Turn | Ball kicked to flipper                |
| Boba Fett Targets  | Short shot, adds time to timers       |

---

### 🎮 Ball Control Positions

| Position             | How to Achieve                 | Use For                         |
| -------------------- | ------------------------------ | ------------------------------- |
| Left Flipper Cradle  | Catch from right ramp          | Plan next shot, wait for timers |
| Right Flipper Cradle | Catch from left ramp           | Set up backhand shots           |
| Post Pass            | Tap flipper as ball rolls down | Transfer ball between flippers  |

### ⚙️ Setup Recommendations

#### Pitch & Level
- **Pitch:** 6.5° is standard. Steeper = faster, more drains. Shallower = slower, easier control.
- **Level:** Ensure side-to-side level is perfect. Use a bubble level on the playfield glass.

#### Left Orbit Fix
If balls dump SDTM from the left orbit <kcite ref="54"/><kcite ref="55"/>:
1. Remove "Targets Light Hunter" plastic screws
2. Loosen the ball guide bracket underneath
3. Push ball guide as far RIGHT as possible (past the post)
4. Tighten firmly with ratcheting screwdriver
5. Consider reducing flipper power to 245 to prevent bricked shots
