## Scoring Philosophy: The Big Three

The proven formula for massive scores is:

1. **Playfield Multiplier (SNIKT)** — Multiplies ALL scoring 2x–5x
2. **Colossus Team-Up** — Doubles ALL points during a battle
3. **Stacked Multiball** — Extra balls + ball saver during dangerous modes

Combine all three during a battle for maximum scoring potential.

---

## SNIKT Playfield Multiplier

The **single most important scoring mechanic** in Deadpool.

| Type                  | How to Activate                                                       | Duration                           |
| --------------------- | --------------------------------------------------------------------- | ---------------------------------- |
| **Temporary (2x–5x)** | Ricochet ball off the SNIKT standup target onto the right katana ramp | Timed (hit SNIKT target to extend) |
| **Permanent (+1x)**   | Complete Lil' Deadpool Bounce mode                                    | Rest of ball                       |
| **Permanent (+1x)**   | Collect 100 weapons                                                   | Rest of game                       |

**Pro tip**: Activate SNIKT right before collecting jackpots or finishing battles for massive multiplied payouts.

## Team-Ups

Team-Ups are **essential for high scores**. Qualify at least 2 before starting a battle.

| Team-Up       | Qualifying Shot (2x) | Battle Perk                          |
| ------------- | -------------------- | ------------------------------------ |
| **Colossus**  | U-Turn               | **Doubles ALL points** during battle |
| **Wolverine** | Center spinner       | Doubles all damage dealt             |
| **Dazzler**   | Left orbit           | More health + time                   |
| **Domino**    | Right orbit          | Bonus scoring on specific features   |

**Important**: Team-Up perks don't activate until you hit their shot again *during* the battle. With Team-Ups active, you can clear a battle in as few as 2 shots and collect a Final Blow bonus in the 75M range.

## Starting Every Mode

### Battles (Main Villain Modes)

| Step | Action                                                                                    |
| ---- | ----------------------------------------------------------------------------------------- |
| 1    | Scoop is **automatically lit at ball start**, or relit by completing D-E-A-D drop targets |
| 2    | Shoot the scoop                                                                           |
| 3    | Select your opponent (Sabretooth, Juggernaut, or Mystique)                                |

### Quests (Chimichanga Missions)

| Step | Action                                                                                |
| ---- | ------------------------------------------------------------------------------------- |
| 1    | Shoot the **right orbit** to advance the chimichanga truck and light orange shots     |
| 2    | Collect **chimichangas** by hitting the lit orange shots                              |
| 3    | Collect **3 chimichangas** (+1 per Quest already completed, max 8) to fill punch card |
| 4    | Shoot the **scoop** to start the Quest                                                |

### Lil' Deadpool Multiball

| Step | Action                                                                          |
| ---- | ------------------------------------------------------------------------------- |
| 1    | Knock down the **OOF drop targets** (3 targets in front of Lil' Deadpool)       |
| 2    | Hit the **standup target** behind them **2 times** (+1 per mode already played) |
| 3    | Hit the **drop targets again** to release the ball and start multiball          |

After playing a Lil' Deadpool mode, you must complete a Battle or Quest before you can qualify the next one.

### Ninja Multiball

| Step | Action                                         |
| ---- | ---------------------------------------------- |
| 1    | Locks are **lit by default** at game start     |
| 2    | Shoot the **right katana ramp** to lock a ball |
| 3    | **Lock 3 balls** to start Ninja Multiball      |

Relighting locks: After playing once, complete the **POOL targets** to relight locks one at a time.

### Disco Multiball

| Step | Action                                                      |
| ---- | ----------------------------------------------------------- |
| 1    | Shoot the **left spinner** repeatedly                       |
| 2    | Accumulate **60 spins** (+60 per activation)                |
| 3    | Spinner shot will **strobe** when ready — shoot it to start |

Tip: Work on this passively. Beginning of the game is a great time — trap up, get the ball on the right flipper, and rip the left orbit.

### Mechsuit Multiball

| Step | Action                                                                                 |
| ---- | -------------------------------------------------------------------------------------- |
| 1    | Shoot **major shots** to light weapons (white inserts)                                 |
| 2    | Shoot the **scoop** to cash out lit weapons                                            |
| 3    | Cash out a total of **45 weapons**                                                     |
| 4    | Shoot the scoop **when no other modes are running** to start 4-ball Mechsuit Multiball |

### Quick Mode Start Reference

| Mode                 | Start Requirement                             |
| -------------------- | --------------------------------------------- |
| **Battle**           | Scoop (lit at ball start or via DEAD targets) |
| **Quest**            | Right orbit → 3+ chimichangas → scoop         |
| **Lil' Deadpool MB** | OOF targets → standup 2x → drop targets       |
| **Ninja MB**         | Lock 3 balls at right ramp                    |
| **Disco MB**         | 60 left spinner shots                         |
| **Mechsuit MB**      | Cash out 45 weapons at scoop                  |

## Battle Strategy & Recommended Order

### Recommended Order

| Order   | Villain        | Reasoning                                                                                                       |
| ------- | -------------- | --------------------------------------------------------------------------------------------------------------- |
| **1st** | **Sabretooth** | Safest. Orbits feed pop bumpers for free damage. Builds Disco MB and Wolverine Chef Mode via spinners.          |
| **2nd** | **Mystique**   | Choose when Wolverine Team-Up + BOOM are ready. Center spinner (Double Damage) → BOOM can defeat her instantly. |
| **3rd** | **Juggernaut** | Most dangerous but most valuable. Stack with Lil' Deadpool Multiball for safety.                                |

### Battle Details

| Battle         | Lit Shots                                   | Strategy                                                               |
| -------------- | ------------------------------------------- | ---------------------------------------------------------------------- |
| **Sabretooth** | Orbits, spinners, bumpers                   | Shoot orbits → feeds pop bumpers for extra damage                      |
| **Mystique**   | Center spinner, then ramps                  | With Wolverine + BOOM: hit center spinner, then BOOM to defeat quickly |
| **Juggernaut** | Hit DEAD/POOL targets, then shoot lit shots | Stack with Lil' Deadpool MB; BOOM the dangerous side targets           |

### Final Shot Bonus

After defeating a villain, you have **15 seconds** to shoot the scoop for the Final Shot. The bonus depends on remaining health and how quickly you shoot it. Can be worth ~30M+.

## What Happens After Defeating All Villains

### Sauron Multiball (1st Wizard Mode)

Lights at the scoop after defeating Sabretooth, Mystique, and Juggernaut.

| Phase | Objective                                                    |
| ----- | ------------------------------------------------------------ |
| 1     | Shoot left ramp and right ramp **2x each** for Jackpots      |
| 2     | Shoot U-Turn → Colossal Jackpot target for **Super Jackpot** |

**Tip**: Use Big BOOM to collect all 4 initial jackpots instantly!

### Level 2 Battles (Sauron Interference)

After Sauron Multiball, you must **replay all 3 battles** with Sauron interfering:

- During battles, Sauron may appear and **block your progress**
- Shoot **both ramps once** to damage Sauron before continuing to damage the villain
- Winning all 3 Level 2 battles unlocks **Clone Multiball**

## Multiball Modes

### Ninja Multiball

- Jackpot values are based on **ninja stars collected** before starting
- The longer you wait to start, the higher the jackpots (but risk losing the ball)
- Collect ninja stars from pop bumper spawns, then lock 3 balls at the right ramp
- **Can stack into Battles and Quests**

### Disco Multiball

- Grind mode — work on it passively while doing other things
- 60 spins to qualify (+60 each subsequent time)
- **Cannot stack** with Battles/Quests

### Mechsuit Multiball

- 4-ball multiball requiring 45 weapons cashed out
- Can only start when **no other modes are running**
- Required for Final Battle wizard mode

## BOOM Button Deep Dive

### Qualifying BOOM

| Step | Action                                                                   |
| ---- | ------------------------------------------------------------------------ |
| 1    | Roll through **inlanes/outlanes** to light B-O-O-M letters               |
| 2    | **Cycle letters with flippers** so you hit *unlit* lanes                 |
| 3    | Complete BOOM → button flashes **yellow**                                |
| 4    | Stack **4 BOOMs** without pressing → button turns **red** for "Big BOOM" |

### BOOM Types

| Type                            | Effect                                                  |
| ------------------------------- | ------------------------------------------------------- |
| **Normal BOOM** (yellow)        | Spots the **highest value lit shot** (except the scoop) |
| **Big BOOM** (red)              | Spots **ALL lit shots** at once                         |
| **Hold button during Big BOOM** | Uses a normal BOOM instead (saves Big BOOM for later)   |

**Priority rule**: If multiple shots have the same value, BOOM prioritizes shots on the **left side** of the playfield.

### BOOM During Battles

| Battle         | BOOM Strategy                                                                              |
| -------------- | ------------------------------------------------------------------------------------------ |
| **Sabretooth** | Less needed — orbits are manageable                                                        |
| **Mystique**   | Hit center spinner (activates Double Damage) → press BOOM → often defeats her in one combo |
| **Juggernaut** | Very useful — spots dangerous side standup targets                                         |

**Tip**: If only one shot remains, BOOM it, then focus on the scoop for the Final Shot jackpot.

### BOOM During Sauron Multiball

Big BOOM can collect **all 4 initial jackpots** at once!

### BOOM During Berserker Rage

| BOOM Type       | Berserker Rage Effect                                      |
| --------------- | ---------------------------------------------------------- |
| **Normal BOOM** | Awards **Berserker BOOM value** + spots highest value shot |
| **Big BOOM**    | Awards **4x Berserker BOOM value** + spots all shots       |

Berserker BOOM value = the **maximum spin session value** you've earned during the current Berserker Rage. Spin hard first, then press BOOM.

### BOOM During Other Modes

Works normally during Quests, Ninja MB, Disco MB, Mechsuit MB, and Lil' Deadpool MB — spots the highest value or all lit shots as usual.

### Mode Select Cancel (Hidden Feature)

**Hold the BOOM button for 2.5 seconds** during Mode Select to **cancel** starting a Battle or Quest. Useful if you accidentally shot the scoop but aren't ready.

### BOOM Strategy Summary

| Situation                        | BOOM Strategy                              |
| -------------------------------- | ------------------------------------------ |
| Battle with 1 shot left          | BOOM it, then shoot scoop for jackpot      |
| Mystique battle                  | Center spinner → BOOM immediately          |
| Juggernaut battle                | Save BOOM for dangerous side targets       |
| Sauron Multiball start           | Big BOOM collects all 4 initial jackpots   |
| Berserker Rage                   | Spin hard first, then BOOM for bonus value |
| Accidentally started mode select | Hold BOOM 2.5 sec to cancel                |

**Don't waste BOOM!** Save it for high-value moments.

## Other Scoring Features

### Berserker Rage (Spinner Frenzy)

- Lit after **4 center loop shots** (+1 each time)
- 45-second mode where spinners award increasing value
- BOOM during Berserker Rage = bonus equal to your best spin session value
- Big BOOM = 4x that bonus

### Katanarama Time

- Collect **3 diamonds at both ramps** to start
- 30-second mode: orbits and ramps score 1M+ increasing by 250K per shot

### Colossal Jackpot

- Make **2 full U-turn shots** (+2 each time) to light
- Value builds from switch hits
- **Strong shot** that hits the target = **doubles the jackpot**
- Further multiplied by Playfield X

### Mystery Awards

Complete **DEADPOOL targets** to light Mystery at scoop:

- Add-A-Ball (during multiball)
- Light Ninja Lock
- Ghost Ninja Stars / Collect Weapons
- Advance Chimichanga Truck
- Rare: Extra Ball or Special

### Extra Ball Requirements

- Collect **25 weapons**
- Collect **50 ninja stars**
- Rare mystery award

## Dangerous vs Safe Shots

### ⚠️ Dangerous Shots

| Shot                        | Why It's Risky                                          |
| --------------------------- | ------------------------------------------------------- |
| **Juggernaut side targets** | Unpredictable side-to-side bounces                      |
| **Right ramp (lock shot)**  | Often bricks or rejects; can sling into post below ramp |
| **Right orbit**             | Quite tight — easy to miss                              |
| **Both orbits**             | Feed into pop bumpers which send the ball anywhere      |

**Right ramp tip**: Hit the shot **more to the left** so it hits the bend rather than riding the ball guide.

### ✅ Safe/Controllable Shots

| Shot                           | Why It's Safe                                              |
| ------------------------------ | ---------------------------------------------------------- |
| **Center spinner (Wolverine)** | Good alternative to tight right orbit; controllable return |
| **Left orbit (Dazzler)**       | Feed back is generally controllable                        |
| **Ramps in general**           | Few shots return balls to dangerous areas                  |

**Good news**: Deadpool has relatively little side-to-side movement. Slingshots aren't super sensitive, so missed shots often don't ricochet straight to the outlanes.

## Ball Control & Handling Tips

### Deadpool-Specific Tips

- **Cradling is hard** on Deadpool compared to other games — don't expect easy traps
- Slings are generally your friend — not overly sensitive on most machines

### General Pinball Techniques

| Technique                   | How It Works                                                                         | When to Use                         |
| --------------------------- | ------------------------------------------------------------------------------------ | ----------------------------------- |
| **Cradle**                  | Hold flipper up, let ball settle. Plan your next shot.                               | Need to slow down and think         |
| **Dead Flip (Bounce Pass)** | Let ball bounce off a *raised* flipper without flipping. Transfers to other flipper. | Fast ball coming at flipper center  |
| **Live Catch (Drop Catch)** | *Release* the flipper just as ball hits, absorbing momentum and stopping it.         | Fast ball, need to gain control     |
| **Post Pass**               | Short, quick flip that bounces ball off the slingshot post to the other flipper.     | Ball on wrong flipper for your shot |

### BOOM Button for Control

Use the BOOM button to reduce difficult shots:

1. Cycle the lanes before the ball goes through an inlane
2. Don't press immediately when BOOM is ready
3. Save for spotting a difficult shot during a battle or finishing off an enemy

## Machine Setup Notes

| Setting             | Recommendation                                   |
| ------------------- | ------------------------------------------------ |
| **Playfield pitch** | 6.5°–7° is the sweet spot                        |

| **Too steep**       | Ramp shots brick                                 |
| **Too shallow**     | Ball doesn't return properly                     |
| **Flipper coils**   | Factory strength recommended                     |

## Full Wizard Mode Progression
```
Defeat 3 Villains → Sauron Multiball
       ↓
Level 2 Battles (with Sauron) → Clone Multiball
       ↓
Complete Both Quests + Sauron MB → Megakrakolodonus Rex
       ↓
All Above + Mechsuit MB → THE FINAL BATTLE (Mr. Sinister)
```

### Megakrakolodonus Rex

Unlocks after playing Sauron Multiball AND completing both Quests:

- Single-ball mode with short ball saver
- Shoot flashing **blue shots** to build jackpot and reveal weak spot
- When vulnerable: **bash Lil' Deadpool target** repeatedly for 1x, 2x, 3x jackpots
- 15 target hits defeats it → shoot scoop for Mode Jackpot

### The Final Battle (Mr. Sinister)

- Requires: Megakrakolodonus Rex + Clone Multiball + Mechsuit Multiball
- 45-second timed single-ball mode
- All 4 Team-Up shots must be made
