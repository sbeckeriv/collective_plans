# 🎩 Capersville - Strategy Guide

*Bally, 1966 — Designed by Ted Zale*

> **"Close the gap. Pull the caper."**

Capersville is a historically important EM — one of the first pinball machines to feature true 3-ball multiball — built around four distinct "Capers" (Code Zapper, Deep 4, Sea Ray, and Sea Ray Super Bonus) and the signature "Flipper Zipper" mechanism that closes the center drain gap. Our goal: complete all four Capers and ride the Super Bonus wheel to a free replay.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Tap the Red Mushroom Bumper to close the Flipper Zipper** — this shuts the center drain gap and is your main defensive tool
- [ ] **Tap any of the 3 Yellow Mushroom Bumpers to reopen the flippers** — useful once you no longer need the closed-gap protection
- [ ] **Hit the White Mushroom Bumper to release captive balls** and advance Deep 4 steps — it's a dual-purpose shot
- [ ] **Light all 6 Sea Ray portholes** to advance the Super Bonus wheel — scores of 5–8 award a Free Replay or bonus points
- [ ] **Only one of Deep 4 Alley or the two Kick-Out-Holes can hold locked balls at a time** — know which one is currently active before planning your multiball setup
- [ ] **Watch the outlanes** — they advance the Code Zapper symbol, but that's no consolation for losing the ball

---

## 🎪 Core Mechanics: The Four Capers

### Code Zapper Caper

1. Tap the **Blue Mushroom Bumper** or cross the **Rollover Button** to open the Code Zapper Gate
2. Entering the gate crosses the Code Zapper Kicker Rollover Switch, launching the ball up the playfield's left side
3. Crossing the **Left Rebound Gate** scores based on the current Code Symbol:

| Symbol | Value |
|---|---|
| Triangle | 100 |
| Circle | 300 |
| **Square** | **500** |

> 🎯 **Pro Tip:** Advance the Code Symbol toward Square before committing to the Left Rebound Gate shot — the difference between Triangle and Square is 5x the points for the same shot.

### Deep 4 Caper

1. Access the **Deep 4 Alley Gate** when open
2. The ball stops at the first of **four steps**
3. Tap the **White Mushroom Bumper** to advance it one step at a time
4. After 4 White Bumper hits, the ball exits to the plunger lane

### Sea Ray Caper

1. **Six Sea Ray Porthole lights** sit on the ship in the middle of the playfield
2. Light them by crossing the Rollover Button or hitting the **Blue or Yellow** Mushroom Bumpers
3. Each full completion advances the **Super Bonus wheel** by one

### Sea Ray Super Bonus Caper

The Super Bonus wheel tracks how many times you've completed the Sea Ray Caper. At a wheel score of **5, 6, 7, or 8**, you receive either a **Free Replay** or bonus points.

## 🔧 Zipper Flippers & Ball Locking

**Flipper Zipper:**
- **Red Mushroom Bumper** closes the gap between flippers (prevents center drains)
- Any of the **3 Yellow Mushroom Bumpers** reopens it

**Ball Locking (Two Independent Systems — Only One Active at a Time):**
- **Deep 4 Alley** and the **two Kick-Out-Holes** can both trap balls for multiball play, but **if one system is active, the other is not**
- Release trapped balls via the **White Mushroom Bumper**

> 🏆 **Historical note:** Capersville was one of the first machines to feature genuine 3-ball multiball — understanding which lock system is currently live is essential to setting it up correctly.

## 🍄 Mushroom Bumper Reference

| Bumper | Points | Effect |
|---|---|---|
| **Red** | None | Closes flipper gap |
| **White** | 10 | Releases captive balls, advances Deep 4 steps |
| **Yellow** (×3) | 100 each | Advances Code Zapper, lights Sea Ray portholes, opens flippers |
| **Blue** | 10 | Opens Code Zapper Gate, lights gate indicator |

## ⚠️ Dangerous Shots & Drain Patterns

| Risk | Mitigation |
|---|---|
| **Outlanes** (50 points each) | Advance the Code Zapper symbol, but that's a poor trade for losing the ball | Close the Flipper Zipper (Red bumper) proactively rather than relying on outlane "compensation" |
| **Losing track of which lock system is active** | Wasted shots trying to lock balls in the inactive system | Watch the playfield state before committing to either Deep 4 or the Kick-Out-Holes |
| **Aggressive bumper play during multiball** | Can lose flipper control while juggling multiple balls | Balance bumper-chasing against maintaining solid flipper positioning |

## 💰 Scoring Strategy

### Early Ball

1. Hit the Rollover Button and Blue/Yellow bumpers to start lighting Sea Ray portholes
2. Work the Code Zapper symbol toward Square before taking the Left Rebound Gate shot
3. Close the Flipper Zipper early for safety if you're still learning the layout

### Mid Ball

1. Push through Deep 4's four steps via the White Mushroom Bumper
2. Continue completing Sea Ray Capers to advance the Super Bonus wheel
3. Set up ball locks in whichever system (Deep 4 or Kick-Out-Holes) is currently active

### Late Ball

1. Aim for a Super Bonus wheel score of 5–8 for a Free Replay or bonus points
2. Cash in high-value Code Zapper shots (Square symbol) whenever lined up
3. Release locked balls for multiball scoring opportunities

## ⚙️ Setup Recommendations

| Setting | Recommendation | Reason |
|---|---|---|
| **Playfield pitch** | 5.5°–6.0° | Standard for mid-60s Bally EM machines |
| **Mushroom bumper sensitivity** | Verify consistent activation | Nearly every core mechanic routes through one of the 6 mushroom bumpers |
| **Lock mechanism switching** | Check that only one system locks at a time, as designed | Confirms intended multiball behavior rather than a malfunction |

---

## 📋 Quick Reference: Strategic Priorities

**Early:** Light Sea Ray portholes, advance the Code Zapper symbol, close the Flipper Zipper for safety.
**Mid:** Push Deep 4 steps, keep completing Sea Ray Capers, set up ball locks.
**Late:** Chase a 5–8 Super Bonus wheel score, cash high-value Code Zapper shots, release locks for multiball.

**Target Outcome:** Completing all 4 Capers in a single game, with a Super Bonus wheel score of 5+ for a Free Replay, represents a strong, well-played game on this historically significant machine.
