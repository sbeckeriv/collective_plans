# 🤖 RoboCop Pinball Strategy Guide
## Data East, 1989 | "The Future in Law Enforcement"

---

> ⚠️ **Important Note on Scoring Goals**
> 
> RoboCop uses an alphanumeric display with 1989-era scoring. **1 billion points is not achievable** on this machine. The highest recorded score on Pinside is approximately **80.4 million** <kcite ref="61"/>. A realistic "monster game" target is **20-30 million**, while a solid competitive game is **8-12 million** <kcite ref="61"/><kcite ref="62"/>.

## 🎯 Core Mechanics

### Skill Shot
The skill shot is indicated by two illuminated lane guides flanking the target lane <kcite ref="26"/><kcite ref="34"/>.

| Ball   | Value   |
| ------ | ------- |
| Ball 1 | 100,000 |
| Ball 2 | 200,000 |
| Ball 3 | 300,000 |

**How to Execute:**
- Plunge for the lane surrounded by the two lit guides <kcite ref="26"/>
- Lane change affects the TOP lanes but **NOT** the lit skill shot guides <kcite ref="26"/>
- The plunge often feeds the right lane, so if center or left is lit, you need a power plunge and nudging <kcite ref="26"/>

> 💡 **Pro Tip:** Going through an UNLIT top lane spots one target of that color AND lights that lane <kcite ref="30"/>. This is crucial for building locks safely!

---

### The Three Suspects (Prime Directives)

The playfield has three colored target banks representing suspects:

| Color    | Suspect            | Targets    | Benefit When Complete                                                         |
| -------- | ------------------ | ---------- | ----------------------------------------------------------------------------- |
| 🟢 Green  | Clarence Boddicker | 3 standups | Lights lock, increases spinner value                                          |
| 🟡 Yellow | Emil Antonowsky    | 3 standups | Lights lock, increases spinner value                                          |
| 🔴 Red    | Dick Jones         | 3 standups | Lights lock, increases spinner value, **relights kickback** <kcite ref="29"/> |

**Critical Strategy:** 
> ⚠️ **NEVER shoot the standups directly!** <kcite ref="28"/> The targets are deadly and will drain your ball. Instead, use the top lanes to spot targets safely.

**How Top Lanes Work:**
- Each top lane corresponds to a target color <kcite ref="28"/>
- Rolling through an UNLIT lane spots one target of that color <kcite ref="30"/>
- Completing ALL top lanes increases Bonus Multiplier (but doesn't spot targets) <kcite ref="28"/>

---

### The Spinner 🌀

The spinner is your primary scoring shot and the heart of the game <kcite ref="74"/>.

| Locks Lit | Spinner Value |
| --------- | ------------- |
| 0         | Base value    |
| 1         | Increased     |
| 2         | Higher        |
| 3         | Maximum       |

**Spinner Strategy:**
- Rip the spinner constantly from the right flipper <kcite ref="34"/><kcite ref="74"/>
- Spinner value increases as you light more locks <kcite ref="74"/><kcite ref="76"/>
- Try backhanding the spinner for variety <kcite ref="74"/>
- The spinner feeds the top lanes and pops, which can be dangerous <kcite ref="34"/>

> ⚠️ **Warning:** Do NOT shoot early to the spinner and hit the post—you'll drain straight down the middle (SDTM) <kcite ref="28"/>

## 🔒 Multiball

### Qualifying Locks

Locks are lit by completing suspect target banks <kcite ref="74"/><kcite ref="77"/>:

1. Complete a colored target bank (via top lanes or direct hits)
2. That suspect's lock lights on the right ramp
3. Shoot the right ramp to lock the ball
4. Repeat for all three suspects

**Lock Indicators:** Green, yellow, and red lamps on the back panel (upper left) show which balls are locked <kcite ref="34"/>.

> 📝 **Note:** This is NOT a lock-stealing game, but other players can empty your locked balls during their multiball, requiring you to relock <kcite ref="34"/>.

---

### Starting Multiball

Lock 3 balls on the right ramp to start 3-ball multiball <kcite ref="34"/><kcite ref="78"/>.

---

### Jackpot Structure

| Feature              | Details                                       |
| -------------------- | --------------------------------------------- |
| **Jackpot Location** | Right ramp <kcite ref="78"/>                  |
| **Starting Value**   | ~500,000 <kcite ref="28"/>                    |
| **Maximum Value**    | 1,000,000 <kcite ref="28"/><kcite ref="34"/>  |
| **Jackpots Per MB**  | **Only 1** <kcite ref="28"/><kcite ref="78"/> |

**Building Jackpot Value:**
- During multiball, switch hits build the jackpot toward maximum <kcite ref="34"/>
- Targets build value faster, but spinner is safer <kcite ref="34"/>
- Wait until jackpot reaches 1M before collecting! <kcite ref="28"/>

> 💡 **Pro Tip:** Do NOT immediately shoot the ramp for jackpot! Shoot the spinner at least once and let bounces off standups increase the jackpot to max 1M first <kcite ref="28"/>.

---

### Post-Jackpot: Lighting Special

After collecting the jackpot:
1. The game wants you to complete 5 targets to light Special on the ramp <kcite ref="28"/>
2. **Don't do this!** The targets are still dangerous <kcite ref="34"/>
3. Instead, keep shooting spinner and ramp until down to one ball <kcite ref="34"/>
4. Special (if lit) is worth 1,000,000 when set to points <kcite ref="27"/>

## 📈 Progression Systems

### Spinner Value Progression

Spinner value increases as you light locks <kcite ref="74"/><kcite ref="76"/>. This is cumulative throughout your game.

### Jumpmaster & Everything is Lit

| Feature               | Details                                                                         |
| --------------------- | ------------------------------------------------------------------------------- |
| **Jumpmaster**        | Tracks successful jump ramp shots                                               |
| **Awards**            | Up to 500,000 points, various awards including Double Scoring <kcite ref="34"/> |
| **Everything is Lit** | Achieved by tying the Jumpmaster record <kcite ref="27"/>                       |
| **Default Record**    | 14 jumps (adjustable down to 5) <kcite ref="27"/>                               |

**Double Playfield Scoring:**
- Two consecutive right ramp shots starts Double Playfield <kcite ref="27"/>
- Based on switches, not a timer <kcite ref="27"/>
- If the ski jump and post-transfer to left inlane is working, you can use that route <kcite ref="27"/>

## 🤖 ED-209 Mode

This is the closest thing to a "wizard mode" on RoboCop.

**How to Start:**
1. Complete any standup target bank <kcite ref="27"/>
2. A lit crosshair insert cycles between targets <kcite ref="27"/>
3. Hit 4 cycling crosshair targets (default setting) <kcite ref="27"/>
4. ED-209 mode starts with a short timer <kcite ref="27"/>

**During ED-209:**
- Quickly shoot the right ramp for **1,000,000 points** <kcite ref="27"/>

> ⚠️ **Warning:** Do NOT actively shoot for the cycling crosshairs—standups will kill you! Let lucky bounces hit them <kcite ref="27"/>.

## 🛡️ Kickback Management

The left outlane kickback is crucial for survival.

**How to Relight:**
1. **Far right captive ball** <kcite ref="27"/>
2. **Lighting suspect Jones (Red)** <kcite ref="29"/> — Much safer option!

> 💡 **Advanced Strategy:** If your kickback is lit, consider avoiding lighting Jones. When you use the kickback, THEN light Jones to get it back <kcite ref="33"/>. However, you lose 1K per spinner rip by not having Jones lit <kcite ref="33"/>.

## 🏆 Scoring Strategy

### Early Game (Ball 1)
1. ✅ Nail the skill shot (100K)
2. ✅ Rip spinner to feed top lanes
3. ✅ Use top lanes to spot targets (go through UNLIT lanes)
4. ✅ Light first lock
5. ✅ Lock first ball

### Mid Game (Ball 2)
1. ✅ Skill shot (200K)
2. ✅ Continue spinner/top lane strategy
3. ✅ Light and lock remaining balls
4. ✅ Start multiball
5. ✅ Build jackpot to 1M, then collect
6. ✅ Work toward Jumpmaster record

### Late Game (Ball 3+)
1. ✅ Skill shot (300K)
2. ✅ Repeat multiball cycle
3. ✅ Achieve Everything is Lit if possible
4. ✅ Trigger ED-209 for 1M bonus
5. ✅ Maximize bonus multiplier

## ⚙️ Setup Recommendations

### Pitch & Level
- This is a **super fast and deadly game** <kcite ref="34"/>
- Standard pitch (6.5°) is already challenging
- Steeper pitch makes it even more brutal

### Tilt Sensitivity
- **No debounce** on tilt—your first warning often results in a tilt <kcite ref="34"/>
- Be very careful with nudging

### Operator Adjustments to Check
- Jumpmaster record (default 14, can be set as low as 5) <kcite ref="27"/>
- Special set to points (1M) vs extra ball <kcite ref="27"/>
- Spinner lit timing (default: lit after completing bank; can be set to require locked ball) <kcite ref="28"/>

## 🚫 Common Mistakes to Avoid

1. **Shooting standups directly** — Use top lanes instead!
2. **Collecting jackpot too early** — Wait for 1M max
3. **Shooting spinner too early** — Hit the post = death
4. **Aggressive nudging** — No tilt debounce!
5. **Ignoring kickback** — Keep it lit via Jones or captive ball

---

## 🎬 What Completing Modes Changes

RoboCop is a 1989 game with simpler rules than modern machines:

- **Completing target banks:** Lights locks, increases spinner value
- **Multiball:** Resets after completion; can be repeated
- **Everything is Lit:** One-time achievement per game
- **ED-209:** Can be triggered multiple times

There are no permanent mode completions or wizard mode progressions in the modern sense.
