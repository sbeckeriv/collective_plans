## 🎯 Core Mechanics

### Skill Shots

At the start of every ball, multiple skill shots are available. Hitting any major switch disables them (except combo skill shot which is timed).

| Skill Shot              | How to Execute                             | Reward                                                  |
| ----------------------- | ------------------------------------------ | ------------------------------------------------------- |
| 🎸 **Rock-O-Meter**      | Soft plunge, time it when top light blinks | Instantly qualifies playfield multiplier at shown level |
| 🚐 **Van Mod-ulator**    | Plunge into drop targets                   | Instantly awards flashing van mod                       |
| 📻 **Super Sonic Radio** | Hit radio targets on left                  | Mystery award + advance radio dial                      |
| ✋ **Combo Hi-Five**     | Hit left crossover before timer expires    | Start combo chain (up to 6x)                            |
| 🔒 **Secret #1**         | *Hint: Instantly lights Overdrive*         | Ball save at left outlane                               |
| 🔒 **Secret #2**         | *Hint: Awards short ball saver*            | Temporary ball save                                     |

> **💡 Pro Tip:** On tournament copies without ball save, prioritize the drop target skill shot to boost your first mode. If plunge is too weak, go for radio targets for mystery award.

---

### Van Mods (The Mod-ulator)

The Mod-ulator consists of 6 lights in the center playfield. Before starting a van mode, upgrade your van for perks:

**How to Mod:**
1. Three lights cycle on the left (Engine, Speaker, Bomb from top to bottom)
2. Hit any drop target to lock in selection
3. Complete all 3 drops WITHOUT hitting the paddle
4. Hit the paddle repeatedly to level up further (timed)

| Mod                     | Perk          | Level 1           | Additional Levels  |
| ----------------------- | ------------- | ----------------- | ------------------ |
| ⚡ **Engine** (Top)      | Extra Time    | +20 seconds       | +10 sec per level  |
| 🔊 **Speakers** (Middle) | Extra Score   | +500,000 per shot | +250,000 per level |
| 💣 **Bomb** (Bottom)     | Spotted Shots | +2 shots spotted  | +1 shot per level  |

**Mod Level Colors:** Yellow → Light Green → Green → Cyan → Blue → Magenta → Red (Levels 1-7)

> **💡 Pro Tip:** Later modes practically REQUIRE mods to complete. Always prioritize modding before starting difficult modes like Roswell or Chicago.

### Van Modes (City Modes)

**How to Start:** Spell V-A-N by shooting lit white arrows → Shoot left ramp → Select destination

All modes require hitting lit shots to fill the meter, then make the final shot at the Overlord to finish.

**Time Bonus Formula:** 1% of total mode points × seconds remaining (minimum 10x multiplier)

| City              | Song                     | Objective                                      | Starting Value | Increment   | Special Shots                              |
| ----------------- | ------------------------ | ---------------------------------------------- | -------------- | ----------- | ------------------------------------------ |
| 🟣 **New Orleans** | "I'll Stick Around"      | All shots lit, shoot one then repeat that shot | 2,000,000      | 200,000     | Side loop/ramp = 2x progress               |
| 🔵 **Seattle**     | "This is a Call"         | Shots move right to left                       | 2,000,000      | 250,000     | Side loop = 2x progress, 3x score          |
| 🟢 **Los Angeles** | "Learn to Fly"           | Ramp combos                                    | 1,000,000      | 100,000     | Side ramp = 2x value; Combos = 2x progress |
| 🟠 **Roswell**     | "Run"                    | Chase orange shot, avoid green (Overlord)      | 1,750,000      | 150,000     | Orange = 3x score, 2x progress             |
| 🟡 **New York**    | "Breakout"               | Spinner builds value, center collects          | 500,000        | 10,000/spin | Roving shot = 3x value, 2x progress        |
| 🩵 **Chicago**     | "Something From Nothing" | Hurry-up spreads if missed                     | 3,500,000      | 500,000     | Single lit shot = 2x progress              |

> **💡 Pro Tip:** Complete modes in one try for FooBot part upgrades and higher Austin wizard mode jackpots. Aim for 50M+ per mode!

**Recommended Mode Order:**
1. **New Orleans** - Stack with first Overlord multiball
2. **Seattle** or **Los Angeles** - Quick completions
3. **Austin** (if jackpot >1.5M) or **D.C.**
4. **New York** - Stack with Combotron multiball
5. **Roswell** then **Chicago** - Save hardest for last with maxed mods

## 🎱 Multiballs

### Overlord Multiballs

All three share jackpot values that persist and can be increased:
- **Starting Jackpot:** 1,000,000
- **Overlord target hit:** +50,000
- **Captive ball during re-lock:** +150,000

**Re-Lock Bonus:** Hit both Overlord targets → Lock ball → 2x jackpots for 20 seconds + spot a jackpot (All My Life)

#### 🎸 All My Life (1st Overlord MB)

| Requirement | Details                                                                  |
| ----------- | ------------------------------------------------------------------------ |
| **Qualify** | Shoot both targets (or Overlord to spot) → Lock ball → Bash captive ball |
| **Balls**   | 3-ball multiball                                                         |

**Jackpot Sequence:**
1. Left ramp, Right ramp, Spinner
2. Left crossover, Right orbit
3. Side loop, Side ramp
4. **Overlord back target = SUPER JACKPOT**

> **💡 Pro Tip:** Use action button to spot difficult jackpots. Re-lock for 2x jackpots!

#### 🔧 Monkey Wrench (2nd Overlord MB)

| Requirement | Details                                                                     |
| ----------- | --------------------------------------------------------------------------- |
| **Qualify** | Shoot Overlord → Hit both targets 2x → Shoot Overlord → Lock → Bash captive |
| **Balls**   | 3-ball multiball                                                            |

**Jackpot Pattern:**
- All shots lit for jackpots
- Same shot again = **Double Jackpot** (+150k to base)
- Complete all jackpots + doubles = **Super Jackpot at Overlord**

#### 🎭 The Pretender (3rd Overlord MB)

| Requirement | Details                                                                     |
| ----------- | --------------------------------------------------------------------------- |
| **Qualify** | Shoot Overlord → Hit both targets 3x → Shoot Overlord → Lock → Bash captive |
| **Balls**   | 3-ball multiball                                                            |

**Jackpot Pattern:**
- All shots lit for jackpot
- Any hit relights other jackpots
- Hit all unique shots = **Super Jackpot at Overlord**

### 👽 Area 51 Multiball

**Premium/LE Qualification:**
1. Right ramp → Upper playfield
2. Complete green lock targets
3. Shoot upper playfield spinner (builds jackpot)
4. Shoot upper playfield outer loop
5. Shoot under diverter to back target

**Pro Qualification:** 5 right ramp shots

| Phase             | Objective                                                                 |
| ----------------- | ------------------------------------------------------------------------- |
| **Jackpots**      | All shots lit (light blue) - 8 jackpots to advance                        |
| **Super Jackpot** | Prem/LE: Secret Files target on upper PF / Pro: Right ramp → Captive ball |

> **💡 Pro Tip:** Stack Area 51 with van modes for utility. Can be started during Overlord multiballs!

---

### 🤖 Combotron Multiball

**Qualification:** Cash out 6 combos at the Overlord → Side ramp lights

**How Combos Work:**
- Hit shots with Combotron inserts in succession
- Colors progress: White → Yellow → Green → Blue → Red
- Cash out at Overlord: 3x value (5x for back target direct hit)

**During Multiball:**
- Keep hitting lit combos to raise levels
- Red combo = Jackpot lit at Overlord
- Multiple red shots = Up to 5x jackpot multiplier!

**Band Member Combos (collect all 6 for Toy Time MB):**

| Member | Shot 1         | Shot 2    | Shot 3            |
| ------ | -------------- | --------- | ----------------- |
| Taylor | Side Ramp      | Spinner   | Side Ramp         |
| Dave   | Right Ramp     | Left Ramp | Right Orbit       |
| Nate   | Side Loop      | Side Loop | Side Loop         |
| Chris  | Right Orbit    | Left Ramp | Right Orbit       |
| Pat    | Left Ramp      | Spinner   | Side Ramp         |
| Rami   | Left Crossover | Side Loop | MOD-ulator Paddle |

## 📈 Progression Systems

### FooBot Parts

Collect all 6 parts to light FooBot Multiball at left ramp.

| Part            | Member | Shot Multiplier   | How to Collect               | Upgrade Criteria                      |
| --------------- | ------ | ----------------- | ---------------------------- | ------------------------------------- |
| 💜 **Torso**     | Taylor | 2x Side Ramp      | Start Combotron MB           | Score 25M in Combotron MB             |
| 🧠 **Head**      | Dave   | 2x Right Ramp     | Start Area 51 MB             | Get Super Jackpot in Area 51          |
| 💪 **Left Arm**  | Nate   | 2x Side Loop      | Super Jackpot in Overlord MB | 2 supers or 1 2x super in All My Life |
| 💪 **Right Arm** | Chris  | 2x Right Orbit    | Start Bot Frenzy             | Score 25M in Bot Frenzy               |
| 🦵 **Left Leg**  | Pat    | 2x Left Ramp      | Complete van mode in one try | Score 50M in a van mode               |
| 🦵 **Right Leg** | Rami   | 2x Left Crossover | Start Supersonic Radio       | Score Radio Jackpot                   |

**Part Status:**
- **Collected (solid):** 2x multiplier active for current ball only
- **Upgraded (flashing):** 2x multiplier active for ENTIRE GAME

### Rock-O-Meter (Playfield Multiplier)

| Step | Action                                         |
| ---- | ---------------------------------------------- |
| 1    | Shoot right orbit when "battery" insert is lit |
| 2    | Roll through Rock-O-Meter lane 3 times         |
| 3    | **2x Playfield** for 40 seconds!               |
| 4    | Repeat while 2x running = **3x Playfield**     |
| 5    | Repeat while 3x running = +10 seconds          |

**Relighting Battery:** Shoot MOD-ulator paddle target

> **💡 Pro Tip:** Get one shot away from 2x, then start Overlord MB + van mode, THEN make the final right orbit shot for massive multiplied scoring!

### Ray Gun Multipliers

| Action                               | Result            |
| ------------------------------------ | ----------------- |
| Alley pass through ray gun target    | 2x next shot      |
| Immediately hit OTHER ray gun target | **4x next shot!** |

**Stacking:** Ray gun multipliers stack with FooBot parts, playfield multiplier, AND 2x jackpot multiplier during Overlord MBs!

**Relight:** Roll through respective inlane

## 🔗 Stacking & Combos

### ✅ What Stacks Well

| Combination                      | Why It Works                                 |
| -------------------------------- | -------------------------------------------- |
| **Van Mode + Overlord MB**       | Utility multiball helps complete modes       |
| **Van Mode + Area 51 MB**        | Especially New Orleans - all shots lit!      |
| **New York + Combotron MB**      | Both heavily use spinner/side ramp           |
| **Bot Frenzy + Any Multiball**   | Spinner scoring explodes with multiple balls |
| **Supersonic Radio + Any Mode**  | Pink shots add to everything                 |
| **2x/3x Playfield + Everything** | Obvious massive multiplier                   |
| **Combotron MB + Area 51 MB**    | Can start Combotron during Area 51           |

### ❌ What NOT to Stack

| Combination                      | Why to Avoid                           |
| -------------------------------- | -------------------------------------- |
| **Combotron MB + Overlord MB**   | Cannot start Combotron during Overlord |
| **Multiple Overlord MBs**        | Only one at a time                     |
| **Difficult modes without mods** | Roswell/Chicago need van mods          |

### ⏰ Optimal Timing

1. **Ball 1:** Get one shot from 2x playfield → Start Overlord MB + New Orleans → Make final right orbit shot
2. **Ball 2:** Complete Seattle/LA → Start Area 51 → Work toward Austin/D.C.
3. **Ball 3:** Stack Bot Frenzy + Supersonic Radio with remaining multiballs

## 🧙 Wizard Modes

### 🌮 Austin (Mini-Wizard Mode)
*Song: "Times Like These"*

**Requirement:** Complete 3 Van Modes

**Gameplay:**
- 6-ball multiball
- Red shots = Jackpots (1.5M + 500k per mode completed in one try)
- Blue shot = Super Jackpot (worth jackpot total × multiplier)
- Overlord moves the blue shot location
- 4 jackpots before super = 5x super + Add-a-ball
- 6x and 7x supers also award add-a-balls

> **💡 Pro Tip:** Play Austin when jackpot value exceeds 1.5M for best returns.

---

### 🏛️ D.C. (Mini-Wizard Mode)
*Song: "Holding Poison"*

**Requirement:** Complete 3 Van Modes (alternate to Austin)

**Gameplay:**
- 6-ball multiball
- Band member shots = Jackpots (1M + 50k per mod + 25k per subsequent)
- Complete drops → Super Jackpot at corresponding band member
- Left shots (crossover, ramp, loop) = 2x Super Jackpots
- Right shots (side ramp, right ramp, orbit) = Add-a-balls
- 6 unique supers = Victory Laps!

> **💡 Pro Tip:** Mod out your van heavily before D.C. - jackpots scale with mod levels!

---

### 🤖 FooBot Multiball (Mini-Wizard Mode)
*Song: "My Hero"*

**Requirement:** Collect all 6 FooBot parts

**Gameplay:**
- Starts with 2 balls
- All band member shots lit for jackpots
- Jackpot value = 3M base + % of related feature scoring (up to 10M each)
- Shoot band member → That shot + Overlord lit
- Keep shooting same shot to increase multiplier OR shoot Overlord for jackpot + add-a-ball
- Complete all 6 = Super Jackpot at Overlord (10% of all jackpots × balls in play)

**Jackpot Contributions:**
| Shot           | Source             | Percentage |
| -------------- | ------------------ | ---------- |
| Left Crossover | Supersonic Radio   | 10%        |
| Left Ramp      | All Van Modes      | 2%         |
| Upper Loop     | Overlord MBs       | 10%        |
| Side Ramp      | Combotron Jackpots | 10%        |
| Right Ramp     | Area 51 MB         | 10%        |
| Right Orbit    | Bot Frenzy         | 10%        |

---

### 🧸 Toy Time Multiball (Mini-Wizard Mode)

**Requirement:** 
- Score all 6 band member combos
- Complete all 6 van modes (if no topper)
- *With topper: Progress carries across games!*

**Gameplay:**
- 3-ball multiball
- Shoot band member → Collect switch hits to light jackpot
- Score jackpot = Add-a-ball
- All 6 jackpots = FooBot Super Jackpot (total of all jackpots + awards/upgrades FooBot part)

**Lock Bonus:** Hit both Overlord targets → Lock ball → 2x switch hits and jackpot increases

---

### ⚔️ The Final Battle (Ultimate Wizard Mode)

**Requirements:**
1. ✅ Save all cities (including Austin AND D.C.)
2. ✅ Play all 3 Overlord Multiballs
3. ✅ Play FooBot Multiball

**Start:** Left ramp when all requirements met

#### Phase 1: Escape
- Rescue Pat and Dave via left/right ramp hurry-ups
- Bot targets or upper PF shots spot right ramp
- Hurry-up: 10M, +5M per shot
- 3 shots each ramp → Overlord shot to advance
- **Unlimited ball save ~1 minute**

#### Phase 2: Rescue
- Rescue other 4 band members at their shots
- Each shot increases multiplier for next
- Complete all 4 → Overlord to advance
- **Unlimited ball save ~1 minute**

#### Phase 3: Shred Battle
- All shots = 3% damage for 5M
- Shots unlight after hitting (not Overlord)
- Multiplier = unlit shots + 1
- Overlord relights all shots (resets multiplier)
- Deplete Overlord HP → Final blow at Overlord
- **100 second time limit (Dave's HP bar)**

#### Phase 4: Learn to Fly
- 3-ball multiball
- All shots deal 5 HP damage for 1M
- Complete Overlord targets → Super Jackpot + Add-a-ball
- All scoring × balls in play
- 40 shots → Final blow at Overlord
- **150 second time limit**

#### 🎉 Epilogue
- Overlord defeated and redeemed!
- All progress resets
- **FooBot shot multipliers increased by +1x for next playthrough!**

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                                      | Danger                        | Mitigation                         |
| ----------------------------------------- | ----------------------------- | ---------------------------------- |
| **MOD-ulator targets from right flipper** | Ball rebounds SDTM            | Backhand from left flipper instead |
| **Overlord targets (early game)**         | Fast drains if not controlled | Prioritize clearing them early     |
| **Right orbit weak shots**                | Returns through spinner SDTM  | Full power shots only              |
| **Upper loop exit**                       | Ball can kick out dangerously | Nudge when ball exits              |
| **UFO captive ball**                      | Rebounds unpredictably        | Nudge after contact                |

### ✅ Safe Shots

| Shot                                | Why Safe            | Best Use                          |
| ----------------------------------- | ------------------- | --------------------------------- |
| **Center spinner**                  | Feeds upper flipper | Combotron building, New York mode |
| **Side ramp**                       | Controlled return   | Bail-out shot, Combotron          |
| **Left ramp (backhand)**            | Short, consistent   | Mode start, Bot Frenzy start      |
| **Left crossover (from held ball)** | Feeds upper flipper | Raygun multiplier setup           |

### 🎮 Flipper Selection

| Shot             | Recommended Flipper | Notes                       |
| ---------------- | ------------------- | --------------------------- |
| Left Crossover   | Right (held ball)   | Precise angle required      |
| Left Ramp        | Left (backhand)     | Short ramp, easy backhand   |
| Side Loop        | Upper               | From left crossover feed    |
| Side Ramp        | Upper               | From spinner feed           |
| Right Ramp       | Right               | Long, steep - full power    |
| Right Orbit      | Right               | Feeds Rock-O-Meter          |
| Overlord Captive | Left (backhand)     | More consistent than direct |

## 💰 Scoring Strategy

### Path to 1 Billion Points

#### 🌅 Early Game (Ball 1)
1. **Skill shot:** Drop targets for van mod OR Rock-O-Meter for multiplier progress
2. **Clear Overlord targets** immediately (dangerous if left)
3. **Get 2 Rock-O-Meter shots** (one away from 2x)
4. **Spell V-A-N** and select New Orleans
5. **Lock ball in Overlord** (one shot from multiball)
6. **Start multiball + mode together**
7. **Make final right orbit shot** for 2x playfield
8. **Complete mode during 2x** = 50M+ easily

#### ☀️ Mid Game (Ball 2)
1. **Complete Seattle or Los Angeles** quickly
2. **Start Area 51 multiball** during a mode
3. **Work toward Austin** (if jackpot >1.5M)
4. **Collect FooBot parts** via feature completion
5. **Build Combotron combos** passively

#### 🌙 Late Game (Ball 3+)
1. **Stack Bot Frenzy + Supersonic Radio** with multiballs
2. **Complete D.C.** with maxed van mods
3. **Finish Roswell and Chicago** with mod support
4. **Play FooBot Multiball**
5. **Reach The Final Battle**

### 💎 Highest Value Opportunities

| Opportunity                                   | Potential Value | How to Maximize                       |
| --------------------------------------------- | --------------- | ------------------------------------- |
| **Van modes during 2x/3x PF**                 | 100M+           | Stack with playfield multiplier       |
| **Austin with 4+ modes completed in one try** | 150M+           | High jackpot base + super multipliers |
| **D.C. with maxed mods**                      | 100M+           | Mod heavily before starting           |
| **Bot Frenzy during multiball**               | 50M+            | 4 unique targets = 8x spinner value   |
| **The Final Battle Phase 4**                  | 200M+           | Keep balls in play, hit supers        |
| **FooBot MB with good feature scores**        | 100M+           | Max out contributing features first   |
