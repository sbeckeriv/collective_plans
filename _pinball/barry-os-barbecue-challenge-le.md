# 🍖 Barry O's Barbecue Challenge - Strategy Guide

*American Pinball, 2024 — A tribute to designer Barry Oursler*

> **"Cook your recipes. Stack your sauce. Don't let the grill get too hot."**

Barry O's BBQ Challenge is a tribute machine built around six Cook-Off modes ("Stars of BBQ"), each demanding multiple ingredients before completion, feeding into Pitmaster Multiball and the Meat Coma mini-wizard. Our goal: win all 3 cook-offs needed for Meat Coma, then stack Pitmaster's progressive jackpot ladder.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Pick your skill shot deliberately** — Left Outlane (3M base + FIRE boost) and Upper Right Saucer (2M base + 1M jackpot boost) are the highest-value options if you're confident in your aim
- [ ] **Complete recipes via orange shots** during Cook-Offs — don't drain mid-recipe, since recipe value resets on drain
- [ ] **Use both flippers on the BOSS/CHEF lanes** — right flipper controls the bottom row (CHEF), left flipper the top row (BOSS)
- [ ] **Don't ignore Side Dishes** — your first two completions per cook-off earn stars toward winning it
- [ ] **Win 3 cook-offs to unlock Meat Coma Multiball** — this is your first major mini-wizard target
- [ ] **Build your Bonus X early** (max 21x) — it multiplies your entire end-of-ball bonus

## 🔥 Core Mechanics

### Skill Shot (Pick Before Plunge)

| Shot | Base Value | Bonus Effect |
|---|---|---|
| BBQ Targets | 500K | Easier pit targets |
| Left Saucer | 1M | +500K to recipe & multiball jackpot |
| Left Ramp | 750K | Recipe values boosted |
| Right Orbit | 750K | BOSS Assistant (easier top lanes) |
| Right Ramp | 1M | Playfield X time expander |
| Upper Right Saucer | 2M | +1M to jackpot |
| Lower Right Saucer | 1.5M | +750K to jackpot |
| Left Inlane | 2.5M | Challenge scoring increases |
| **Left Outlane** | **3M** | **FIRE boosted (+1M to hurry-ups)** |

> 🎯 **Pro Tip:** Skill shot values increase the more unique shots you've hit throughout the game — revisit different skill shot choices across balls rather than always picking the same one.

### Cook-Offs (Main Mode)

Face six "Stars of BBQ" by completing recipes via lit **orange shots**. Recipe value grows with every switch hit during the cook-off — but **resets entirely if you drain**, so finish quickly once you start.

**Earn up to 5 stars per cook-off:**
1. Complete the main recipe (orange shots)
2. Apply "Spinner Sauce" via the spinners
3. Find the correct spice level via post targets
4. Complete **two** side dish drop target sets

**Win 3 cook-offs to unlock Meat Coma Multiball** — your first major milestone.

### BOSS and CHEF Lanes

Top lanes split by flipper control: **right flipper = bottom row (CHEF)**, **left flipper = top row (BOSS)**.

| Completion | Reward |
|---|---|
| **CHEF** | Bumpers +1K, recipe +150K (+25K each additional), one lit orange shot toward recipe |
| **BOSS** | Bumpers +1K, recipe +300K (+50K each additional), random award (first one each ball is always a Playfield X upgrade) |

**BOSS Assistant** (from Right Orbit skill shot or the Drag Race challenge) makes your next BOSS/CHEF completion easier.

### FIRE Hurry-Up

A short timed feature with a rolling score (200K–1M base). Complete by hitting **either blue orbit**.

> ⚠️ **Failing a FIRE hurry-up halves your current recipe value** — only let one expire if you have no other choice.

Each completed FIRE hurry-up in a ball adds **+0.5x multiplier** to the next one (max **5x**). The base value (not the multiplied bonus) adds directly to your recipe.

### Side Dishes

Complete drop targets near the right orbit. Your **first two completions per cook-off earn stars**; further completions add to your eventual cash-out value. Benefits include Bonus X increases, challenge value boosts, spinner value boosts, and a **sweep bonus** (double effect for clearing the whole bank at once). Collect all three flashing inserts to start **Side Dish Multiball**.

## 🏁 Hot Rod Challenges (9 Timed Modes)

Qualify by hitting orbits with the "Qualify Challenge" insert lit, then start at an open saucer. Raise shot values beforehand via purple arrows or drop targets.

| Challenge | Shots | Award |
|---|---|---|
| Ignition | BBQ PIT targets | Easier pit targets |
| Rally Ramps | Ramps | Ramps worth millions |
| Cruise Night | Bumpers → PIT bank | Bumpers worth more |
| Bike Night | Orbits | Orbits worth millions |
| Custom Upgrades | B/I/Mild/Spicy targets | Spicy targets worth more |
| Hot Sauce Station | Saucers | Saucers worth millions |
| Drag Race | BOSS/CHEF lanes | BOSS Assistant |
| Lift Off | Two flashing shots | FIRE Boosted (+1M) |
| Speedy Delivery | 5-shot sequence | Equipment repairs 2x |

Every shot made during any challenge also adds **350K to your end-of-ball bonus**.

## 🎁 Multiballs

### Pitmaster Multiball

1. Lock ball(s) in the side saucers
2. Hit the back-right saucer to start
3. **Progressive jackpot** (500M standard, carries over outside tournament mode); base jackpot determined by your recipe value
4. Ladder progression: **1x → 1.5x → 2x → ... → 10x (jackpot 19) → 15x (jackpot 20) → progressive jackpot chance**
5. **Super Jackpot:** lock balls at two saucers, scores the next two jackpot values
6. **Mega Jackpot:** lock at all three saucers, scores two values above the Super
7. Relight jackpots via the upper-right saucer (the same one that starts multiball initially)

> 💰 **Pro Tip:** Building recipe value to its maximum *before* starting Pitmaster directly increases your base jackpot — don't rush into multiball with a low recipe value if you can help it.

### Side Dish Multiball

1. Collect all three flashing Side Dish inserts (complete drop targets 3 times)
2. **Red shots = jackpots** at the base multiball value, incrementing after switch hits
3. Relight by completing the drop targets again
4. **A second Side Dish Multiball requires two complete insert sets**

### Firework Frenzy

1. **Bash-lock** balls already locked for Pitmaster (left or lower-right saucer)
2. Collect **10 fireworks** to start
3. Jackpots start at **5M**, increasing via left ramp or saucer collections
4. **Countdown timer** — locked balls eject if it reaches zero; reset it by locking/hitting balls
5. Lock all remaining balls before the timer expires for an **Add-A-Ball + Super Jackpot**

## 🏆 Mini-Wizard Mode: Meat Coma Multiball

**Requirement:** Win 3 cook-offs.

- Six lit areas (ramps, orbits, standup banks) show **red jackpots**
- Hitting one area unlights it; hitting a bank target unlights the entire bank
- Once all six are lit, the saucers light red — shoot or bash-lock them to relight all six areas again
- **Add-A-Balls** awarded for the first three jackpots collected
- Mode runs **20 seconds** per ball, continuing if an Add-A-Ball is available

## 📊 Scoring Systems

### Bonus X

Maximum **21x** (built as 1x + 2x + 3x + 4x + 5x + 6x stages). Increase via drop targets, BOSS awards, or **Double Bonus X** from BBQ PIT targets.

### Playfield Multiplier

Right ramp shots increase the playfield multiplier by **+0.5x** (max **4x**) for ~30 seconds. Extend it further via right ramp skill shots, BBQ PIT awards, or BOSS awards.

### Combos & Combomania

Hitting shots in succession scores combos (shown via white arrows). **Combomania** starts requiring 15 combos, with each subsequent activation requiring more — a 60-second feature where one-way combos score; it can carry into multiball but can't be started during one.

### End-of-Ball Bonus (× Bonus X)

| Component | Value |
|---|---|
| Balls played | 250K each |
| Cook-offs won | 500K each |
| FIRE completions | 350K each |
| Side dishes | 200K each |
| Challenge shots made | 350K each |
| BBQ PIT awards | 1M each |
| Plus: challenge bonus, Pitmaster bonus, bonus tip |  |

## ⚠️ Dangerous Moments & Mistakes

| Risk | Mitigation |
|---|---|
| **Draining mid-recipe** | Resets your entire recipe value progress | Finish recipes efficiently rather than chasing side objectives mid-cook-off |
| **Letting FIRE hurry-up expire** | Halves your current recipe value | Always have a plan to hit a blue orbit before the timer runs out |
| **Starting Pitmaster with low recipe value** | Caps your base jackpot low for the whole multiball | Build recipe value as high as practical before triggering it |
| **Letting the Firework Frenzy timer expire** | Locked balls eject, ending the mode early | Keep locking/hitting balls to reset the timer if you're not ready to finish |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Choose a high-value skill shot you're comfortable making
2. Start your first cook-off and work toward completing the main recipe
3. Build Bonus X via drop targets and BOSS/CHEF lane completions

**Target Score:** 50M–150M

### Mid Game (Ball 2)

1. Win your second cook-off, working toward the 3 needed for Meat Coma
2. Qualify and run a few Hot Rod Challenges for end-of-ball bonus padding
3. Build toward Pitmaster Multiball with a strong recipe value banked

**Target Score:** 150M–400M

### Late Game (Ball 3+)

1. If you've won 3 cook-offs, dive into **Meat Coma Multiball**
2. Stack Pitmaster's jackpot ladder as high as you can, chasing Super/Mega Jackpots
3. Push into Firework Frenzy if Pitmaster balls are already locked

**Target Score per ball:** 200M–600M+
