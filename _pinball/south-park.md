# 🏔️ South Park - Strategy Guide

*Sega Pinball, 1999 — Designed by Joe Balcer & Joe Kaminkow*

> **"Oh my God, they killed Kenny! ...But not your multiball."**

South Park (Sega, 1999) is a 6-player Sega machine featuring all four main boys from the show, character-based modes, multiball, and the infamous Cartman voice package. The game supports up to 6 players — unique among pinball machines of its era. Our goal: **150M–350M** through character mode completion and multiball jackpot exploitation.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Shoot the right ramp** to start character modes (Kyle, Stan, Kenny, Cartman)
- [ ] **Complete character modes** to unlock multiball and wizard mode
- [ ] **Hit the center spinner** for fast base points and mode advance
- [ ] **Lock balls** via the left scoop to start South Park Multiball
- [ ] **Trigger the Chef Award** via pop bumpers for mystery bonuses
- [ ] **Avoid the right outlane** — it's the primary drain path on this machine

## 🧒 Core Mechanics

### Skill Shot

| Skill Shot | How | Reward |
|---|---|---|
| **Right ramp** | Time the plunge to hit the ramp | Mode start + character select |
| **Lane** | Hold flipper to light lane | Bonus multiplier step |

### Character Modes (4 Boys)

Each character has a unique mode triggered by the right ramp:

| Character | Mode Theme | Key Shot |
|---|---|---|
| **Stan** | Football — hit targets in sequence | Center targets |
| **Kyle** | Dodgeball — timed target hits | Left and right standups |
| **Kenny** | Kenny Dies — race against ball drain timer | Right ramp fast |
| **Cartman** | Respect My Authoritah — bash specific targets | Center bash toy |

Completing a character's mode awards a character bonus and advances toward South Park Multiball.

### Chef Award (Pop Bumpers)

Hit the pop bumpers repeatedly to accumulate Chef hits. At specific thresholds, a Chef Award is granted:
- Mystery points
- Mode extension
- Extra ball lighting
- Multiball lock advance

### Spinner (Center)

The center spinner is the fastest scoring shot on the machine. During active modes, spinner hits advance mode objectives passively. Always shoot through it when passing.

## 🎳 Multiball

### South Park Multiball

**How to start:** Complete 2 character modes, then lock 2 balls in the left scoop.

**Jackpot structure:**
- 3-ball multiball
- Right ramp = jackpot (5M–20M, escalating)
- **Super Jackpot:** Complete all 4 character modes during multiball → shoot left scoop for super jackpot (50M–100M+)
- Spinner feeds right ramp; use it to chain jackpot shots

### Kenny Multiball (Mini)

Triggered specifically by completing Kenny's mode — a brief 2-ball multiball where every shot scores the "Kenny Die" value × 2. Shorter but starts faster than the main multiball.

## 🔄 Progression Systems

- **Character completion** is cumulative — each completed character stays done for the game
- **Chef Award** milestones increase in value per threshold reached
- **6-player format** — in multiplayer games, character mode progress is per-player (not shared)

## 🧙 Wizard Mode

### Cartman's Wizard Mode

**Requirement:** Complete all 4 character modes.

**Gameplay:** Cartman takes over the DMD with escalating demands. All shots lit at 5× values. The right ramp repeatedly awards the "Respect My Authoritah" mega jackpot.

## 💰 Scoring Strategy

### Ball 1
1. Right ramp → start Stan mode (easiest objectives)
2. Complete Stan's mode
3. Spinner passes for base points
4. Chef Award via bumpers

### Ball 2
1. Right ramp → Kenny mode (short timer; prioritize speed)
2. Right ramp → Kyle mode
3. Lock first ball for multiball

### Ball 3
1. Lock second ball → South Park Multiball
2. Right ramp jackpots during multiball
3. Complete Cartman mode if possible → wizard

### Score Targets

| Performance | Score |

| Beginner | 30M–80M |
| Solid | 100M–200M |

## 💡 Pro Tips

> **Kenny mode is timed and unforgiving.** When it starts, immediately shoot the right ramp repeatedly — don't pause.

> **6-player games change the score context.** In a 6-player game, conserving ball time is even more critical; each player gets fewer opportunities.

> **Chef Awards compound.** The 3rd and 4th Chef Awards in a ball are worth dramatically more than the 1st. Build bumper hits in clusters, not one at a time.
