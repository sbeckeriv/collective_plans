## 🎯 Core Mechanics

### Skill Shots

| Type                       | How to Execute                                                                               | Reward                                                |
| -------------------------- | -------------------------------------------------------------------------------------------- | ----------------------------------------------------- |
| 🟠 **Standard Skill Shot**  | Soft plunge into orange skill shot target                                                    | 1M + 1 EDDIE letter + 5 sec ball save                 |
| 🔴 **Super Skill Shot**     | Hold left flipper, full plunge to inner orbit, shoot Super Jackpot target with upper flipper | 5M + Instant Playfield X qualified + 10 sec ball save |
| 🟣 **Secret Skill Shot #1** | Soft plunge directly into left outlane                                                       | Awards vary                                           |
| 🟣 **Secret Skill Shot #2** | Soft plunge into left inlane (shatz)                                                         | Awards vary                                           |
| 🟣 **Secret Skill Shot #3** | Full plunge around top, hit captive ball with upper flipper                                  | Awards vary                                           |

> 💡 **Pro Tip:** The Super Skill Shot is extremely valuable—even if you miss the target, you'll often hit one or both upper X targets for playfield multiplier progress <kcite ref="60"/>.

### Playfield Multipliers (2X/3X)

Shoot the four scattered **X targets** around the playfield to light a return lane for 2X scoring <kcite ref="31"/>:

1. Complete X targets once → Light 2X at return lanes
2. Complete X targets again → Light 3X at return lanes
3. Roll through lit return lane to **activate** multiplier (20 seconds)
4. Hit flashing X targets during multiplier to add +5 seconds

> 💡 **Pro Tip:** Qualified Playfield X carries between balls! Save it for high-value moments <kcite ref="31"/>.

### REVIVE (Outlane Ball Save)

Shoot the left spinner to spell **R-E-V-I-V-E** and light outlane ball saves <kcite ref="31"/>:
- First REVIVE: Both outlanes lit
- Subsequent: One outlane lit
- Lit Revives carry between balls
- Cannot stack Revives while both outlanes are lit

> ⚠️ **Warning:** The left spinner is a dangerous shot that can drain quickly. Work on REVIVE during multiballs or with ball save active <kcite ref="31"/>.

## ⚔️ Eddie Battle Modes

Spell **E-D-D-I-E** by shooting white arrow shots, then shoot the center ramp to start the flashing mode <kcite ref="31"/>.

- You're spotted 2 EDDIE letters toward your first mode
- Shooting the left spinner changes the flashing mode (only before EDDIE is complete)
- Completing a mode lights **Soul Shard** at bullseye (10 second hurry-up)
- Single-ball modes can stack with Trooper/Mummy Multiball

### 1. ✈️ Aces High (2-Ball Multiball)

| Phase   | Objective                                                | Scoring                                         |
| ------- | -------------------------------------------------------- | ----------------------------------------------- |
| Stage 1 | Shoot 4 blue shots to defeat fighters                    | Jackpots                                        |
| Stage 2 | Shoot either ramp, then bullseye (repeat for both ramps) | Bomber jackpots                                 |
| Stage 3 | Shoot strobing shot, then bullseye within 5 seconds      | Final jackpot (multiplied by bullseye position) |

- **Add-a-Ball:** First Mystery Award during multiball
- **Soul Shard:** Awarded after defeating the Ace

### 2. 🌙 Fear of the Dark (Single Ball)

| Step | Action                                | Scoring                   |
| ---- | ------------------------------------- | ------------------------- |
| 1    | Shoot any of 4 purple arrow shots     | Lights both spinners      |
| 2    | Rip either spinner for massive points | REVIVE spinner = 2X value |
| 3    | Repeat 3 times total                  | Soul Shard lights         |

> 💡 **Pro Tip:** Shoot the right ramp before hitting spinners to add +2X to spinner value! Getting BOTH spinners going awards 4X on the left spinner <kcite ref="31"/>.

### 3. ⚓ Rime of the Ancient Mariner (2-Ball Multiball)

| Phase      | Objective                                     | Scoring                |
| ---------- | --------------------------------------------- | ---------------------- |
| Hurry-Up   | Shoot center shot to lock in countdown value  | Starting at 1M         |
| Left Side  | Shoot 3 left shots to move them toward center | Shot value + 500K each |
| Jackpot    | Shoot bullseye when shots are centered        | Total shot values      |
| Right Side | Repeat for right side including drop targets  | Soul Shard             |

- **Add-a-Ball:** First Mystery Award during multiball

### 4. ⛪ Hallowed Be Thy Name (Single Ball)

Complete shots **in order** as quickly as possible <kcite ref="31"/>:

| Order | Shot               | Value                         |
| ----- | ------------------ | ----------------------------- |
| 1     | Orange arrow shot  | 3M                            |
| 2     | Captive ball       | 5M                            |
| 3     | Orange arrow shot  | 3M                            |
| 4     | All 3 drop targets | 10M                           |
| 5     | Orange arrow shot  | 3M                            |
| 6     | Bullseye target    | 10M + 1M per second remaining |

> 💡 **Pro Tip:** This mode stacks excellently with Trooper Multiball—the structured shots become easier with multiple balls <kcite ref="66"/>.

### 5. 🔥 Flight of Icarus (Single Ball)

| Objective                    | Scoring                           |
| ---------------------------- | --------------------------------- |
| Alternate ramp shots         | 2M (+150K increment per ramp)     |
| Combo ramps within 5 seconds | 2X, 3X, 4X... multiplied scoring  |
| Score 20M total              | Lights bullseye to end mode early |

> 💡 **Pro Tip:** You can "risk it" and keep playing for more points, or escape early via the bullseye once 20M is reached <kcite ref="31"/>.

### 6. ⏰ 2 Minutes to Midnight (Mini-Wizard Mode)

**Requirement:** Complete all 5 Eddie Battle modes

This is a single-ball, timed mode with a timer counting UP to midnight (effectively 60 seconds) <kcite ref="31"/>:

| Feature         | Details                                                                    |
| --------------- | -------------------------------------------------------------------------- |
| Shots           | 5 major shots lit for 1M + 15% of respective mode score + Soul Shard value |
| Super Jackpot   | Collect all 5 shots to light at Super Jackpot target                       |
| Timer Extension | Shoot any X target to add 5 seconds (one time only)                        |
| Eddie Card      | Level 1 for starting; Level 2 for 2X Super Jackpot                         |

## 🎱 Multiballs

### 🪖 Trooper Multiball (Main Multiball)

**How to Qualify:**
1. Complete drop targets when "Light Lock" is flashing (2 completions first time)
2. Lock 3 balls at lit orbit shots
3. Multiball starts as 3-ball (4-ball if Mummy lock was also lit)

**Jackpot Structure:**

| Jackpot Level | Color  | Multiplier |
| ------------- | ------ | ---------- |
| 1             | Blue   | 1X         |
| 2             | Green  | 2X         |
| 3             | Yellow | 3X         |
| 4             | Orange | 4X         |
| 5             | Red    | 5X         |

- Collect 3 jackpots → Light Super Jackpot
- Super Jackpot relights collected jackpots at +1X
- **Cannon Shot:** Complete drops during multiball to light bullseye for 1-3 jackpots + add-a-ball (first time)

**Eddie Cards:**
- Level 1: Score a Super Jackpot
- Level 2: Advance all jackpots to 3X (yellow) OR score a single 5X jackpot <kcite ref="31"/>

> 💡 **Pro Tip:** Jackpot levels carry over between Trooper Multiballs! Build them up over multiple attempts <kcite ref="31"/>.

### 🧟 Mummy Multiball

**How to Qualify:**
1. Hit captive ball 2 times (5 times for subsequent) to light Mummy Lock
2. Lock ball at left ramp (Premium/LE)
3. Spell M-U-M-M-Y at captive ball (5 more hits)
4. Hit captive ball again to start

**Jackpot Structure:**
- Score 10 switch hits (+10 per jackpot) to qualify jackpot at captive ball
- Each jackpot spots a MUMMY letter
- Spell MUMMY → Light Super Jackpot at bullseye

**Add-a-Balls (up to 3 per multiball):**

| Level | Shots Required                                     |
| ----- | -------------------------------------------------- |
| 1     | Both ramps                                         |
| 2     | Both ramps + both orbits                           |
| 3+    | Both ramps, orbits, loops, center ramp, SJP target |

**Eddie Cards:**
- Level 1: First Super Jackpot
- Level 2: Second Super Jackpot (per game) <kcite ref="31"/>

> 💡 **Pro Tip:** Mummy MB stacks well with Flight of Icarus—the ramp shots for add-a-balls are the same shots needed for the mode <kcite ref="31"/>.

### 🤖 Cyborg Multiball

**How to Qualify:**
Complete all 5 Power Features, then shoot bullseye/center ramp

**During Multiball:**
1. Hit each Power Feature type once to light Super Jackpot at Orb
2. Super Jackpot value determined by shots collected (starts at 4M)
3. Each Super Jackpot adds +1X multiplier
4. First two Super Jackpots add-a-ball

**Eddie Cards:**
- Level 1: Start Cyborg Multiball
- Level 2: Score 5X Super Jackpot <kcite ref="31"/>

> ⚠️ **Note:** Playfield X cannot be qualified during Cyborg MB—bring one in with you! <kcite ref="31"/>

## 📈 Progression Systems

### Power Features

Make shots to activate and complete Power Features <kcite ref="31"/>:

| Feature          | Shots           |
| ---------------- | --------------- |
| 🔄 Power Spinners | Spinner hits    |
| 🔵 Power Orbits   | Orbit shots     |
| 🟢 Power Ramps    | Ramp shots      |
| 🟡 Power Targets  | Target hits     |
| 🔴 Power Bumpers  | Pop bumper hits |

- **Activate:** Make required hits (shown on screen)
- **Complete:** Continue hitting to fill the meter
- Each completed feature adds +3M to Power Jackpot base
- Complete all 5 → Light Cyborg Multiball

### Power Jackpot

Collect at Orb target after completing any Power Feature <kcite ref="31"/>:

| Boost Source                       | Value Added    |
| ---------------------------------- | -------------- |
| Complete Power Feature             | +3M            |
| Level 1 Eddie Card                 | +5M            |
| Level 2 Eddie Card                 | +15M           |
| Tomb Treasure #3                   | +15M           |
| Additional Power Feature while lit | +1X multiplier |
| Mystery Award                      | +1X multiplier |
| Tomb Treasure #8                   | +5X multiplier |

> 💡 **Pro Tip:** Power Jackpot resets on drain! Collect it before you lose the ball <kcite ref="31"/>.

### Eddie Cards

Collect all 4 to qualify **Number of the Beast** <kcite ref="31"/>:

| Card            | Level 1 Requirement         | Level 2 Requirement                     |
| --------------- | --------------------------- | --------------------------------------- |
| 🧟 Mummy Eddie   | Mummy MB Super Jackpot      | 2nd Mummy Super (per game)              |
| 🤖 Cyborg Eddie  | Start Cyborg MB             | Cyborg 5X Super Jackpot                 |
| 🪖 Trooper Eddie | Trooper MB Super Jackpot    | All jackpots at 3X OR single 5X jackpot |
| 💂 Soldier Eddie | Start 2 Minutes to Midnight | 2MTM 2X Super Jackpot                   |

- **Level 1 cards:** +5M to Power Jackpot, lost on failed Number of the Beast
- **Level 2 cards:** +15M to Power Jackpot, retained permanently

### Soul Shards

Collected by shooting bullseye within 10 seconds after completing any Eddie Battle <kcite ref="31"/>:
- Worth 20% of mode score × 2X or 3X (based on bullseye position)
- Increase end-of-ball bonus
- Light Tomb Treasures
- Re-scored when defeating the Beast

### Tomb Treasures

Collected at left ramp (Premium/LE) after completing objectives <kcite ref="31"/>:

| #   | Award                                         |
| --- | --------------------------------------------- |
| 1   | 15M                                           |
| 2   | Super Slings (250K + 1K increment per hit)    |
| 3   | +15M Power Jackpot OR Can I Play With Madness |
| 4   | Super Combos (5X combos/deathblows)           |
| 5   | Light Extra Ball                              |
| 6   | Light Revive                                  |
| 7   | Collect 2X Bonus                              |
| 8   | +5X Power Jackpot                             |
| 9   | 50M + Spot random Level 2 Eddie Card          |
| 10  | **Run to the Hills** (Super Wizard Mode)      |

**Objectives that light Tomb Treasures:**
- Each unique Soul Shard (5 total)
- Loop Jackpot
- 6-Way Combo
- Mummy MB Super Jackpot
- Trooper MB Super Jackpot
- Win Number of the Beast
- Collect all 4 Level 2 Eddie Cards

## 🔗 Stacking & Combos

### ✅ What Stacks Well

| Stack                             | Why It Works                                                     |
| --------------------------------- | ---------------------------------------------------------------- |
| **Hallowed + Trooper MB**         | Structured shots easier with multiple balls <kcite ref="66"/>    |
| **Fear of the Dark + Trooper MB** | Spinner rips during multiball chaos                              |
| **Flight of Icarus + Mummy MB**   | Ramp shots count for both add-a-balls and mode <kcite ref="31"/> |
| **Any mode + Playfield X**        | Double or triple all scoring                                     |
| **Power Jackpot + Playfield X**   | Massive multiplied payouts                                       |

### ❌ What Does NOT Stack

| Combination                           | Reason                                              |
| ------------------------------------- | --------------------------------------------------- |
| Trooper + Mummy + Cyborg              | Multiballs are mutually exclusive <kcite ref="77"/> |
| Aces High + Any MB                    | It's already a 2-ball multiball                     |
| Rime of the Ancient Mariner + Any MB  | It's already a 2-ball multiball                     |
| Cyborg MB + Playfield X qualification | X targets used for Super Jackpot                    |

### Optimal Timing

1. **Start mode BEFORE multiball** to carry it into multiball
2. **Qualify Playfield X** before starting valuable modes/multiballs
3. **Activate Playfield X** right before collecting Power Jackpot or Tomb Treasure
4. **Lock 2 balls for Trooper**, then start a single-ball mode, then lock 3rd ball <kcite ref="45"/>

### Combos & Deathblows

Combos don't time out—they end when you repeat a shot or hit targets <kcite ref="31"/>:

| Combo Shots | Value                      |
| ----------- | -------------------------- |
| 2-Way       | 500K                       |
| 3-Way       | 2M                         |
| 4-Way       | 5M                         |
| 5-Way       | 10M                        |
| 6-Way       | 20M (lights Tomb Treasure) |

**Deathblow Shots:** Captive ball, Bullseye, Extra Ball target
**Super Deathblow (3X):** Center bullseye, Super Jackpot target, REVIVE spinner

> 🏆 **Maximum Deathblow:** 6-Way Combo → Super Deathblow → 3X Playfield → Super Combos = **900M!** <kcite ref="31"/>

## 🧙 Wizard Modes

### ⚔️ Number of the Beast (Wizard Mode)

**Requirement:** Collect all 4 Eddie Cards (any level)

**Gameplay:**
1. Single-ball mode with 30-second ball save
2. Three red shots lit—shoot any one
3. Shoot bullseye within 5 seconds to enable counter-attacks
4. All shots light yellow for 10 seconds—score counter-attacks
5. Cannot shoot same shot consecutively
6. 15 counter-attacks required to defeat the Beast

**Rewards:**
- 100M + all Soul Shard values
- Flippers disable, ball drains, fresh ball save
- Cannot challenge Beast again until Run to the Hills is played

> ⚠️ **Warning:** If you drain, you lose all Level 1 Eddie Cards and must re-qualify! Level 2 cards are retained <kcite ref="31"/>.

### 🏔️ Run to the Hills (Super Wizard Mode)

**Requirement:** Collect 10th Tomb Treasure

**Gameplay:**
- 6-ball multiball with generous ball save
- All shots lit for jackpots
- Complete Power Features in order (top-left to bottom-right) to light Super Jackpot
- Super Jackpot adds +1X to all jackpots
- Spell MUMMY at captive ball to add balls

**After Completion:**
- All Tomb Treasures reset
- All Soul Shards cleared
- All Eddie Cards reset <kcite ref="31"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                     | Risk                                 | Mitigation                                                         |
| ------------------------ | ------------------------------------ | ------------------------------------------------------------------ |
| **Left Spinner**         | Drains to left outlane on miss       | Only attempt with Revive lit or during multiball <kcite ref="60"/> |
| **Big Loop (late flip)** | Rattles into left outlane            | Keep upper left flipper UP for first few loops <kcite ref="57"/>   |
| **Right Orbit**          | Can rattle to right outlane          | Use controlled shots                                               |
| **Center Ramp (weak)**   | Rolls back to flippers unpredictably | Full power shots only                                              |

### ✅ Safe Shots

| Shot             | Why It's Safe              | Best Use               |
| ---------------- | -------------------------- | ---------------------- |
| **Captive Ball** | Ball returns to flipper    | Mummy progress, combos |
| **Drop Targets** | Controlled return          | Lock progress, awards  |
| **Right Ramp**   | Clean feed to left flipper | Mode progress, combos  |
| **Left Ramp**    | Feeds to right flipper     | Tomb Treasures, locks  |

### Flipper Selection

| Situation                   | Flipper                           | Notes                                |
| --------------------------- | --------------------------------- | ------------------------------------ |
| Ball on left flipper        | Right ramp, center ramp, drops    | Safest options                       |
| Ball on right flipper       | Left ramp, left orbit             | Avoid left spinner unless Revive lit |
| Ball on upper left flipper  | Mini loop, big loop, captive ball | Hold flipper up for loop returns     |
| Ball on upper right flipper | Drops, captive ball               | Use for Power Features               |

> 💡 **Pro Tip:** The upper left flipper should be held UP for the first 2-3 loop shots to make them easier. Drop it after to continue scoring <kcite ref="57"/>.

## 📚 Additional Info

### Extra Ball Opportunities

| Method           | Requirement                              |
| ---------------- | ---------------------------------------- |
| Power Features   | Complete 3 of 5                          |
| Tomb Treasure #5 | Collect 5th treasure                     |
| Loops            | 30 total loops (excluding Loop Jackpots) |

- Extra Ball consolation: 10M (not multiplied by Playfield X) <kcite ref="31"/>

### Mystery Awards (Orb)

Complete drops when "Light Orb" is lit to enable Mystery Award <kcite ref="31"/>:
- Collect Light Orb again before shooting to increase level (up to Level 3)
- Level 3 after 2MTM can award "Spot Soul Shard"

### Setup Recommendations

| Setting        | Recommendation    | Reason                |
| -------------- | ----------------- | --------------------- |
| Pitch          | 6.5°              | Standard play         |
| Outlane Posts  | IN                | Reduces brutal drains |
| Revive Setting | Both outlanes lit | More forgiving        |
| Extra Balls    | ON                | Extends play          |
| Ball Save      | 10+ seconds       | Learn the game        |

### Can I Play With Madness (Premium/LE)

Third Tomb Treasure on Premium/LE models <kcite ref="31"/>:
- 2-ball multiball with moving red shot
- Flip to move lit shot left/right
- Make lit shot to add ball and light 2 moving shots
- Turn red shots green, avoid re-hitting green shots
- Shoot Orb to lock in lit shots for 10 seconds

### Easter Eggs

- **Secret Skill Shots:** Three hidden skill shots available <kcite ref="98"/>
- **Song Selection:** 12 Iron Maiden songs available (cosmetic only)
- **DJ Mixer Mode:** Hold both flippers in attract mode for jukebox

## ✅ Quick-Start Checklist

### First Ball
- [ ] Go for Super Skill Shot (hold left flipper, full plunge)
- [ ] Hit X targets to qualify Playfield X
- [ ] Sweep drops for double awards
- [ ] Start working on EDDIE letters
- [ ] Hit captive ball for Mummy progress

### Every Ball
- [ ] Check if Playfield X is qualified—activate before big scores
- [ ] Work on REVIVE during safe moments
- [ ] Collect Power Jackpot before draining
- [ ] Stack modes with multiballs when possible

### Before Multiball
- [ ] Start a single-ball mode first (if possible)
- [ ] Qualify Playfield X
- [ ] Know the add-a-ball rules for that multiball

### High Score Attempts
- [ ] Save 3X Playfield for Power Jackpots and Tomb Treasures
- [ ] Build mode scores for 2 Minutes to Midnight
- [ ] Collect all Soul Shards for Number of the Beast bonus
- [ ] Work toward Level 2 Eddie Cards for permanent progress
