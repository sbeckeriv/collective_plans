# 🏰 Medieval Madness Remake LE - Complete Strategy Guide
## *Path to 1 Billion Points*

Medieval Madness is widely considered one of the greatest pinball machines ever made <kcite ref="79"/>. This guide covers the 2015 Chicago Gaming Company Limited Edition remake, which features LED lighting and a shaker motor among its 1,000 units produced <kcite ref="88"/>.

## 🎯 Core Mechanics

### Skill Shots

| Type                  | How to Execute                                               | Award                                                     |
| --------------------- | ------------------------------------------------------------ | --------------------------------------------------------- |
| **Normal Skill Shot** | Plunge to lit rollover lane (change with flippers)           | 50K + 10K per successive + **5X Bonus** <kcite ref="39"/> |
| **Super Skill Shot**  | Hold LEFT flipper while plunging, then hit any flashing shot | 100K + starts Castle Hurry-Up <kcite ref="39"/>           |

> 💡 **Pro Tip:** The Super Skill Shot gives quick points and a Hurry-Up, but the normal Skill Shot is safer and the +5X Bonus can be worth more on a long ball <kcite ref="39"/>.

### Mode Objectives, Scoring & Tips

#### 🧌 Trolls! Mode

**Start:** Hit Troll Standups 8 times, then shoot Merlin's Saucer <kcite ref="43"/>

**Objective:** Hit each Troll 3 times within 30 seconds <kcite ref="43"/>

**Scoring:**
- Completing Trolls! = 1M
- Each Troll destroyed counts toward Master of Trolls (need 3 hits = 1 Troll)

**Tips:**
- Bank shots off one Troll into Lock or Merlin's Saucer = 1 Troll Bomb <kcite ref="43"/>
- Save Troll Bombs for Battle for the Kingdom!
- Starting Trolls! simultaneously with Multiball Madness is extremely valuable <kcite ref="56"/>

## 🎱 Multiballs

### Castle Multiball

**Qualify:** 3 Lock shots (first time), 6 Lock shots (subsequent) <kcite ref="42"/>

**Locks are virtual** - no balls held by machine

#### Jackpot Structure

| Stage       | Shots Required                          | Award          | Value                               |
| ----------- | --------------------------------------- | -------------- | ----------------------------------- |
| **Stage 1** | Either ramp (5 total)                   | Jackpot        | 750K base <kcite ref="42"/>         |
| **Stage 2** | Lock shot                               | Super Jackpot  | 1.5M (2x Jackpot) <kcite ref="42"/> |
| **Stage 3** | Both ramps, both orbits, Lock, Catapult | Super Jackpots | 1.5M each <kcite ref="42"/>         |

**Complete all Stage 3 shots:** Return to Stage 1 with Jackpot +250K permanently <kcite ref="42"/>

**Full cycle value:** 14.25M at base values <kcite ref="43"/>

> 💡 **Pro Tip:** First Super Jackpot lights Extra Ball at Merlin's Saucer (default settings) <kcite ref="43"/>

### Multiball Madness

**The signature multiball mode!**

**Start:** Shoot Merlin's Saucer with at least 1 Madness light flashing <kcite ref="55"/>

#### Ball Count by Madnesses Lit

| Madnesses Lit | Balls in Play                                 |
| ------------- | --------------------------------------------- |
| 1             | 2 balls                                       |
| 2-4           | 3 balls                                       |
| 5             | **4 balls** + special intro <kcite ref="56"/> |

#### Scoring Values

| Award                | Formula                 | Min  | Max                    |
| -------------------- | ----------------------- | ---- | ---------------------- |
| Jackpot              | 75K + 25K × Madnesses   | 100K | 175K                   |
| Super Jackpot        | 250K + 150K × Madnesses | 400K | **1M**                 |
| Double Super Jackpot | 2 × Super Jackpot       | 800K | **2M**                 |
| Troll Super Jackpot  | ½ × Super Jackpot       | 200K | 500K <kcite ref="56"/> |

**Shot Assignments:**
- Shots corresponding to lit Madnesses = **Super Jackpot**
- Other shots = Jackpot
- Lock shot = **Double Super Jackpot** <kcite ref="56"/>
- Catapult Standup = Also Double Super Jackpot!

> ⚠️ **Critical:** You can only play Multiball Madness ONCE until you complete Royal Madness! Plan accordingly! <kcite ref="56"/>

#### Maximizing Multiball Madness

1. **Light ALL 5 Madnesses before starting** - 4 balls + maximum jackpot values
2. **Start Trolls! at the same time** - Completing Trolls! during MM adds Troll Madness, returns all balls, resets ball saver! <kcite ref="56"/>
3. **Shoot left ramp → right ramp** - Keeps ball at top of playfield, builds Bonus X and Super Jets <kcite ref="66"/>
4. **Park balls in bumpers** - Orbits and lit right ramp feed bumpers during MM <kcite ref="63"/>

## 🔄 Stacking & Combos

### ✅ What Stacks WELL

| Combination                         | Why It's Great                                                                                           |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------- |
| **Trolls! + Multiball Madness**     | Complete Trolls! during MM = adds Troll Madness, returns all balls, ball saver resets! <kcite ref="56"/> |
| **Castle Multiball + Castle shots** | Use extra balls to pound the Castle - can destroy 2-3 Castles in one multiball <kcite ref="63"/>         |
| **Super Jets + Royal Madness**      | Orbit shots feed bumpers = safer ball control <kcite ref="57"/>                                          |
| **Smack-A-Troll + any Multiball**   | Can be combined with any multiball <kcite ref="40"/>                                                     |

### ❌ What NOT to Stack / Avoid

| Situation                               | Why It's Bad                                                                           |
| --------------------------------------- | -------------------------------------------------------------------------------------- |
| **Starting weak Multiball Madness**     | Only 1 Madness = 2 balls, low jackpots, wastes your ONE shot at MM <kcite ref="62"/>   |
| **EB lit + weak MM lit at Merlin's**    | Tough choice - consider feeding bumpers via orbits to get EB instead <kcite ref="63"/> |
| **Troll Madness during MM + Lock shot** | Left Troll blocks the Double Super Jackpot shot! Kill Troll first <kcite ref="63"/>    |

### Optimal Timing

1. **Light 4-5 Madnesses** before starting Multiball Madness
2. **Have Trolls! ready but not started** - start it WITH Multiball Madness
3. **Save Troll Bombs** for Battle for the Kingdom
4. **Start Castle Multiball** when you want to focus on Castle destruction

## 🧙 Wizard Modes

### Royal Madness (Mini-Wizard Mode)

**Qualify:** Light all 5 Madness lights AND have already played Multiball Madness <kcite ref="57"/>

**Start:** Merlin's Saucer

**Objective:** Make 10 shots in 20 seconds <kcite ref="57"/>:
- 2 shots to each ramp
- 2 shots to orbits (combined)
- 2 shots to Catapult
- 1 shot to each Troll

**Scoring:**
| Shot                    | Award                                                                |
| ----------------------- | -------------------------------------------------------------------- |
| 1st shot to each target | 200K + 1 completion (Catapult Slam, etc.)                            |
| 2nd shot                | 1.5M Jackpot (+250K each subsequent Royal Madness) <kcite ref="57"/> |

**Timer:** Resets to 20 seconds on each shot made

**Completion Reward:** **Extra Ball** (or 15M on hard settings) <kcite ref="57"/>

> 💡 **Pro Tips:**
> - Brief ball saver at start - shoot Catapult immediately! <kcite ref="57"/>
> - Lock shot awards one shot, starting with Catapult <kcite ref="57"/>
> - Catapult Standup counts as Catapult shot <kcite ref="57"/>
> - Use Troll Bombs if needed for final shots <kcite ref="58"/>

---

### Battle for the Kingdom (Main Wizard Mode)

**Qualify:** Complete ALL 6 blue objectives (Master of Trolls, Defender of Damsels, Patron of the Peasants, Catapult Ace, Joust Champion, Castle Crusher) <kcite ref="58"/>

**Start:** Main Entrance (single-ball play only)

**Phase 1 - Battle Jackpots:**
- 4-ball multiball with ~15 second ball saver <kcite ref="58"/>
- Hit: Catapult, both ramps, both orbits, Lock
- Each shot = **2.5M Battle Jackpot** <kcite ref="59"/>

**Phase 2 - Castle Assault:**
- After completing Phase 1, pound the Gate
- Trolls pop up (use Troll Bombs to remove them!)
- Each Gate hit = **5M Battle Jackpot** <kcite ref="59"/>
- 6 Gate hits required

**Phase 3 - Final Shot:**
- Gate raises, Main Entrance flashes
- Enter Castle = **50M BONUS!** <kcite ref="59"/>

**Total BfK Value:** 15M (Phase 1) + 30M (Phase 2) + 5M (final gate) + 50M = **100M+**

**Victory Laps:**
After winning BfK, all major shots light for Victory Laps <kcite ref="59"/>:
- First Victory Lap = 500K
- Ultra Combo (any shot during timer) = 2× Victory Lap value + 20K added
- Values continue increasing throughout ball

**Subsequent BfK:** Worth 50M more each time! <kcite ref="60"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                            | Risk                          | Mitigation                                                          |
| ------------------------------- | ----------------------------- | ------------------------------------------------------------------- |
| **Main Entrance (Castle)**      | Center drain if missed        | Prepare for slap save; safer than AFM's saucer <kcite ref="36"/>    |
| **Lock shot (right flipper)**   | Risky angle, can drain        | Only shoot with ball save lit or during multiball <kcite ref="69"/> |
| **Orbits on the fly**           | Fast return = loss of control | Use loop pass technique <kcite ref="108"/>                          |
| **Double Super Jackpot (Lock)** | Tempting but dangerous        | Shoot ramps instead for consistent scoring <kcite ref="66"/>        |

### ✅ Safe Shots

| Shot                         | Why It's Safe                                                   |
| ---------------------------- | --------------------------------------------------------------- |
| **Left Ramp (Peasant)**      | Split ramp design - misses feed right flipper <kcite ref="35"/> |
| **Right Ramp (Damsel)**      | Split ramp design - misses feed right flipper <kcite ref="37"/> |
| **Orbits (with Super Jets)** | Feed bumpers, ball returns controlled <kcite ref="42"/>         |
| **Catapult**                 | Ball returns to left inlane <kcite ref="34"/>                   |

### Ball Control Techniques

**Loop Pass:** Hold flipper up as ball returns from orbit - slows ball and places it on opposite flipper <kcite ref="108"/>

**Drop Catch:** Let ball drop onto raised flipper, lower flipper as ball arrives to kill momentum

**Post Pass:** Bounce ball off raised flipper post to opposite flipper

> 💡 **Pro Tip:** "Trying to shoot the loops on the fly is death" - Keith Elwin <kcite ref="108"/>

## 💰 Scoring Strategy

### Path to 1 Billion Points

The author of the main rulesheet achieved **1.11 billion** with unlimited extra balls, and **350M** with limited EBs. Expect **200M** to be a very good competition score <kcite ref="64"/>.

### Early Game (Ball 1-2)

| Priority | Action                       | Target Score        |
| -------- | ---------------------------- | ------------------- |
| 1        | Super Skill Shot             | 100K + Hurry-Up     |
| 2        | Light Trolls! (hit standups) | Setup for later     |
| 3        | Shoot ramps for Madnesses    | Light 2-3 Madnesses |
| 4        | Work on Castle #1-2          | 6M+                 |
| 5        | Get 2 Troll Bombs            | Save for BfK        |

### Mid Game (Building Resources)

| Priority | Action                | Why                                          |
| -------- | --------------------- | -------------------------------------------- |
| 1        | Light ALL 5 Madnesses | Maximum Multiball Madness value              |
| 2        | Have Trolls! ready    | Start with MM for ball return                |
| 3        | Destroy Castles 3-4   | 14M+ and progress to King                    |
| 4        | Collect Extra Balls   | Castle MB Super Jackpot, 2 Castles destroyed |

### Late Game (Big Points)

| Priority | Action                          | Potential Value         |
| -------- | ------------------------------- | ----------------------- |
| 1        | **5-Madness Multiball Madness** | 10-30M+                 |
| 2        | **Royal Madness**               | 15M+ and Extra Ball     |
| 3        | **Destroy King's Castle**       | 27M (gate hits + bonus) |
| 4        | **Battle for the Kingdom**      | 100M+                   |
| 5        | **Victory Laps**                | Unlimited potential     |

### Highest Value Opportunities

| Opportunity                 | Value    | Difficulty |
| --------------------------- | -------- | ---------- |
| Battle for the Kingdom      | 100M+    | Very Hard  |
| King's Castle destruction   | 27M      | Hard       |
| 5-Madness Multiball Madness | 10-30M   | Medium     |
| Royal Madness completion    | 15M + EB | Medium     |
| Castle Multiball full cycle | 14.25M   | Medium     |
| Stacked Hurry-Ups (5x)      | 12M      | Medium     |

### The Billion Point Strategy

1. **Maximize Extra Balls** - Every EB is another chance at BfK
2. **Complete BfK multiple times** - 50M, 100M, 150M... <kcite ref="60"/>
3. **Stack Hurry-Ups** - 5+ stacked = 12M+ per collection
4. **Victory Laps** - After BfK, values keep climbing
5. **Bonus X** - Can exceed 100X; 1M switch bonus × 100X = 100M+ <kcite ref="60"/>

## 🏆 Summary: The Path to Glory

```
Ball 1-2: Build Resources
├── Light Trolls! (don't start yet)
├── Get 2+ Troll Bombs
├── Light 3-4 Madnesses
└── Destroy Castles 1-2

Ball 2-3: Execute
├── Light 5th Madness
├── Start Trolls! + Multiball Madness together
├── Destroy Castles 3-5
└── Collect Extra Balls

Ball 3+: Wizard Modes
├── Complete Royal Madness (EB!)
├── Destroy King's Castle
├── Battle for the Kingdom (100M+)
└── Victory Laps → Repeat BfK
```

> 🎯 **Remember:** "The average value of a shot to the Castle is worth more than the average value of a shot during almost all Multiball Madnesses" <kcite ref="62"/>. Don't neglect Castle destruction!
