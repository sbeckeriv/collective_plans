## 🎯 CORE MECHANICS

### Skill Shots

| Option                   | How to Execute                                                                                                                     | Award               |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------- |
| 🎱 **Bonus X Skill Shot** | Normal plunge → Magnet catches ball above B-U-G rollovers → Use flippers to move flashing letter → Ball drops through lit rollover | +1 Bonus Multiplier |
| 🚀 **Ramp Skill Shot**    | Hold LEFT flipper + plunge → Hit right ramp                                                                                        | **5 Million**       |
| 🔒 **Orbit MB Letter**    | Hold LEFT flipper + soft plunge into Orbit MB sinkhole                                                                             | 1 Orbit MB Letter   |

> 💡 **Pro Tip:** The Ramp Skill Shot is the safest and most valuable option for beginners. Holding the left flipper disables the magnet. <kcite ref="31"/>

### Main Objective: Liberate 6 Planets

Your mission is to eliminate all bugs on each planet, then either **Advance Planet** or start **Planet Multiball**. <kcite ref="31"/>

**How to Kill Bugs:**
- Hit **flashing** Bug standup targets
- Each bug species has a color:

| Bug Type     | Color  | Target Locations        |
| ------------ | ------ | ----------------------- |
| 🟡 Warrior    | Yellow | 4 standups scattered    |
| 🟢 Hopper     | Green  | 4 standups scattered    |
| 🔵 Plasma Bug | Blue   | 2 standups near bumpers |
| 🔴 Tanker     | Red    | 2 standups scattered    |

> 💡 **Pro Tip:** Using the **mini-flipper** to hit bugs counts as **2 kills**! You can also "cheat" by flipping the mini-flipper after making a shot with a regular flipper. <kcite ref="31"/>

---

### Mode Flow
Kill All Bugs → Brain Bug Appears → Capture/Escape →
Choose: Advance Planet OR Planet Multiball

**Brain Bug:**
- Huge pop-up target appears after clearing a planet
- Hit it 1 time (first planet), 2 times (second), etc.
- Each hit = **1 Million**
- Timed mode - if you miss, it escapes

> ⚠️ **Warning:** The Brain Bug shot is risky. Many players trap the ball and let it escape - the 1-2M isn't worth the drain risk. <kcite ref="31"/>

**After Brain Bug:**
- **Arm Nukes hole** = Start Planet Multiball
- **Recon Scoop** = Advance to next planet
- **Hurry-up** starts at 5M → counts down to 1M

### Recon Awards System

Shoot ramps to increase Recon Level, then collect at the scoop. <kcite ref="31"/>

| Ramp       | Letters Spelled |
| ---------- | --------------- |
| Left Ramp  | S-T-A-R-S-H-I-P |
| Right Ramp | T-R-O-O-P-E-R-S |

**Recon Level Awards:**

| Level | Possible Awards                                                 |
| ----- | --------------------------------------------------------------- |
| 1     | Ground Attack, Psychic Test, Spot Species, Bonus Hold, 1M       |
| 2     | Ground Attack, Orbits Lit, Super Pops (50K/hit), 2M             |
| 3     | **Double Scoring**, Spot Planet, Multiball Ready, 5M            |
| 4     | Ramp Hurry-up, Air Attack, Nuke Chase, 8M                       |
| 5     | **Multiball**, Multiball Ready, **Orbit Multiball**, Extra Ball |
| 6     | Extra Ball, Special, **25M**                                    |

> 💡 **Pro Tip:** Level 5 typically awards Regular MB if you haven't had one, otherwise Orbit MB. **Avoid Level 6** - Orbit MB is worth far more than 25M! <kcite ref="31"/>

**Key Recon Awards Explained:**

| Award            | Effect                                                   |
| ---------------- | -------------------------------------------------------- |
| 🎯 Ground Attack  | 2-ball MB, standups cycle 100K-400K rapidly              |
| 🔮 Psychic Test   | Video mode - match cards for Special                     |
| 🪲 Spot Species   | Kills all bugs of majority species                       |
| ⏱️ Double Scoring | All points doubled for 20 seconds                        |

## 🎱 MULTIBALLS

### 1. Regular Multiball (Live Fire Assault Range)

**How to Start:**
- Shoot the **Warrior target** (moving standup in center)
- 4 hits for first MB, 6 hits for second, etc.
- Warrior moves backward as you hit it
- When 1 hit remains, it moves back and forth

**Gameplay:**
- **4-ball multiball**
- Both ramps lit for **Jackpots** (start at 3M, max 4M)
- Increase jackpot by hitting Bug standups
- Light all standups during MB = **Extra Ball**
- **5 Jackpots** → lights **Super Jackpot** at Warrior sinkhole

| Jackpot Type  | Value   | Location         |
| ------------- | ------- | ---------------- |
| Jackpot       | 3M-4M   | Left/Right Ramps |
| Super Jackpot | **15M** | Warrior Sinkhole |

> 💡 **Pro Tip:** The Warrior standup ALWAYS counts as the sinkhole for Super Jackpot collection, even when blocking it. <kcite ref="31"/>

**Restart Mechanic:** If you fail to collect any jackpots, you get 15-20 seconds to restart MB at the Warrior sinkhole (2-ball restart).

---

### 2. Planet Multiball (Nuke Multiball)

**How to Start:**
- Clear all bugs on a planet
- Shoot **Arm Nukes hole** (instead of Advance Planet)

**The Key Strategy:** Wait to nuke multiple planets at once! <kcite ref="48"/><kcite ref="73"/>

| Planets Nuked | Nuke Jackpot | Super Nuke | Balls |
| ------------- | ------------ | ---------- | ----- |
| 1             | 1M           | 10M        | 2     |
| 2             | 2.5M         | 25M        | 2     |
| 3             | 4M           | 40M        | 3     |
| 4             | 7M           | 70M        | 4     |
| 5             | **12M**      | **120M**   | 4     |

**Lit Shots During Planet MB:**
- Left Loop, Left Ramp, Recon Scoop, Right Ramp, Right Loop, Arm Nukes

**Objectives:**
1. Make 5 Nuke shots to light **Super Nuke** at Warrior
2. Recon Scoop = **Double Nuke** (2x jackpot)
3. Arm Nukes hole = Spots a Nuke + awards jackpot
4. After Super Nuke, all shots relight!

> 🎯 **This is your biggest scoring opportunity!** A 5-planet Planet MB can yield **270M+** in a single multiball. <kcite ref="31"/>

---

### 3. Orbit Multiball

**How to Start:**
- Shoot **3 lit loops** to lock first ball (spells O-N-E)
- Shoot **3 more lit loops** to lock second ball (spells T-W-O)
- Inlanes light opposite loop for ~5 seconds
- Total: 6 loop shots (doesn't increase)

**Gameplay:**
- Starts as **2-ball multiball**
- Both loops lit for **5M Jackpots** (remain lit throughout)
- First 2 loop shots **add a ball** (even after ball saver expires)
- Maximum **4 balls** in play

> 💡 **Pro Tip:** You can make 3 consecutive loop shots for a lock - the loop stays lit after each shot! <kcite ref="31"/>

| -------------- | -------------- | ------------- |

## 🔄 STACKING & COMBOS

### ✅ What Stacks Well

| Combination                      | Why It Works                 |
| -------------------------------- | ---------------------------- |
| **5-Planet MB + Double Scoring** | 12M jackpots become 24M!     |
| **Recon Level 5 + Low MB count** | Guarantees Regular MB award  |
| **Hold Bonus + High Multiplier** | THE key to billions          |
| **Orbit MB + Ramp Hurry-up**     | Safe loop shots + ramp value |

### ❌ What NOT to Stack

| Avoid                                             | Reason                                                         |
| ------------------------------------------------- | -------------------------------------------------------------- |
| **Early Planet MB (1-2 planets)**                 | Only 1-2.5M jackpots, 2 balls - not worth it <kcite ref="48"/> |
| **Orbit MB from Recon when Regular MB available** | Regular MB typically worth more                                |
| **Brain Bug attempts on steep machines**          | High drain risk for low reward                                 |

### Recon Combos

Shooting **Right Ramp → Left Ramp → Recon Scoop** in sequence can boost your Recon Level by +3 additional levels. <kcite ref="31"/>

## 🏆 WIZARD MODE: KLENDATHU

### Requirements
- Clear and **Nuke all 5 planets** (Antilles through Planet P)
- Shoot **Recon Scoop** when Klendathu is flashing

### Gameplay
- **Unlimited balls for 30 seconds** (ball saver)
- ALL Bug standups are lit
- ALL Nuke shots are lit

### Objectives

| Task                 | Requirement                  | Points          |
| -------------------- | ---------------------------- | --------------- |
| Kill Bugs            | 9 of EACH species (36 total) | ~points per bug |
| Nuke Shots           | All 5 Nuke locations         | 5M each         |
| Double Nuke          | Recon Scoop                  | 10M             |
| **Completion Bonus** | All bugs + all nukes         | **50M**         |

> ⚠️ **Note:** After the 30-second timer expires, the mode continues with balls still in play until all drain. <kcite ref="31"/>

### Klendathu Shortcut

At **Recon Level 6**, you can sometimes be awarded Klendathu directly as a "mystery" award, bypassing the need to clear all 5 planets! <kcite ref="83"/>

## 🎯 SHOT ANALYSIS

### ⚠️ Dangerous Shots

| Shot                  | Danger                       | Mitigation                                                 |
| --------------------- | ---------------------------- | ---------------------------------------------------------- |
| **Weak Left Ramp**    | Dribbles back SDTM           | Nudge RIGHT immediately <kcite ref="75"/>                  |
| **Bug Standups**      | Unpredictable rebounds       | Use mini-flipper, accept randomness                        |
| **Arm Nukes Hole**    | Eject goes to dangerous spot | Don't chase EB here <kcite ref="73"/>                      |
| **Recon Scoop Eject** | Fast, unpredictable          | Use mini-flipper backhand to right orbit <kcite ref="73"/> |
| **Center Drain**      | Wide flipper gap             | Nudge to slow ball, practice trapping <kcite ref="111"/>   |

### ✅ Safe Shots

| Shot                       | Why Safe                  | Best Use               |
| -------------------------- | ------------------------- | ---------------------- |
| **Right Ramp**             | Clean feed to left inlane | Recon levels, jackpots |
| **Left Ramp (full power)** | Returns to right flipper  | Recon levels, jackpots |
| **Loops (when lit)**       | Magnet catches ball       | Orbit MB progress      |
| **Warrior Target**         | Center of playfield       | Regular MB progress    |

### Flipper Selection

| Target       | Flipper          | Notes                                      |
| ------------ | ---------------- | ------------------------------------------ |
| Left Ramp    | Right Flipper    | Full power only                            |
| Right Ramp   | Left Flipper     | Primary safe shot                          |
| Left Loop    | Right Flipper    | Backhand possible                          |
| Right Loop   | Left Flipper     | Or mini-flipper backhand                   |
| Arm Nukes    | **Mini-Flipper** | Trap on right, flip mini <kcite ref="31"/> |
| Bug Standups | Either/Mini      | Mini = 2x kills                            |

### Mini-Flipper Mastery

The mini-flipper is **essential** for ball control: <kcite ref="31"/>
- **Drop-catch** balls from Recon scoop eject
- **Slow down** balls from right inlane
- **Backhand** to right orbit from scoop ejects
- **Arm Nukes shot**: Trap on right flipper, flip mini-flipper

## 💰 SCORING STRATEGY

### The Path to 1 Billion

Top Pinside scores exceed **1.4 billion**, confirming 1B is achievable. <kcite ref="76"/>

### 🔑 THE SECRET: Hold Bonus

> **"The only way to score multiple billions is the hold-bonus feature."** <kcite ref="73"/>

**How it works:**
1. Build up bonus (kill bugs) and multiplier (B-U-G rollovers)
2. Get **Bonus Hold** from Recon Level 1
3. Your ENTIRE multiplied bonus carries to next ball
4. Multiply it AGAIN on the next ball
5. Hold it AGAIN
6. Repeat exponentially!

### Early Game (Ball 1)

| Priority | Action          | Goal                                 |
| -------- | --------------- | ------------------------------------ |
| 1        | Ramp Skill Shot | 5M start                             |
| 2        | Shoot ramps     | Build Recon to Level 5               |
| 3        | Collect Recon 5 | Get Regular MB                       |
| 4        | During MB       | Collect jackpots, light all standups |
| 5        | Clear Antilles  | Don't start Planet MB yet!           |
| 6        | Advance Planet  | Move to Zegema Beach                 |

### Mid Game (Balls 2-3)

| Priority | Action               | Goal                                                           |
| -------- | -------------------- | -------------------------------------------------------------- |
| 1        | Grind Recon Level 1  | Fish for **Bonus Hold** and **Spot Species** <kcite ref="73"/> |
| 2        | Clear planets safely | Use Spot Species when possible                                 |
| 3        | Build Orbit MB       | 6 loop shots total                                             |
| 4        | Stack planets        | Get to 4-5 planets before nuking                               |
| 5        | Start 5-Planet MB    | 12M jackpots, 120M Super Nuke                                  |

### Late Game

| Priority | Action              | Goal                     |
| -------- | ------------------- | ------------------------ |
| 1        | Reach Klendathu     | Complete wizard mode     |
| 2        | Maximize held bonus | Keep multiplying         |
| 3        | Double Scoring + MB | Stack for maximum points |

### Scoring Hierarchy

| Feature               | Potential Value  | Difficulty        |
| --------------------- | ---------------- | ----------------- |
| 5-Planet MB           | 200-300M         | High              |
| Regular MB (full run) | 100-150M         | Medium            |
| Orbit MB              | 50-100M          | Medium            |
| Klendathu Completion  | 50M + bugs/nukes | Very High         |
| Held Bonus (stacked)  | **Unlimited**    | Requires luck     |

| --------------- | -------------------------------------- |

### Does Completing Modes Change Gameplay?

- **Planets:** Once nuked, they stay nuked (lit solid)
- **Multiballs:** Hit requirements increase (4→6→8 for Regular MB)
- **Recon:** Resets to Level 1 each ball, but awards can shift levels
- **Klendathu:** One-time wizard mode per game
