# 🦕 JURASSIC PARK (Pro) Pinball Machine - Complete Strategy Guide
## *Mastering Paddocks, Multiballs & Dino Capture*

> **"When dinosaurs ruled the earth..."**

Welcome to Nublar! This comprehensive guide covers mastery of Stern's 2019 Jurassic Park Pro pinball machine, from skill shots and paddock modes through chaotic multiballs and the ultimate wizard modes <kcite ref="1"/>.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Ball 1:** Soft plunge for "C" target (6M + ball save) OR standard skill shot sequence for 2M-12M
- [ ] **Spell MAP** by shooting white targets (left ramp, right ramp, side ramp)
- [ ] **Start paddock mode** at left ramp — hit rescue shots (red) first
- [ ] **Capture your first dinosaur** to unlock Control Room
- [ ] **Hit truck repeatedly** to build T-Rex Multiball jackpot
- [ ] **Start Chaos Multiball** by spelling CHAOS at the bumper target
- [ ] **Build amber bonus** via bumpers and dinosaur captures
- [ ] **Pursue Raptor Tri-Ball** for massive super jackpot potential
- [ ] **Stack modes before multiballs** for multiplied scoring
- [ ] **Reach wizard modes** (Visitor's Center, Museum Mayhem, Escape Nublar, final mode)

## 🎱 Skill Shots

### Standard Skill Shot Sequence

Start with a full plunge and hit shots in order for escalating values:

| Shot                | Points      | Notes                              |
| ------------------- | ----------- | ---------------------------------- |
| **Left Ramp**       | 2,000,000   | Base value, increases by 1M/ball   |
| **+ Right Ramp**    | 4,000,000   | Doubles left ramp value            |
| **+ Side Ramp**     | 8,000,000   | Quadruples left ramp value         |
| **+ Right Orbit**   | 12,000,000  | Sextuples left ramp value (1M max) |

> 💡 **Pro Tip:** The skill shot value increases by 1 million per ball, starting at 2M. This sequence can reach 12M+ on later balls! <kcite ref="2"/>

### Secret Skill Shot (Soft Plunge)

Perform a soft plunge, letting the ball roll back to the left flipper, then immediately shoot the "C" target:

| Shot              | Points      | Bonus                 |
| ----------------- | ----------- | --------------------- |
| **"C" Target**    | 6,000,000   | +3 second ball save   |

> 💡 **Intermediate Tip:** The soft plunge C-shot gives you 6M and a longer ball save, plus early CHAOS progress. This is often superior to the standard skill shot if you want to begin paddock modes or multiballs immediately <kcite ref="2"/>.

## 🗺️ Core Mechanics: The MAP System

The fundamental game loop revolves around **spelling MAP** and entering paddock modes:

### MAP Spelling
- **M** = Left ramp (white arrow)
- **A** = Right ramp (white arrow)
- **P** = Side ramp (white arrow)

Once MAP is spelled, the **left ramp** lights for a **Paddock Mode** entry.

## 🦖 Paddock Modes (Dinosaur Capture)

Each dinosaur paddock presents a unique challenge. The sequence progresses through different dinosaurs with increasing complexity.

### Paddock Mechanics

| Phase | Action                           | Points         | Progress                      |
| ----- | -------------------------------- | -------------- | ----------------------------- |
| 1     | **Rescue Phase**                 | 500K–2M/shot   | Hit red rescue shots           |
| 2     | **Trap Setup**                   | 750K–3M/shot   | Hit yellow trap targets        |
| 3     | **Dinosaur Capture**             | 2M–5M          | Hit green dinosaur shot        |
| Bonus | **Perfect Paddock** (no misses)  | +1x multiplier | Stacks amber bonus multiplier  |

> 💡 **Pro Tip:** Always complete rescue phases cleanly — Perfect Paddock bonuses stack your amber multiplier for massive end-of-ball bonuses! <kcite ref="1"/>

### Dinosaur Types

- **Carnivores (T-Rex, Velociraptors):** Aggressive behavior, faster timer
- **Herbivores (Triceratops, Brachiosaurus):** Docile, longer timer windows
- **Unique (Spinosaurus):** Unlocks King of the Island Multiball <kcite ref="1"/>

---

## 🚛 T-Rex Events (Truck Modes)

Hit the **truck captive ball** repeatedly to build T-Rex Multiball:

| Event Number | Truck Hits Needed | Reward                                |
| ------------ | ----------------- | ------------------------------------- |
| 1–3          | 3–5 hits          | Build hurry-up value (500K base)      |
| 4 (T-REX)    | Complete sequence | Unlock 2-ball multiball at left ramp  |

**Feed T-Rex Multiball** awards jackpots across five orange CHAOS shots. Build value before the left ramp by hitting orange rescue targets (each adds to jackpot multiplier) <kcite ref="2"/>.

## 🔫 Control Room Modes

**Activated by:** Capturing your first dinosaur, spelling CHAOS twice, or hitting 15 tower shots.

Three selectable modes:

| Mode               | Objective              | Award          | Points    |
| ------------------ | ---------------------- | -------------- | --------- |
| **System Boot**    | Collect 4 fossils      | Fossils + time | 3M–10M    |
| **Secure Protocol**| Hit 5 major shots      | Bonus jackpot  | 5M–15M    |
| **Lockdown**       | Hit bumpers in sequence| Timer bonus    | 2M–8M     |

> 💡 **Pro Tip:** Always have a Control Room mode active before starting a multiball — it doubles jackpot values when modes are stacked! <kcite ref="2"/>

## 💰 Scoring Systems

### Amber Bonus
- **Built by:** Bumper hits
- **Collected at:** Amber mine target (left inlane)
- **Left flipper:** 10M + 3s ball save
- **Right flipper:** 20M + 12s ball save + increased amber value
- **Multiplier stacking:** Each Perfect Paddock adds +1x (up to 6x+)
- **End-of-ball:** Carried over entire game <kcite ref="1"/>

### DNA Combos
**11 unique shot sequences** that each award:
- **2,000,000 base points**
- **+400K per combo** in end-of-ball bonus
- **Extra ball** lit at 8 DNA combos collected
- **Wildcard fossil** for completing all 11 combos <kcite ref="1"/>

### 6-Way Combos
Complete spinner, upper loop, tower ramp, helipad, left ramp, and right ramp in any order:

| Progress     | Points per Shot |
| ------------ | --------------- |
| 1st combo    | 250,000         |
| 2nd combo    | 500,000         |
| 3rd+ combos  | 1,000,000       |

Multipliers apply when other modes are active <kcite ref="1"/>.

### Fossils
- **11 dinosaur fossils** grouped into 4 collectible sets
- **Set completion awards:** 20M–250M depending on set
- **Bonus:** Extra balls, multiplier bonuses, Control Room advances <kcite ref="1"/>

## 🎰 Multiballs

### Chaos Multiball (3-ball)

**Qualification:** Spell CHAOS letters in order at the bumper target.

**Mechanics:**
1. Hit truck to switch which side scores jackpots
2. **Side-switching jackpots:** Amber mine scores jackpot on lit side
3. **Super Jackpot:** Bumpers count down add-a-balls (up to 6 balls possible)
4. **Value building:** Ramp shots boost spinner value for massive spinner rips <kcite ref="2"/>

> 💡 **Pro Tip:** Set up a Control Room mode before starting Chaos Multiball, then activate 2x or 3x playfield scoring. Work the spinner at max value for exponential returns! <kcite ref="2"/>

---

### Raptor Tri-Ball (4-phase, up to 6-ball)

**Qualification:** Hit raptor pen targets (right side), then weaken the fence.

**Phases:**

| Phase         | Action                  | Jackpot Multiplier |
| ------------- | ----------------------- | ------------------ |
| 1             | Hit raptor shots        | 1x–2x              |
| 2             | CHAOS jackpots          | 2x–4x              |
| 3             | Super CHAOS jackpots    | 3x–6x + extra ball |
| 4             | Raptor Rampage          | Up to 9x           |

> ⚠️ **Difficulty:** This is the hardest multiball to start but offers the highest value potential. Back-hand the raptor pit from the left flipper cradle for safer shot execution <kcite ref="2"/>.

**Key Rule:** Balls in play multiply jackpot values — 6 balls = 6x multiplier on base jackpots! <kcite ref="1"/>

---

### King of the Island Multiball (3-ball)

**Qualification:** Capture Spinosaurus dinosaur.

**Mechanics:**
- **Switch-based progression:** Complete shots in any order
- **Multiplier scaling:** 1x → 2x → 3x → 4x → 5x for sequential jackpots
- **Value accumulation:** Each switch hit adds to running jackpot total

## 🎪 Wizard Modes (Mini & Final)

### Visitor's Center (1st Mini-Wizard)

**Prerequisite:** Complete initial paddock progression.

**Mechanics:**
- **2-ball multiball mode**
- **Objective:** Lock raptors in cold storage before timer expires
- **Lockup shots:** Helipad, tower ramp, side ramp (roving lit shot)
- **Reward:** Fossil advancement, DUNE-like letter progression <kcite ref="1"/>

---

### Museum Mayhem (2nd Mini-Wizard)

**Prerequisite:** Play all four T-Rex events (spell T-REX).

**Mechanics:**
- **Objective:** Defeat raptors with T-Rex
- **Raptors summoned by:** Hitting specific targets
- **T-Rex damage:** Amber mine shots
- **Completion:** Awards major point bonuses and letter progression <kcite ref="1"/>

---

### Secure Control Room (3rd Mini-Wizard)

**Mechanics:**
- **Objective:** Collect hurry-ups from all major shots in sequence
- **Shots required:** Left ramp, right ramp, side ramp, right orbit, tower, helipad
- **Time pressure:** 30–60 second timer
- **Completion:** Awards letter and massive bonus <kcite ref="1"/>

---

### Escape Nublar (4th Mini-Wizard)

**Mechanics:**
- **Single-ball mode** during volcanic eruption
- **Paddock navigation:** Choose path through paddocks
- **Rescue phases:** Hit rescue shots while avoiding dinosaurs
- **Path-finding:** Each choice affects difficulty and available shots
- **Completion:** Awards letter and prepares final wizard mode <kcite ref="1"/>

---

### When Dinosaurs Ruled the Earth (Final Wizard Mode)

**Prerequisites:** Complete all four mini-wizard modes in sequence.

**Mechanics:**
- **6-ball multiball ultra mode**
- **All amber features active:** Bumpers build bonus, rescues available
- **Objective:** Defeat all dinosaur species simultaneously
- **Scoring:** Exponential multipliers on all shots
- **Victory condition:** Achieve targets to end game with massive final bonus

> 🏆 **Final Milestone:** This mode represents the pinnacle of Jurassic Park Pro scoring. Reaching it and completing it can exceed 300–500 million points alone! <kcite ref="1"/>

## 🛠️ Advanced Features

### Smart Missile
- **Activation:** Lit via rescue shots at left inlane
- **Use:** Press action button to select awards before shooting bumper
- **Available awards:** Extra rescues, clears, fossils, multiball starts, extra balls <kcite ref="1"/>

### Supply Drops (Blue Targets)
- **Awards:** Mystery bonuses
- **Possible awards:** Add-a-balls, CHAOS advances, 2x scoring activation <kcite ref="1"/>

### Pteranodon Attack (Right Ramp Frenzy)
- **Activation:** Right ramp shots
- **Mechanic:** Switch hits build awards during 30-second frenzy
- **Threshold multipliers:** Carry over between attempts
- **Danger:** Very fast ball speed on upper flipper <kcite ref="1"/>

## 📈 Stacking & Combos

### ✅ Optimal Stacks

| Combination                        | Benefit                                           |
| ---------------------------------- | ------------------------------------------------- |
| **Control Room + Multiball**       | Doubles jackpot values during active multiballs   |
| **DNA Combos + Amber Multiplier**  | 400K per combo × 6x+ amber = 2.4M+ per combo    |
| **T-Rex Multiball + 2x Scoring**   | Ramps × 2x = exponential jackpot growth           |
| **Perfect Paddock chains**         | Each amber multiplier stacks (multiplicative!)    |
| **Supply Drop 2x + Chaos**         | 2x on already-multiplied chaos jackpots           |

### ❌ What NOT to Stack

| Avoid                                   | Reason                                            |
| --------------------------------------- | ------------------------------------------------- |
| **Right ramp during Control Room**      | Pteranodon attack interrupts mode focus           |
| **Multiple unprogressed paddock modes** | Ball saves conflict, scoring diluted              |
| **Raptor Tri-Ball unpracticed**         | Too difficult to execute properly, high drain risk |

## 🎮 Pro Tips & Advanced Tactics

**Ball Control:**
- Cradle on left flipper between major shots to set up precise angles
- Left flipper is strongest for left ramp and control room entry
- Right flipper excels at amber mine (20M+ value) and side ramp entry <kcite ref="2"/>

**Mode Management:**
- Always have Control Room or T-Rex running before starting a multiball
- Never waste a multiball without an active background mode
- Use Supply Drop 2x strategically during Chaos for exponential growth <kcite ref="2"/>

**Spinner Strategy:**
- Track shots hit since last spinner — each ramp/tower/helipad increases value by 100K–250K
- Combine ramp hits to max spinner, then cash out during multiball for huge returns
- Complete all 4 ramps = max spinner value (lasts 1 rip only) <kcite ref="2"/>

**Playfield Awareness:**
- "C" target (control room) is right of center — aim slightly left with right flipper
- Raptor pit is dangerous; always back-hand from left flipper cradle
- Truck (captive ball) builds value incrementally — don't neglect it <kcite ref="2"/>
