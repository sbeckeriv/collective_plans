# 🔥 Firepower II Strategy Guide
## The Path to Maximum Points

> **Target Score:** While Firepower II doesn't have a 1 billion point ceiling (it's a 1983 machine with 7-digit displays maxing at 9,999,990), the goal is to maximize every ball through multiball domination and bonus stacking. Top competitive scores reach into the **2-4 million range**.

## 🎯 Core Mechanics

### Skill Shot / Plunge Options

Firepower II does not have a traditional skill shot with variable values <kcite ref="72"/>. The plunge feeds the ball into the top lanes area.

**Plunge Strategy:**
- Use lane change (right flipper) to position unlit lanes before plunging
- A medium-strength plunge drops the ball into the A-B-C-D lanes
- A hard plunge may send the ball around the orbit

### Top Lanes (A-B-C-D)

| Action                  | Points              | Bonus Advance |
| ----------------------- | ------------------- | ------------- |
| Any lane (lit or unlit) | 1,000               | +2            |
| Complete all 4 lanes    | Advances multiplier | —             |
| Complete at 5X          | 25,000              | —             |

- **Lane Change:** Right flipper only rotates lit lanes <kcite ref="73"/>
- Completing lanes advances bonus multiplier: **2X → 3X → 5X**
- Once at 5X, each completion scores 25,000 points

> 💡 **Pro Tip:** Getting to 5X bonus multiplier is your #1 priority every ball. This turns your 99K max bonus into 495K!

## 🎱 Multiball

### How to Start Multiball

Firepower II features a simple but effective 2-ball multiball system <kcite ref="74"/>:

1. **Lock a Ball:** Shoot the upper right lane (always lit during single-ball play)
2. **Release:** Hit the center standup target above the bumpers
3. **Multiball Begins!** All scoring is doubled

### Lock & Release Values

| Multiball # | Lock Value | Release Value |
| ----------- | ---------- | ------------- |
| 1st         | 10,000     | 50,000        |
| 2nd         | 20,000     | 100,000       |
| 3rd+        | 50,000     | 150,000       |

> ⚠️ **Important:** During multiball, shooting the lock scores only 5,000 (not doubled) and immediately kicks the ball out <kcite ref="74"/>. There are no jackpots—just doubled scoring on everything else!

### Multiball Strategy

- **UTAD (Up The And Down):** Focus on the right side to feed balls into top lanes
- Left orbit shots whip around and come back down fast—dangerous!
- Keep both balls in play as long as possible
- Lock/release values carry over between balls, so starting multiball 3+ times per game is huge

## 🔤 FIREPOWER Targets

Nine standup targets spell F-I-R-E-P-O-W-E-R across the playfield <kcite ref="75"/>.

### Target Scoring

| Action           | Points | Bonus      |
| ---------------- | ------ | ---------- |
| Hit unlit target | 1,000  | +1 advance |
| Hit lit target   | 1,000  | —          |

### Three-Letter Bank Completions (F-I-R, E-P-O, W-E-R)

Completing any 3-letter bank:
- ✅ Lights one bumper for **1,000 pts** (up from 100) for rest of ball
- ✅ Lights left orbit for **Hold Bonus** for 10 seconds

### Full FIREPOWER Completions

| Completion | Award                                                          |
| ---------- | -------------------------------------------------------------- |
| 1st        | Spinner lit (1,000/spin), one in lane lit for 3 bonus advances |
| 2nd        | Other in lane also lit                                         |
| 3rd        | **Extra Ball** lit at ramp                                     |
| 4th        | Outlanes lit alternately for **Special**                       |
| 5th        | Instant **Special**                                            |
| 6th+       | 50,000 points each                                             |

> 💡 **Pro Tip:** Progress on current spelling carries over ball-to-ball, but completions reset each ball <kcite ref="75"/>.

## 🎰 Bonus System

### Building Bonus

| Shot                      | Bonus Advances |
| ------------------------- | -------------- |
| Flashing FIREPOWER target | +1             |
| Top lane (any)            | +2             |
| Unlit orbit / Lit in lane | +3             |
| Lit orbit (Hold Bonus)    | +10            |

### Bonus Multiplier Progression

| Top Lane Completions | Multiplier |
| -------------------- | ---------- |
| 1                    | 2X         |
| 2                    | 3X         |
| 3+                   | 5X         |

**Maximum Bonus:** 99,000 × 5X = **495,000 points** <kcite ref="79"/>

### Hold Bonus

- Lit for 10 seconds after completing any 3-letter bank (F-I-R, E-P-O, or W-E-R)
- Shoot the left orbit when lit to hold base bonus to next ball
- Second Hold Bonus collection in same ball scores 25,000 instead
- **Multiplier NEVER carries over** <kcite ref="79"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                    | Risk                                                           |
| ----------------------- | -------------------------------------------------------------- |
| Left Orbit (strong)     | Whips around to right side, can center drain <kcite ref="23"/> |
| All 9 FIREPOWER targets | Rebound risk to center/outlanes                                |
| Release target          | Ball can bounce straight down the middle                       |
| Bumper area exits       | Balls coming down between bumpers can center drain             |

### ✅ Safe Shots

| Shot                      | Notes                       |
| ------------------------- | --------------------------- |
| Lock saucer (upper right) | Controlled feed, always lit |
| Weak left orbit           | Falls into top lanes safely |
| Right side orbit          | Feeds to top lanes          |
| Ramp shot                 | Controlled return           |

### Flipper Selection

| Target             | Flipper | Notes               |
| ------------------ | ------- | ------------------- |
| Lock saucer        | Right   | Clean backhand      |
| Release target     | Either  | Up the middle       |
| Left orbit/spinner | Left    | Direct shot         |
| Ramp               | Right   | Backhand preferred  |
| Top lanes          | Either  | Via orbit or plunge |

## 📈 Scoring Strategy

### Early Game (Ball 1)

1. 🎯 **Get to 5X multiplier ASAP** - Top lanes are priority #1
2. 🔒 **Start multiball** - Lock + Release = doubled scoring
3. 📝 Work on FIREPOWER letters naturally during multiball

### Mid Game (Ball 2)

1. 🔄 **Repeat multiball** - Lock/release values have increased
2. 🔥 **Complete FIREPOWER once** - Lights spinner for 1,000/spin
3. 💫 **Rip the spinner** during multiball with 5X bonus building

### Late Game (Ball 3+)

1. 🎱 **Multiball is king** - 50K lock + 150K release + doubled scoring
2. 🎰 **Max bonus every ball** - 495K is massive
3. 🎁 **Collect mystery scores** - Right in lane → Ramp for 20K-99K

### Highest Value Opportunities

| Opportunity                  | Value               | Priority |
| ---------------------------- | ------------------- | -------- |
| Max bonus (5X × 99K)         | 495,000             | ⭐⭐⭐⭐⭐    |
| 3rd+ Multiball release       | 150,000 (×2 = 300K) | ⭐⭐⭐⭐⭐    |
| Lit spinner during multiball | 2,000/spin          | ⭐⭐⭐⭐     |
| Top lane completion at 5X    | 25,000 (×2 = 50K)   | ⭐⭐⭐⭐     |
| Mystery score                | 20,000-99,000       | ⭐⭐⭐      |

## 🎁 Additional Features

### Extra Ball

- Lit at ramp after **3rd FIREPOWER completion** in one ball <kcite ref="77"/>
- Cannot be assigned point value in competition play

### Mystery Award

- Right in lane lights ramp for **Mystery Score** (7 seconds)
- Awards random value: **20,000 - 99,000 points** <kcite ref="77"/>

### Special

- 4th FIREPOWER completion lights outlanes alternately
- 5th completion awards instant Special
- In competition/novelty play: **50,000 points** <kcite ref="80"/>

### Pop Bumpers

| State                              | Value |
| ---------------------------------- | ----- |
| Unlit                              | 100   |
| Lit (after F-I-R, E-P-O, or W-E-R) | 1,000 |

### Spinner

| State                     | Value      |
| ------------------------- | ---------- |
| Unlit                     | 100/spin   |
| Lit (after 1st FIREPOWER) | 1,000/spin |

## ⚙️ Setup Recommendations

### Pitch & Level

- **Standard pitch:** 6.5° recommended for balanced play
- Steeper pitch (7°+) makes the game faster and more dangerous
- Ensure side-to-side level is perfect—this game drains fast

## 🎮 Game Notes

### What This Game Doesn't Have

As a 1983 solid-state machine, Firepower II does **not** feature:
- ❌ Wizard modes
- ❌ Complex mode stacking
- ❌ Progressive jackpots
- ❌ Video modes
- ❌ Skill shot with variable values
- ❌ Ball save

### What Makes It Special

- ✅ Fast, aggressive gameplay <kcite ref="22"/>
- ✅ Simple but deep strategy
- ✅ Multiball always accessible (lock always lit)
- ✅ Massive bonus potential (495K)
- ✅ Classic Williams sound by Eugene Jarvis <kcite ref="38"/>
