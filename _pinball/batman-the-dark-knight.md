# 🦇 Batman: The Dark Knight - Strategy Guide

*Stern Pinball, 2008 — Designed by George Gomez*

> **"Why so serious?"**

Batman: The Dark Knight is built around three core features — Joker Multiball, Scarecrow Multiball, and the Batmobile — that all feed into a final combined Bat Signal Multiball. Stacking is the name of the game here: nearly everything can run alongside everything else. Our goal: **3–5 million points** on a well-played game, with elite "billion point" Batmobile Hurry-Up combos for the truly dedicated.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Watch the plunger lane inserts** (Joker, Scarecrow, Batmobile cycle) — whichever is lit when you plunge becomes **2x for the rest of the ball**; aim for Joker or Scarecrow first
- [ ] **The flashing top rollover also scores a skill shot** if you don't want to gamble on the cycling insert
- [ ] **Lock 3 balls via the Joker lock target** to start Joker Multiball
- [ ] **Use the Scarecrow scoop to start the crane**, then hit it over the 5 yellow inserts to begin Scarecrow Multiball
- [ ] **Shoot Alfred during multiball** for an Add-A-Ball
- [ ] **Stack everything you can** — all three multiballs can run simultaneously, and Batman mode progress continues even during multiball

---

## 🃏 Core Mechanics

### Skill Shot

Plunge when **Joker, Scarecrow, or Batmobile** is lit in the cycling plunger-lane insert — that shot becomes **2x for the rest of the ball**. The flashing top rollover lane also scores a standard skill shot if you'd rather not gamble on timing the cycle.

### Joker Multiball

1. Shoot the **Joker lock target** to light locks
2. Lock **3 balls** to start Joker Multiball
3. Shoot the **5 playfield shots** for jackpots
4. Shoot **Joker** for the Super Jackpot

### Scarecrow Multiball

1. Shoot the **Scarecrow scoop** to start the Scarecrow Attack crane
2. Hit the crane over the **5 yellow inserts** to start Scarecrow Multiball
3. During multiball, keep hitting the crane over the 5 yellow inserts for jackpots
4. Shoot the **Scarecrow scoop** again for the Super Jackpot

> ⚠️ **You can't start a fresh Scarecrow Attack crane sequence during another multiball** — but you *can* finish a crane sequence already in progress and start Scarecrow Multiball while stacked with something else.

### Batmobile Multiball

1. Shoot the **right orbit 4 times**, then the **center ramp**, to start the 2-ball Batmobile Multiball
2. Shoot the right orbit for jackpots

This multiball isn't worth much by itself — its real value is the progress it contributes toward Bat Signal (below).

### Harvey Dent

Complete **20 pop bumper hits (50K)** then **20 spinner spins (100K)**.

## 🚨 Bat Signal Multiball (The Real Endgame)

1. Complete the three main features — **Joker Multiball, Scarecrow Multiball, and the Batmobile Hurry-Up** — to light Bat Signal at the left orbit
2. **Phase 1 (Build):** single-ball phase — shoot **Joker, center ramp, and Scarecrow** to build the jackpot value
3. **Risk/reward:** you can keep building this value higher, but **draining during Phase 1 loses the multiball entirely**
4. Shoot the **left orbit** to lock in your built value and start Phase 2
5. **Phase 2 (Bat Signal Multiball):** hit Joker, center ramp, and Scarecrow for the value you banked in Phase 1

> 🎯 **Pro Tip:** Decide your risk tolerance before Phase 1 — a conservative player banks early for a safe multiball, while a confident player keeps pushing the build value knowing a drain costs everything.

---

## 🔗 Stacking Rules

- **All three multiballs can run together** — this is the single biggest scoring strategy in the game
- You can't start a *new* Scarecrow Attack crane sequence mid-multiball, but a crane sequence already in motion can complete and trigger Scarecrow Multiball while stacked
- You **can** start a multiball during an active Batman mode
- You **cannot** start a Batman mode during multiball — but BATMAN standup target progress made during multiball still counts, and the mode becomes available the moment multiball ends

## 💰 Advanced Strategy: Billion-Point Batmobile Hurry-Up

1. Get **2x Batmobile** from the skill shot
2. Shoot the **center ramp** to start the Batmobile Hurry-Up
3. Keep hitting the **right orbit** to build the hurry-up value as high as you're comfortable with
4. Shoot the **left orbit** to **double the hurry-up value** (once)
5. Shoot the **center ramp** to collect

> 💎 This combo, stacked with the 2x skill shot multiplier and the left orbit double, is the single highest-leverage scoring play in the entire game — practice the right orbit grind specifically for this.

## ⚠️ Dangerous Shots & Drain Patterns

| Shot | Risk | Mitigation |
|---|---|---|
| **Draining during Bat Signal Phase 1** | Loses the entire built-up jackpot value and the multiball itself | Set a personal "good enough" threshold and bank early rather than over-pushing |
| **Right orbit grinding** | Repeated orbit shots under pressure can lead to careless drains | Only grind the orbit when you have a controlled, comfortable rhythm |
| **Scarecrow crane positioning** | Mistimed crane hits waste the limited window before it resets | Watch the yellow insert pattern carefully before committing to a flipper shot |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Take the skill shot, aiming for Joker or Scarecrow if you can time it
2. Start working toward Joker locks and the Scarecrow scoop
3. Build BATMAN standup target progress whenever convenient

**Target Score:** 200K–500K

### Mid Game (Ball 2)

1. Stack Joker Multiball with Scarecrow Multiball if both are available
2. Push Batmobile progress toward completion — its real value is unlocking Bat Signal
3. Complete a Batman mode if you have a clean single-mode window

**Target Score:** 500K–1.5M

### Late Game (Ball 3+)

1. Once all three features are complete, light and run Bat Signal — decide your risk tolerance for Phase 1 build value
2. Attempt the Billion-Point Batmobile Hurry-Up combo if you have the shot control for it
3. Stack every multiball you can for maximum simultaneous jackpot scoring

**Target Score per ball:** 1M–3M+

## ⚙️ Setup Recommendations

| Setting | Recommendation | Reason |
|---|---|---|
| **Playfield pitch** | 6.5°–7.0° | Stern factory standard |
| **Scarecrow crane mechanism** | Verify smooth, accurate movement | Crane positioning accuracy is critical to Scarecrow Multiball qualification |
| **Batmobile bridge/teeter-totter** | Check mechanical reliability | This unique mechanism affects ball diversion to the upper playfield |

---

## 📋 Quick Reference: Strategic Priorities

**Ball 1:** Skill shot toward Joker/Scarecrow, start working toward locks and the scoop.
**Ball 2:** Stack multiballs, build Batmobile progress, complete a Batman mode if possible.
**Ball 3+:** Light and run Bat Signal, attempt the Billion-Point Hurry-Up, stack everything.

**Target High-Score Ballpark:** **3–5 million points** is a strong, competitive game — the Billion-Point Batmobile Hurry-Up combo (despite the name) is really about maximizing a single huge collection, not a literal billion points.
