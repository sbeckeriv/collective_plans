# 🤡 Funhouse 2.0: Rudy's Nightmare - Strategy Guide

*Pedretti Gaming, 2021 — Officially licensed conversion kit for Williams' 1990 Funhouse*

> **"Same Rudy. New nightmare."**

Rudy's Nightmare is a complete software, display, and sound conversion kit for the original 1990 Funhouse cabinet — same physical playfield (Mirror, Clock, trapdoor, Rudy), but an entirely new modern ruleset with original modes and a brand-new wizard mode. Our goal: complete each primary mode to unlock "Defeat Evil Rudy" multiball, and master the trapdoor/Rudy combo that the community has identified as the biggest scoring play.

## 🎃 Core Mechanics

### Hardware Carryover from Original Funhouse

Since this is a **non-permanent conversion kit** (the Mystery Mirror, display, CPU, and sound boards are swapped, with all original parts able to be restored later), the physical shots remain identical to the 1990 original:
- The **Mirror** scoop shot
- The **trapdoor** on the upper loop
- **Rudy's** animatronic mouth
- The standard **Clock** progression mechanic

What's changed is entirely in the **software layer** — new modes, new display graphics, new music, and a new wizard mode structure built on top of the same hardware.

### New Primary Modes

Completing each of these primary modes progressively works toward the game's central wizard mode goal.

### Evil Rudy Multiball & Defeat Evil Rudy

- **Finishing each primary mode** lights **"Defeat Evil Rudy"** — a dedicated **3-ball multiball**
- This is the central wizard-mode-equivalent goal of Rudy's Nightmare, replacing the original Funhouse's simpler Midnight Multiball as the game's main objective

### The Trapdoor/Rudy Combo

The growing player community for this code has specifically identified the **trapdoor and Rudy combo** as the source of the **biggest scores** in Nightmare. Mastering the timing and sequencing of this combo is the single highest-leverage skill to develop.

> 🎯 **Pro Tip:** Since this mod kit reuses the same physical hardware as the original 1990 Funhouse, any muscle memory you have for the Mirror shot, trapdoor loop, and Rudy's mouth carries over directly — it's the software goals and mode structure that are entirely new to learn.

| Risk | Mitigation |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Start working through whichever primary mode is active
2. Use your existing Funhouse shot knowledge (Mirror, trapdoor, Rudy) since the hardware is unchanged
3. Begin learning the new mode-specific objectives shown on the full-color display

### Mid Game (Ball 2)

1. Complete additional primary modes (Duck Hunt, Evil Clones, Hot-Dog Combat, etc.)
2. Start practicing the trapdoor/Rudy combo for high-value scoring
3. Track your progress toward lighting Defeat Evil Rudy

### Late Game (Ball 3+)

1. If enough primary modes are complete, dive into Defeat Evil Rudy multiball
2. Lean on the trapdoor/Rudy combo as your primary scoring engine
3. Finish any remaining primary modes for full completion
