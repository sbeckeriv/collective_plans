# 🦇 Batman Forever Pinball - Ultimate Strategy Guide
## *Path to 1 Billion Points*

> **Target Score:** 1,000,000,000+ points

## 📋 Quick-Start Checklist

- [ ] Select **Light Extra Ball** or **Start Gotham Event** at skill shot
- [ ] Lock in desired mode with launch button before bumpers change it
- [ ] Spell **CAVE** to light multiball locks
- [ ] Match **Two-Face good heads** for Bonus Held
- [ ] Collect all **Bat Signals** for Circus Countdown
- [ ] Hit **Mr. E** for 20x Bonus (game-changer!)
- [ ] Complete all 7 Gotham Events for **Forever** wizard mode

## 🎯 Core Mechanics

### Skill Shot Options

At ball launch, use flippers to cycle through 4 awards before plunging:

| Award                    | Description                                          | Priority |
| ------------------------ | ---------------------------------------------------- | -------- |
| 🔒 **Light Lock**         | Lights VUK for multiball lock (25M if all locks lit) | ⭐⭐⭐⭐     |
| ❓ **Mr. E Award**        | Random mystery award                                 | ⭐⭐⭐      |
| 🎬 **Start Gotham Event** | Starts currently lit mode                            | ⭐⭐⭐⭐⭐    |
| 🪙 **Two-Face 2M Pops**   | Bumpers worth 2M for rest of ball                    | ⭐⭐       |

> **💡 Pro Tip:** If Light Extra Ball mode is showing, ALWAYS choose Start Gotham Event. Otherwise, choose Light Lock and use bumpers to change to a better mode, then lock it in with the launch button.

---

### Gotham Events (Main Modes)

All modes are **30 seconds** and **cannot be stacked**. No progress toward other features during modes.

**Starting Modes:**
- Select from skill shot, OR
- Complete all 5 "?" targets → Shoot right orbit
- **Lock in** desired mode by pressing launch button (prevents bumper changes)

| Mode                   | Objective                                          | Scoring                                                 | Difficulty |
| ---------------------- | -------------------------------------------------- | ------------------------------------------------------- | ---------- |
| 🚗 **Rooftop Chase**    | Video mode - steer Batmobile avoiding obstacles    | 100M+ if completed, 50M flawless bonus                  | ⭐⭐⭐        |
| ❓ **?????**            | Hit ? targets repeatedly                           | 20M + 3M per hit                                        | ⭐⭐         |
| 🪙 **Face Two-Face**    | Hit coin targets 2x each                           | 12M + 2M per target, 60M completion                     | ⭐⭐         |
| 🚨 **Crime Spree**      | Build jackpot with switches, collect at right ramp | 250K + 250K per 8 switches                              | ⭐⭐⭐⭐       |
| 🧠 **Brain Drain**      | Spell C-A-V-E in order (bottom to top)             | 20M + 20M per correct hit, 50M completion + EB hurry-up | ⭐⭐⭐⭐⭐      |
| 🔐 **Safe Trap**        | Hit all 4 ramps                                    | 35M per ramp, 50M completion = 190M total               | ⭐⭐⭐⭐       |
| ⚡ **Light Extra Ball** | Lights EB at mini-ramp                             | Extra Ball                                              | ⭐⭐⭐⭐⭐      |

> **⚠️ Important:** You CANNOT lock in "Light Extra Ball" mode - the game says "I don't think so!" You must catch it naturally.

**Mode Priority (Best to Worst):**
1. 🥇 Light Extra Ball
2. 🥈 Rooftop Chase
3. 🥉 Crime Spree
4. Safe Trap
5. Brain Drain
6. Face Two-Face
7. ????? (Too Many Question Marks)

> **💡 Pro Tip:** Time out the bottom 2 modes by trapping the ball. They score poorly and have dangerous recovery shots.

## 🎱 Multiballs

### Standard Multiball (3-Ball)

**Qualifying:**
1. Spell **CAVE** at drop targets OR select Light Lock from skill shot
2. Shoot VUK to lock balls (up to 3)
3. Lock 3rd ball to start multiball

**Jackpot Sequence:**

| Phase | Shot                     | Value                                 |
| ----- | ------------------------ | ------------------------------------- |
| 1️⃣     | Left Ramp (Save Chase)   | 50M base jackpot                      |
| 1️⃣     | Right Ramp (Save Robin)  | 50M base jackpot                      |
| 2️⃣     | Side Ramp                | Double Jackpot (sum of both jackpots) |
| 3️⃣     | Load Batwing → Mini-Ramp | **300M Super Jackpot**                |

- CAVE targets and ? targets near ramps add **5M** to jackpot value
- After 1st multiball, must complete Two-Face targets to light Double Jackpot

> **💡 Pro Tip:** The Super Jackpot is extremely difficult. Load the Batwing, then either time your launch perfectly OR launch and quickly shoot the mini-ramp manually as backup.

### 4-Ball Multiball (Advanced)

**How to achieve:**
1. Light Batwing (4 ramp shots)
2. When 3rd lock is lit, load the Batwing via left ramp
3. Fire Batwing into VUK with timed launch button press
4. Multiball starts with **4 balls** and **75M base jackpot** (instead of 50M)

## 📈 Progression Systems

### Bat Signals & Circus Countdown

- One shot is marked with a roving Bat Signal
- Collect each signal: **5M + 5M per signal spotted**
- Spot ALL signals → **Circus Countdown** starts
- Shoot right orbit for hurry-up starting at **99M**

> **Note:** Completing Circus Countdown is required for Forever wizard mode.

### Two-Face Coins

Two target banks (center and far right). Each has Good Head (left) and Bad Head (right).

| Match       | Reward                  |
| ----------- | ----------------------- |
| Good + Good | **Bonus Held** ⭐        |
| Bad + Bad   | **22M**                 |
| Mismatch    | 2M + Relight Laser Kick |

> **💡 Pro Tip:** On ball 3 with ball save active, go for Good Heads match first - Bonus Held pays TWICE if you drain on last ball!

### Bat Gadgets

- Shoot right orbit (when mode not lit) or VUK during normal play
- Collect **5 gadgets** → **Super Laser Kick** (unlimited kickback)

### Laser Kick (Kickback)

**Light by:**
- Completing B-A-T top lanes
- Matching Two-Face coins (after first match)
- 2x Bonus multiplier

## 🏆 Wizard Mode: FOREVER

### Requirements

Complete ALL of the following:

- [ ] Play all 7 Gotham Events
- [ ] Match Good Heads (Two-Face coins)
- [ ] Match Bad Heads (Two-Face coins)
- [ ] Spot all Bat Signals (triggers Circus Countdown)

### Gameplay

- **4-ball multiball** with **unlimited ball save** for ~60 seconds
- Every switch hit adds **1M × balls in play** to Forever Jackpot
- Jackpot starts at **50M**, collected by loading the moving Batwing
- Batwing moves left-to-right - time your left ramp shot!
- Jackpot resets to 50M after collection
- Mode ends when all balls drain after ball save expires

> **💡 Pro Tip:** Keep all 4 balls in play as long as possible - the jackpot builds faster with more balls!

## 🎯 Shot Analysis

### 🔴 Dangerous Shots

| Shot                  | Risk                               | Mitigation                    |
| --------------------- | ---------------------------------- | ----------------------------- |
| **CAVE Drop Targets** | Ball rejection to outlanes         | Soft shots, be ready to nudge |
| **? Standup Targets** | Unpredictable rebounds             | Avoid during critical moments |
| **Side Ramp**         | Requires late upper flipper timing | Practice the timing           |
| **Left Orbit**        | Can feed bumpers unpredictably     | Strong shots only             |

### 🟢 Safe Shots

| Shot                   | Why It's Safe                                    |
| ---------------------- | ------------------------------------------------ |
| **Right Ramp**         | Clean feed to right flipper, enables ski passing |
| **Mini-Ramp (U-turn)** | Quick feed to upper flipper                      |
| **VUK**                | Controlled ball, advances multiple features      |

### 🎮 Flipper Selection

| Target      | Flipper               | Notes                                   |
| ----------- | --------------------- | --------------------------------------- |
| Left Ramp   | Left                  | Steep shot, similar to STTNG right ramp |
| Right Ramp  | Left                  | Best for ski passing                    |
| Side Ramp   | Upper Right           | Late flip required                      |
| Mini-Ramp   | Left (early) or Upper | Early flip from left works              |
| VUK         | Either Lower          | Backhand possible from right            |
| Right Orbit | Left or Right         | Strong shot for saucer                  |

### 🔄 Batarangs (Combo Shot)

- Left Orbit → Side Ramp = **10M + Batarang**
- Beat the champion = **+100M bonus**
- Inner right inlane lights **2x Batarang**

## 💰 Scoring Strategy

### Path to 1 Billion Points

#### Early Game (Ball 1)
1. Select **Start Gotham Event** if Light Extra Ball is showing
2. Otherwise, select **Light Lock** and work toward multiball
3. Complete B-A-T lanes for kickback and bonus multiplier
4. Hunt for **Mr. E 20x Bonus** - this is the key to huge scores!

#### Mid Game (Ball 2)
1. Continue mode completion toward Forever
2. Match **Good Heads** for Bonus Held
3. Collect Bat Signals
4. Start multiball and go for Double Jackpot + Super Jackpot

#### Late Game (Ball 3)
1. If you have 20x Bonus + Bonus Held: **SHOOT RAMPS**
2. Each ramp = 500K × 20 = 10M in bonus alone
3. Gotham Events played = 3M × 20 = 60M each in bonus
4. If Forever is qualified, start it for massive scoring

### Highest Value Opportunities

| Opportunity              | Value            | Difficulty |
| ------------------------ | ---------------- | ---------- |
| 20x Bonus + Ramps        | 100M-500M+ bonus | ⭐⭐⭐⭐       |
| Forever Wizard Mode      | 200M-500M+       | ⭐⭐⭐⭐⭐      |
| Super Jackpot            | 300M             | ⭐⭐⭐⭐⭐      |

> **💡 The 20x Bonus Strategy:** Go for mystery awards until you get 20x bonus. Then just shoot ramps. If possible, get Bonus Held. This is the easiest path to beating high scores!
