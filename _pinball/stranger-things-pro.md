## 📺 Chapter Modes

### Starting Chapters
- Shoot **left orbit** or **left ramp** 3× → Start Season 1 chapter
- Shoot **right ramp** or **right orbit** 3× → Start Season 2 chapter
- Pop bumpers cycle which chapter is selected <kcite ref="21"/>

> 💡 **Pro Tip:** Until 2 chapters have been played, the left standup lane spots progress toward starting a chapter <kcite ref="21"/>.

## 🐕 Demodog Attack Modes

**How to Start:** Spell D-E-M-O-D-O-G at standup targets → shoot left standup lane <kcite ref="21"/>

| Mode                      | Objective                              | Scoring                    | Drawing                   |
| ------------------------- | -------------------------------------- | -------------------------- | ------------------------- |
| **Break Out!** (40 sec)   | Hit 15 DEMODOG targets                 | 1M/2M/4M per wave          | +1 for completing         |
| **Junk Yard** (40 sec)    | 3 phases: orbits → ramps → inner shots | Varies by phase, up to 18M | +1 for completing phase 3 |
| **It's a Trap!** (40 sec) | Shoot all 8 lit shots                  | 3M each                    | +1 for all 8 shots        |

> ⚠️ **Note:** Starting "It's a Trap!" solidly lights the Demodog! objective <kcite ref="21"/>.

## 🧠 Multiballs

### Telekinesis Multiball

**How to Qualify:**
1. Shoot green LIGHT LOCK targets on each side of left ramp
2. First MB: Both targets = all 3 locks lit
3. Later MBs: 2 target completions per lock <kcite ref="21"/>

**How to Start:** Lock 3 balls on left ramp

**Jackpot Structure:**

| Phase         | Shot                                  | Value                  |
| ------------- | ------------------------------------- | ---------------------- |
| Jackpots      | 5 lit shots                           | 1.1M (+100K per Super) |
| Super Jackpot | Left saucer                           | 6M base                |
| 2× Super      | Lock ball on left ramp first (20 sec) | 12M                    |

**Multiplier Stacking (Pro):**
- 2× from left ramp lock
- 2× from playfield multiplier
- 2× from combo shot
- **Maximum: 48M Super Jackpot** <kcite ref="21"/>

> 💡 **Pro Tip:** Cradle a ball on each flipper. Shoot left ramp with right flipper for 2× lock, then tip the cradled ball on left flipper into the saucer for easy double super <kcite ref="55"/>.

### 🎰 End-of-Ball Bonus

Bonus **carries over between balls** (except chapter scoring) <kcite ref="21"/>:

| Component                   | Value         |
| --------------------------- | ------------- |
| 5-Way Combo achieved        | +10M          |
| Demogorgon Modes started    | ×2M each      |

## 🔀 Stacking & Combos

### ✅ What Stacks Well

| Stack                           | Why It Works                                            |
| ------------------------------- | ------------------------------------------------------- |
| Chapter + TK Multiball          | Ball save protects you while completing mode objectives |
| Chapter + Demodog + TK MB       | Maximum scoring potential                               |
| 2× Playfield + Any Multiball    | Double all jackpots                                     |
| Demogorgon MB + locked TK balls | Swipe balls to extend multiball (Premium/LE only)       |

**Stacking Sequence:** Start Chapter → Start Demodog → Start TK Multiball <kcite ref="21"/>

### ❌ What NOT to Stack

| Avoid                          | Reason                                          |
| ------------------------------ | ----------------------------------------------- |
| TK MB + Single-ball Demogorgon | Ramp flap blocks Burn it Back and Mystery shots |
| Multiple Chapters              | Cannot stack chapter modes                      |
| Starting modes during TK MB    | Chapters cannot be started during TK Multiball  |

> 💡 **Pro Tip:** Start your chapter with the SAME shot that starts multiball (left ramp) for perfect timing <kcite ref="71"/>.

## 🏆 Wizard Modes

### Mini-Wizard Modes

#### 🌀 Total Isolation 1
**Requirement:** Play 4 chapters (one on each major shot) <kcite ref="21"/>

| Phase         | Objective                   | Scoring                          |
| ------------- | --------------------------- | -------------------------------- |
| Jackpots      | All major shots lit         | 1M (+100K per shot), 2× if combo |
| Super Jackpot | Center scoop (after 3+ JPs) | Sum of all jackpots              |
| Add-a-Ball    | 3rd Super Jackpot           | +1 ball                          |

#### 🌀 Total Isolation 2
**Requirement:** Play 8 chapters (two on each major shot)

| Phase         | Objective                                    | Scoring                     |
| ------------- | -------------------------------------------- | --------------------------- |
| Jackpots      | Shoot flashing shot, then shoot again for 2× | Base + combo doubles        |
| Super Jackpot | Center scoop (all shots made)                | Combined value of all shots |

> 💡 **Pro Tip:** Total Isolation 2 can yield 200M+ Super Jackpots with good combo play <kcite ref="21"/>.

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                   | Risk Level | Drain Path                      |
| ---------------------- | ---------- | ------------------------------- |
| **Drop Targets**       | 🔴 HIGH     | SDTM (straight down the middle) |
| **Light Lock Targets** | 🔴 HIGH     | Rebounds to outlanes            |
| **Demogorgon Mouth**   | 🔴 HIGH     | Misses go anywhere              |
| **Right Orbit**        | 🟡 MEDIUM   | Can feed slings unpredictably   |
| **Pop Bumpers**        | 🟡 MEDIUM   | Random exits                    |

> ⚠️ **"Missed shots are dangerous shots"** - This game is unforgiving <kcite ref="84"/>.

### ✅ Safe Shots

| Shot             | Safety   | Notes                                        |
| ---------------- | -------- | -------------------------------------------- |
| **Left Ramp**    | 🟢 HIGH   | Returns to left flipper, backhandable on Pro |
| **Right Ramp**   | 🟢 MEDIUM | Returns to right flipper                     |
| **Left Orbit**   | 🟢 HIGH   | Good for looping inlanes                     |
| **Burn it Back** | 🟢 MEDIUM | Inner loop, good progression                 |

### 🎮 Flipper Selection

| Target       | Recommended Flipper                                        |
| ------------ | ---------------------------------------------------------- |
| Left Ramp    | Right flipper (forehand) OR Left flipper (backhand on Pro) |
| Right Ramp   | Right flipper                                              |
| Left Orbit   | Left flipper                                               |
| Right Orbit  | Right flipper                                              |
| Drop Targets | Either (but dangerous!)                                    |
| Demogorgon   | Right flipper (when ramp is down)                          |

> 💡 **Pro Tip:** Post-passing feels awkward on this game due to sling placement. Practice catching and cradling instead <kcite ref="65"/>.

## 💰 Scoring Strategy: Path to 1 Billion

### 🌅 Early Game (Ball 1)

1. **Go for MXV Skill Shot** - Sets up drawing collection <kcite ref="71"/>
2. **Light locks during ball save** - Both lock targets are dangerous <kcite ref="54"/>
3. **Start a chapter** - Prioritize chapters over everything <kcite ref="53"/>
4. **Stack chapter + TK Multiball** - Use same left ramp shot to start both <kcite ref="71"/>
5. **Avoid drop targets** - Only shoot during Upside Down <kcite ref="56"/>

### 🌞 Mid Game (Balls 2-3)

| Priority | Goal                        | Why                                    |
| -------- | --------------------------- | -------------------------------------- |
| 1        | Reach Total Isolation       | 4 chapters = first wizard mode         |
| 2        | Stack modes with multiballs | Maximize scoring windows               |
| 3        | Build Bonus X               | Loop left orbit for inlane completions |
| 4        | Collect 2× Playfield        | Time it for multiballs                 |
| 5        | Progress Demogorgon modes   | Big points in modes 2, 4, 5            |

### 🌙 Late Game (Final Ball)

1. **If bonus is huge:** Focus on completing inlanes for max Bonus X <kcite ref="58"/>
2. **If close to wizard mode:** Push for Total Isolation 2 or season wizard modes
3. **Demogorgon Mode 2 (Trap 'Em!):** Can yield 100M+ with 2× playfield <kcite ref="66"/>

### 💎 Highest Value Opportunities

| Opportunity                         | Potential Value |
| ----------------------------------- | --------------- |
| Total Isolation 2 Super Jackpot     | 150-200M+       |
| Demogorgon Mode 2 with 2× Playfield | 100-150M        |
| End-of-Ball Bonus (maxed)           | 300-500M+       |
| Can't Stop This! (Mode 5)           | 100M+           |
| Stacked TK MB + Chapter + 2×        | 50-100M         |

> 💡 **The Path to 1 Billion:** Get to Total Isolation ASAP, stack everything possible, and build massive bonus. Players report 1B+ games with 500M+ in bonus alone <kcite ref="62"/><kcite ref="66"/>.

## ✅ Quick-Start Checklist

### 🆕 First Game Priorities

- [ ] **Skill Shot:** Go for MXV (soft plunge → rebound → saucer)
- [ ] **Ball Save:** Hit both LIGHT LOCK targets while safe
- [ ] **First Mode:** Shoot left ramp 3× to start Season 1 chapter
- [ ] **First Multiball:** Lock 3 balls on left ramp for TK Multiball
- [ ] **Stack:** Start chapter BEFORE starting multiball
- [ ] **Avoid:** Drop targets (except during Upside Down)
- [ ] **Collect:** Super Jackpot in TK MB (left saucer after 5 jackpots)

### 🎯 Key Shots to Practice

1. Left ramp (forehand and backhand)
2. Left orbit loops
3. Burn it Back inner loop
4. Catching/cradling from ramp returns

| < 50M    | Learning     |
| 50-150M  | Beginner     |
| 150-400M | Intermediate |
