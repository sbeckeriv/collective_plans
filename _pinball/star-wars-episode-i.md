# 🌟 Star Wars Episode I — Complete Player's Guide
### Williams, 1999 | Pinball 2000 | 🎯 Target: 1 Billion Points

> **📜 Historic Note:** Star Wars Episode I is the **last pinball machine ever manufactured by WMS Industries** (Williams/Bally labels). Production ended when WMS announced the closure of their pinball division on **October 25, 1999**. Version 1.4 of the ROM encodes this date as an Easter egg in the Jar Jar scoring: **19,992,510 points** = 25-10-1999. A true piece of pinball history worth understanding deeply.

---

## ⚡ Quick-Start Checklist

Use this before and during your first serious game:

- [ ] Nail the **skill shot** on every plunge
- [ ] Complete **C-3PO Orbit twice** early (Extra Ball + JEDI letter)
- [ ] Hit **both saucers** for random awards before locking
- [ ] **Lock both balls** in the saucers to start Multiball
- [ ] **Chain ramp shots** within 2–3 seconds — this is how you reach 1 billion
- [ ] Start **R2-D2 Astrodroid** or **Space Battle** mode during ramp runs
- [ ] Spell **J-E-D-I** → shoot the center target → **Saber Battle**
- [ ] Grab **Hold Bonus** whenever available — preserves your 5x multiplier

## 🎯 Core Mechanics

### Skill Shots

Plunge the ball up the slope with **just enough force** to climb it — but not so much it spills into the right ramp return lane. The target is the **R2-D2 "Sneaky" Lane** at the top of the slope.

| Attempt # |    Points | Bonus                 |
| --------- | --------: | --------------------- |
| 1st       | 1,000,000 | —                     |
| 2nd       | 2,000,000 | —                     |
| 3rd       | 3,000,000 | ✨ JEDI Letter awarded |
| 4th+      | 3,000,000 | —                     |

**Key rules:**
- Attempts count at any ball launch **except** ball-saver restores
- During Jedi Battles: only the **first plunged ball** is eligible per battle
- The R2-D2 lane is the **only reliable path to collect Extra Balls**
- All lit Extra Balls **persist between balls**

> 🏆 **Pro Tip:** Skill shot attempts are cumulative across the entire game. Your 3rd total attempt — even if spread across different balls — awards the JEDI letter. Never skip a plunge attempt.

---

### Ball Saver

- **~5 seconds** after ball hits either inlane OR completes the skill shot
- **No ball saver** during Multiball
- Jedi Battles use their own separate ball-feed system

## 🎭 Modes

Modes launch from the **Holoprojector**. Normally you must hit it **3 times** first — or use the "Activate Projector" saucer award to skip this requirement.

**Mode rotation rules:**
- Only ~5 modes available at game start
- Each completed mode unlocks another until all 13 are accessible
- After all 13 complete: **all modes reset and bonuses DOUBLE** on the 2nd pass
- 2nd pass: Perfect Shooting = **6,000,000 pts** | Completion = **2,000,000 pts**

## 🎱 Multiball

### Qualifying Multiball

1. Shoot **Watto's Junk Shop** (left saucer) — locks ball 1
2. Shoot **Mos Eisley Market** (right saucer) — locks ball 2
3. Auto-plunger fires ball 3 — **Multiball begins!**

**Relighting saucers for subsequent multiballs:**

| Multiball # | Pre-hits Required Before Locking      |
| ----------- | ------------------------------------- |
| 1st         | None — lock directly                  |
| 2nd         | Hit either saucer **1 time** first    |
| 3rd         | Hit either saucer **2 times** first   |
| Nth         | Hit either saucer **N-1 times** first |

### Jackpot Structure

| Step                  | Shot Required |                     Value |
| --------------------- | ------------- | ------------------------: |
| Jackpot 1             | Center Target |                 1,000,000 |
| Jackpot 2             | Left Ramp     |                 1,000,000 |
| Jackpot 3 + **SUPER** | Right Ramp    | 1,000,000 + **5,000,000** |
| *Cycle repeats...*    |               |                           |

Full 3-shot cycle value: **7,000,000 points** (1M + 1M + 1M + 5M Super)

### Multiball Rules & Notes

- **No ball saver** during Multiball
- If 2 balls drain (3 if ball saver still running) before hitting Jackpot 1: offered multiball restart by shooting either saucer
- Restarted Multiball has only **2 balls** (no auto-plunger)
- **Any mode running before Multiball continues during Multiball** — multiple balls can complete mode shots faster
- After the mode ends, Projector stays down until Multiball ends — no new modes can start

> 🏆 **Pro Tip:** Start **R2-D2 Astrodroid** or **Space Battle** immediately before locking the 2nd ball. When Multiball starts, chaotic ball movement often completes mode shots automatically while you chase jackpots.

> ⚠️ **Warning:** Never start Multiball while a **Jedi Battle (Saber Battle)** is active — the ball interaction is described as "absolutely horrible" and can cost you both modes.

## 📈 Progression Systems

### 🏆 JEDI Rank — The Primary Goal

The game's narrative goal is to become a **Jedi Spirit** by advancing through 4 ranks:

```
YOUTH ──► KNIGHT ──► MASTER ──► 🌟 SPIRIT
```

**To advance each rank:** Spell J-E-D-I → Center Retractable Target drops → shoot it → **Saber Battle**

> ✅ **Key Rule:** JEDI letters are **cumulative across all balls** — they never reset between balls.

**Ways to earn JEDI letters:**

| Method                        | Notes                                                        |
| ----------------------------- | ------------------------------------------------------------ |
| Mode completions              | Completing/performing well in modes awards letters           |
| Skill shot (3rd total)        | Cumulative — earns a letter on the 3rd successful skill shot |
| C-3PO Orbit (2nd completion)  | Letter awarded on 2nd complete                               |
| C-3PO Orbit (8th completion)  | Letter on 8th complete                                       |
| C-3PO Orbit (13th completion) | Letter on 13th complete                                      |
| Saucer random award           | Both saucers can randomly award a letter                     |

---

Hit the C-3PO rollover target to add pieces (8 total; 4 lit at game start, ~2-second pause required between hits). Awards cycle through this pattern:

|     2nd      | ✨ JEDI Letter                |

> 🏆 **Pro Tip:** C-3PO orbit is efficient and low-risk. Complete it early in each ball. The 1st completion lights an Extra Ball. The 4th completion's **Hold Bonus** is one of the most valuable awards in the game.

| Component                  |     Max Value |
| -------------------------- | ------------: |

| Bonus Multiplier           |            ×5 |

### 🐟 Gungan Jets / JAR JAR Progression

- Hit Gungan Jets (jet bumpers) to light letters G-U-N-G-A-N
- Completing GUNGAN transforms the right spinner into a fish creature
- Shoot the right ramp to spell J-A-R-J-A-R
- Completing JAR JAR = **2,000,000 pts** (standard ROMs)
- **Version 1.4 ROM Easter Egg:** JAR JAR = **19,992,510 pts** 🎉

> ⚠️ GUNGAN resets each ball by default (check settings — may be adjustable).

### ✅ What Stacks Well

| Stack Combination                  | Why It Works                                                            |

| Scenario                                  | Why to Avoid                                                   |
| ----------------------------------------- | -------------------------------------------------------------- |
| **Multiball during Jedi Battle**          | Ball interaction is chaotic and damaging to both modes         |
| **Starting new modes mid-Multiball**      | Impossible — Projector won't reappear until Multiball ends     |
| **Chasing Gungan/Jar Jar at key moments** | GUNGAN resets each ball — risky distraction unless on v1.4 ROM |

### 🔥 Ramp Combo System — The Road to 1 Billion

This is the most important mechanic on the machine:

- Shoot either ramp **within 2–3 seconds** of any other ramp shot = **Combo awarded**
- Each consecutive combo: **combo number × 100,000 points**
- Combo #1 = 100,000 | #10 = 1,000,000 | #50 = 5,000,000 | #100 = 10,000,000

**Cumulative combo chain totals:**

| Chain Length | Total Points Scored |
| :----------: | ------------------: |
|    10-way    |           5,500,000 |
|    20-way    |          21,000,000 |
|    30-way    |          46,500,000 |
|    50-way    |         127,500,000 |
|   100-way    |         505,000,000 |
| **141-way**  |  **~1,000,000,000** |

> ⚠️ **Tournament Note:** This combo system was so powerful it was **specifically restricted in tournament ROMs**. On a standard location machine, this is your primary route to 1 billion and beyond.

> 🏆 **Pro Tip:** Cradle the ball on the left flipper, then alternate: Left ramp → Right ramp → Left ramp → repeat. The 2–3 second window is generous. During **R2-D2 Astrodroid** mode, every ramp shot is triple-dipping: combo points + hurry-up reset + mode progress.

---

### ⏱️ Optimal Timing for Starting Features

1. **Ball start:** Complete C-3PO orbit once or twice before anything else
2. **Before locking for Multiball:** Hit saucers first for random awards ("Activate Projector" is the jackpot award here)
3. **Start a mode BEFORE the 2nd lock:** Mode runs during Multiball and can complete faster with 3 balls flying
4. **Ramp combos:** Build these continuously — never waste a ramp shot opportunity
5. **Saber Battle:** Trigger when you have fresh lasers and are in control — not during chaotic moments

## ⚔️ Wizard Modes

### Saber Battle (Jedi Battle) — Rank Advancement

**How to qualify:**
1. Earn JEDI letters through modes, skill shots, C-3PO orbit, and saucer awards
2. Spell **J-E-D-I** — the Center Retractable Target drops
3. Shoot the center target to start the **Saber Battle**

**Rank Progression:**

| Rank   | Battle # | Win Award | Loss Award | Outcome            |
| ------ | :------: | --------: | ---------: | ------------------ |
| Youth  |   1st    | 5,000,000 |  1,000,000 | → Knight           |
| Knight |   2nd    | 5,000,000 |  1,000,000 | → Master           |
| Master |   3rd    | 5,000,000 |  1,000,000 | → 🌟 Jedi Spirit    |
| Spirit |   4th+   | 5,000,000 |  1,000,000 | Cycles (4 locales) |

> ✅ **You advance rank whether you win OR lose** — but winning = 5× the points. Always push to win.

### Saber Battle Gameplay

- Up to **3 balls** in play simultaneously
- Lost balls are **re-fed to the plunger** — you manually plunge each one
- Only the **first plunged ball** is eligible for a skill shot
- Darth Maul attacks Qui-Gon on his own — your shots power Qui-Gon's counterattacks
- **Race against time:** Lower Darth Maul's health to zero before time expires
- **Best shots:** Ramp combos — they deal the most damage and score the most points
- Darth Maul **speeds up** with each successive battle

> 🏆 **Pro Tip:** You can still attempt the skill shot on your first plunge during a Saber Battle. Do it — points plus potential JEDI letter toward the next battle.

### 🌟 Grand Goal: Jedi Spirit

Reaching **Jedi Spirit** after 3 Saber Battles is the game's narrative completion. After reaching Spirit, the Saber Battle locales cycle through 4 environments every 4 battles — you can keep farming **5,000,000 per battle win** indefinitely.

## 📋 Additional Information

### 🔄 Does Completing Modes Change Gameplay?

Yes — significantly:
- Completing each mode **unlocks another** until all 13 are available
- After all 13 complete: all modes reset and **all bonuses double** on the 2nd pass
- Mode selection within the available pool is semi-random
- "Activate Projector" saucer award removes the 3-hit Holoprojector requirement for the next mode start

### 🎮 Easter Eggs & Hidden Features

#### 🐟 The Jar Jar Date Easter Egg (v1.4 ROM Only)
Spelling JAR-JAR via Gungan Jets + Right Ramp awards **19,992,510 points** — encoding 25-10-1999 (October 25, 1999), the day WMS Industries announced the closure of its pinball division.

#### 💰 Free Credits Flipper Code

At the attract mode / start screen, enter:

`B → 24R → 1L → 9R → 1L → 14R → 1L → 1R → 1L`

*(Spells "XINA" = Xina Is Not Appl)*

> **Note:** B = Start Button, L = Left Flipper, R = Right Flipper. Numbers indicate how many presses (e.g., "24R" = press right flipper 24 times).

#### 🔫 Super Duper Secret STAP Bonus
In Ground Battle, press **both laser buttons simultaneously** the instant a wave begins (before any druids spawn) to earn **1,000,000 bonus points** in one action.

#### 🧲 Magnet Saucer Trick
The Sith Probe Triad magnet has reversed balls heading toward the drain and launched them directly into the Market saucer — locking a ball for free without targeting it intentionally.
