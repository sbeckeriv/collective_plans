## 🎯 Core Mechanics

### Skill Shots (Plunge Options)

Use the flipper buttons to cycle through options while the ball is in the plunger lane. Launch when your desired choice is lit <kcite ref="63"/>.

| Option                   | Effect                                                              | Best For                                                          |
| ------------------------ | ------------------------------------------------------------------- | ----------------------------------------------------------------- |
| ⭐ **Warp Factor 4**      | Start at Warp 4 (or Warp 2 if you've hit Warp 9)                    | Early game — lights spinner, builds toward artifacts              |
| 🚀 **Start Mission**      | Immediately starts the flashing mission                             | Mission grinding, Final Frontier progress                         |
| 🔒 **Light Lock**         | Lights and auto-locks a ball (2x only, then becomes Light Holodeck) | Multiball setup                                                   |
| 🎯 **Launch Probe**       | Ball loads into cannon, shoot lit target for random award           | Risky but can yield Extra Ball                                    |
| 🎰 **Flipper Skill Shot** | Ball feeds to left flipper, shoot right ramp for displayed award    | Awards include: Extra Ball, 10M, Hold Multipliers, Double Spinner |

> **💡 Pro Tip:** Choose **Warp Factor 4** if available — it lights the spinner (100K per spin × unit value), and reaching Warp 5 holds your bonus multipliers, Warp 7 doubles the spinner, and Warp 8 awards an artifact <kcite ref="105"/>.

### Missions (Modes)

There are **7 missions** plus the **Final Frontier** wizard mode. Start missions by shooting the center **Start Mission** sinkhole when lit. The flashing mission is advanced by bumpers and the Q target <kcite ref="61"/>.

#### Mission Summary Table

| Mission                 | Timer          | Objective                                                               | Artifact Requirement                                  | Max Points          |
| ----------------------- | -------------- | ----------------------------------------------------------------------- | ----------------------------------------------------- | ------------------- |
| ⏰ **Time Rift**         | 20s            | Shoot any lit shot (10M each). Time Rift targets add 5s and 5M per shot | ~5 shots                                              | 25M/shot max        |
| 🌀 **Worm Hole**         | 20s            | Right ramp adds 10M, spinner adds 1M/spin. Left ramp collects           | Shoot left ramp (Delta Quadrant)                      | 30M+ base           |
| 🔭 **Search the Galaxy** | 40s            | Shoot ramps: Center→Right→Left for 10M each                             | Any 3 ramps = 1 artifact; In order + NZ = 2 artifacts | 50M (NZ bonus)      |
| ⚔️ **Battle Simulation** | 20s            | Use cannons to hit lit targets. Advance levels 1-6                      | Level 5                                               | 100M (Level 6) + EB |
| ❓ **Q's Challenge**     | No fixed timer | Hit lit shots (10M→100M). Shots time out individually                   | 5 shots                                               | 100M/shot max       |
| 🚑 **Rescue**            | 30s            | Load 50 people (switches), rescue via ramps                             | 25 people loaded/rescued                              | 100M total          |
| ☄️ **Asteroid Threat**   | 20s            | Part 1: Shoot center for countdown value. Part 2: Hit all 7 shots       | 3 shots in Part 2                                     | 20M/shot            |

### Mission Details

#### ⏰ Time Rift
- All shots lit for **10M each**
- Time Rift targets (3-bank): Add **5 seconds** and **5M** to shot value (max 25M/shot)
- Very generous timer — one of the easiest missions <kcite ref="61"/>

> **💡 Pro Tip:** "Hit your shots and yawn" — the time additions are extremely generous <kcite ref="105"/>.

#### 🌀 Worm Hole
- Value starts at **30M**
- Right ramp: +10M to value
- Spinner: +1M per spin
- **Left ramp collects value AND artifact** <kcite ref="72"/>

#### 🔭 Search the Galaxy
- Shoot ramps in order: **Center (Alpha) → Right (Beta) → Left (Delta)**
- Each ramp: **10M**
- Complete in order: Lights **Neutral Zone for 50M** + second artifact
- Any 3 ramps (any order): 1 artifact <kcite ref="61"/>

> **⚠️ Warning:** The left ramp (Delta Quadrant) from the upper flipper is a high drain risk. Consider just getting 3 ramps in any order for the artifact <kcite ref="91"/>.

#### ⚔️ Battle Simulation
- Timer **pauses** while cannons are loaded
- Each level: **10M × level number** (Level 3 = 30M)
- **Level 5: Artifact**
- **Level 6: 100M + Extra Ball**
- Right cannon → Neutral Zone targets (just hitting targets counts)
- Left cannon → Start Mission hole <kcite ref="72"/>

> **💡 Pro Tip:** Pick a visual reference point on the playfield for when to fire. Don't try to "aim" — time your trigger pull to a consistent spot <kcite ref="105"/>.

#### ❓ Q's Challenge
- Shots light randomly, hitting one lights another
- Q target adds additional shots
- Scoring: **10M → 20M → 30M...** up to **100M per shot**
- **This is the ONLY mission that continues during multiball!** <kcite ref="72"/>
- Shots time out after ~10 seconds if not hit

> **🔥 Key Strategy:** Start Q's Challenge, then lock your 3rd ball for Borg Multiball. Q's shots won't time out during multiball = massive points! <kcite ref="106"/>

#### 🚑 Rescue
- Every switch hit = 1 person loaded + **1M**
- Ramps rescue loaded people
- Start Mission rescues 5 people regardless
- **25 people = Artifact**
- **50 people = 100M total** <kcite ref="72"/>

> **💡 Pro Tip:** Right ramp → Left orbit at start = guaranteed artifact from switch hits <kcite ref="105"/>.

#### ☄️ Asteroid Threat
- **Part 1:** 20M counts down quickly. Shoot center to collect.
- **Part 2:** All 7 shots lit for collected value. 20 seconds to hit them.
- If countdown reaches 5M, Part 2 starts automatically at 5M/shot
- **3 shots = Artifact** <kcite ref="72"/>

> **💡 Pro Tip:** Don't stress about the countdown. Getting 5M vs 20M per shot is minor — focus on getting the artifact <kcite ref="105"/>.

## 🎱 Multiballs

### Borg Multiball (Main Multiball)

**How to Start:**
1. Shoot **right orbit** to light lock
2. Shoot **right orbit** again (or side ramp) to lock ball
3. Lock **3 balls** via right orbit to start multiball <kcite ref="70"/>

**Jackpot Structure:**

| Jackpot | Value                   | Shot                                          |
| ------- | ----------------------- | --------------------------------------------- |
| Single  | Base value (starts 20M) | Start Mission                                 |
| Double  | 2× base                 | Start Mission (2nd hit)                       |
| Triple  | 3× base                 | Start Mission (3rd hit) OR Side Ramp (always) |

- **Jackpot value increases throughout the entire game** (never resets!)
- Pop bumpers increase jackpot value
- By late game: **100M-300M base** = **300M-900M triple jackpots!** <kcite ref="71"/>

**Shields System:**
- Green lights in center = shield strength
- Shields deplete as Borg attacks
- **Spinner recharges shields**
- Jackpot only lit when shields are up
- **Side ramp ALWAYS scores triple jackpot** regardless of shields <kcite ref="70"/>

**Cannon Bonus:**
- At multiball start, left cannon loads
- Shoot jackpot FROM cannon = **+10M to jackpot value** before collecting
- Can chain: Single → Double → Triple from cannons <kcite ref="70"/>

> **🔥 Pro Strategy:** Focus on Borg Multiball for massive scores. One player achieved **160 billion** primarily through Borg Multiball with base jackpots reaching **1 billion** (3B triple jackpots!) <kcite ref="110"/>

### Neutral Zone Multiballs

Shoot the **Neutral Zone hole** (not targets) to cycle through three mini-multiballs <kcite ref="69"/>:

#### 👽 Ferengi Multiball
- Right cannon loads → Shoot NZ targets to start **3-ball multiball**
- Any NZ target hit adds another ball (max 4) for **10 seconds**
- Jackpot at Start Mission: **10M × balls in play**
- Yellow standups (left): +2M to next jackpot only

> **💡 Pro Tip:** If you start Ferengi near Warp 9, immediately go for Warp 9 mode. The NZ ball-adder stays active throughout Warp 9, giving you a long guaranteed 4-ball multiball! <kcite ref="105"/>

#### 🛸 Romulan Mode
- 3 shots lit = 3 Romulan ships on display
- Hit shot = ship cloaks (disappears) with timer
- Timer expires = shot relights
- **Cloak all 3 simultaneously = 30M** <kcite ref="69"/>

#### 🚀 Cardassian Multiball
- **2-ball multiball**
- Jackpot at NZ targets: Starts at **50M**
- Enterprise shields affect jackpot (0% shields = 5M jackpot)
- **Spinner increases shield strength** (and jackpot value) <kcite ref="69"/>

> **💡 Pro Tip:** For Romulan/Cardassian, consider trapping the ball and waiting them out. The NZ shots have nasty drain paths <kcite ref="105"/>.

## 📈 Progression Systems

### Warp Factors

Advance warp by shooting the **left ramp (Delta Quadrant)** <kcite ref="82"/>.

| Warp | Benefit                                     |
| ---- | ------------------------------------------- |
| 4    | Lights spinner (100K × unit value per spin) |
| 5    | **Hold Bonus Multipliers** between balls    |
| 6    | Doubles spinner unit value, lights Hurry Up |
| 7    | Doubles spinner value again                 |
| 8    | **Artifact**                                |
| 9    | Starts **Factor 9 Mode**                    |

#### Factor 9 Mode
- Each spinner/left ramp shot advances warp by 0.1
- **9.1 = 20M, 9.2 = 25M... 9.9 = 60M**
- 10 seconds per shot (20 seconds for final 9.9 shot)
- After completion, return to Warp 1; left ramp no longer advances warp
- Plunge option becomes "Warp Factor 2" <kcite ref="82"/>

> **⚠️ Warning:** Timer does NOT pause during Factor 9 — don't lock balls or shoot sinkholes! <kcite ref="82"/>

> **💡 Pro Tip:** You can collect each artifact **multiple times**. Stack them for massive Final Frontier payouts! <kcite ref="83"/>

### Holodeck

**Light Holodeck:** 3 right ramps, then 9, etc. (or via skill shot)

Shoot right orbit when lit to choose:
- **25 Million** (10M on older ROMs)
- **Shuttle Simulation** (video mode)

#### Shuttle Simulation Pattern
The cavern layout **never changes**. Memorize this pattern <kcite ref="81"/>:

```
L-L-R-R-L-L-R-L-L-L
```

**Awards:** 10M - 10M - 0M - 30M - 30M - 30M - 0M - 10M or EB - 30M - **Artifact**

**Total: ~159M + Artifact** (149M if you get the EB)

> **🎮 Easter Egg:** Hold the trigger and press the right flipper button during Holodeck selection to play poker with Riker! <kcite ref="80"/>

## 🏆 Wizard Modes

### Final Frontier (Main Wizard Mode)

**Requirements:** Complete all 7 missions <kcite ref="72"/>

**Gameplay:**
- **6-ball multiball** (balls launch from cannons)
- Every major shot (except Command Decision) lit for jackpot
- Jackpot value based on **total artifacts collected**

**Scoring Formula:**

| Artifacts | Starting Bonus       | Per-Shot Jackpot |
| --------- | -------------------- | ---------------- |
| 4         | 400M + **1 Billion** | 100M             |
| 8         | 800M + **2 Billion** | 200M             |
| 12        | 1.2B + **3 Billion** | 250M (max)       |

- **100M per artifact** at start
- **+1 Billion for every 4 artifacts**
- Jackpot = **25M × artifacts** (max 250M/shot) <kcite ref="72"/>

> **🔥 Pro Strategy:** With 8+ artifacts, Final Frontier starts with **2+ Billion** before you even make a shot! Focus on artifact collection throughout the game <kcite ref="97"/>.

**Final Frontier Tips:**
- Go for center shots when 5-6 balls are in play (chaos management)
- Switch to orbit/ramp shots when down to 2-3 balls (precision scoring)
- Highest FF scores come from **extended 2-3 ball play**, not initial chaos <kcite ref="105"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                                   | Danger                               | Mitigation                                                      |
| -------------------------------------- | ------------------------------------ | --------------------------------------------------------------- |
| **Left Ramp (from upper flipper)**     | Miss = straight down left outlane    | Flip early if you must miss <kcite ref="105"/>                  |
| **Right Ramp (late flip)**             | Bounces off left post → left outlane | Flip late if you must miss                                      |
| **Right Orbit (for locks)**            | Exit through bumpers → right outlane | Nudge RIGHT when ball hits top of right sling <kcite ref="90"/> |
| **Neutral Zone shots**                 | Missed cannon shots → right outlane  | Trap and wait out Romulan/Cardassian                            |
| **Start Mission (from right flipper)** | Harder angle, inconsistent           | Prefer left flipper for this shot <kcite ref="105"/>            |

### ✅ Safe Shots

| Shot                               | Why It's Safe                   |
| ---------------------------------- | ------------------------------- |
| **Right Ramp (from left flipper)** | Clean feed back to left flipper |
| **Left Orbit → Left Ramp combo**   | Controlled ball movement        |
| **Center Ramp**                    | Returns to flippers predictably |
| **Spinner (partial orbit)**        | Even failed shots stay in play  |

### Ball Control Positions

- **Left flipper trap:** Best for Start Mission, right ramp
- **Right flipper trap:** Best for left orbit → left ramp combo
- **Dead pass:** Let ball bounce from one flipper to trap on other

### Nudging Strategy

> **💡 Pro Tip:** STTNG punishes missed shots more than most tables. Use nudges liberally! <kcite ref="105"/>

- **Right nudge** when ball exits bumpers toward right outlane
- **Left nudge** when ball heads toward left outlane
- Nudge **early** when ball is high on playfield to redirect before it gains speed
- Keep ball **inside** the angled guides above bumpers <kcite ref="108"/>

## 💰 Scoring Strategy

### Path to 1 Billion Points

#### Early Game (Ball 1)
1. Choose **Warp Factor 4** on plunge
2. Shoot **left ramp** to advance toward Warp 9
3. Complete **Warp 5** (holds multipliers) and **Warp 8** (artifact)
4. Start first mission via Start Mission hole
5. Focus on **artifact collection** in every mission

#### Mid Game (Balls 2-3)
1. Continue mission completion toward Final Frontier
2. **Lock balls** for Borg Multiball when convenient
3. Complete **Shuttle Simulation** for easy artifact + 150M
4. Build **bonus multiplier to 10x** (top lanes)
5. Advance **Rank** via Command Decision

#### Late Game (Final Balls)
1. **Stack Q's Challenge + Borg Multiball** for massive points
2. Trigger **Final Frontier** with maximum artifacts
3. Exploit **late-game Borg Multiball** (jackpots now 100M-300M+)
4. Each triple jackpot = **300M-900M!** <kcite ref="71"/>

### Two Paths to Billions

| Strategy                | Focus                                     | Potential                                |
| ----------------------- | ----------------------------------------- | ---------------------------------------- |
| **Final Frontier Path** | Complete all missions, maximize artifacts | 4-7 Billion per FF <kcite ref="105"/>    |
| **Borg Multiball Path** | Ignore missions, spam multiball           | 160+ Billion possible <kcite ref="110"/> |

> **🔥 For 1 Billion:** Either path works. For **maximum scores**, pure Borg Multiball grinding with late-game billion-point jackpots is mathematically superior, but less fun <kcite ref="110"/>.

### Highest Value Opportunities

| Opportunity                     | Value              | Difficulty              |
| ------------------------------- | ------------------ | ----------------------- |
| Final Frontier (8+ artifacts)   | 2B+ starting bonus | Hard                    |
| Late-game Triple Jackpot        | 300M-900M          | Medium                  |
| Q's Challenge + Multiball stack | 1B+                | Medium                  |
| Shuttle Simulation              | 159M + artifact    | Easy (memorize pattern) |
| Factor 9 Mode complete          | 285M total         | Medium                  |
| Battle Simulation Level 6       | 100M + EB          | Hard                    |

| ------------------ | ---------------------------------- |

| Shuttle Simulation | Collect in cavern (before Special) |

> **💡 Pro Tip:** Get Shuttle Simulation **before** reaching 450M Special. After Special, the simulation EB stops appearing <kcite ref="105"/>.

### Command Decision Awards

Shoot the small sinkhole next to left ramp to choose <kcite ref="116"/>:
- **Rerun a completed mission** (operator adjustable: 1x, never, or unlimited)
- **Advance Rank** (when lit)
- Random awards

### Mystery/Random Awards

Available through:
- Launch Probe skill shot
- Flipper Skill Shot
- Command Decision

Possible awards: Extra Ball, Light Extra Ball, Special, 10M, Hold Multipliers, Double Spinner, Advance Rank, Warp Factor X

### Setup Recommendations

| Setting       | Recommendation  | Reason                  |
| ------------- | --------------- | ----------------------- |
| Pitch         | 6.5° - 7.0°     | Standard for widebody   |
| Outlane posts | Middle position | Balanced difficulty     |
| Ball saver    | 10-15 seconds   | Fair for learning       |
| Mode reruns   | 1 or Unlimited  | Allows artifact farming |
| Extra balls   | ON              | More play time          |

### Tournament Settings

- Ball saver ("Return to Duty") disabled
- "Light Lock" skill shot replaced with "Bonus Multiplier"
- Ferengi ball-add window: 8 seconds only
- No mode reruns allowed <kcite ref="43"/>
