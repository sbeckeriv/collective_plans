## 📋 Quick-Start Checklist for New Players

| Step | Action                                                                   | Why                                              |
| ---- | ------------------------------------------------------------------------ | ------------------------------------------------ |
| 1️⃣    | Select **Paladin (Ardor)** as your first character                       | Easier Dragon Multiball completion               |
| 2️⃣    | Choose **Arabel** or **Kobold Town** as starting location                | Best early modes                                 |
| 3️⃣    | Ignore blue travel shots — hit **center spinner** and **orange targets** | Start Dungeon Crawl ASAP                         |
| 4️⃣    | Spell **DRAGON** at the 3-bank targets                                   | Qualify Dragon Multiball                         |

## 🎯 Core Mechanics

### Skill Shots

| Skill Shot           | How to Execute                              | Value            |
| -------------------- | ------------------------------------------- | ---------------- |
| **Standard**         | Short plunge → Upper loop                   | 2M × ball number |
| **Cube Shot**        | Short plunge → Gelatinous cube switch       | 2M × ball number |
| **Super Skill Shot** | Left eject → Side ramp → Right ramp         | 6M+              |
| **MXV Skill Shot**   | Hit hidden temple lane behind upper flipper | 6M+              |

> 💡 **Pro Tip:** The super skill shot is risky but worth attempting on ball 1 for the extra points and early momentum.

### Class Selection

Select your character with the **right flipper** and starting location with the **left flipper** at game start.

| Character            | Class              | Perk                      | Best For         |
| -------------------- | ------------------ | ------------------------- | ---------------- |
| **Ardor**            | Dragonborn Paladin | Fewer hits to defeat Rath | 🌟 New players    |
| **Anya**             | Halfling Rogue     | Better treasure/luck      | Treasure hunting |
| **Frey Curlyfoot**   | Dwarf Cleric       | Bonus damage to undead    | Campaign modes   |
| **Helenir Weavhart** | High Elf Wizard    | Longer shield timer       | Ball saves       |

**Unlockable Characters:**
- **Rhyme** (Ranger) — 2 ranged arrows at once
- **Juniper Amari** (Bard) — Increased 2x scoring timer
- **Goregath** (Barbarian) — Rage ability (collects all lit shots once per ball)
- **Rowan Freewander** (Fighter) — Requires topper connection

### Battle System

During battles, **red shots** deal damage based on your party's D20 dice rolls.

| Action            | How                                                      | Effect                                    |
| ----------------- | -------------------------------------------------------- | ----------------------------------------- |
| **Attack**        | Shoot red arrows                                         | Party rolls dice for damage               |
| **Ranged Attack** | Hold action button, release when blue light is on target | Spots the shot without shooting it        |
| **Critical Hit**  | Hit captive ball twice quickly                           | Spots a damage shot + advances multiballs |
| **Heal**          | Shoot green shots (when available)                       | Uses healing potion                       |

> 💡 **Pro Tip:** Don't risk dangerous shots during battles! Use ranged attacks or critical hits to spot difficult shots safely.

### Travel Mode & Location Modes

1. **Travel Mode** — Hit 3 blue shots (or drain) to reach your destination
2. **In Town** — Shoot yellow "choose action" shots to start modes
3. **Complete Level 1 + Level 2 modes** of the same color → Unlocks Level 3 mode
4. **Complete all town modes** → Unlocks that town's **Town Dungeon**

## 🎱 Multiballs

### 🔥 Dragon Multiball

**How to Qualify:**
1. Hit the 3-bank standup targets under Rath to spell **DRAGON**
2. Shoot **right orbit** or **side ramp** (when no mode running) to start

**Gameplay (6-8 ball multiball):**
- Unlimited ball save until you win or lose
- Shoot 3-bank → Hit Rath's head to deal damage
- Jackpots at flashing shots: 1x → 2x → 3x value
- Super jackpot at upper loop after collecting all 1x jackpots
- **Win condition:** Hit Rath enough times before too many balls drain

| Dragon Level | Drains Allowed | Hits Required | Ardor Hits |
| ------------ | -------------- | ------------- | ---------- |
| 1            | 16             | 8             | 6          |
| 2            | 14             | 10            | 8          |
| 3            | 12             | 12            | 10         |
| 4            | 10             | 14            | 12         |
| 5            | 8              | 16            | 14         |

**Town Celebration Multiball:**
After defeating Rath, remaining balls continue with victory lap jackpots worth 500k × jackpots collected during Dragon MB.

> 💡 **Pro Tip:** After completing Dragon Multiball, your next Fizmo purchase is **50% off**!

### 🟢 Gelatinous Cube Multiball

**How to Qualify:**
1. Shoot **left orbit** to lower the ramp
2. Shoot the **left ramp** (1-4 times depending on difficulty setting)
3. Ball locks at the cube magnet
4. Optional: Lock additional balls at right orbit and left ramp for bigger multiball

**Gameplay:**
- Hurry-up value starts at 5M, decreases over time
- Lock more balls = +5M to hurry-up value
- Jackpots worth hurry-up value + 3M per jackpot (max 25M)
- Super jackpot at upper loop worth 20M after collecting all jackpots

> ⚠️ **Important:** Only ONE Gelatinous Cube Multiball per game unless you complete a map mode to requalify it!

> 💡 **Pro Tip:** Stack Cube MB with a mode so it requalifies immediately after the mode ends.

### 🍺 Tavern Brawl (Mini-Wizard Mode)

**Requirements:**
- ✅ Start Dragon Multiball
- ✅ Start Gelatinous Cube Multiball  
- ✅ Start a Dungeon Crawl

**Start:** Shoot center spinner when no other mode is running

**Gameplay (2-ball multiball):**
- All switches score 10k
- Jackpots at lit shots: Start at 4M + 200k per jackpot
- Hit 3-bank to spell DRAGON → Lights add-a-ball at shortcut loop
- Shoot center spinner 3× → Enables looping jackpots at left ramp (1x, 2x, 3x)
- Super jackpot worth 3× normal jackpot

> 🎯 **THIS IS YOUR PRIMARY SCORING MODE FOR 1 BILLION!**

> 💡 **Pro Tip:** Trap the ball against the shield, hold on right flipper, and repeatedly shoot the left ramp for massive looping jackpot points!

## 🏰 Dungeon Crawls

**How to Start:**
1. Complete orange dungeon targets OR shoot center spinner to spot them
2. When lit, shoot center spinner again (Pro) or hit the pop-up scoop (Premium/LE)

**Gameplay:**
- 20-second ball save at each level start
- **Orange shots** = Move forward
- **Purple shots** = Choose direction at intersections
- Encounters: Battles, traps (end run), treasure rooms, stairs down

**Dungeon Levels:**
| Level | Reward                               |
| ----- | ------------------------------------ |
| 1     | Extra Ball in Award Room             |
| 2-5   | Legendary Equipment in Treasure Room |

**Legendary Equipment (10 uses each):**
- 🗡️ **Glass Sword of Unmaking** — One-shots non-boss enemies
- 🏹 **Glass Bow of All-Shot** — Ranged attack hits ALL shots
- 🛡️ **Glass Armor of Immortality** — Negates next enemy damage
- ⚔️ **Glass Shield of Endurance** — Shield stays up 2× longer

> 💡 **Pro Tip:** Dungeon maps reset every Sunday and are the same for all players that week.

## 📈 Progression Systems

### Character Leveling

- Gain XP from battles, treasure, completing modes
- Party members match your character's level
- Max level: 20
- Higher levels = harder Dragon Multiball qualification but higher starting Dragon level

### Equipment Priority

```
Legendary (Dungeon Crawl) > Artifacts (Level 3 modes) > Regular Equipment
```

### Artifacts (from Level 3 Modes)

| Artifact                   | Source                    | Effect                      |
| -------------------------- | ------------------------- | --------------------------- |
| **Dragon Heart Shield**    | Prison Break (Orange 3)   | Store up to 5 shield uses   |
| **Mercy the Broadsword**   | Graveyard (Blue 3)        | 2× damage + healing on hits |
| **Oculus Bow of True Aim** | Slaad Assassin (Purple 3) | Ranged covers 2 shots       |
| **Lifeblood Armor**        | Acquire Armor (White 3)   | +2 AC + reflects damage     |

### Insider Connected Features

- Campaign progress saves per character
- Character level/items persist across games
- "One-shot" mode: Use saved character without saving progress
- Legendary equipment does NOT save between games

## 🔄 Stacking & Combos

### ✅ What Stacks Well

| Combination                    | Why                                             |
| ------------------------------ | ----------------------------------------------- |
| **Mode + Dragon Multiball**    | Complete mode objectives during multiball chaos |
| **Mode + Gelatinous Cube MB**  | Requalifies Cube MB when mode ends              |
| **2× Scoring + Any Multiball** | Double all jackpot values                       |
| **Tavern Brawl + Add-a-ball**  | More balls = more looping jackpots              |

### ❌ What NOT to Stack

| Avoid                             | Reason                                         |
| --------------------------------- | ---------------------------------------------- |
| **Town Dungeons + Multiballs**    | Town dungeons block multiball starts           |
| **Multiple modes simultaneously** | Only one mode runs at a time                   |
| **Dungeon Crawl + Tavern Brawl**  | Must exit dungeon before starting Tavern Brawl |

### Optimal Timing

1. **Early game:** Focus on Dungeon Crawl first for foundation score and legendary equipment
2. **Mid game:** Qualify Dragon MB and Cube MB while playing modes
3. **Late game:** Stack 2× scoring with Tavern Brawl for maximum points

## 🧙 Wizard Modes

### 🎲 Tiny's Dice Game (Mini-Wizard)

**Requirement:** Complete 3 modes (location modes or town dungeons)

**Gameplay:**
- Timed mode at center spinner
- Spinner value: 25k × dice number (1-20)
- Hit other shots to add time and increase dice number
- At dice = 20: Spinner worth 500k per spin!

**Premium/LE Difference:** Pop-up scoop increases spinner value instead of the spinner itself.

### 🏆 Main Wizard Modes (Character Unlocks)

#### Ritual of the Chosen (Unlocks Bard)
**Requirement:** Complete The Lich of Dragonspear Castle → Choose "Go after the Scepter"

**Gameplay:** 5 dragon battles (White → Green → Blue → Black → Red)
- Each phase: 15-second timer with ball save
- Lock balls at green shots to extend timer
- Multiball phase: Hit 5 red damage shots per dragon

#### Rescue the King (Unlocks Barbarian)
**Requirement:** Complete Battle of Gauntlgrym → Choose "Stay and help Nighdaar"

**Gameplay:** 7-phase fight incorporating all prior mode mechanics
- Stealth missions, battles, hybrid phases
- Final battle against Balinor

#### The Undermountain (Unlocks Ranger)
**Requirement:** 
- Lich of Dragonspear: "Stay and fight Sammaster" → Defeat him, OR
- Battle of Gauntlgrym: "Help King Raddok" → Defeat Balinor

**Gameplay:** 8-phase journey to Xanathar's lair
- Navigate Yawning Portal, Skullport, Mezzoloth ambushes
- Final battle against Xanathar with special attacks (flipper reversal, darkness, etc.)

## 🎯 Shot Analysis

### Flipper Coverage

| Flipper           | Shots Available                                                                      |
| ----------------- | ------------------------------------------------------------------------------------ |
| **Left Flipper**  | Right orbit, Right ramp, Shortcut loop, Center spinner, Critical hit captive ball    |
| **Right Flipper** | Fizmo/left eject, Left orbit, Critical hit, Center spinner, Shortcut loop (backhand) |
| **Upper Flipper** | Upper loop, Side ramp, Critical hit, Center spinner, Dungeon pop-up                  |

### ⚠️ Dangerous Shots

| Shot               | Risk                                 | Mitigation                                 |
| ------------------ | ------------------------------------ | ------------------------------------------ |
| **Left orbit**     | Tough angle, can reject to outlane   | Backhand from left flipper or shield-stage |
| **Left ramp**      | Only from right flipper, steep angle | Use shield to slow ball first              |
| **3-bank targets** | Ball can bounce unpredictably        | Shoot from left flipper for better control |
| **Right orbit**    | Fast return to left flipper          | Be ready for quick catch                   |

### ✅ Safe Shots

| Shot                 | Why Safe                               |
| -------------------- | -------------------------------------- |
| **Center spinner**   | Returns to right flipper               |
| **Shortcut loop**    | Feeds upper flipper with magnet assist |
| **Fizmo/left eject** | Controlled kickout                     |
| **Right ramp**       | Clean feed to left flipper             |

### Shield Staging Technique

1. Press action button to raise shield
2. Ball slows against shield
3. Flip from the side the ball is resting against
4. **Left side of shield** → Right ramp shot
5. **Right side of shield** → Left orbit shot

## 💰 Scoring Strategy: Path to 1 Billion

### Scoring Benchmarks

| Score | Level     | What It Takes                                |
| ----- | --------- | -------------------------------------------- |
| 50M   | Beginner  | Complete travel mode + 1 mode                |
| 200M  | Good      | Dragon MB + 1 mode                           |
| 500M  | Excellent | Multiple multiballs + Tavern Brawl           |
| 1B+   | Expert    | Repeated Tavern Brawls with optimal stacking |

### The 1 Billion Strategy

#### Phase 1: Foundation (Ball 1)
1. **Immediately start Dungeon Crawl** — ignore travel mode blue shots
2. Find legendary equipment if possible (levels 2-5)
3. Spell DRAGON at 3-bank
4. Progress Gelatinous Cube (left orbit → left ramp)

#### Phase 2: Multiball Qualification (Ball 2)
1. Start a simple mode (Retrieve Potion for Nighdaar is quick)
2. Stack Dragon Multiball with the mode
3. Complete Dragon MB for 50% off Fizmo discount
4. Start Gelatinous Cube MB (stack with another mode to requalify)

#### Phase 3: Tavern Brawl Farming (Ball 3+)
1. **Light Tavern Brawl** (all three requirements met)
2. Start 2× Scoring (secret door → hidden temple)
3. **Start Tavern Brawl during 2× Scoring**
4. Get add-a-ball (complete DRAGON → shortcut loop)
5. **Farm looping jackpots** at left ramp (spinner 3× → ramp repeatedly)
6. Repeat Tavern Brawl via Fizmo purchase (1250 gold)

#### Key Scoring Opportunities

| Feature                              | Potential Score              |
| ------------------------------------ | ---------------------------- |
| Tavern Brawl (with looping jackpots) | 100M+ per run                |
| Dragon Multiball + Town Celebration  | 30-50M                       |
| Gelatinous Cube Multiball            | 20-40M                       |
| Darkhold Bonus Round                 | 100M+ (unlimited ball save!) |
| Dungeon Crawl (deep run)             | 20-50M                       |

> 🎯 **The Secret:** Tavern Brawl can be purchased from Fizmo for 1250 gold, allowing unlimited runs. Stack with 2× scoring and farm looping jackpots for massive points!
