# 🎲 Monopoly - Strategy Guide

*Stern Pinball, 2001 — Designed by Pat Lawlor*

> **"Pass Go. Buy properties. Build your empire."**

Monopoly was the first Stern title from Pat Lawlor Design, and it deliberately avoided the "hit-the-flashing-shots" multiball formula common at the time in favor of a deeper, board-game-driven ruleset. Move around the board, buy properties, build houses and hotels, and eventually collect every property to unlock the wizard mode. Our goal: advance around the board efficiently via the Railroad ramp, build multiball jackpots through the left ramp, and collect every property for the timed building-frenzy finale.

---

## 🎯 Quick-Start Checklist for New Players

- [ ] **Shoot the Railroad ramp to advance quickly** — it moves you forward up to **8 board spaces** and can be shot comfortably from either lower flipper, making it your fastest movement tool
- [ ] **Lock 2 balls via the center ramp, then shoot the right ramp** to start multiball — the autoplunger kicks it off with a cool animation sequence
- [ ] **During multiball, shoot the steep left ramp from the upper flipper** to score a jackpot
- [ ] **Relight the jackpot by shooting the Railroad ramp**, then go for a **double jackpot**
- [ ] **Collect every property on the board** to unlock the wizard mode — a timed multiball where you build as many houses and hotels as possible
- [ ] **This game deliberately avoids easy continuous multiball** — expect a deeper, more deliberate ruleset than typical early-2000s Stern titles

## 🎩 Core Mechanics

### Moving Around the Board

The **Railroad ramp** is your primary movement tool, advancing you **up to 8 spaces** with a comfortable shot from either lower flipper — use it liberally to progress around the board quickly and land on new properties to buy.

### Multiball

1. **Lock 2 balls** by shooting up the **center ramp**
2. Shoot the **right ramp** to trigger the **autoplunger** and start multiball
3. During multiball, shoot the **steep left ramp from the upper flipper** for a **jackpot**
4. **Relight the jackpot** via the **Railroad ramp**, then chase a **double jackpot**

> 🎯 **Pro Tip — Core Multiball Loop:** Once in multiball, the repeatable cycle of left-ramp jackpot → Railroad-ramp relight → double jackpot is your main scoring engine. Master this loop rather than chasing random shots.

### Wizard Mode

**Requirement:** Collect every property on the Monopoly board.

- A **timed multiball**
- Goal: **build as many houses and hotels as possible** before time runs out

> 💰 This is a genuine full-game achievement — collecting every single property requires sustained board-movement progress across multiple balls, not a single lucky run.

## ⚠️ Dangerous Shots & Drain Patterns

| Risk | Mitigation |
|---|---|
| **Steep left ramp (upper flipper)** | A genuinely difficult shot given its steep angle | Only attempt when you have clean control from the upper flipper |
| **Rotating flipper** | A unique mechanical feature requiring its own learning curve | Practice timing with the rotating flipper before relying on it for precision shots |
| **6 pop bumpers** | A notably bumper-heavy layout for this era | Treat extended bumper time as incidental rather than something to force |

## 💰 Scoring Strategy

### Early Game (Ball 1)

1. Use the Railroad ramp aggressively to advance around the board quickly
2. Start buying properties as you land on them
3. Begin building toward your first multiball via center ramp locks

**Target Score:** Build steadily through board progression and early multiball

### Mid Game (Ball 2)

1. Run multiball using the left-ramp jackpot / Railroad-ramp relight loop
2. Continue collecting properties, working toward full board ownership
3. Build houses and hotels on properties where you've completed a color group (monopoly)

### Late Game (Ball 3+)

1. Push to finish collecting every remaining property
2. If you've collected the full board, run the wizard mode and build as many houses/hotels as possible within the time limit
3. Keep cycling the multiball jackpot loop whenever it's active

## 📋 Quick Reference: Strategic Priorities

**Ball 1:** Advance around the board via Railroad ramp, start buying properties.
**Ball 2:** Run the multiball jackpot loop, keep collecting properties and building houses.
**Ball 3+:** Finish collecting the full board, run the wizard mode's timed building frenzy.

**Target High-Score Ballpark:** A strong game centers on completing multiple multiball jackpot cycles and steady property collection — reaching the full-board wizard mode represents genuine mastery.
