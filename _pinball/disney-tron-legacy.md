## 🎱 Core Mechanics

### Skill Shots

TRON offers three skill shot options at ball launch <kcite ref="21"/>:

| Skill Shot                   | How to Execute                                                | Base Value            | Bonus                                           |
| ---------------------------- | ------------------------------------------------------------- | --------------------- | ----------------------------------------------- |
| **Scoop Skill Shot** 🏆       | Soft plunge to bounce ball into Flynn's Arcade scoop          | 500K (+50K each time) | Collects Mystery Award                          |
| **Left Ramp Skill Shot**     | Soft plunge, ball rolls to right flipper → shoot left ramp    | 250K (+50K each time) | Collects Light Cycle insert + starts Find Flynn |
| **Upper Flipper Skill Shot** | Hold left flipper + full plunge → upper flipper to right ramp | 300K (+50K each time) | Collects Light Cycle insert                     |

> 💡 **Pro Tip:** The scoop skill shot is the most valuable because it also awards a mystery. Aim for this first, use the left ramp as backup. <kcite ref="22"/>

### The Progress Ladder

The central ladder of nine inserts is your roadmap to the wizard mode <kcite ref="21"/>:

| Insert          | Feature               | How to Flash                      | How to Light Solid                |
| --------------- | --------------------- | --------------------------------- | --------------------------------- |
| **Flynn**       | Find Flynn            | Roll over Zen switch              | Shoot the roving red arrow        |
| **Gem**         | Follow Gem            | Shoot Gem shot 3 times            | Collect one Gem award during mode |
| **Clu**         | Terminate Clu         | Complete C-L-U inlanes + scoop    | Collect all 4 Clu helmet shots    |
| **Zuse**        | Zuse Fast Scoring     | Complete Z-U-S-E targets          | Complete Z-U-S-E once during mode |
| **Quorra**      | Quorra Multiball      | Shoot left spinner 5x + scoop     | Collect one Super Jackpot         |
| **Disc**        | Disc Multiball        | Lower Recognizer + 7 disc shots   | Collect one Super Jackpot         |
| **Light Cycle** | Light Cycle Multiball | Shoot 4 Light Cycle shots + scoop | Collect Double Super Jackpot      |
| **Recognizer**  | Progress to Disc      | Hit Recognizer bank once          | Lower the Recognizer bank         |
| **TRON**        | TRON Modes            | Complete T-R-O-N once             | Start all 3 TRON modes            |

> ⚡ **Key Insight:** Flashing = started, Solid = completed. This is opposite of some other Stern games! <kcite ref="21"/>

### Mode Objectives & Scoring

#### 🔵 Find Flynn
- **Start:** Roll over Zen switch (right of scoop)
- **Objective:** Shoot the roving red arrow
- **Scoring:** 275K (+50K each time)
- **Tips:** Arrow moves left-to-right across major shots; not timed <kcite ref="21"/>

#### 💚 Follow Gem
- **Start:** 3 shots to Gem (right inner loop), then 5, then 7...
- **Duration:** 25 seconds (extendable)
- **Objective:** Keep shooting Gem shot
- **Scoring:** 750K first shot, +250K each subsequent (max 2.5M)
- **Timer Extension:** Either spinner adds 10 seconds (max 40 sec)
- **Tips:** Best combined with multiballs; Gem shot is HARD <kcite ref="21"/><kcite ref="22"/>

#### 🔴 Terminate Clu
- **Start:** Complete C-L-U inlanes → shoot scoop
- **Duration:** 25 seconds
- **Objective:** Shoot 4 Clu helmet shots (left orbit, left inner loop, right orbit, scoop)
- **Scoring:** 250K first shot, 500K for remaining three
- **Shortcut:** Re-completing C-L-U spots the leftmost unlit Clu shot <kcite ref="21"/>

#### ⚪ Zuse Fast Scoring
- **Start:** Complete Z-U-S-E standup targets
- **Duration:** 25 seconds (extendable)
- **Objective:** Hit switches for points; complete Z-U-S-E to boost value
- **Scoring:** All switches worth 10K (+1K per Z-U-S-E target)
- **Timer Extension:** Complete all 4 Z-U-S-E targets = +10 seconds + reset
- **Tips:** Great during multiball! Runs through other modes <kcite ref="21"/>

#### 🟡 TRON Modes (Double Scoring / Super Pops / Super Spinners)
- **Start:** Complete T-R-O-N targets OR use Zen + 1 target hit
- **Duration:** 30 seconds each
- **Modes:**
  - **Double Scoring** 🏆 — Doubles ALL scoring
  - **Super Pops** — Triples pop bumper value
  - **Super Spinners** — Triples spinner value

> 💡 **Pro Tip:** Bank Zen shots! Roll over Zen multiple times without hitting T-R-O-N, then one target hit starts multiple modes in sequence. <kcite ref="21"/>

## 🎰 Multiballs

### ⚡ Light Cycle Multiball

| Aspect            | Details                                          |
| ----------------- | ------------------------------------------------ |
| **Qualification** | Shoot 4 flashing Light Cycle shots (6 available) |
| **Start**         | Flynn's Arcade scoop                             |
| **Balls**         | 2                                                |
| **Add-a-Ball**    | ❌ None available                                 |

**Jackpot Structure:**
- All Light Cycle shots lit for 350K jackpots
- **Super Jackpot:** Left ramp → Left inner loop (Quorra) = 750K
- **Double Super Jackpot:** Left ramp → Quorra → Right ramp = 1.5M <kcite ref="21"/>

> 💡 **Pro Tip:** During ball save, deliberately drain balls so they're kicked to upper flipper for End of Line combos! <kcite ref="22"/>

---

### 💚 Quorra Multiball

| Aspect            | Details                                   |
| ----------------- | ----------------------------------------- |
| **Qualification** | 5 shots to left spinner (Quorra shot)     |
| **Start**         | Flynn's Arcade scoop                      |
| **Balls**         | 2                                         |
| **Add-a-Ball**    | ✅ 2 available (3 hits to Recognizer each) |

**Jackpot Structure:**
- Quorra shot (left spinner): 250K (+50K each, max 500K)
- Each jackpot adds to Super Jackpot value
- **Super Jackpot:** Gem shot (right inner loop) = 1M + accumulated jackpots <kcite ref="21"/>

> 💡 **Pro Tip:** Quorra's add-a-balls make this the most sustainable multiball. Prioritize getting them during ball save! <kcite ref="22"/>

---

### 🔵 Disc Multiball

| Aspect            | Details                                              |
| ----------------- | ---------------------------------------------------- |
| **Qualification** | 4 hits to Recognizer bank → 7 shots to spinning disc |
| **Start**         | Automatic when qualified                             |
| **Balls**         | 3                                                    |
| **Add-a-Ball**    | ❌ None, but RESTART available                        |

**Jackpot Structure:**

**Stage 1 - Collect 10 Discs:**
- All blue disc shots lit for 250K (+25K each, max varies)
- Shooting disc relights all shots but only scores 100K
- Disc shot also increases next multiball's jackpot base

**Stage 2 - Double Jackpots:**
- Recognizer rises; 3 shots = 500K each

**Stage 3 - Super Jackpot:**
- Shoot spinning disc = Sum of all jackpots collected

**Restart Rule:** If you drain without collecting Super Jackpot, one shot to disc restarts multiball! <kcite ref="21"/><kcite ref="22"/>

> ⚠️ **Important:** Disc Multiball CANNOT stack with Light Cycle or Quorra! <kcite ref="21"/>

## 📈 Progression Systems

### Ladder Completion States

| State        | Meaning        | Visual            |
| ------------ | -------------- | ----------------- |
| **Off**      | Not started    | Dark insert       |
| **Flashing** | Started/played | Blinking insert   |
| **Solid**    | Completed      | Steady lit insert |

### Progress Persistence
- Ladder progress carries across balls
- After Portal wizard mode, all inserts reset to off
- Bonus multiplier resets each ball, but bonus base carries over <kcite ref="21"/>

### Bonus Multiplier
- Each Gem shot adds +1X to bonus multiplier
- Multiplier resets each ball
- Can be doubled with Double Scoring active at drain <kcite ref="21"/>

## 🧙 Wizard Modes

### 🌊 Sea of Simulation (Mini-Wizard)

**Requirements:** All 9 ladder inserts at least FLASHING <kcite ref="21"/>

**Start:** Flynn's Arcade scoop

**Gameplay:**
- Single-ball, untimed mode
- Complete any flashing (incomplete) features to make them solid
- For each SOLID insert at start, receive bonus:
  - Flynn = 1M, Gem = 2M, Clu = 3M... up to TRON = 9M
- Chain continues until hitting a flashing insert

**Maximum Payout:** 45M (if all solid) + Double Scoring = 90M! <kcite ref="22"/>

> 💡 **Pro Tip:** Having lower ladder inserts solid is MORE valuable because the chain pays out sequentially! <kcite ref="22"/>

---

### 🌀 Portal Multiball (Main Wizard Mode)

**Requirements:** All 9 ladder inserts SOLID <kcite ref="21"/>

**Start:** Flynn's Arcade scoop

**Instant Award:** 1M + Sea of Simulation bonuses (if not already played)

**Stage 1 - Collect Jackpots:**
- 4-ball multiball
- Each shot has multiple inserts (Light Cycle, Disc, Arrow, sometimes Clu)
- Shoot flashing inserts for jackpots (500K base, +50K each)
- Collecting all inserts at a shot adds a ball
- 21 total shots to collect
- **Super Jackpot:** Spinning disc = 1M + sum of all jackpots

**Stage 2 - Victory Lap:**
- All red arrow shots worth 1M each
- Spinning disc worth 1M
- Double Scoring still available! <kcite ref="21"/>

> ⚠️ **Note:** Portal is extremely difficult to complete. The Gem shots alone require 3 successful hits! <kcite ref="22"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot                | Danger Level | Why                                                         |
| ------------------- | ------------ | ----------------------------------------------------------- |
| **Right Orbit**     | 🔴 HIGH       | Often SDTM if doesn't hit bumpers cleanly <kcite ref="87"/> |
| **Left Orbit**      | 🟠 MEDIUM     | Can go SDTM if misses bumpers                               |
| **Recognizer Bank** | 🟠 MEDIUM     | Center shot; rebounds unpredictably                         |
| **Gem Shot**        | 🟠 MEDIUM     | Hard to hit; bad misses drain                               |

### ✅ Safe Shots

| Shot                      | Safety Level | Notes                                  |
| ------------------------- | ------------ | -------------------------------------- |
| **Left Ramp**             | 🟢 SAFE       | Loopable; feeds right flipper          |
| **Quorra (Left Spinner)** | 🟢 SAFE       | Feeds upper flipper; good control      |
| **Flynn's Arcade Scoop**  | 🟢 SAFE       | Bailout shot from left flipper         |
| **Right Ramp**            | 🟢 SAFE       | From upper flipper; feeds left flipper |

### Flipper Selection Guide

| Ball Position          | Best Shots                           | Avoid              |
| ---------------------- | ------------------------------------ | ------------------ |
| **Right Flipper**      | Left ramp (loop it!), Quorra spinner | Right orbit        |
| **Left Flipper**       | Scoop, Recognizer (carefully)        | Left orbit (risky) |
| **Upper Left Flipper** | Right ramp 🏆, Gem                    | —                  |

> 💡 **Pro Tip:** Catch balls from the scoop kickout by holding the right flipper up. <kcite ref="87"/>

## 💰 Scoring Strategy

### 🏆 The Secret Weapon: End of Line Jackpot

**This is the #1 scoring opportunity in TRON!** <kcite ref="22"/>

| Aspect             | Details                         |
| ------------------ | ------------------------------- |
| **How to Collect** | Side ramp → Scoop (immediately) |
| **Base Value**     | 500K                            |
| **Growth**         | +value for every combo made     |
| **Resets?**        | NO! Value carries through ball  |

> 🔥 **Critical:** End of Line is almost ALWAYS worth more than anything else. After ANY side ramp shot, forget everything else and shoot the scoop! <kcite ref="22"/>

### Path to 1 Billion Points

#### Early Game (0-50M)
1. ✅ Collect scoop skill shot + mystery
2. ✅ Start building End of Line value with combos
3. ✅ Light Quorra Multiball (left spinner x5)
4. ✅ Light Light Cycle Multiball (4 shots)
5. ✅ Start Double Scoring before multiball

#### Mid Game (50-200M)
1. ✅ Stack Quorra + Light Cycle multiballs
2. ✅ Keep Double Scoring active
3. ✅ Collect End of Line jackpots repeatedly
4. ✅ Use add-a-balls at Recognizer
5. ✅ Complete ladder inserts toward Sea of Simulation

#### Late Game (200M+)
1. ✅ Reach Sea of Simulation with maximum solid inserts
2. ✅ Have Double Scoring active for Sea of Simulation entry (doubles all bonuses!)
3. ✅ Complete Portal Multiball
4. ✅ Continue End of Line farming
5. ✅ Repeat the cycle!

### Scoring Priorities Table

| Priority | Feature                   | Potential Value      |
| -------- | ------------------------- | -------------------- |
| 1️⃣        | End of Line Jackpot       | 500K-5M+ per collect |
| 2️⃣        | Sea of Simulation Entry   | 45M-90M (with 2X)    |
| 3️⃣        | Disc Multiball Super      | 2.5M-20M+ (with 2X)  |
| 4️⃣        | Portal Multiball          | 50M+                 |
| 5️⃣        | Stacked Multiballs + Zuse | Variable, very high  |

## 📋 Additional Information

### Mystery Awards (Flynn's Arcade)

Light mystery by shooting right orbit; collect at scoop <kcite ref="21"/>

| Award               | Effect                    |
| ------------------- | ------------------------- |
| 500K                | Instant points            |
| Advance Recognizer  | -1 hit needed on bank     |
| Complete Recognizer | Instantly lowers bank     |
| Advance Quorra      | +1 Quorra shot            |
| Start Quorra        | Lights Quorra MB at scoop |
| Advance Clu         | Spots 1 C-L-U letter      |
| Advance Gem         | +1 Gem shot               |
| Start Gem           | Starts Follow Gem mode    |
| Light Special       | Rare; percentaged         |
| Light Extra Ball    | Very rare; percentaged    |

### Extra Ball Opportunities

Extra balls are difficult to earn in TRON <kcite ref="65"/>:
- Mystery award (rare, percentaged)
- LE only: Complete DAFT PUNK on left ramp
- Operator setting: Can set score-based extra ball

> ⚠️ **Tournament Warning:** If extra balls are ON and you have a multiball lit at the scoop, ask the TD to drain your extra ball for you! Plunging may accidentally start the multiball. <kcite ref="22"/>

### Setup Recommendations

| Setting           | Recommendation                                                                 |
| ----------------- | ------------------------------------------------------------------------------ |
| **Pitch**         | 6.5° (factory standard) <kcite ref="51"/>                                      |
| **Level**         | Ensure proper left-to-right level to prevent Gem→SDTM drains <kcite ref="65"/> |
| **Insult Level**  | HIGH (more speech!) <kcite ref="65"/>                                          |
| **Extra Ball**    | Set to 15M score-based for home play <kcite ref="65"/>                         |
| **Outlane Posts** | Factory (hard) or adjust to preference                                         |

### LE-Only Features

The Limited Edition includes <kcite ref="22"/>:
- **End of Line Multiball (Daft Punk Multiball):** Spell DAFT PUNK on left ramp twice
- Moving Recognizer
- Drop targets instead of standups for T-R-O-N
- Fiber optic light tubes on ramps
- Additional chrome hardware

### Easter Eggs & Fun Facts

- **Zen Award:** Scores exactly 42K points (Hitchhiker's Guide reference!) <kcite ref="21"/>
- **Music:** Features 3 Daft Punk tracks: Tron Legacy (End Titles), C.L.U., and Derezzed <kcite ref="21"/>
- **Display Timers:** Three bars at bottom-left show TRON mode timers; vertical bars show Clu/Gem/Zuse timers <kcite ref="22"/>

## ⚡ Quick-Start Checklist

### Before You Play
- [ ] Check pitch is ~6.5°
- [ ] Verify machine is level left-to-right
- [ ] Note if extra balls are enabled (tournament consideration)

### Ball 1 Goals
- [ ] Soft plunge for scoop skill shot
- [ ] Start building End of Line value
- [ ] Shoot Quorra spinner 5x
- [ ] Collect 4 Light Cycle shots
- [ ] Start Double Scoring

### Every Ball Goals
- [ ] Prioritize End of Line combos (side ramp → scoop)
- [ ] Stack multiballs when possible
- [ ] Keep Double Scoring active during big scoring
- [ ] Work toward solid ladder inserts
- [ ] Don't tilt! Bonus can be huge

### The Golden Rule
> 🏆 **After ANY side ramp shot, immediately shoot the scoop for End of Line!**
