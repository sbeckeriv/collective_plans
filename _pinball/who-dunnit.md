## 🎯 Core Mechanics

### Skill Shots

| Shot                 | Award                        | Notes                        |
| -------------------- | ---------------------------- | ---------------------------- |
| **T**                | 10M + 10M per repeat         | Increases each time scored   |
| **A**                | Next clue + lights phone     | ⭐ Best for progression       |
| **X**                | +1x Bonus Multiplier         | ⭐ Best for scoring           |
| **I**                | Lights Roulette              | Skip if roulette already lit |
| **Super Skill Shot** | 20M + clue + 1x bonus + lock | 🏆 **BEST OPTION**            |

> **💡 Pro Tip:** The Super Skill Shot requires a soft plunge into the left saucer. If you drain immediately after (without flipping), the ball returns to the plunger for another attempt! Repeat 3x for instant Basement Multiball <kcite ref="26"/>.

**Skill Shot Selection:** Press left flipper to change which TAXI letter is lit <kcite ref="26"/>.

### Elevator System

The elevator is the central mechanic. Shoot the **UP ramp** (left) or **DOWN ramp** (right) to change floors, then shoot **EXIT** to collect the floor award <kcite ref="28"/>.

| Floor             | Award                         | Priority |
| ----------------- | ----------------------------- | -------- |
| **B** (Basement)  | 🎱 Basement Multiball          | ⭐⭐⭐      |
| **M** (Main)      | Lights Spin shots + Roulette  | ⭐        |
| **2**             | Clue + lights phone           | ⭐⭐       |
| **3**             | Evidence OR Who Dunnit        | ⭐⭐       |
| **4**             | Spin Slots                    | ⭐        |
| **5**             | Evidence OR Who Dunnit        | ⭐⭐       |
| **6**             | Equipment (Revolver priority) | ⭐⭐       |
| **7**             | 🎱 Elevator Madness (2-ball)   | ⭐⭐       |
| **8**             | Interrogate OR Who Dunnit     | ⭐⭐       |
| **P** (Penthouse) | 🎉 Penthouse Party             | ⭐⭐⭐      |

> **⚠️ Important:** After visiting all 10 floors, the next EXIT shot goes directly to the **Roof** to catch the killer <kcite ref="28"/>.

**3-Bank Target:** Blocks elevator after collecting an award. Complete it to re-enable elevator access. It auto-lowers when starting Roof Multiball or collecting Second Chance <kcite ref="28"/>.

### Solving Cases

There are **5 cases** in order: Missing Bullet → Broken Mirror → Bloody Dagger → Sticky Fingers → The Tattoo <kcite ref="23"/>.

**Steps to Solve:**
1. **Collect Clues** - Spell TAXI or use Taxi Chase
2. **Interrogate** - Shoot lit phone after collecting a clue
3. **Who Dunnit?** - Select the killer at the phone
4. **Catch the Killer** - Chase on the Roof → Roof Multiball

#### 🔍 Identifying the Guilty Suspect

**Listen to the first interrogation carefully!** The guilty suspect is determined by specific dialogue:

| Dialogue Contains    | Guilty Suspect       | Notes                     |
| -------------------- | -------------------- | ------------------------- |
| **"Victoria"**       | Victoria the Dancer  | Most common               |
| **"Tony"**           | Tony the Crooked Cop |                           |
| **"Trixie"**         | Trixie the Palmist   |                           |
| **"Fingers"**        | Fingers the Gambler  |                           |
| **"Mona"**           | Mona the Lover       |                           |
| **"Biff"**           | Biff the Boxer       |                           |
| **"Walter" / "Tex"** | Keep interrogating   | ⚠️ These are never guilty! |

> **💡 Pro Tip:** The guilty suspect's name is **always mentioned in the first interrogation** unless they mention "Walter" (Butler) or "Tex" (Bruno), who are red herrings. If you hear Walter or Tex mentioned first, keep interrogating until you hear one of the actual suspects <kcite ref="27"/>.

**Example:** If the detective says "I spoke with Victoria..." → Victoria is guilty!

## 🎱 Multiballs

### Basement Multiball (3-Ball) ⭐⭐⭐

**How to Start:**
- Lock 3 balls at left saucer (Super Skill Shot helps!)
- Spin "Multiball" on slots
- Exit on Basement floor <kcite ref="29"/>

**Jackpot Shots:**
- Elevator ramps (all 3 for one jackpot, then disabled)
- Spinner (repeatable)
- Sewer (repeatable)
- Mystery target (if lit)

**Jackpot Value:**
- Starts at **75M**
- +1M per bumper hit (or **+5M with Revolver**)
- Maximum: **500M** <kcite ref="29"/>

> **🏆 Strategy:** This is your **primary scoring source**! Keep shooting balls into bumpers to max the jackpot, then collect repeatedly at spinner/sewer <kcite ref="73"/>.

---

### Roof Multiball (4-Ball)

**How to Start:**
1. Correctly identify the killer at Who Dunnit
2. Chase on Roof: Hit lit orbit (alternates every 12 sec)
3. Shoot phone during 100M hurry-up
4. Hurry-up value becomes jackpot for all shots <kcite ref="27"/>

**Jackpot:** 25M-100M per shot (locked at hurry-up value when caught)

> **💡 Pro Tip:** Don't rush to catch the killer! Collect all 5 clues first for the Extra Ball, then solve the case <kcite ref="73"/>.

---

### Elevator Madness (2-Ball)

**How to Start:** Exit on Floor 7 or spin "?" on slots <kcite ref="28"/>

**Scoring:** Each elevator shot scores 10M-35M based on floor. UP/DOWN ramps change floor and value <kcite ref="24"/>.

> **⚠️ Warning:** After ~20 seconds, the 3-bank target starts cycling up/down, blocking shots <kcite ref="24"/>.

## 💰 Scoring Strategy

### Target: 1 Billion Points

**Phase 1: Opening (0-100M)**
1. Master Super Skill Shot (3x for instant Basement MB)
2. Collect 3 clues + 3 locks + 3x bonus multiplier
3. Play Basement Multiball, build jackpot in bumpers

**Phase 2: Building (100M-500M)**
1. Max clues (5) for Extra Ball
2. Solve case → Roof Multiball
3. Access Penthouse Party (~750M potential) <kcite ref="23"/>
4. Train Combos for equipment

**Phase 3: Scoring (500M-1B+)**
1. Repeat case cycle with Extra Balls
2. Stack Penthouse Party with Roof Multiball
3. Build end-of-ball bonus (can reach 1B+) <kcite ref="23"/>

**Phase 4: Risk/Reward (1B+)**
1. Consider Roulette above 2.5B
2. Max bet can double your score
3. Tournament mode: wins on even attempts (2,4,6...) <kcite ref="31"/>

### Highest Value Opportunities

| Feature                  | Potential    | Difficulty |
| ------------------------ | ------------ | ---------- |
| **Basement MB Jackpots** | 500M each    | Medium     |
| **Penthouse Party**      | 750M-1B      | Easy       |
| **Roof Multiball**       | 100M × shots | Medium     |
| **Roulette**             | Double score | High risk  |
| **End-of-Ball Bonus**    | 1B+          | Long game  |
