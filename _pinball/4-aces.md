# 🃏 4 Aces (Williams, 1970) — Complete Strategy Guide

## 🔧 Unique Features

### Stop Magnets
When the ball hits certain hanging targets that are lit for bonus, an electromagnet catches and stops the ball completely, holds it while bonus points are scored, then releases it. This was an innovative feature for 1970!

### Flipper Sequencing
Pressing the flipper buttons triggers the **upper flippers first**, then the lower ones. This requires different timing than modern games.

### Ball Launch
Player presses a button on the lockdown bar to shoot the ball into play **from between the flippers** — not a traditional plunger lane.

## 🎯 Core Scoring Mechanics

### Building Bonus

> 💡 **Pro Tip:** Bonus is your primary scoring opportunity on this game!

| Method                    | Action                               |
| ------------------------- | ------------------------------------ |
| **Spinning Disk**         | Spin the center spinning disk        |
| **White Standup Targets** | Hit the white targets along the back |

### Collecting Bonus

The bonus collection system uses a **color-matching mechanic**:

| To Collect...   | Hit These Targets       | Then Hit...               |
| --------------- | ----------------------- | ------------------------- |
| **Black Bonus** | Spade + Club targets    | White target between them |
| **Red Bonus**   | Heart + Diamond targets | White target between them |

### Card Advance (Carryover Feature)

1. Hit **both A and B targets** to light the near outlanes for **card advance**
2. Card positions are shown on the backglass
3. **Carryover feature:** Card progress carries from game to game!
4. Lighting all playing cards on the backglass awards **Special** (free game)

> 💡 **Pro Tip:** In multiplayer games, players can "steal" each other's bonus — this was an original feature of the game!

## ⚠️ Shot Analysis

### 🔴 Dangerous Areas

| Hazard             | Location         | Why It's Dangerous                |
| ------------------ | ---------------- | --------------------------------- |
| **Dual Outlanes**  | Both sides       | 4 total drain paths instead of 2! |
| **Spinning Posts** | Playfield        | Unpredictable ball deflection     |
| **Wide Body**      | Entire playfield | More area = harder to control     |

### 🟢 Safer Shots

| Shot                     | Benefit                                         |
| ------------------------ | ----------------------------------------------- |
| **Center spinning disk** | Builds bonus, relatively safe                   |
| **Upper flipper shots**  | More control before ball reaches lower flippers |

### 🎯 Flipper Tips

- Remember: **Upper flippers fire first** when you press the buttons
- Use upper flippers to redirect ball before it reaches the more dangerous lower playfield
- The 4-flipper layout gives you more save opportunities

## 🏆 Scoring Strategy

### Priority Order

| Priority | Action                                                        | Why                    |
| -------- | ------------------------------------------------------------- | ---------------------- |
| 🥇        | Build bonus via spinning disk and white standup targets       | Primary scoring method |
| 🥈        | Collect bonus by qualifying and hitting correct color targets | Cashes in your bonus   |
| 🥉        | Work on card advance for carryover Special award              | Long-term goal         |
| ⚠️        | Avoid the dual outlanes                                       | This game drains fast! |

### Early Game
- Focus on learning ball behavior off spinning posts
- Practice upper flipper timing
- Build small bonuses and collect them

### Mid Game
- Maximize bonus building before collection
- Work on A+B targets for card advance
- Develop consistent shots to white targets

### Late Game
- Go for maximum bonus collection
- If card advance is close, prioritize completing it
- Play conservatively to avoid dual outlanes

## 🎲 Fun Facts

- **Designer Steve Kordek** invented the dual flipper layout used in almost all pinball machines
- **Stop magnets** were cutting-edge technology in 1970
- **Widebody cabinet** makes this game play differently than standard-width machines
- **Card carryover** means your progress persists between games — unusual for EM era!
