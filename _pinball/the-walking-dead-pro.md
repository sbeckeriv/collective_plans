# 🧟 The Walking Dead Pro - Ultimate Strategy Guide
## Goal: 1 Billion Points (and Beyond!)

> **💀 Fair Warning:** TWD is notorious for its brutality. The layout punishes mistakes severely, and the flipper gap is wider than most machines. On a mean copy, 50M is commendable. On a friendly one, 250M+ is achievable. The world record involves rolling the score TWICE (9+ billion)! <kcite ref="56"/>

## 🎯 Core Mechanics

### Skill Shots

Plunge into the flashing top lane (cycle with flippers). Hit rollovers on the way up for bonus points <kcite ref="23"/>.

| Skill Shot # | Base Value | Rollover Bonus (each) | Maximum Total |
| ------------ | ---------- | --------------------- | ------------- |
| 1st          | 250K       | 150K                  | 550K          |
| 2nd          | 750K       | 200K                  | 1,150K        |
| 3rd          | 1,250K     | 250K                  | 1,750K        |
| 4th          | 1,750K     | 300K                  | 2,350K        |

> **💡 Pro Tip:** Missing the lit lane doesn't reset the base value - it stays at 250K until you hit it. Only successful skill shots advance the value.

### Walker Kills

White crosshair inserts indicate walker kill shots. Each kill awards ~250K and counts toward various features <kcite ref="56"/>.

- **Shot Multiplier:** Every 2 kills lights/advances the X on inlanes
- **Bicycle Girl:** Lit after 5 kills, then every 20 kills
- **Extra Ball:** Lit at Woodbury after killing enough walkers
- **Last Man Standing:** Lit after 115 total kills

## 🎮 Dead Features (Modes)

### How to Start Modes

1. Clear the drop target bank (1 completion for first mode, 2 for subsequent) <kcite ref="29"/>
2. Shoot any flashing mode shot to start that mode
3. Modes CAN be lit and started during multiball
4. Modes CANNOT be stacked with each other

> **💡 Pro Tip:** Clearing drops during any mode adds 10 seconds to the timer. Shooting a previously-completed mode shot increases current mode value.

### Mode Breakdown

| Mode         | Start Shot  | Objective                               | Multi-Kill Requirement |
| ------------ | ----------- | --------------------------------------- | ---------------------- |
| 🏚️ **Barn**   | Left Orbit  | Loop left orbit repeatedly              | 3 shots                |
| 🔬 **CDC**    | Left Ramp   | Complex multi-phase mode                | 3 shots                |
| ⚔️ **Riot**   | Inner Loop  | Shoot Riot, value builds on switch hits | 2 shots                |
| 🚇 **Tunnel** | Right Orbit | All shots lit, tunnel doubles value     | 2 shots                |
| 🏟️ **Arena**  | Right Ramp  | Loop right ramp for escalating values   | 3 shots                |

---

### 🔬 CDC Mode (Best Single-Ball Mode!)

**Phase 1:**
- Loop left ramp for 750K (+25K per shot)
- Each ramp lights another shot (left to right)
- Shooting any lit shot advances to Phase 2 <kcite ref="29"/>

**Phase 2:**
- All lit shots (except left ramp) score the sum of Phase 1 points
- Hit all lit shots to relight them
- First left ramp shot = Phase 2 sum value
- Second left ramp shot = ENTIRE Phase 2 total (ends mode)

> **💡 Pro Tip:** Don't finish CDC early! Loop the left ramp until you miss, then shoot everything else, then cash out with the left ramp. Use your shot multiplier on the final cash-out shot for massive points! <kcite ref="56"/>

---

### ⚔️ Riot Mode (Best Multiball Mode!)

- Shoot Riot shot repeatedly for points
- Value increases by 75K per switch hit (not per shot!)
- Perfect for multiball stacking - switch hits from chaos build value
- One player scored 255M on a single Riot shot with 2X playfield + shot X! <kcite ref="45"/>

---

### 🚇 Tunnel Mode

**Phase 1:**
- All shots lit for 5M
- Shooting Tunnel first adds 2.5M to all shot values
- Shooting any non-Tunnel shot locks in value, advances to Phase 2

**Phase 2:**
- All shots lit except Tunnel
- Collect all shots to relight them
- Shooting Tunnel ends the mode

> **⚠️ Warning:** Don't accidentally shoot Tunnel during Phase 2!

---

### 🏟️ Arena Mode

- Loop right ramp for escalating values (3M base, +3M/6M/9M...)
- Breaking the combo halves the next shot value
- Woodbury briefly lights after each ramp to end mode and collect ~20% bonus <kcite ref="29"/>

## 🎱 Multiballs

### Stacking Rules

| Stack                    | Allowed?               |
| ------------------------ | ---------------------- |
| Blood Bath + Prison      | ✅ Yes                  |
| Blood Bath + Well Walker | ✅ Yes                  |
| Prison + Well Walker     | ❌ No                   |
| Any MB + Mode            | ✅ Yes (modes continue) |

> **💡 Pro Tip:** You cannot progress toward Prison while Well Walker is running, and vice versa <kcite ref="24"/>.

---

### 🏛️ Prison Multiball

**Starting:**
1. Bash prison doors to spell P-R-I-S-O-N (P-R-I-S-O spotted by default)
2. Hit prison again to engage magnet and start "clearing the yard"
3. Shoot all 5 major shots + prison to re-spell PRISON
4. Hit prison one more time to start multiball <kcite ref="25"/>

**During Multiball (3 balls):**

| Phase            | Objective                                         | Reward                      |
| ---------------- | ------------------------------------------------- | --------------------------- |
| Phase 1          | Hit major shots, then prison to relight           | ~1M jackpots                |
| Phase 1 Complete | 6 jackpots collected                              | Super Jackpot at Riot       |
| Phase 2          | Ramps lit for 500K (+50K each), collect at Prison | Jackpots build toward Siege |
| Siege Qualified  | 12 total jackpots (across all Prison MBs)         | Lights Siege wizard mode    |

> **💡 Pro Tip:** Progress carries over between Prison Multiballs! If you got 5 jackpots last time, you only need 1 more for the super.

---

### 🪣 Well Walker Multiball

**Starting:**
1. Hit Well Walker to spell W-E-L-L
2. Hit Well Walker 5 more times while letters strobe (55K-75K per hit based on letters lit)
3. Multiball begins <kcite ref="26"/>

**During Multiball (2 balls):**

| Phase         | Objective                                   | Details                         |
| ------------- | ------------------------------------------- | ------------------------------- |
| Phase 1       | Switch hits light W-E-L-L letters           | W:25, E:30, L:35, L:40 switches |
| Jackpots      | Hit Well Walker when letter complete        | Value = switches × 25K          |
| Super Jackpot | Complete all WELL letters                   | Sum of all jackpots collected   |
| Phase 2       | Ramps build jackpot, collect at Well Walker | 500K base (+50K per ramp)       |

> **💡 Pro Tip:** The Super Jackpot can be MASSIVE. Save your shot multiplier for this! On Premium/LE, the spinner helps rack up switch hits quickly <kcite ref="26"/>.

> **🎮 Advanced Tech:** If you're one hit from starting Well Walker MB and about to drain, hip-check the machine! The Well Walker is sensitive enough to register hits from a firm nudge. This is legal per IFPA rules <kcite ref="56"/>.

---

### 🩸 Blood Bath Multiball

**Starting:**
1. Collect all 3 supplies (First Aid, Weapons, Food) by clearing drops
2. Supplies rove while all drops are up - time your shot!
3. Clear drops one more time to start Blood Bath <kcite ref="27"/>

**During Multiball (2 balls):**

| Feature       | Value                                                 |
| ------------- | ----------------------------------------------------- |
| Switch hits   | 15K each (increases by 2.5K per bank clear)           |
| Jackpot       | Accumulated switch value, collected at drops          |
| Add-a-balls   | 2 available from drop clears                          |
| Super Jackpot | Collect all 3 supplies again = UNLIMITED add-a-balls! |

> **💡 Pro Tip:** Blood Bath is the ONLY multiball with add-a-balls. Stack it with Prison or Well Walker for maximum chaos and points! <kcite ref="56"/>

## 📈 Progression Systems

### Shot Multiplier (X)

- Every 2 walker kills lights X on inlanes
- Control which inlane is lit with flippers
- Rolling over lit inlane multiplies NEXT shot
- If already lit, value increases (+1X per 2 kills)
- No apparent ceiling! <kcite ref="34"/>

> **💡 Pro Tip:** DON'T waste your multiplier on small shots! Build it up to 4X-6X and use it on a CDC cash-out, Well Walker Super, or big Riot shot. One multiplied shot can be worth more than an entire ball of normal play <kcite ref="56"/>.

---

### 2X Playfield

1. Complete stab targets (beside Prison) to light 2X
2. Shoot Tower standup (upper right) to start 20-second 2X scoring
3. Completing stab targets again extends timer by 10 seconds <kcite ref="30"/>

> **💡 Pro Tip:** 2X playfield STACKS with shot multiplier! A 3X shot during 2X playfield = 6X total! <kcite ref="56"/>

---

### Multi-Kills

Yellow inserts at mode shots indicate Multi-Kill availability. Value = 1/20th of your total mode points <kcite ref="39"/>.

**How to Light Multi-Kills:**

| Objective      | Requirement                |
| -------------- | -------------------------- |
| Barn Mode      | 3 shots                    |
| CDC Mode       | 3 shots                    |
| Riot Mode      | 2 shots                    |
| Tunnel Mode    | 2 shots                    |
| Arena Mode     | 3 shots                    |
| Prison MB      | 4 jackpots                 |
| Well Walker MB | 3 jackpots                 |
| Blood Bath MB  | 3 jackpots                 |
| Bicycle Girl   | 3 shots                    |
| Fish Tanks     | Collect 3                  |
| Drop Targets   | 10 bank clears             |
| Combos         | 10                         |
| Crossbow       | 3 shots                    |
| Woodbury       | Collect "Start Multi-Kill" |

> **💡 Pro Tip:** Multi-Kill value carries over until you drain. Multiple Multi-Kills stack their values together. Light all 6 weapon inserts to qualify Horde! <kcite ref="39"/>

## 🧙 Wizard Modes

### 🧟‍♂️ Horde (Mini-Wizard)

**Qualification:** Light all 6 Multi-Kill weapon inserts

**Gameplay:**
- 2-ball multiball at Woodbury
- Walkers advance toward you at 5 major shots
- Closer walkers = higher jackpot multiplier (up to 4X)
- Complete wave (3 kills first wave, +1 each subsequent) = Super Jackpot + Add-a-ball
- After 3 waves: Choose FIGHT or ESCAPE at Woodbury <kcite ref="40"/>

> **⚠️ Competition Mode Warning:** If a walker reaches you, YOUR BALL ENDS! In normal play, it just ends Horde <kcite ref="44"/>.

---

### 🏰 Siege (Mini-Wizard)

**Qualification:** Score 12 jackpots during Prison Multiball(s)

**Gameplay:**
- Phase 1: Barn, Prison, Arena lit for jackpots
- Phase 2: All 5 mode shots + Prison lit
- Carries over between balls <kcite ref="41"/>

---

### 🚂 Escape Terminus (Mini-Wizard)

**Qualification:** Play all 5 Dead Features, then relight modes via drops

**Gameplay:**
- All 5 mode shots lit for hurry-ups
- Values based on your mode scores (20% cumulative):
  - Shot 1: 20% of 5th mode
  - Shot 2: 20% of (5th + 4th)
  - Shot 3: 20% of (5th + 4th + 3rd)
  - Shot 4: 20% of (5th + 4th + 3rd + 2nd)
  - Shot 5: 20% of ALL modes combined <kcite ref="42"/>

> **💡 Pro Tip:** Play out your modes fully, especially later ones! A monster 5th mode makes Terminus incredibly valuable.

---

### 🏆 Last Man Standing (Final Wizard Mode)

**Qualification:** Kill 115 walkers total

**Gameplay:**
- 4-ball multiball
- Kill 115 walkers AGAIN (count goes down)
- Shot values ramp up as mode progresses
- Add-a-balls at checkpoints
- Completing it awards 100M bonus + epic ending sequence <kcite ref="43"/>

## 🎯 Shot Analysis

### ⚠️ Dangerous Shots

| Shot            | Danger Level | Risk                              |
| --------------- | ------------ | --------------------------------- |
| Prison (center) | 🔴 HIGH       | Magnet throws balls SDTM          |
| Well Walker     | 🔴 HIGH       | Backhand can send to left outlane |
| Pop Bumper Exit | 🔴 HIGH       | Completely unpredictable feed     |
| Drop Targets    | 🟡 MEDIUM     | Awkward angle, rebounds dangerous |
| Left Orbit      | 🟡 MEDIUM     | Feeds unpredictable pops          |

### ✅ Safer Shots

| Shot               | Safety Level | Notes                          |
| ------------------ | ------------ | ------------------------------ |
| Left Ramp (CDC)    | 🟢 SAFE       | Loopable, good control         |
| Right Ramp (Arena) | 🟢 SAFE       | Loopable, returns to flipper   |
| Riot/Tower         | 🟡 MEDIUM     | Feeds pops but worth it for 2X |

### 🎮 Ball Control Tips

- **Pop Bumper Exit:** Don't try to control immediately - just keep it out of the middle!
- **Flipper Gap:** Wider than normal - be ready for SDTM drains
- **Trap Up:** Always trap when possible on this brutal layout
- **Nudging:** Essential for survival - learn to save from the pops <kcite ref="56"/>

## 💰 Scoring Strategy: Path to 1 Billion

### Early Game (0-100M)

1. **Survive the plunge** - get control ASAP
2. **Clear drops** during ball save to light first mode
3. **Start CDC** if comfortable with left ramp
4. **Build toward Blood Bath** while playing modes
5. **Progress Prison or Well Walker** opportunistically

### Mid Game (100M-500M)

1. **Stack Blood Bath + Prison/Well Walker**
2. **Start Riot mode** during multiball stack
3. **Build shot multiplier** to 4X+
4. **Light 2X playfield** before big scoring opportunities
5. **Cash in multiplied shots** on Riot or CDC finishers

### Late Game (500M-1B+)

1. **Repeat multiball stacks** with modes
2. **Target Horde** via Multi-Kills
3. **Use 2X + shot X** on every major jackpot
4. **Play Escape Terminus** if modes are complete
5. **Chase Last Man Standing** for the 100M completion bonus

## 📝 Additional Information

### Extra Balls

- Lit at Woodbury after killing a percentage of walkers (default ~10%, then 60 more)
- When disabled, awards 2.5M points instead <kcite ref="58"/>

### Woodbury Awards (Mystery)

Lit by completing Fish Tank (3 right ramps, 2 spotted by default). Plunge and select award with flippers:
- Points (1M+)
- Light Bit
- Various small awards
- Value increases with each completion <kcite ref="56"/>

### BIT (Outlane Bonus)

- Draining out a lit outlane collects BIT value
- Starts at 500K, increases 250K per top lane completion <kcite ref="35"/>

### Crossbow Mode

- Lit at right ramp after collecting Weapons supply
- 5 shots lit for 1M+ each
- Hitting Well Walker or certain targets ends mode early
- 3 successful shots starts Crossbow Frenzy (Fibonacci-style scoring!) <kcite ref="37"/><kcite ref="38"/>

- Extra balls award points
