# 🏰 Big Guns Strategy Guide
## Williams, 1987 | "Save the Queen!"

> ⚠️ **Important Note:** Big Guns uses 7-digit displays, making the **maximum displayable score 9,999,999** <kcite ref="21"/>. The goal of "1 billion points" isn't possible on this machine. Instead, this guide focuses on **rolling the score multiple times** and achieving the highest possible totals. Skilled players have reported scores around 45 million (rolling 4+ times) <kcite ref="21"/>.

## 🎯 Skill Shot

### How It Works
The plunger feeds the ball to a raised horseshoe-shaped path. Your target is the center path at the top of the horseshoe <kcite ref="21"/>.

| Result           | Outcome                                               |
| ---------------- | ----------------------------------------------------- |
| ✅ **Hit**        | Awards Tower Bonus + loads Right Cannon automatically |
| ❌ **Overshoot**  | Ball falls toward flippers via Right Cannon Load path |
| ❌ **Undershoot** | Ball returns to plunger (try again!)                  |

### 🏆 Tower Bonus Values

| Successful Skill Shots | Award               |
| ---------------------- | ------------------- |
| 1st                    | 100,000             |
| 2nd                    | 200,000             |
| 3rd                    | 300,000             |
| ...                    | +100K each          |
| **Maximum**            | 900,000 - 1,000,000 |

> 💡 **Pro Tip:** Tower Bonus **carries through Extra Balls and Multiball plunges** on the same ball. It only resets when advancing to ball 2 or 3 <kcite ref="21"/>. Master the skill shot to consistently max this out!

## ⚔️ Core Mechanics

### 🛡️ Palace Guards (10 Targets)

The Guards are your path to safety features and big points.

**Layout:**
- **Left Side:** 5 targets (2 near outlane, 3 below left upper flipper)
- **Right Side:** 5 targets (2 near outlane, 3 below right upper flipper)

**Guard Value Progression:**

| Guards Cleared    | Award       | Bonus                        |
| ----------------- | ----------- | ---------------------------- |
| All 10            | 100,000     | Lights Kickback + Opens Gate |
| All 10 (2nd time) | 150,000     | Relights Kickback + Gate     |
| All 10 (3rd time) | 200,000     | Relights Kickback + Gate     |
| All 10 (4th time) | 250,000     | Relights Kickback + Gate     |
| All 10 (5th time) | **Special** | Relights Kickback + Gate     |

> ⚠️ Guard value **does not carry** from ball to ball <kcite ref="21"/>.

---

### 🧌 Troll Targets & Drop Targets

Each side has a bank of 3 drop targets protecting a Troll target behind them.

**Scoring Mechanism:**
1. Value arrows cycle in front of drop targets: 150K → 150K+200K → 200K → 200K+250K → 250K → 250K+150K
2. Hit a drop target when its lamp is lit to "set up" the Troll for that value
3. A timer begins - hit the Troll before it expires!
4. All drops reset after Troll hit OR timer expiration <kcite ref="21"/>

**Extra Ball Opportunity:**
- Hit **3 Trolls** during the game → lights inlane for Extra Ball
- Use lane change to alternate which inlane is lit <kcite ref="21"/>

---

### 🔄 Bonus Loops (Bonus X Multiplier)

The U-turn loops behind each Troll target build your end-of-ball bonus multiplier.

**How to Build Bonus X:**
1. Pass through a Bonus Loop
2. Timer starts (pulsing faster as it expires)
3. Hit the opposite loop before timer expires
4. Repeat in figure-8 pattern!

| Bonus X                | Special Opportunity                          |
| ---------------------- | -------------------------------------------- |
| 2X - 9X                | One value flashes for Special                |
| Reach flashing X       | Lights Outlane Special for remainder of ball |
| **9X + 99K EOB Bonus** | Special musical salute! <kcite ref="21"/>    |

> 💡 **Pro Tip:** The figure-8 loop pattern is very fast due to the short upper playfield. Master this for rapid Bonus X building!

---

### 🎯 Center Target (Raise Post)

1. Either inlane lights the Center Target
2. Hit Center Target when lit → raises Center Post (~10 seconds)
3. Timer-based - light pulses faster as it expires <kcite ref="21"/>

The Center Post provides crucial ball save protection between the lower flippers!

## 🔫 Cannon System & Ball Locks

### Loading Cannons

| Cannon            | How to Load                           | Notes                          |
| ----------------- | ------------------------------------- | ------------------------------ |
| **Right Cannon**  | Hit Right Cannon Load saucer when lit | Skill shot auto-loads this!    |
| **Left Cannon**   | Hit Left Cannon Load saucer when lit  | Always lit at game start       |
| **Center Cannon** | Hit Center Cannon path                | Only lit after both L+R loaded |

**Lock Sequence:**
1. **Game Start:** Both Left and Right Cannon Load are lit
2. **After 1st Multiball:** Only Left Cannon lit initially
3. Load Left → Right Cannon lights
4. Load Right → Center Cannon lights
5. Load Center → **MULTIBALL!** <kcite ref="21"/>

> ⚠️ **Common Issue:** Aging machines often have erratic locks that fire prematurely. If you experience this, try to load all three cannons quickly before locks fail <kcite ref="21"/>.

## 🎱 Multiball

### Starting Multiball
Load all three cannons (Left, Right, Center) to begin 3-ball Multiball <kcite ref="22"/>.

### 🌟 INVINCIBLE Mode

During Multiball, a "traveling" Load Cannon light moves: Left → Center → Right → repeat

**When you load the lit cannon:**
1. **INVINCIBLE** mode activates (~15 seconds, countdown displayed)
2. Kickback, Gate, and Center Post become **permanent** (won't expire)
3. The 6 center Guards (3 green + 3 red) light up
4. Hit all 6 to score **Queen's Rescue!** <kcite ref="21"/>

### 👑 Queen's Rescue Scoring

| Queen's Rescue # | Award                                             |
| ---------------- | ------------------------------------------------- |
| 1st              | 300,000                                           |
| 2nd+             | Increases gradually during each Multiball session |
| **Maximum**      | 3,000,000                                         |

> 💡 **Pro Tip:** After scoring Queen's Rescue, you can score it **again** in the same Multiball by reloading the lit cannon after the animation finishes! <kcite ref="21"/>

### 2X Scoring
**All scores are doubled during Multiball** (except Queen's Rescue) <kcite ref="21"/>.

## 🏰 King's Chamber (Backbox Mini-Game)

The backbox contains a pachinko-style playfield with pins and 5 slots <kcite ref="22"/>.

### How It Works
1. Every Center Cannon load fires the mini-ball
2. Mini-ball drops through pins into one of 5 slots
3. Corresponding shield lights up
4. Use flippers to "steer" lit shields left/right
5. Light all 5 shields → next King's Chamber shot lights Extra Ball at inlanes <kcite ref="21"/>

### End-of-Game Bonus Shot
After your last ball drains, you get a **free King's Chamber shot**:
- One shield is lit (cannot be steered)
- If mini-ball lands in the lit slot → **Extra Ball!** <kcite ref="21"/>

> 💡 **Pro Tip:** If the mini-ball mechanism fails (common issue), the game awards one shield automatically. Always steer shields to the right when you know it won't fire, so the auto-awarded leftmost shield helps your progress <kcite ref="21"/>.

## 📊 Progression Systems

### Per-Ball Progress (Resets Each Ball)
- Tower Bonus value (resets to 100K)
- Guard value (resets to 100K)
- Bonus X multiplier
- Center Target/Post timer

### Cumulative Progress (Carries Between Balls)
- King's Chamber shields (until Extra Ball collected)
- Troll count toward Extra Ball
- Cannon lock status (until Multiball played)

## 🎯 Shot Analysis

### Playfield Layout
The playfield is **almost completely symmetrical** left-to-right, with two distinct areas <kcite ref="21"/>:

**Upper Area (~1/3 of playfield):**
- Two upper flippers (far apart)
- Drop target banks + Troll targets
- Bonus loops
- Center Target

**Lower Area:**
- Standard lower flippers
- Slingshots
- Guard targets
- Cannon Load saucers

### ✅ Safe Shots

| Shot                   | From                         | Notes                 |
| ---------------------- | ---------------------------- | --------------------- |
| Left/Right Cannon Load | Opposite flipper             | Open shot, consistent |
| Left/Right Cannon Load | Same-side flipper (backhand) | Reliable backhand     |
| Bonus Loops            | Upper flippers               | Short, fast shots     |
| Guard targets          | Lower flippers               | Wide target area      |

### ⚠️ Dangerous Shots

| Shot                       | Risk                                 | Mitigation                              |
| -------------------------- | ------------------------------------ | --------------------------------------- |
| Center Cannon              | Longest shot on playfield            | Forgiving angle, but miss = fast return |
| Missed upper flipper shots | Ball returns to lower playfield fast | Keep Center Post raised                 |
| Outlane drains             | Standard risk                        | Complete Guards for Kickback/Gate       |

### 🎮 Flipper Selection

| Target            | Best Flipper                                          |
| ----------------- | ----------------------------------------------------- |
| Left Cannon Load  | Right lower flipper (direct) or Left lower (backhand) |
| Right Cannon Load | Left lower flipper (direct) or Right lower (backhand) |
| Center Cannon     | Either lower flipper                                  |
| Left Bonus Loop   | Right upper flipper                                   |
| Right Bonus Loop  | Left upper flipper                                    |

## 💰 Scoring Strategy

### 🎯 Path to Maximum Score

Since the display maxes at 9,999,999, your goal is to **roll the score as many times as possible**.

### Early Game Priorities
1. ✅ **Nail the Skill Shot** - Start Tower Bonus at 100K
2. ✅ **Complete Guards** - Get Kickback + Gate protection
3. ✅ **Load Left Cannon** - Begin lock sequence
4. ✅ **Build Bonus X** - Figure-8 the loops

### Mid Game Priorities
1. ✅ **Load Right Cannon** - Continue lock sequence
2. ✅ **Hit Trolls** - Work toward Extra Ball (3 needed)
3. ✅ **Keep Center Post raised** - Hit Center Target after inlanes
4. ✅ **Build King's Chamber shields** - Extra Ball opportunity

### Late Game / Multiball Priorities
1. ✅ **Start Multiball** - Load Center Cannon
2. ✅ **Activate INVINCIBLE** - Load the traveling lit cannon
3. ✅ **Score Queen's Rescue** - Hit all 6 center Guards
4. ✅ **Repeat Queen's Rescue** - Reload lit cannon after animation
5. ✅ **Cycle back to Multiball** - Keep locking balls!

### 🏆 Highest Value Opportunities

| Opportunity                    | Value              | Priority |
| ------------------------------ | ------------------ | -------- |
| Maxed Tower Bonus (Skill Shot) | 900K-1M per plunge | ⭐⭐⭐⭐⭐    |
| Queen's Rescue (maxed)         | Up to 3M           | ⭐⭐⭐⭐⭐    |
| Multiball 2X scoring           | Doubles everything | ⭐⭐⭐⭐     |
| Guard completion (Special)     | Special award      | ⭐⭐⭐      |
| Troll targets                  | 150K-250K+         | ⭐⭐⭐      |
| 9X Bonus + 99K EOB             | Variable           | ⭐⭐       |

> 💡 **Pro Tip:** "Extra Balls are sparse on this game, but Multi-Balls are a piece of cake, and you can get quite a score just off the Tower Bonus alone" <kcite ref="21"/>. Focus on cycling Multiballs and maxing Tower Bonus!

## 🎮 Quick-Start Checklist for New Players

### Before You Play
- [ ] Check ceiling height - this machine has an oversized backbox! <kcite ref="22"/>
- [ ] Verify cannon locks are working properly
- [ ] Note which inlane is lit for Extra Ball

### Ball 1 Strategy
1. [ ] **Plunge for Skill Shot** - aim for center path
2. [ ] **Clear Guards** - get Kickback + Gate
3. [ ] **Load Left Cannon** (if not already loaded via Skill Shot)
4. [ ] **Load Right Cannon**
5. [ ] **Build Bonus X** via loops
6. [ ] **Load Center Cannon** → Start Multiball!

### During Multiball
1. [ ] **Watch for traveling Load Cannon light**
2. [ ] **Load lit cannon** → INVINCIBLE activates
3. [ ] **Hit 6 center Guards** → Queen's Rescue!
4. [ ] **Reload lit cannon** → Score Queen's Rescue again!
5. [ ] **Keep balls in play** - 2X scoring on everything

### Key Callouts to Listen For
- *"You are Invincible! Save the Queen!"* <kcite ref="21"/>

## 🎪 Easter Eggs & Trivia

- 🎨 The backbox is **4 inches taller** than standard Williams machines because artist Python Anghelo needed more room for the castle artwork <kcite ref="22"/>
- 👑 King Tyrant was inspired by designer Mark Ritchie's brother, **Steve Ritchie** (designer of High Speed, F-14 Tomcat, etc.), who also voiced the character <kcite ref="22"/>
- 🔫 This was the **first pinball machine to use catapults**, invented by Mark Ritchie <kcite ref="22"/>
- 🎰 The King's Chamber pachinko concept came from Williams' 1967 game *Apollo* <kcite ref="22"/>
- 💰 A June 1988 software revision increased earnings at some locations by 20% <kcite ref="22"/>
- 🔄 The game awards replays **each time** the replay score is passed - so rolling the score and hitting replay again gives another credit! <kcite ref="21"/>
