# 🌟 Star Wars Trilogy Pinball Strategy Guide
## Sega Pinball, 1997

> ⚠️ **Scoring Note:** This 1997 machine scores in the **millions**, not billions. Top recorded scores are in the 30-60 million range. A score of **50+ million** represents elite play on this machine!

---

## 📋 Quick-Start Checklist for New Players

- [ ] Learn the skill shot timing (watch Vader's head scan)
- [ ] Focus on completing ONE Heroic mode per ball
- [ ] Shoot drop targets to work toward Star Wars Multiball
- [ ] Collect FALCON letters via ramp shots
- [ ] Complete F-O-R-C-E targets for Hurry-ups
- [ ] Work toward the 6 Darth Vader tasks for Wizard Mode

## 🎱 Multiballs

### ⚔️ Star Wars Multiball (3-Ball)

**How to Qualify:**
1. A 4×4 matrix of TIE Fighter lights sits beside the 4 drop targets
2. At game start, a solid row cycles among the 4 drops
3. Hit the lit drop target to destroy that row of TIE Fighters
4. Destroy all 16 TIE Fighters (4 rows × 4 targets)
5. Ramp and Mode Start VUK light for "Load X-Wing"

**How to Start:**
1. Shoot the ramp (or Mode Start VUK) to load the X-Wing
2. Shoot through the gap in the drop targets to launch!

| Phase           | Shot                    | Value                          |
| --------------- | ----------------------- | ------------------------------ |
| 1️⃣ Jackpot       | Ramp                    | 500K + 2K/switch               |
| 2️⃣ F-O-R-C-E     | 5 targets               | Jackpot value each (+10K/drop) |
| 3️⃣ Super Jackpot | Load X-Wing → Shoot gap | 1M+ switch value               |

> 💡 **Pro Tip:** The 'C' and 'E' FORCE targets are the hardest to hit. Aim slightly left to avoid the rubber guard on 'C'. Pop bumper action sometimes hits them for free!

**Multiball Restart:** If you drain to 1 ball without collecting the ramp Jackpot, you get 15 seconds to reload the X-Wing for a 2-ball restart.

---

### 🚀 Falcon Multiball (4-Ball)

**How to Qualify:**
- Shoot the ramp to spell F-A-L-C-O-N (6 shots)
- Cannot be done during Heroic modes
- After first Falcon MB, letters begin timing out

**How to Play:**

| Jackpot # | Switches Needed | Base Value       |
| --------- | --------------- | ---------------- |
| 1st       | 40 switches     | 200K + 2K/switch |
| 2nd       | 60 switches     | 600K + 2K/switch |
| 3rd       | 80 switches     | 800K + 2K/switch |

- First Jackpot adds a 5th ball (if you haven't drained one)
- Ramp collects Jackpots when lit

> 💡 **Pro Tip:** Keep balls in the pop bumpers to rack up switch hits quickly!

---

### 🤖 Probe Droid Multiball (2-Ball)

**How to Start:**
- Complete F-O-R-C-E targets → Shoot Han Hole → Select Probe Droid from Hurry-up options

**How to Play:**
- One drop target is raised at a time
- Shoot it for 200K, then another appears
- Value increases by 50K each hit

## 📊 Progression Systems

### The 6 Darth Vader Tasks

Complete all 6 to unlock the **Wizard Mode**. Progress is shown between the flippers.

| Task                  | How to Complete                          | Difficulty |
| --------------------- | ---------------------------------------- | ---------- |
| 🎯 **Super Jackpot**   | Collect Super Jackpot in Star Wars MB    | ⭐⭐⭐⭐ Hard  |
| ⏱️ **Hurry-up**        | Light, start, and collect any Hurry-up   | ⭐⭐ Medium  |
| 🚀 **Falcon Jackpot**  | Score a Jackpot in Falcon MB             | ⭐⭐ Medium  |
| 🏎️ **Landspeeder**     | Reach first Landspeeder award (4 orbits) | ⭐⭐ Medium  |
| ✅ **Complete Heroic** | Start AND complete any Heroic mode       | ⭐⭐ Medium  |
| 🧊 **Thaw Han**        | 3 shots to Han Hole (not during modes)   | ⭐ Easy     |

> ⚠️ **Critical:** Super Jackpot is BY FAR the hardest task. Prioritize this early when you have fresh focus!

### Landspeeder Progression

| Orbits | Award                   |
| ------ | ----------------------- |
| 1-3    | 50K each                |
| 4      | 300K + Darth Vader task |
| 5-8    | 50K each                |
| 9      | **Extra Ball**          |
| 10-14  | 50K each                |
| 15     | 300K                    |

- Orbits are lit by inlanes or by making an orbit shot

### Thaw Han Solo

| Shots to Han Hole | Result                                     |
| ----------------- | ------------------------------------------ |
| 1-2               | 50K each                                   |
| 3                 | 300K + Thaw Han + Trivia Question          |
| 4+                | Additional Thaws require +1 shot each time |

- Trivia questions test your Star Wars knowledge (32 questions in rotation)
- Left flipper = Answer A, Right flipper = Answer B

## 🔄 Hurry-Ups

Complete F-O-R-C-E targets → Shoot Han Hole to start

| Hurry-Up            | Objective                       | Value                     |
| ------------------- | ------------------------------- | ------------------------- |
| 💥 **Extra Ball**    | Shoot Mode Start in 15 sec      | Extra Ball                |
| 🎰 **Special**       | Shoot Special Lane under X-Wing | Free Game                 |
| 🍺 **Cantina**       | Loop the ramp                   | Countdown from ?K to 100K |
| 🎯 **Bounty Hunter** | Boba Fett points to award       | Usually Video Mode        |
| 🤖 **Probe Droid**   | 2-ball multiball                | See above                 |

### Video Mode (Bounty Hunter)
- Drive Landspeeder, run over Stormtroopers, avoid obstacles
- Pattern identical to Guns N' Roses pinball video mode
- Perfect score (all Stormtroopers) = Extra Ball

## 🔀 Stacking & Combos

### ✅ What Stacks Well

| Combination                  | Why It Works                                            |
| ---------------------------- | ------------------------------------------------------- |
| Heroic Mode + Star Wars MB   | Mode continues through MB; double scoring opportunities |
| Falcon letters + Normal play | Ramp shots always count toward FALCON                   |
| Landspeeder + Everything     | Orbit shots always count                                |

### ❌ What Does NOT Stack

| Combination                  | Why It Doesn't Work                        |
| ---------------------------- | ------------------------------------------ |
| Heroic Mode + FALCON letters | FALCON letters don't register during modes |
| Heroic Mode + Hurry-ups      | Can't start Hurry-ups during modes         |
| Multiple Heroic Modes        | Only one mode at a time                    |

### 🎯 Optimal Stacking Strategy

1. Get close to Star Wars Multiball (15-16 TIE Fighters destroyed)
2. Start a Heroic mode (preferably Leia or C3P0/R2D2)
3. Trigger Multiball during the mode
4. Complete mode objectives while collecting Jackpots

## 🧙 Wizard Mode: The Redemption of Darth Vader

### Requirements
Complete all 6 Darth Vader tasks (see Progression section)

### Gameplay
- **Type:** 2-ball multiball
- **Duration:** 50 seconds
- **Ball Saver:** Active entire time

### Objective
1. Load the X-Wing cannon with one ball
2. Use the other ball to knock down drop targets
3. Fire through the gap you create
4. Each successful shot: **3-4 Million points**

> 💡 **Pro Tip:** You can't sight the shot by looking over the gun—the cannon rotates! Time your shot based on the cannon's position, not visual alignment.

### Strategy
- Keep one ball controlled while the other loads
- Knock down 2-3 drop targets just before firing
- The mode may end after enough successful shots (unconfirmed)

### ⚠️ Dangerous Shots

| Shot             | Danger                             | Mitigation                    |
| ---------------- | ---------------------------------- | ----------------------------- |
| **X-Wing Gap**   | Ball can rattle out to outlanes    | Only shoot when necessary     |
| **Han Hole**     | Deceptively large; balls skip over | Use controlled flipper timing |
| **Right Orbit**  | Can feed SDTM                      | Nudge right on weak shots     |
| **Drop Targets** | Rebounds unpredictable             | Be ready for quick save       |

### ✅ Safe Shots

| Shot                 | Why It's Safe                 | Best Use                    |
| -------------------- | ----------------------------- | --------------------------- |
| **Ramp**             | Clean feed to left flipper    | FALCON, Jackpots, Leia mode |
| **Left Orbit**       | Usually feeds right flipper   | Landspeeder, Chewie mode    |
| **Mode Start VUK**   | Ball held, then fed to inlane | Starting modes, C3P0 mode   |
| **Light Saber Bank** | Low risk, left side           | Building kickback           |

| Mode Start   | Either           | Left is safer         |

## 📈 Path to 50+ Million (Elite Score)

### The Strategy

1. **Ball 1:** Thaw Han + Complete 1 Heroic + Work drops (~5-8M)
2. **Ball 2:** Star Wars MB with Super Jackpot + Hurry-up (~10-15M)
3. **Ball 3:** Falcon MB + Complete remaining tasks (~10-15M)
4. **Wizard Mode:** 4-5 successful shots (~15-20M)
5. **Bonus:** Built multiplier throughout (~2-5M)

### Key Success Factors

> 🎯 **Super Jackpot is everything.** Practice the F-O-R-C-E targets and X-Wing gap shot. This single achievement often separates good games from great ones.

> 🔄 **Stack modes with Multiball.** The extra balls give you more chances to complete mode objectives while also collecting Jackpots.

> 🛡️ **Light Saber Saver is your friend.** Hit the left bank early and often. The virtual kickback saves games.

> 🎱 **Control the ball.** This game rewards patience. Catch, aim, shoot. Don't flip wildly.
