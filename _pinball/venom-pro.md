# 🕷️ Venom Premium Pinball: The Ultimate Strategy Guide

**Goal: 1 Billion Points and Beyond!**

> *"We Are Venom!"* — This guide will transform you from symbiote host to apex predator. Venom is one of Stern's most demanding machines with incredibly fast shots and deep rules that reward strategic planning. <kcite ref="34"/>

## 📋 Quick-Start Checklist for New Players

| Step | Action                                                                                     |
| ---- | ------------------------------------------------------------------------------------------ |
| 1️⃣    | Choose **Eddie Brock** as your first host (starts with 2 locks lit)                        |
| 2️⃣    | Select "Spot Mini-Mode" skill shot with right flipper                                      |
| 3️⃣    | Shoot flashing colored shots (yellow → light blue → white)                                 |
| 4️⃣    | When you hear "BALL 2 LOCKED" — **hold the action button** to delay multiball              |
| 5️⃣    | During Mayhem Multiball, hit Carnage captive ball + center ramp to stack Carnage Multiball |
| 6️⃣    | Use Spider-Sense (hold action button) during multiballs for instant super jackpots         |

## 🎯 Core Mechanics

### Host Selection

At the start of each ball, choose your host with the **left flipper**. Each host plays differently: <kcite ref="21"/><kcite ref="22"/>

| Host               | Starting Locks | Special Ability                                         | Best For                  |
| ------------------ | -------------- | ------------------------------------------------------- | ------------------------- |
| **Eddie Brock**    | 2              | Higher Mayhem MB scoring                                | New players, early game   |
| **Flash Thompson** | 0              | Extra-wide Bell Tower (left loop counts)                | High scoring, multipliers |
| **Gwen Stacy**     | 1              | 2X Infected defeated, higher Doppelganger scoring       | Progression, late game    |
| **Peter Parker**   | 1              | Easier Carnage qualification, higher Carnage MB scoring | Late game, Carnage focus  |

**Venomized Hosts** (unlocked by defeating Knull): <kcite ref="21"/>
- **Venomized Hulk**: HULK SMASH! (action button awards ALL shots once per ball)
- **Venomized Captain America**: Always gets "ALL OF THE ABOVE" in Mystery
- **Venomized Wolverine**: Longer ball savers, 2X Infected defeated

### Skill Shots 🎱

Use the **left flipper** to cycle between 3 skill shots per host, then plunge: <kcite ref="21"/><kcite ref="23"/>

| Host  | Option 1              | Option 2                     | Option 3                        |
| ----- | --------------------- | ---------------------------- | ------------------------------- |
| Eddie | Top Lanes (2 Bonus X) | 180 Ramp (Spot Mini-Mode) ⭐  | Right Orbit (Big Points)        |
| Flash | Top Lanes (2 Bonus X) | Mystery Scoop                | Right Ramp (Spot Mini-Mode) ⭐   |
| Gwen  | Top Lanes (2 Bonus X) | Left Loop (Spot Mini-Mode) ⭐ | Right Loop (Start Doppelganger) |
| Peter | Top Lanes (2 Bonus X) | 180 Ramp (Spot Mini-Mode) ⭐  | Mystery Scoop                   |

> **💡 Pro Tip:** Always go for "Spot Mini-Mode" — it instantly completes the highest mini-mode on your ladder, accelerating your path to multiball. <kcite ref="23"/>

**Secret Hideout Skill Shot:** Drop the ball into the top hole during plunge for **4 Bonus Multipliers**! <kcite ref="21"/>

### Mini-Modes & The Grid

The 12-insert grid between the slingshots tracks your progress. Each host has a column of 3 lights: <kcite ref="21"/><kcite ref="24"/>

**To Complete a Mini-Mode:**
1. Shoot any solid **yellow**, **light blue**, or **white** arrow (untimed first shot)
2. Quickly shoot the blinking second arrow of the same color before timeout
3. Completing all 3 colors finishes your host's ladder

**Grid Bonuses** (complete boxes of 4 adjacent lights): <kcite ref="21"/>

| Bonus                      | Effect                                     |
| -------------------------- | ------------------------------------------ |
| ⏱️ Extra Time               | Longer timers on all modes                 |
| 🕸️ 2X Web Combos            | Each combo counts as 2                     |
| 🛡️ Extra Ball Save          | Longer ball saves                          |
| ✖️ +1X Playfield Multiplier | Base multiplier increased                  |
| 🔥 Flamethrower             | Shoot between 2 lit shots to collect all 3 |
| 📈 2X Experience            | Level up twice as fast                     |

> **💡 Pro Tip:** Completing Eddie + Flash's ladders gives you Extra Time + +1X Playfield — an excellent foundation for high scoring! <kcite ref="24"/>

## 🎰 Multiballs

### Mayhem Multiball (2-6 Balls)

**How to Qualify:** <kcite ref="26"/>
1. Complete mini-modes to light locks (1 mini-mode = 1st lock, 2 = 2nd, 3 = 3rd+)
2. Shoot green arrows to lock balls in left or right tanks
3. After 2nd lock, game tries to start multiball — **HOLD ACTION BUTTON** to delay!

**Lock Benefits:**

| Left Tank                         | Right Tank                          |
| --------------------------------- | ----------------------------------- |
| 1st ball: Agony joins battle      | 1st ball: Riot joins battle         |
| 2nd ball: Lasher joins battle     | 2nd ball: Phage joins battle        |
| 3rd ball: 2X Experience during MB | 3rd ball: +1X Playfield during MB ⭐ |

> **💡 Pro Tip:** Prioritize the right tank's 3rd lock for +1X playfield — this is crucial for high scoring! <kcite ref="26"/>

**Multiball Phases:** <kcite ref="21"/>
1. **Quick Jackpots:** Single ball, alternate left ramp ↔ scoop for quick jackpots
2. **Main Phase:** All balls release, complete all shots for jackpots → Super Jackpot
3. **Escalation:** After Super, shots require 2 hits (Double Jackpots), then 3 hits (Triple), etc.

**Super Jackpot** defeats one minor boss brought into the multiball!

### Carnage Multiball

**How to Start:** <kcite ref="27"/>
1. Hit the Carnage captive ball repeatedly until he's pushed all the way back to Ravencroft
2. Shoot the center ramp (Police Station) when lit

**Scoring:**
- Blue shots build Super Jackpot value
- Push Carnage back to Ravencroft again → shoot scoop for **Super Jackpot**

> **💡 Pro Tip:** Use Spider-Sense during Carnage Multiball — it instantly counts as pushing Carnage all the way back and lights your Super Jackpot! <kcite ref="27"/>

**Carnage Modes Progression:** <kcite ref="21"/>
1. **Carnage Multiball** (1st Carnage completion)
2. **Carnage Hurry-Up** (2nd completion) — shoot center ramp to start, add value with captive ball/ramps
3. **Blood Lust** (3rd completion) — flip-count mode, balance adding flips vs. collecting points

### Quick Doppelganger Multiball (Premium/LE Only) 🎮

**How to Start:** <kcite ref="67"/>
1. Play as **Gwen Stacy** or **Venomized Wolverine**
2. Make **8 shots** to the right loop during single-ball play
3. Ball locks at left loop entrance
4. Quickly shoot the 180 ramp with the new ball for jackpot + multiball start

**Scoring:**
- Hit Doppelganger targets to lock in hurry-up value
- 4 shots light for jackpots worth the locked-in value
- Complete all 4 → Super Jackpot at scoop (4X value)

### Million Plus Pops (Premium/LE Only) 🎮

**How to Activate:** <kcite ref="68"/>
- Play as **Flash Thompson** or **Venomized Hulk**
- Make **8 shots** to the target below the U-turn ramp
- All pop bumper hits worth **1M × Playfield Multiplier** for limited time
- All orbit shots send balls to bumpers during this mode

## 📈 Progression Systems

### Experience Points (XP) & Levels

Everything you do earns XP. Higher levels = more damage in battles. <kcite ref="60"/>

| Level Threshold | Unlocks                                         |
| --------------- | ----------------------------------------------- |
| Level 10        | Minor Boss Battles (Riot, Agony, Phage, Lasher) |
| Level 20        | Grendel (Major Boss)                            |
| Level 30        | Knull (Final Boss)                              |
| Level 99        | Maximum level                                   |

**XP Boosters:**
- 2X Experience grid bonus
- Lock 3 balls in left tank (2X during that Mayhem MB)

> ⚠️ **Warning:** Defeating Knull resets your XP to zero! <kcite ref="57"/>

### Team-Ups

Unlock allies to assist during multiballs and battles: <kcite ref="21"/>

| Team-Up           | Ability                               | How to Unlock                                 |
| ----------------- | ------------------------------------- | --------------------------------------------- |
| **Hybrid**        | Super Bomb + Max 7X Playfield         | Defeat all 4 minor bosses                     |
| **Sleeper**       | Add-a-Ball (MB) / Extra Time (Battle) | Defeat enough Infected, OR choose Flash first |
| **Miles Morales** | +1 Web Combo per combo                | Complete a 4-way combo, OR choose Peter first |

Press the **action button** during mode intros to select your team-up.

### Infected & Doppelganger

**Defeating Infected:** <kcite ref="28"/>
1. Hit any target to start it flashing
2. Hit it again while flashing to defeat an Infected

**Infected Rewards:**
- Triggers Doppelganger attacks (~every 5 defeated)
- Awards Bonus Multipliers
- Unlocks Sleeper
- Lights Extra Ball
- Lights Scream video mode

**Doppelganger Mode:** <kcite ref="28"/>
- Shoot the left loop when flashing red/blue before time runs out
- 1st time: 1 target, 2nd: 2 targets, 3rd: 3 targets
- Defeat 3 Doppelgangers → lights **Rampage**

## 🔄 Stacking & Combos

### What Stacks Well Together ✅

| Stack                            | Why It's Powerful                                                                         |
| -------------------------------- | ----------------------------------------------------------------------------------------- |
| **Mayhem MB + Carnage MB**       | Double the balls, double the chaos! Hit Carnage during Mayhem ball save <kcite ref="35"/> |
| **Mayhem MB + Carnage Hurry-Up** | Cash in hurry-up with 4-7X playfield for 300-500M+ <kcite ref="125"/>                     |
| **Any Multiball + Spider-Sense** | Instant super jackpot qualification <kcite ref="32"/>                                     |
| **Playfield X + Host Hurry-Up**  | 500M-1B+ possible with multipliers active <kcite ref="30"/>                               |
| **Toxin Team-Up + Sleeper**      | Add-a-ball safety net, build shot values with captive ball <kcite ref="125"/>             |

### What NOT to Stack ❌

| Avoid                              | Reason                                                               |
| ---------------------------------- | -------------------------------------------------------------------- |
| Starting Battle when MB is ready   | Battle locks out multiball progress until won/lost <kcite ref="25"/> |
| Ignoring playfield X during stacks | Multipliers are the key to billion-point games                       |
| Using Spider-Sense too early       | Save it for when you're in trouble or need super jackpot             |

### Optimal Timing

1. **Build playfield X first** (center ramp → bell tower)
2. **Start Mayhem MB** with 5-6 balls locked
3. **Stack Carnage MB** during Mayhem ball save
4. **Activate Spider-Sense** to light both super jackpots
5. **Cash in Carnage Hurry-Up** while multiplier is active

## 🧙 Wizard Modes

### Toxin Team-Up (Mini-Wizard Mode)

**Requirements:** <kcite ref="56"/>
- Start Mayhem Multiball
- Start Carnage Multiball  
- Defeat 1 Doppelganger

**Gameplay:**
1. 3 random shots lit for 2 jackpots each (starting 1M, +1M per shot)
2. Hit Carnage for 2X/3X jackpot multiplier (temporary)
3. Complete 6 jackpots → Super Jackpot at scoop (25M base)
4. Collecting Super adds a ball → 2-ball multiball continues
5. Subsequent supers add 25M to value

> **💡 Pro Tip:** Select Sleeper team-up — press action button to add-a-ball even during single-ball phase! <kcite ref="56"/>

### Rampage Mode

**Requirements:** Defeat 3 Doppelgangers <kcite ref="28"/>

**Gameplay:** <kcite ref="21"/>
1. Timed mode — shoot blue shots to build value
2. When time expires, Doppelganger appears
3. Hit Doppelganger repeatedly to collect built-up value

> **💡 Pro Tip:** A great Rampage can be worth 300M+! <kcite ref="28"/>

### Minor Boss Battles (Riot, Agony, Phage, Lasher)

**Requirements:** Complete host's mini-mode ladder + Level 10+ <kcite ref="25"/>

| Boss       | Host  | Mechanic                                                                          |
| ---------- | ----- | --------------------------------------------------------------------------------- |
| **Riot**   | Eddie | Red/blue shots move across playfield; hit both colors together for special attack |
| **Agony**  | Flash | Shoot red shots; prevent green poison from taking over                            |
| **Phage**  | Gwen  | Right orbit → bell tower → 3 center shots for special attack                      |
| **Lasher** | Peter | Track which orbit Lasher jumps to; deal damage through orbits                     |

**Battle Tips:** <kcite ref="21"/>
- Higher level = more damage per hit
- Web combos multiply damage
- Bell tower always adds time
- Special Attack deals massive damage

> **💡 Pro Tip:** Battles aren't worth pursuing for score — focus on multiballs and hurry-ups unless you're at level 20+ and want to progress. <kcite ref="25"/>

### Grendel (Major Boss)

**Requirements:** Defeat 1 minor boss + Level 20+ <kcite ref="57"/>

**Phases 1-2:** Shoot lit shots through 3 color levels (blue → orange → red). If Grendel attacks, flippers reverse — shoot blue shot to fix.

**Phases 3-4:** Shifts to 2-ball multiball. Lock balls at left/center ramps. If a ball drains after ball save, sudden death begins!

### Knull (Final Boss)

**Requirements:** Defeat Grendel + Level 30+ <kcite ref="57"/>

**5 Phases:** Shoot blue (small damage) → yellow (medium) → red (large + special attack)

**Knull's Attacks:**
- 🛡️ **Shield:** All shots turn green, must shoot to remove
- 🌑 **Darkness:** Playfield dims temporarily
- ⚡ **Kill Flipper:** Left flipper dies for a few seconds

**Phase 5:** 3-ball multiball with TIMED FLIPPERS (can't hold flipper >1.75 seconds)!

**Rewards for Defeating Knull:**
- Speed Run clock stops
- Unlock a Venomized host (Hulk, Cap, or Wolverine)
- XP resets to zero

### Klyntar Multiball (Final Wizard Mode)

**Requirements (ALL in one game):** <kcite ref="59"/>
1. ✅ Defeat Knull
2. ✅ Start Rampage
3. ✅ Start Blood Lust
4. ✅ Start Toxin Team-Up
5. ✅ Start Scream video mode
6. ✅ Start Mayhem Multiball
7. ✅ Complete Host Hurry-Up
8. ✅ Start Bell Tower Frenzy

**Gameplay:** <kcite ref="59"/>
- Timed 2-ball multiball with unlimited ball save
- 4 shots lit (left ramp, center ramp, scoop, right orbit)
- Each shot has 4-stage hurry-up; value carries forward
- Playfield dims with each completion
- Bonus for seconds remaining if all 16 shots completed

## 🎯 Shot Analysis

### Dangerous Shots ⚠️

| Shot                     | Danger                                 | Mitigation                                                      |
| ------------------------ | -------------------------------------- | --------------------------------------------------------------- |
| **Carnage Captive Ball** | Very dangerous recovery, ball can SDTM | Only shoot during ball save or when essential <kcite ref="27"/> |
| **Left Loop**            | Fast return, can catch you off-guard   | Nudge and attempt live catch on right flipper <kcite ref="33"/> |
| **Right Orbit**          | Can rattle around and drain            | Use during multiball when safer                                 |

### Safe Shots ✅

| Shot                             | Why It's Safe                                 |
| -------------------------------- | --------------------------------------------- |
| **Center Ramp (Police Station)** | Clean feed to left flipper, builds Bell Tower |
| **Left Ramp (Alchemax)**         | Reliable feed, good for combos                |
| **Scoop (Daily Bugle)**          | Ball held, gives you time to recover          |

### Ball Control Positions

- **Left flipper hold:** Safe position, plan your next shot
- **Right flipper hold:** Good for center ramp or right orbit
- **Post transfer:** Use slingshot posts to move ball between flippers

> **💡 Pro Tip:** This game rewards controlled play. Don't panic-flip — trap up and plan your shots! <kcite ref="33"/>

## 💰 Scoring Strategy: Path to 1 Billion

### Scoring Tiers

| Score    | Skill Level  | Strategy Focus                              |
| -------- | ------------ | ------------------------------------------- |
| 100-300M | Beginner     | Basic multiball, some combos                |
| 300-500M | Intermediate | Stacked multiballs, playfield X             |
| 500M-1B  | Advanced     | Host Hurry-Up with multipliers              |
| 1-3B+    | Expert       | Full stack optimization, multiple hurry-ups |

### The Billion-Point Game Plan <kcite ref="125"/><kcite ref="35"/>

**Early Game (Ball 1):**
1. Start as **Eddie Brock** (2 locks lit)
2. Skill shot: Spot Mini-Mode at left loop
3. Complete Eddie's mini-mode ladder quickly
4. Change to **Flash Thompson** at left ramp
5. Complete Flash's ladder (now have Extra Time + +1X Playfield bonuses)

**Mid Game (Ball 1-2):**
1. Lock 6 balls for Mayhem (hold action button after each lock!)
2. Prioritize right tank 3rd lock for +1X playfield
3. Start Mayhem MB → immediately hit Carnage → stack Carnage MB
4. Build playfield X to 4-7X during multiball
5. Use Spider-Sense to light both super jackpots
6. Cash in Carnage Hurry-Up while multiplier active (300-500M possible!)

**Late Game (Ball 2-3):**
1. Play Toxin Team-Up with Sleeper (add-a-ball safety)
2. Keep multiplier alive with bell tower shots
3. Complete Host Combo → Host Hurry-Up with multiplier (500M-1B!)
4. If time permits: Rampage, Blood Lust, or Boss Battles

### Key Scoring Opportunities

| Feature                  | Potential Score | Requirements                                                    |
| ------------------------ | --------------- | --------------------------------------------------------------- |
| Host Hurry-Up + 7X PF    | 500M - 1B+      | Toxin → Host Combo → Hurry-Up with multiplier <kcite ref="30"/> |
| Carnage Hurry-Up + 4X PF | 300-500M        | Stack with Mayhem, maintain multiplier <kcite ref="125"/>       |
| Rampage                  | 200-300M        | 3 Doppelgangers, good shooting                                  |
| Toxin Team-Up (extended) | 200-500M        | Keep multiball alive, build values <kcite ref="125"/>           |
| Mayhem + Carnage Stack   | 100-200M        | 6-ball Mayhem, stack Carnage                                    |

### Playfield Multiplier Strategy <kcite ref="31"/>

**Building Multiplier:**
1. Loop center ramp repeatedly (2 shots = 1st bell tower, 3 = 2nd, etc.)
2. Shoot bell tower target when lit → +1X (max 7X)
3. Even unlit bell tower adds time to multiplier

**Maintaining Multiplier:**
- Hit bell tower (even unlit) to add time
- Flash/Hulk: Left loop also counts as bell tower!
- Plan your high-value shots while multiplier is active

> **💡 Pro Tip:** The difference between a 500M game and a 2B game is multiplier management. Always know your current X and time remaining! <kcite ref="125"/>

## 📚 Additional Information

### Extra Ball Opportunities <kcite ref="63"/>

- Defeat enough Infected (threshold varies)
- Mystery award (rare, percentage-based)
- Grid bonus completion

### Mystery Awards <kcite ref="21"/>

Complete bottom 4 lanes 3 times → Mystery lit at scoop

**Possible Awards:**
- Big Points, +1-3 Bonus Multipliers
- +1X Playfield Multiplier
- +1000-2000 XP, +2-3 Levels
- Light Locks, Light Extra Ball
- Start Doppelganger, Start Scream, Start Bell Tower Frenzy
- Complete Grid Boxes, Unlock Team-Ups
- Light 2-3 More Mysteries

> **💡 Pro Tip:** In Competition Mode, Mystery awards are fixed sequence: Bonus X → XP → Light 3 Locks → +1X PF → Bell Tower Frenzy → Defeat 5 Infected... <kcite ref="21"/>

### Setup Recommendations

| Setting               | Recommendation      | Reason                                 |
| --------------------- | ------------------- | -------------------------------------- |
| **Pitch**             | 6.5° - 7.0°         | Game plays fast; steeper = harder      |
| **Outlane Posts**     | Standard position   | Adjust based on skill level            |
| **Competition Mode**  | ON for tournaments  | Removes randomness from Mystery        |
| **Insider Connected** | OFF for tournaments | Prevents unfair progression advantages |

### Game Play Modes <kcite ref="21"/>

Access by holding both flippers between games:

| Mode                 | Description                                                  |
| -------------------- | ------------------------------------------------------------ |
| **Cooperation Play** | All players share scores, progress, and XP                   |
| **Impossible Play**  | Very Hard + timed flippers (1.75s max hold) + no extra balls |
| **Monster Play**     | Player 1 is "Monster" on Impossible; others normal           |
| **Team Play**        | 2v2, 3v1, or 2v1 score combinations                          |
| **Competition Play** | Removes randomness (fixed Mystery sequence)                  |
| **DJ Mixer**         | Jukebox mode (Free Play only)                                |

### Easter Eggs & Secrets

- **Speed Run:** Game tracks time from first plunge to defeating Knull. Beat your best time!
- **Hulk Smash:** As Venomized Hulk, press action button to instantly award ALL shots (once per ball)
- **Secret Hideout:** Drop ball in top hole during skill shot for 4 Bonus X
- **Symbiote Mania (Topper Mode):** Requires topper; play all 8 characters in random order <kcite ref="58"/>

### Web Combo Multipliers <kcite ref="21"/>

Shooting combos builds shot multipliers:

| Combo Length | Shot Multiplier |
| ------------ | --------------- |
| 2-way        | 2X              |
| 4-way        | 3X              |
| 6-way        | 4X              |
| 10-way       | 5X (maximum)    |

**Host Combo Ability:**
- Eddie/Wolverine: Average
- Flash/Hulk/Cap: Poor
- Gwen/Peter: Good (more shots light for combos)

### Host Signature Combos <kcite ref="21"/>

After playing Toxin Team-Up, complete your host's signature combo to light Host Hurry-Up:

| Host  | Combo Sequence                                  |
| ----- | ----------------------------------------------- |
| Eddie | Center Ramp → Left Ramp → 180 → Right Orbit     |
| Flash | Right Ramp → Scoop → Left Ramp → Scoop          |
| Gwen  | Left Loop → Center Ramp → Left Loop → Left Ramp |
| Peter | Left Orbit → Left Ramp → 180 → Scoop            |

## 🏆 Summary: The Path to Victory

1. Eddie → Flash (complete both ladders)
2. Lock 6 balls (prioritize right tank 3rd lock)
3. Stack Mayhem + Carnage + Carnage Hurry-Up
4. Build 4-7X playfield multiplier
5. Spider-Sense for instant super jackpots
6. Cash Carnage Hurry-Up with multiplier (300-500M)
7. Toxin Team-Up with Sleeper
8. Host Combo → Host Hurry-Up with multiplier (500M+)
9. Rampage / Blood Lust for bonus points
10. Repeat multiball stacks as available               

> **🎯 Remember:** Venom rewards strategic planning over frantic flipping. Know your host, manage your multipliers, and stack your features. The symbiote will do the rest!
